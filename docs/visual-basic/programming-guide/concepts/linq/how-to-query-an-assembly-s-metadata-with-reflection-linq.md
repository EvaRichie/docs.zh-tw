# <a name="how-to-query-an-assemblys-metadata-with-reflection-linq-visual-basic"></a><span data-ttu-id="366e7-102">如何：使用反映查詢元件的中繼資料（LINQ）（Visual Basic）</span><span class="sxs-lookup"><span data-stu-id="366e7-102">How to: Query An Assembly's Metadata with Reflection (LINQ) (Visual Basic)</span></span>
<span data-ttu-id="366e7-103">下列範例示範如何搭配使用 LINQ 與反射，來擷取符合所指定搜尋準則之方法的特定中繼資料。</span><span class="sxs-lookup"><span data-stu-id="366e7-103">The following example shows how LINQ can be used with reflection to retrieve specific metadata about methods that match a specified search criterion.</span></span> <span data-ttu-id="366e7-104">在此情況下，查詢會尋找組件中所有方法的名稱，而這些方法會傳回陣列這類可列舉類型。</span><span class="sxs-lookup"><span data-stu-id="366e7-104">In this case, the query will find the names of all the methods in the assembly that return enumerable types such as arrays.</span></span>  
  
## <a name="example"></a><span data-ttu-id="366e7-105">範例</span><span class="sxs-lookup"><span data-stu-id="366e7-105">Example</span></span>  
  
```vb
Imports System.Linq
Imports System.Reflection  

Module Module1  
    Sub Main()  
        Dim asmbly As Assembly =
            Assembly.Load("System.Core, Version=3.5.0.0, Culture=neutral, PublicKeyToken= b77a5c561934e089")  
  
        Dim pubTypesQuery = From type In asmbly.GetTypes()
                            Where type.IsPublic
                            From method In type.GetMethods()
                            Where method.ReturnType.IsArray = True
                            Let name = method.ToString()
                            Let typeName = type.ToString()
                            Group name By typeName Into methodNames = Group  
  
        Console.WriteLine("Getting ready to iterate")  
        For Each item In pubTypesQuery  
            Console.WriteLine(item.methodNames)  
  
            For Each type In item.methodNames  
                Console.WriteLine(" " & type)  
            Next  
        Next  
        Console.WriteLine("Press any key to exit... ")  
        Console.ReadKey()  
    End Sub  
End Module  
```  
  
<span data-ttu-id="366e7-106">這個範例會使用 <xref:System.Reflection.Assembly.GetTypes%2A?displayProperty=nameWithType> 方法，以傳回所指定組件中的類型陣列。</span><span class="sxs-lookup"><span data-stu-id="366e7-106">The example uses the <xref:System.Reflection.Assembly.GetTypes%2A?displayProperty=nameWithType> method to return an array of types in the specified assembly.</span></span> <span data-ttu-id="366e7-107">套用[Where 子句](../../../language-reference/queries/where-clause.md)篩選準則，只會傳回公用類型。</span><span class="sxs-lookup"><span data-stu-id="366e7-107">The [Where Clause](../../../language-reference/queries/where-clause.md) filter is applied so that only public types are returned.</span></span> <span data-ttu-id="366e7-108">對於每一個公用類型，會使用從 <xref:System.Type.GetMethods%2A?displayProperty=nameWithType> 呼叫傳回的 <xref:System.Reflection.MethodInfo> 陣列來產生子查詢。</span><span class="sxs-lookup"><span data-stu-id="366e7-108">For each public type, a subquery is generated by using the <xref:System.Reflection.MethodInfo> array that is returned from the <xref:System.Type.GetMethods%2A?displayProperty=nameWithType> call.</span></span> <span data-ttu-id="366e7-109">這些結果會進行篩選，僅傳回其傳回型別為陣列的方法，或為實作 <xref:System.Collections.Generic.IEnumerable%601> 之類型的方法。</span><span class="sxs-lookup"><span data-stu-id="366e7-109">These results are filtered to return only those methods whose return type is an array or else a type that implements <xref:System.Collections.Generic.IEnumerable%601>.</span></span> <span data-ttu-id="366e7-110">最後，會使用類型名稱作為索引鍵來群組這些結果。</span><span class="sxs-lookup"><span data-stu-id="366e7-110">Finally, these results are grouped by using the type name as a key.</span></span>
