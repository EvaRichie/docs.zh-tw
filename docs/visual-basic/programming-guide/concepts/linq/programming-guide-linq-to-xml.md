# <a name="programming-guide-linq-to-xml-visual-basic"></a>程式設計指南（LINQ to XML）（Visual Basic）
本節提供使用 [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] 進行程式設計的概念性和使用說明資訊。  
  
## <a name="who-should-read-this-documentation"></a>本文件的對象  
 本檔的目標是已經瞭解 Visual Basic 的開發人員，以及 .NET Framework 的一些基本層面。  
  
 本文件的目標在於讓各種開發人員容易使用 [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)]。 [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] 可讓 XML 程式設計更為容易。 您不需要是專業開發人員就可以使用本文件。  
  
 [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] 大量依賴泛型類別。 因此，您必須瞭解泛型型別和類別的用法，這點非常重要。 如需詳細資訊，請參閱[Visual Basic 中的泛型型別](../../language-features/data-types/generic-types.md)。
