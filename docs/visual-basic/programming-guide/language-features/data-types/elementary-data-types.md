# <a name="elementary-data-types-visual-basic"></a>基礎資料類型 (Visual Basic)
Visual Basic 會提供一組預先定義的資料類型，您可以將其用於許多程式設計項目。 本節說明這些類型，以及如何使用它們。  
  
> [!NOTE]
> Visual Basic 中的每個基本資料類型都受到結構或命名空間中的類別支援 <xref:System> 。 編譯器會使用每個資料類型關鍵字做為基礎結構或類別的別名。 例如，使用保留字來宣告變數， `Byte` 就和使用完整結構名稱進行宣告一樣 <xref:System.Byte?displayProperty=nameWithType> 。

 [數值資料類型](numeric-data-types.md)  
 描述整數和非整數的數數值型別。  
  
 [字元資料類型](character-data-types.md)  
 描述 `Char` 和 `String` 類型。  
  
 [其他資料類型](miscellaneous-data-types.md)  
 描述 `Boolean` 、 `Date` 和 `Object` 類型。
