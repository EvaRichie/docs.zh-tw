# <a name="type-conversions-in-visual-basic"></a>Visual Basic 中的類型轉換
將某個資料類型的值變更為另一個類型的程式稱為「*轉換*」。 轉換是*擴展*或*縮小*，視相關類型的資料容量而定。 根據原始程式碼中的語法，它們也是*隱含*或*明確*的。
