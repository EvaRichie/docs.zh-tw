# <a name="-help---visual-basic"></a>-help、-?  (Visual Basic) 

顯示編譯器選項。  
  
## <a name="syntax"></a>語法  
  
```console  
-help  
```

或  

```console
-?  
```  
  
## <a name="remarks"></a>備註  

 如果您在編譯中包含這個選項，則不會建立任何輸出檔案，也不會進行編譯。  
  
> [!NOTE]
> `-help`選項無法在 Visual Studio 開發環境中使用; 只有在從命令列編譯時，才能使用此選項。  
  
## <a name="example"></a>範例  

 下列程式碼會從命令列顯示說明。  
  
```console  
vbc -help  
```
