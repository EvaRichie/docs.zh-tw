## <a name="example"></a><span data-ttu-id="cbca3-126">範例</span><span class="sxs-lookup"><span data-stu-id="cbca3-126">Example</span></span>  
 <span data-ttu-id="cbca3-127">下列範例顯示 Visual Basic 編譯器插入 PE 的預設資訊清單。</span><span class="sxs-lookup"><span data-stu-id="cbca3-127">The following example shows the default manifest that the Visual Basic compiler inserts into a PE.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="cbca3-128">編譯器會將標準應用程式名稱 MyApplication 插入資訊清單 XML。</span><span class="sxs-lookup"><span data-stu-id="cbca3-128">The compiler inserts a standard application name MyApplication.app into the manifest XML.</span></span> <span data-ttu-id="cbca3-129">這是讓應用程式在 Windows Server 2003 Service Pack 3 上執行的因應措施。</span><span class="sxs-lookup"><span data-stu-id="cbca3-129">This is a workaround to enable applications to run on Windows Server 2003 Service Pack 3.</span></span>  
  
```xml  
<?xml version="1.0" encoding="utf-8" standalone="yes"?>  
<assembly xmlns="urn:schemas-microsoft-com:asm.v1" manifestVersion="1.0">  
  <assemblyIdentity version="1.0.0.0" name="MyApplication.app"/>  
  <trustInfo xmlns="urn:schemas-microsoft-com:asm.v2">  
    <security>  
      <requestedPrivileges xmlns="urn:schemas-microsoft-com:asm.v3">  
        <requestedExecutionLevel level="asInvoker"/>  
      </requestedPrivileges>  
    </security>  
  </trustInfo>  
</assembly>  
```
