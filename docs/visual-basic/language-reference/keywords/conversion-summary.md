# <a name="conversion-summary-visual-basic"></a>轉換摘要 (Visual Basic)
Visual Basic 語言關鍵字和執行時間程式庫成員是依用途和使用來組織。  
  
|動作|語言項目|  
|------------|----------------------|  
|將 ANSI 值轉換為字串。|<xref:Microsoft.VisualBasic.Strings.Chr%2A>, <xref:Microsoft.VisualBasic.Strings.ChrW%2A>|  
|將字串轉換為小寫或大寫。|<xref:Microsoft.VisualBasic.Strings.Format%2A>, <xref:Microsoft.VisualBasic.Strings.LCase%2A>,<xref:Microsoft.VisualBasic.Strings.UCase%2A>|  
|將日期轉換為序號。|<xref:Microsoft.VisualBasic.DateAndTime.DateSerial%2A>, <xref:Microsoft.VisualBasic.DateAndTime.DateValue%2A>|  
|將十進位數轉換成其他基底。|<xref:Microsoft.VisualBasic.Conversion.Hex%2A>, <xref:Microsoft.VisualBasic.Conversion.Oct%2A>|  
|將數位轉換為字串。|<xref:Microsoft.VisualBasic.Strings.Format%2A>, <xref:Microsoft.VisualBasic.Conversion.Str%2A>|  
|將一種資料類型轉換成另一種。|[CBool](../functions/type-conversion-functions.md)、 [CByte](../functions/type-conversion-functions.md)、 [CDate](../functions/type-conversion-functions.md)、 [CDbl](../functions/type-conversion-functions.md)、 [CDec](../functions/type-conversion-functions.md)、 [CInt](../functions/type-conversion-functions.md)、 [CLng](../functions/type-conversion-functions.md)、 [CSng](../functions/type-conversion-functions.md)、 [CShort](../functions/type-conversion-functions.md)、 [CStr](../functions/type-conversion-functions.md)、 [CType](../functions/ctype-function.md)、 <xref:Microsoft.VisualBasic.Conversion.Fix%2A> 、<xref:Microsoft.VisualBasic.Conversion.Int%2A>|  
|將日期轉換成日、月、工作日或年。|<xref:Microsoft.VisualBasic.DateAndTime.Day%2A>, <xref:Microsoft.VisualBasic.DateAndTime.Month%2A>, <xref:Microsoft.VisualBasic.DateAndTime.Weekday%2A>, <xref:Microsoft.VisualBasic.DateAndTime.Year%2A>|  
|將時間轉換為小時、分鐘或秒。|<xref:Microsoft.VisualBasic.DateAndTime.Hour%2A>, <xref:Microsoft.VisualBasic.DateAndTime.Minute%2A>, <xref:Microsoft.VisualBasic.DateAndTime.Second%2A>|  
|將字串轉換成 ASCII 值。|<xref:Microsoft.VisualBasic.Strings.Asc%2A>, <xref:Microsoft.VisualBasic.Strings.AscW%2A>|  
|將字串轉換為數字。|<xref:Microsoft.VisualBasic.Conversion.Val%2A>|  
|將時間轉換為序號。|<xref:Microsoft.VisualBasic.DateAndTime.TimeSerial%2A>, <xref:Microsoft.VisualBasic.DateAndTime.TimeValue%2A>|
