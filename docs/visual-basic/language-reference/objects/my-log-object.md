# <a name="mylog-object"></a>My.Log 物件
提供屬性和方法，用於將事件和例外狀況 (Exception) 資訊寫入應用程式的記錄檔接聽程式。  
  
## <a name="remarks"></a>備註  
 如需方法和 `My.Log` 物件屬性的資訊，請參閱 <xref:Microsoft.VisualBasic.Logging.AspLog>。  
  
 `My.Log`物件僅適用于 ASP.NET 應用程式。 若是用戶端應用程式，請使用[My.user 物件](my-application-log-object.md)。  
  
## <a name="requirements"></a>需求  
 **命名空間：** <xref:Microsoft.VisualBasic.Logging>  
  
 **類別：**<xref:Microsoft.VisualBasic.Logging.AspLog>  
  
 **元件：** Visual Basic 執行時間程式庫（在 Microsoft 中）
