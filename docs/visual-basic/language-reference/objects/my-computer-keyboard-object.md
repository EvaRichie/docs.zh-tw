# <a name="mycomputerkeyboard-object"></a>My.Computer.Keyboard 物件
提供屬性以存取鍵盤目前的狀態，例如目前已按下哪些按鍵，並提供方法將按鍵輸入傳送至使用中視窗。  
  
## <a name="remarks"></a>備註  
 如需方法和 `My.Computer.Keyword` 物件屬性的資訊，請參閱 <xref:Microsoft.VisualBasic.Devices.Keyboard>。  
  
 如需詳細資訊，請參閱[存取鍵盤](../../developing-apps/programming/computer-resources/accessing-the-keyboard.md)。  
  
## <a name="requirements"></a>需求  
 **命名空間：** <xref:Microsoft.VisualBasic.Devices>  
  
 **類別：**<xref:Microsoft.VisualBasic.Devices.Keyboard>  
  
 **元件：** Visual Basic 執行時間程式庫（在 Microsoft 中）
