# <a name="mycomputerfilesystemspecialdirectories-object"></a>My.Computer.FileSystem.SpecialDirectories 物件
提供屬性，以存取常用的參考目錄。  
  
## <a name="remarks"></a>備註  
 如需方法和 `My.Computer.FileSystem.SpecialDirectories` 物件屬性的資訊，請參閱 <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>。  
  
 如需詳細資訊，請參閱[如何：取出我的文件目錄的內容](../../developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md)。  
  
## <a name="requirements"></a>需求  
 **命名空間：** <xref:Microsoft.VisualBasic.MyServices>  
  
 **類別：** <xref:Microsoft.VisualBasic.MyServices.SpecialDirectoriesProxy>（提供對的存取權 <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories> ）  
  
 **元件：** Visual Basic 執行時間程式庫（在 Microsoft 中）
