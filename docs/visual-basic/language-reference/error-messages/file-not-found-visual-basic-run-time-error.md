# <a name="file-not-found-visual-basic-run-time-error"></a>找不到檔案 (Visual Basic 執行階段錯誤)
在指定的位置找不到檔案。 此錯誤的可能原因如下：  
  
- 語句參考了不存在的檔案。  
  
- 嘗試呼叫動態連結程式庫（DLL）中的程式，但找不到語句的子句中指定的程式庫 `Lib` `Declare` 。  
  
- 您嘗試開啟專案或載入不存在的文字檔。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 檢查檔案名和路徑規格的拼寫。
