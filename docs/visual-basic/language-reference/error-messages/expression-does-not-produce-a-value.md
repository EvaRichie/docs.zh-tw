# <a name="expression-does-not-produce-a-value"></a>運算式沒有產生值
您嘗試使用的運算式不會產生值產生內容中的值，例如 `Sub` 在預期的內容中呼叫 `Function` 。  
  
 **錯誤識別碼：** BC30491  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將運算式變更為會產生值的運算式。
