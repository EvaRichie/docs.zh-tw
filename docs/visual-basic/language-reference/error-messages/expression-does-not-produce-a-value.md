# <a name="expression-does-not-produce-a-value"></a>運算式沒有產生值

您已嘗試使用不會在值產生內容中產生值的運算式，例如 `Sub` 在預期的內容中呼叫 `Function` 。  
  
 **錯誤識別碼：** BC30491  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將運算式變更為一個產生值的運算式。
