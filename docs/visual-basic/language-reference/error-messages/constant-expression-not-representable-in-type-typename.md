# <a name="bc30439-constant-expression-not-representable-in-type-typename"></a>BC30439：常數運算式無法在類型 ' ' 中表示 \<typename>

您正在嘗試評估將無法放入目標型別的常數，通常是因為它溢出範圍。

 **錯誤識別碼：** BC30439

## <a name="to-correct-this-error"></a>更正這個錯誤

1. 將目標型別變更為可以處理常數的型別。
