# <a name="line-is-too-long"></a>行太長

來源文字行不能超過65535個字元。  
  
 **錯誤識別碼：** BC30494  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將行的長度縮短為65535個字元或更少。
