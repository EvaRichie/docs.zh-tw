# <a name="internal-error-happened-at-location"></a>在 \<location> 上發生內部錯誤

發生內部錯誤。 發生錯誤的那一行會包含在錯誤訊息中。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定此錯誤不是由 `Error` 語句或方法所產生 `Raise` ; 如果不是，請聯繫 Microsoft 產品支援服務，以報告訊息出現的條件。
