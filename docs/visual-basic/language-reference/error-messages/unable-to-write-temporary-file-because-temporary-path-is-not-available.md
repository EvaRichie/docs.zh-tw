# <a name="bc30698-unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>BC30698：無法寫入暫存檔，因為無法使用暫存路徑

Visual Basic 無法判斷儲存暫存檔案的路徑。

 **錯誤識別碼：** BC30698

## <a name="to-correct-this-error"></a>更正這個錯誤

1. 重新啟動 Visual Studio。

2. 如果問題持續發生，請重新安裝 Visual Studio。
