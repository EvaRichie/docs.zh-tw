# <a name="property-or-method-not-found"></a>找不到屬性或方法
未定義參考的物件方法或物件屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 您可能拼錯了物件的名稱。 若要查看為物件定義的屬性和方法，請顯示物件瀏覽器。 選取適當的 [物件程式庫]，以查看可用屬性和方法的清單。
