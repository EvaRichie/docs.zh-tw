# <a name="bc30143-unable-to-embed-resource-file-filename-error-message"></a>BC30143：無法內嵌資源檔 ' \<filename> '： \<error message>

Visual Basic 編譯器會呼叫元件連結器 ( # A0，也稱為 Alink) 來產生具有資訊清單的元件。 連結器在將原生 COM+ 資源檔案直接內嵌到組件時回報了錯誤。

 **錯誤識別碼：** BC30143

## <a name="to-correct-this-error"></a>更正這個錯誤

1. 檢查引號的錯誤訊息，並參閱 [Al.exe](../../../framework/tools/al-exe-assembly-linker.md)的主題。 以取得進一步說明和建議。

2. 如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
