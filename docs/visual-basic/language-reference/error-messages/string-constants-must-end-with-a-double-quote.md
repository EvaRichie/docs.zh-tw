# <a name="string-constants-must-end-with-a-double-quote"></a>字串常數必須以雙引號結尾
字串常數的開頭和結尾都必須加上引號。  
  
 錯誤識別碼 **：** BC30648  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定字串常值的結尾是引號（"）。 如果您貼上其他文字編輯器的值，請確定貼入的字元是有效的引號，而不是其中一個類似的字元，例如 "smart" 或 "大引號（" or "）或兩個單引號（' '）。
