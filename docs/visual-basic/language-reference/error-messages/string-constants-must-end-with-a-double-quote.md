# <a name="bc30648-string-constants-must-end-with-a-double-quote"></a>BC30648：字串常數必須以雙引號結尾

字串常數的開頭和結尾都必須加上引號。

 錯誤識別碼 **：** BC30648

## <a name="to-correct-this-error"></a>更正這個錯誤

- 請確定字串常值的結尾是引號 ( ") 。 如果您從其他文字編輯器貼上值，請確定貼上的字元是有效的引號，而不是與它類似的其中一個字元，例如 "smart" 或 "two" 引號 ( "或" ) 或兩個 ( ' ' ) 的單引號。
