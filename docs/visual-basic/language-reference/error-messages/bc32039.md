# <a name="bc32039-array-declared-as-for-loop-control-variable-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="3df9f-102">BC32039：宣告為 for 迴圈控制變數的陣列不能以初始大小宣告</span><span class="sxs-lookup"><span data-stu-id="3df9f-102">BC32039: Array declared as for loop control variable cannot be declared with an initial size</span></span>

<span data-ttu-id="3df9f-103">`For Each`迴圈會使用陣列做為其反覆運算變數，但會初始化該陣列。</span><span class="sxs-lookup"><span data-stu-id="3df9f-103">A `For Each` loop uses an array as its iteration variable but initializes that array.</span></span>

<span data-ttu-id="3df9f-104">**錯誤識別碼：** BC32039</span><span class="sxs-lookup"><span data-stu-id="3df9f-104">**Error ID:** BC32039</span></span>

## <a name="example"></a><span data-ttu-id="3df9f-105">範例</span><span class="sxs-lookup"><span data-stu-id="3df9f-105">Example</span></span>

<span data-ttu-id="3df9f-106">下列範例會產生 bc32039：</span><span class="sxs-lookup"><span data-stu-id="3df9f-106">The following example generates bc32039:</span></span>

```vb
Dim arrayList As New List(Of Integer())
For Each listElement(1) As Integer In arrayList

Next
```

## <a name="to-correct-this-error"></a><span data-ttu-id="3df9f-107">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="3df9f-107">To correct this error</span></span>

<span data-ttu-id="3df9f-108">移除反覆運算變數宣告中的初始化，如下列範例所示：</span><span class="sxs-lookup"><span data-stu-id="3df9f-108">Remove the initialization from the declaration of the iteration variable as shown in the following example:</span></span>

```vb
Dim arrayList As New List(Of Integer())
For Each listElement() As Integer In arrayList

Next
```

<span data-ttu-id="3df9f-109">或者，您可以使用型別推斷：</span><span class="sxs-lookup"><span data-stu-id="3df9f-109">or you can use type inference:</span></span>

```vb
Dim arrayList As New List(Of Integer())
For Each listElement In arrayList

Next
```
