# <a name="expression-too-complex"></a>運算式太複雜

浮點數運算式包含太多的嵌套子運算式。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 視需要將運算式分成許多不同的運算式，以防止發生錯誤。
