# <a name="bc30560-name-is-ambiguous-in-the-namespace-namespacename"></a>BC30560： ' \<name> ' 在命名空間 ' ' 中不明確 \<namespacename>

您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 Visual Basic 編譯器沒有任何衝突解決規則;您必須自行區分名稱。

 **錯誤識別碼：** BC30560

## <a name="to-correct-this-error"></a>更正這個錯誤

- 完整限定名稱。
