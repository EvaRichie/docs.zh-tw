# <a name="device-io-error"></a>裝置 I/O 錯誤
當您的程式使用印表機或磁片磁碟機等裝置時，發生輸入或輸出錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定裝置運作正常，然後再次嘗試操作。
