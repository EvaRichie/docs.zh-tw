# <a name="cant-create-necessary-temporary-file"></a>無法建立必要的暫存檔
磁片磁碟機已滿，其中包含 TEMP 環境變數所指定的目錄，或 TEMP 環境變數指定了無效或唯讀磁片磁碟機或目錄。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 刪除磁片磁碟機中的檔案（如果已滿）。  
  
2. 請在 TEMP 環境變數中指定不同的磁片磁碟機。  
  
3. 請為 TEMP 環境變數指定有效的磁片磁碟機。  
  
4. 從目前指定的磁片磁碟機或目錄移除唯讀限制。
