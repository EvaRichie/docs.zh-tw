# <a name="identifier-expected"></a>必須是識別項

不是可辨識的宣告專案名稱的程式設計專案會在內容需要專案名稱的位置發生。 其中一個可能的原因是，在語句開頭以外的地方指定了屬性。  
  
 **錯誤識別碼：** BC30203  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確認語句中的任何屬性都放在開頭。  
  
- 確認語句中的所有專案名稱拼寫正確。
