# <a name="identifier-expected"></a>必須是識別項
當內容需要專案名稱時，就會出現不是可辨識之宣告專案名稱的程式設計項目。 其中一個可能的原因是，已在語句開頭以外的地方指定了屬性。  
  
 **錯誤識別碼：** BC30203  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確認語句中的任何屬性全都放在一開始。  
  
- 確認語句中的所有元素名稱拼寫正確。
