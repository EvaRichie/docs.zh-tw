# <a name="as-any-is-not-supported-in-declare-statements"></a>'Declare' 陳述式中不支援 'As Any'

`Any`資料類型使用於 `Declare` Visual Basic 6.0 和較早版本中的語句，以允許使用可包含任何資料類型的引數。 但 Visual Basic 支援多載，因此 `Any` 資料類型會過時。  
  
 **錯誤識別碼：** BC30828
