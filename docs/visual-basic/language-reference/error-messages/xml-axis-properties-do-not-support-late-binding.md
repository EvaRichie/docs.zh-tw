# <a name="bc31168-xml-axis-properties-do-not-support-late-binding"></a>BC31168： XML 軸屬性不支援晚期繫結

XML 軸屬性已針對不具類型的物件參考。

 **錯誤識別碼：** BC31168

## <a name="to-correct-this-error"></a>更正這個錯誤

- 請先確定物件是強型別 <xref:System.Xml.Linq.XElement> 物件，再參考 XML 軸屬性。
