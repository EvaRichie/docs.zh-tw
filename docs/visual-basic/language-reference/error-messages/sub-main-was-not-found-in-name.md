# <a name="bc30420-sub-main-was-not-found-in-name"></a>BC30420：在 ' ' 中找不到 ' Sub Main ' \<name>

`Sub Main` 遺漏，或指定的位置不正確。

 **錯誤識別碼：** BC30420

## <a name="to-correct-this-error"></a>更正這個錯誤

1. 提供遺漏的 `Sub Main` 陳述式，或如果存在的話，將它移至程式碼中的適當位置。 如需的詳細資訊 `Sub Main` ，請參閱 [Visual Basic 中的 Main](../../programming-guide/program-structure/main-procedure.md)程式。

2. 在 [**專案設計**工具] 的 [**啟動表單**] 方塊中，指定專案啟始物件的位置。
