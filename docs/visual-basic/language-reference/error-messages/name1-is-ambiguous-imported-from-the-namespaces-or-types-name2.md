# <a name="bc30561-name1-is-ambiguous-imported-from-the-namespaces-or-types-name2"></a>BC30561： ' \<name1> ' 不明確，從命名空間或類型 ' ' 匯入 \<name2>

您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 Visual Basic 編譯器沒有任何衝突解決規則;您必須自行區分名稱。

 **錯誤識別碼：** BC30561

## <a name="to-correct-this-error"></a>更正這個錯誤

- 藉由移除命名空間匯入來區分名稱。

- 完整限定名稱。
