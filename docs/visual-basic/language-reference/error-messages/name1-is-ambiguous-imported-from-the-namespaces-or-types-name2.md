# <a name="name1-is-ambiguous-imported-from-the-namespaces-or-types-name2"></a>從命名空間或類型 '\<name1>' 匯入的 '\<name2>' 模稜兩可

您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 Visual Basic 編譯器沒有任何衝突解決規則;您必須自行區分名稱。  
  
 **錯誤識別碼：** BC30561  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 藉由移除命名空間匯入來區分名稱。  
  
2. 完整限定名稱。
