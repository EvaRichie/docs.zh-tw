# <a name="bc36548-cannot-convert-anonymous-type-to-an-expression-tree-because-a-property-of-the-type-is-used-to-initialize-another-property"></a>BC36548：無法將匿名型別轉換為運算式樹狀架構，因為類型的屬性是用來初始化其他屬性

當使用匿名型別的一個屬性來初始化匿名型別的另一個屬性時，編譯器不會接受匿名型別的轉換成運算式樹狀結構。 例如，在下列程式碼中， `Prop1` 會在初始化清單中宣告，然後用來做為的起始值 `Prop2` 。

**錯誤識別碼：** BC36548

## <a name="example"></a>範例

```vb
Module M2

    Sub ExpressionExample(Of T)(x As Expressions.Expression(Of Func(Of T)))
    End Sub

    Sub Main()
        ' The following line causes the error.
        ExpressionExample(Function() New With {.Prop1 = 2, .Prop2 = .Prop1})
    End Sub
End Module
```

## <a name="to-correct-this-error"></a>更正這個錯誤

將的初始值指派給 `Prop1` 區域變數。 將該變數指派給 `Prop1` 和 `Prop2` ，如下列程式碼所示：

```vb
Sub Main()
    Dim temp = 2
    ExpressionExample(Function() New With {.Prop1 = temp, .Prop2 = temp})
End Sub
```
