# <a name="evaluation-of-expression-or-statement-timed-out"></a>運算式或陳述式評估已經逾時

運算式的評估未及時完成。  
  
 **錯誤識別碼：** BC30722  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 確認輸入的程式碼正確無誤。  
  
2. 簡化您的運算式，以縮短執行的時間。
