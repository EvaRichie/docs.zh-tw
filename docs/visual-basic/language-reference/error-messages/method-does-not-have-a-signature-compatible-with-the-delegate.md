# <a name="method-does-not-have-a-signature-compatible-with-the-delegate"></a>方法沒有與委派相容的簽章
方法的簽章與您嘗試使用的委派之間有不相容的情況。 `Delegate` 陳述式定義委派類別的參數類型和傳回類型。 任何具有相容類型和傳回類型之參數的程式，都可以用來建立這個委派類型的實例。  
  
 **錯誤識別碼**： BC36563
