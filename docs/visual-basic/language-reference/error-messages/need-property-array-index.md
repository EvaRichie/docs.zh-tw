# <a name="need-property-array-index"></a>須提供屬性陣列索引
這個屬性值包含陣列，而不是單一值。 您未指定嘗試存取之屬性陣列的索引。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 檢查元件的文件，找出適合陣列的索引範圍。 在屬性存取陳述式中指定適合的索引。
