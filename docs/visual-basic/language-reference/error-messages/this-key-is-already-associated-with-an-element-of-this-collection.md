# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>此索引鍵已經和此集合的項目建立關聯性

針對集合成員指定的索引鍵已經識別集合的另一個成員。 索引鍵是在方法中指定的字串 `Add` ，可唯一識別集合的特定成員。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 針對這個成員使用不同的金鑰。
