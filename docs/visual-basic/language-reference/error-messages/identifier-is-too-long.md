# <a name="identifier-is-too-long"></a>識別項太長

每個程式設計項目的名稱或識別碼限制為1023個字元。 此外，完整名稱不能超過1023個字元。 這表示 () 的整個識別碼字串 `<namespace>.<...>.<namespace>.<class>.<element>` 長度不能超過1023個字元，包括成員存取運算子 (`.`) 個字元。  
  
 **錯誤識別碼：** BC30033  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 減少識別項的長度。
