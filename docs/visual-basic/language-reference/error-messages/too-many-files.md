# <a name="too-many-files"></a>檔案太多

在跟作業系統允許的根目錄中建立了多個檔案，或是開啟的檔案數目超過您 CONFIG.SYS 檔案中的 **files =** 設定所指定的數目。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 如果您的程式開啟、關閉或儲存根目錄中的檔案，請變更您的程式，使其使用子目錄。  
  
2. 增加您的檔案中所指定的檔案數目 **=** CONFIG.SYS 檔案中的設定，然後重新開機電腦。
