# <a name="too-many-files"></a>檔案太多
在根目錄中建立的檔案比作業系統允許的還要多，或開啟的檔案數超過您設定中的**files =** 設定中指定的數目。SYS 檔案。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 如果您的程式正在開啟、關閉或儲存根目錄中的檔案，請變更您的程式，使其使用子目錄。  
  
2. 增加您的檔案中指定的檔案數 **=** 設定中的。SYS 檔案，然後重新開機您的電腦。
