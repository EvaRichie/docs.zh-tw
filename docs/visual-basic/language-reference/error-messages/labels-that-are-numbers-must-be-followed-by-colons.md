# <a name="bc30801-labels-that-are-numbers-must-be-followed-by-colons"></a>BC30801：數位的標籤後面必須加上冒號

行號與其他類型的標籤遵循相同的規則，而且必須包含冒號。

 **錯誤識別碼：** BC30801

## <a name="to-correct-this-error"></a>更正這個錯誤

- 在程式程式碼的開頭放置數位，後面接著冒號。例如：

    ```vb
    400:    X += 1
    ```
