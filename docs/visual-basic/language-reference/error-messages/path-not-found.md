# <a name="path-not-found"></a>找不到路徑
在檔案存取或磁片存取操作期間，作業系統找不到指定的路徑。 檔案的路徑包含磁片磁碟機規格，加上必須進行遍歷以尋找檔案的目錄和子目錄。 路徑可為相對或絕對路徑。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 驗證並重新指定路徑。
