# <a name="cant-open-filename-for-writing"></a>無法開啟 '\<filename>' 以進行寫入

無法開啟指定的檔案進行寫入，可能是因為它已經開啟。  
  
 **錯誤識別碼：** BC2012  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 關閉檔案，然後將它重新開啟。  
  
2. 檢查檔案的許可權。
