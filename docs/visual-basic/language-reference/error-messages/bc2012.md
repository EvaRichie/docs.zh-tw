# <a name="bc2012-cant-open-filename-for-writing"></a>BC2012：無法開啟 ' \<filename> ' 進行寫入

無法開啟指定的檔案進行寫入，可能是因為它已經開啟。

 **錯誤識別碼：** BC2012

## <a name="to-correct-this-error"></a>更正這個錯誤

1. 關閉檔案，然後將它重新開啟。

2. 檢查檔案的許可權。
