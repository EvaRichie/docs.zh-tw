# <a name="bc42025-access-of-shared-member-constant-member-enum-member-or-nested-type-through-an-instance-qualifying-expression-will-not-be-evaluated"></a><span data-ttu-id="6a8ce-102">BC42025：透過實例存取共用成員、常數成員、列舉成員或巢狀型別;將不會評估合格的運算式</span><span class="sxs-lookup"><span data-stu-id="6a8ce-102">BC42025: Access of shared member, constant member, enum member or nested type through an instance; qualifying expression will not be evaluated</span></span>

<span data-ttu-id="6a8ce-103">類別或結構的執行個體變數用來存取 `Shared` 該類別或結構中所定義的變數、屬性、程式或事件。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-103">An instance variable of a class or structure is used to access a `Shared` variable, property, procedure, or event defined in that class or structure.</span></span> <span data-ttu-id="6a8ce-104">如果執行個體變數用來存取類別或結構的隱含共用成員，例如常數或列舉，或是嵌套的類別或結構，也可能會發生這個警告。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-104">This warning can also occur if an instance variable is used to access an implicitly shared member of a class or structure, such as a constant or enumeration, or a nested class or structure.</span></span>

<span data-ttu-id="6a8ce-105">共用成員的目的是只建立該成員的單一複本，並將該單一副本提供給其宣告所在之類別或結構的每個實例。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-105">The purpose of sharing a member is to create only a single copy of that member and make that single copy available to every instance of the class or structure in which it is declared.</span></span> <span data-ttu-id="6a8ce-106">它與此用途一致 `Shared` ，可透過其類別或結構的名稱來存取成員，而不是透過保存該類別或結構之個別實例的變數。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-106">It is consistent with this purpose to access a `Shared` member through the name of its class or structure, rather than through a variable that holds an individual instance of that class or structure.</span></span>

<span data-ttu-id="6a8ce-107">`Shared`透過執行個體變數存取成員，可讓您的程式碼更難理解，方法是隱藏該成員的事實 `Shared` 。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-107">Accessing a `Shared` member through an instance variable can make your code more difficult to understand by obscuring the fact that the member is `Shared`.</span></span> <span data-ttu-id="6a8ce-108">此外，如果這類存取是執行其他動作的運算式的一部分，例如會傳回 `Function` 共用成員實例的程式，Visual Basic 會略過運算式和其他任何執行的動作。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-108">Furthermore, if such access is part of an expression that performs other actions, such as a `Function` procedure that returns an instance of the shared member, Visual Basic bypasses the expression and any other actions it would otherwise perform.</span></span>

<span data-ttu-id="6a8ce-109">如需詳細資訊和範例，請參閱 [Shared](../modifiers/shared.md)。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-109">For more information and an example, see [Shared](../modifiers/shared.md).</span></span>

<span data-ttu-id="6a8ce-110">根據預設，這個訊息是一個警告。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-110">By default, this message is a warning.</span></span> <span data-ttu-id="6a8ce-111">如需隱藏警告或將警告視為錯誤的詳細資訊，請參閱 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)。</span><span class="sxs-lookup"><span data-stu-id="6a8ce-111">For more information about hiding warnings or treating warnings as errors, see [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).</span></span>

## <a name="example"></a><span data-ttu-id="6a8ce-113">範例</span><span class="sxs-lookup"><span data-stu-id="6a8ce-113">Example</span></span>

<span data-ttu-id="6a8ce-114">下列範例會產生 bc42025：</span><span class="sxs-lookup"><span data-stu-id="6a8ce-114">The following example generates bc42025:</span></span>

```vb
Public Class TestClass
    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        Dim tc As New TestClass()
        tc.SayHello() ' BC42025.
    End Sub
End Module
```

## <a name="to-address-this-warning"></a><span data-ttu-id="6a8ce-115">解決這個警告</span><span class="sxs-lookup"><span data-stu-id="6a8ce-115">To address this warning</span></span>

<span data-ttu-id="6a8ce-116">使用定義成員以存取它的類別或結構的名稱 `Shared` ，如下列範例所示：</span><span class="sxs-lookup"><span data-stu-id="6a8ce-116">Use the name of the class or structure that defines the `Shared` member to access it, as shown in the following example:</span></span>

```vb

    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        TestClass.SayHello()
    End Sub
End Module
```

> [!NOTE]
