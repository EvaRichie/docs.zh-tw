# <a name="bc30766-functionname-is-not-declared-smart-devicevisual-basic-compiler-error"></a>BC30766： ' \<functionname> ' 未宣告 (Smart Device/Visual Basic 編譯器錯誤) 

<`functionname` 未宣告>。 檔案 I/O 功能通常是在 `Microsoft.VisualBasic` 命名空間中，但是目標 .NET Compact Framework 版本並不支援它。

 **錯誤識別碼：** BC30766

## <a name="to-correct-this-error"></a>更正這個錯誤

- 使用 `System.IO` 命名空間中所定義的函式來執行檔案作業。
