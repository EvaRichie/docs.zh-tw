# <a name="overflow-visual-basic-error"></a>溢位 (Visual Basic 錯誤)

常值表示要指派給它的資料類型限制以外的值。  
  
 **錯誤識別碼：** BC30036  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請查閱目標資料類型的值範圍，並重寫常值以符合該範圍。
