# <a name="array-subscript-expression-missing"></a>遺漏陣列註標運算式

陣列初始化會離開一或多個定義陣列界限的注標。 例如，語句可能包含運算式，此運算式會 `myArray (5,5,,10)` 離開第三個注標。  
  
 **錯誤識別碼：** BC30306  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 提供遺漏的注標。
