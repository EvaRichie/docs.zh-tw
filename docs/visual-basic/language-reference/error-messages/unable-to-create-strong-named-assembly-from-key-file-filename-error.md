# <a name="bc31026-unable-to-create-strong-named-assembly-from-key-file-filename-error"></a>BC31026：無法從金鑰檔 ' ' 建立強式名稱的元件 \<filename> ： \<error>

無法從指定的金鑰檔建立強式名稱的元件。

 **錯誤識別碼：** BC31026

## <a name="to-correct-this-error"></a>更正這個錯誤

1. 請確認已指定正確的金鑰檔案，且該檔案未被其他應用程式鎖定。
