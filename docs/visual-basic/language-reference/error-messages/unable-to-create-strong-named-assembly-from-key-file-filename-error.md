# <a name="unable-to-create-strong-named-assembly-from-key-file-filename-error"></a>無法從金鑰檔 '\<filename>': \<error> 建立強式名稱的組件

無法從指定的金鑰檔建立強式名稱的元件。  
  
 **錯誤識別碼：** BC31026  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確認已指定正確的金鑰檔案，且該檔案未被其他應用程式鎖定。
