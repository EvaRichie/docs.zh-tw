# <a name="some-subkeys-cannot-be-deleted"></a>無法刪除某些子機碼

已嘗試刪除登錄機碼，但操作失敗，因為無法刪除某些子機碼。 這通常是因為許可權不足所造成。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定您有足夠的許可權可刪除指定的子機碼。
