# <a name="bc31084-events-cannot-be-declared-with-a-delegate-type-that-has-a-return-type"></a>BC31084：無法使用具有傳回類型的委派類型來宣告事件

針對函數程式指定了委派。

 **錯誤識別碼：** BC31084

## <a name="to-correct-this-error"></a>更正這個錯誤

- 指定程式的委派 `Sub` 。
