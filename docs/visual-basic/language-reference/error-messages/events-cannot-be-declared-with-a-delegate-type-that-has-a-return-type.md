# <a name="events-cannot-be-declared-with-a-delegate-type-that-has-a-return-type"></a>事件不可以宣告為具有傳回類型的委派類型

針對函數程式指定了委派。  
  
 **錯誤識別碼：** BC31084  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 指定程式的委派 `Sub` 。
