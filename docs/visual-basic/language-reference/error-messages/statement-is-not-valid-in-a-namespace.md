# <a name="statement-is-not-valid-in-a-namespace"></a>命名空間中的陳述式無效
語句不能出現在命名空間的層級。 命名空間層級所允許的唯一宣告為模組、介面、類別、委派、列舉和結構宣告。  
  
 **錯誤識別碼：** BC30001  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將語句移至模組、類別、介面、結構、列舉或委派定義內的位置。
