# <a name="error-saving-temporary-win32-resource-file-filename-error-message"></a>儲存暫存 Win32 資源檔 '\<filename>' 時發生錯誤: \<error message>

Visual Basic 編譯器會呼叫元件連結器 ( # A0，也稱為 Alink) 來產生具有資訊清單的元件。 連結器在取得檔案名稱以便用於寫入記憶體內資源時回報了錯誤。  
  
 **錯誤識別碼：** BC30137  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 檢查引號的錯誤訊息，並參閱 [Al.exe](../../../framework/tools/al-exe-assembly-linker.md)的主題。 以取得進一步說明和建議。  
  
2. 如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
