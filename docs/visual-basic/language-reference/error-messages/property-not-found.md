# <a name="property-not-found"></a>找不到屬性
這個物件不支援指定的屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 檢查屬性名稱的拼寫。  
  
2. 請檢查物件的檔，以確定當物件實際上支援「標題」或名稱類似的屬性時，您不會嘗試存取類似 "text" 屬性的內容。
