# <a name="out-of-memory-visual-basic-compiler-error"></a>記憶體不足 (Visual Basic 編譯器錯誤)
需要提供更多的記憶體。  
  
 **錯誤識別碼：** BC2004  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 關閉不必要的應用程式、檔和來源檔案。  
  
- 排除不必要的控制項和表單，使其一次載入較少  
  
- 減少變數的數目 `Public` 。  
  
- 檢查可用的磁碟空間。  
  
- 藉由安裝額外的記憶體或重新配置記憶體來增加可用的 RAM。  
  
- 請確定記憶體在不再需要時釋放。
