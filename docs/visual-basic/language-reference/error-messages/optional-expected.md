# <a name="optional-expected"></a>必須是 'Optional'

程式聲明中的選擇性引數後面接著必要的引數。 選擇性引數後面的每個引數也必須是選擇性的。  
  
 **錯誤識別碼：** BC30202  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 如果需要引數，請將其移至引數清單中的第一個選擇性引數前面。  
  
2. 如果引數是選擇性的，請使用 `Optional` 關鍵字。
