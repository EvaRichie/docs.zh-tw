# <a name="argument-not-optional-visual-basic"></a>引數不是選擇性的 (Visual Basic)

引數的數目和類型必須符合預期。 可能是引數數目不正確，或省略的引數不是選擇性的。 只有在程序定義中宣告引數時，才可以從呼叫使用者定義的程式中省略它 `Optional` 。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 提供所有必要的引數。  
  
2. 請確定省略的引數是選擇性的。 如果不是，請在呼叫中提供引數，或 `Optional` 在定義中宣告參數。
