# <a name="bc31043-arrays-declared-as-structure-members-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="64f03-102">BC31043：宣告為結構成員的陣列無法以初始大小宣告</span><span class="sxs-lookup"><span data-stu-id="64f03-102">BC31043: Arrays declared as structure members cannot be declared with an initial size</span></span>

<span data-ttu-id="64f03-103">結構中的陣列是以初始大小宣告的。</span><span class="sxs-lookup"><span data-stu-id="64f03-103">An array in a structure is declared with an initial size.</span></span> <span data-ttu-id="64f03-104">您無法初始化任何結構專案，而且宣告陣列大小是一種初始化形式。</span><span class="sxs-lookup"><span data-stu-id="64f03-104">You cannot initialize any structure element, and declaring an array size is one form of initialization.</span></span>

<span data-ttu-id="64f03-105">**錯誤識別碼：** BC31043</span><span class="sxs-lookup"><span data-stu-id="64f03-105">**Error ID:** BC31043</span></span>

## <a name="example"></a><span data-ttu-id="64f03-106">範例</span><span class="sxs-lookup"><span data-stu-id="64f03-106">Example</span></span>

<span data-ttu-id="64f03-107">下列範例會產生 BC31043：</span><span class="sxs-lookup"><span data-stu-id="64f03-107">The following example generates BC31043:</span></span>


## <a name="to-correct-this-error"></a><span data-ttu-id="64f03-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="64f03-108">To correct this error</span></span>

2. <span data-ttu-id="64f03-110">如果您需要特定大小的陣列，您可以在程式碼執行時，使用 [ReDim 語句](../statements/redim-statement.md) 來重設動態陣列的維度。</span><span class="sxs-lookup"><span data-stu-id="64f03-110">If you require a certain size of array, you can redimension a dynamic array with a [ReDim Statement](../statements/redim-statement.md) when your code is running.</span></span> <span data-ttu-id="64f03-111">下列範例會加以說明：</span><span class="sxs-lookup"><span data-stu-id="64f03-111">The following example illustrates this:</span></span>

    ```vb
    Structure DemoStruct
        Public demoArray() As Integer
    End Structure
    Sub UseStruct()
        Dim struct As DemoStruct
        ReDim struct.demoArray(9)
        Struct.demoArray(2) = 777
    End Sub
    ```
