# <a name="bc30655-unable-to-find-required-file-filename"></a>BC30655：找不到必要的檔案 ' \<filename> '

Visual Studio 所需要的某個檔案已經遺失或損毀。

 **錯誤識別碼：** BC30655

## <a name="to-correct-this-error"></a>更正這個錯誤

- 請重新安裝 Visual Studio。
