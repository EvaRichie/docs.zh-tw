# <a name="property-array-index-is-not-valid"></a>無效的屬性陣列索引

提供的值對屬性陣列索引而言無效。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查元件的檔，以確定您的索引位於指定屬性的有效範圍內。
