# <a name="option-keyword-statement"></a>Option \<keyword> 陳述式
引進的語句會指定套用至整個原始程式檔的編譯器選項。  
  
## <a name="remarks"></a>備註  
 編譯器選項可以控制是否必須明確宣告所有變數、縮小類型轉換是否必須是明確的，或字串是否應做為文字或二進位數量進行比較。  
  
 `Option` 關鍵字可用於以下內容：  
  
 [Option Compare 陳述式](option-compare-statement.md)  
  
 [Option Explicit 陳述式](option-explicit-statement.md)  
  
 [Option Infer 陳述式](option-infer-statement.md)  
  
 [Long](option-strict-statement.md)
