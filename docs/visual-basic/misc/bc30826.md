# <a name="endif-statements-are-no-longer-supported-use-end-if-instead"></a>不再支援 'EndIf' 陳述式; 請使用 'End If' 代替
不再支援 `EndIf` 陳述式。  
  
 **錯誤 ID︰** BC30826  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用 `End If` 來終止 `If` 陳述式。
