# <a name="mustoverride-cannot-be-specified-on-procedurename-because-it-is-in-a-partial-type-that-is-declared-notinheritable-in-another-partial-definition"></a><span data-ttu-id="1a14d-102">無法針對 '\<procedurename>' 指定 'MustOverride'，因為它所屬的部分類型，在其他部分定義中宣告為 'NotInheritable'</span><span class="sxs-lookup"><span data-stu-id="1a14d-102">'MustOverride' cannot be specified on '\<procedurename>' because it is in a partial type that is declared 'NotInheritable' in another partial definition</span></span>

<span data-ttu-id="1a14d-103">程序或屬性在多個部分宣告定義的類別內宣告為 `MustOverride` ，但其中一個部分定義為類別指定 `NotInheritable` 。</span><span class="sxs-lookup"><span data-stu-id="1a14d-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="1a14d-104">當您分割數個部分宣告中的類別定義時，編譯器會將類別視為其所有部分宣告的聯集。</span><span class="sxs-lookup"><span data-stu-id="1a14d-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="1a14d-105">這不只適用於成員，同時也適用於實作、繼承和存取層級。</span><span class="sxs-lookup"><span data-stu-id="1a14d-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="1a14d-106">若要覆寫程序或屬性，類別必須繼承其基底類別。</span><span class="sxs-lookup"><span data-stu-id="1a14d-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="1a14d-107">因此，若要為基底類別的程序或屬性指定 `MustOverride` ，您必須為類別指定 `MustInherit` 。</span><span class="sxs-lookup"><span data-stu-id="1a14d-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="1a14d-108">因為它們互相矛盾，所以您無法為同一個類別指定 `MustInherit` 和 `NotInheritable` 。</span><span class="sxs-lookup"><span data-stu-id="1a14d-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="1a14d-109">**錯誤識別碼：** BC30927</span><span class="sxs-lookup"><span data-stu-id="1a14d-109">**Error ID:** BC30927</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="1a14d-110">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="1a14d-110">To correct this error</span></span>  
  
- <span data-ttu-id="1a14d-111">如果必須覆寫屬性或程序，則從其出現的部分宣告中移除 `NotInheritable` 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="1a14d-111">If the property or procedure must be overridden, then remove the `NotInheritable` keyword from the partial declaration in which it appears.</span></span>
