# <a name="raiseevent-declaration-must-end-with-a-matching-end-raiseevent"></a>'RaiseEvent' 宣告之後必須搭配相對應的 'End RaiseEvent'
`RaiseEvent` 宣告的結尾必須是 `End RaiseEvent` 陳述式。  
  
 **錯誤 ID︰** BC31117  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查 `RaiseEvent` 宣告的結尾是否為 `End RaiseEvent` 陳述式。
