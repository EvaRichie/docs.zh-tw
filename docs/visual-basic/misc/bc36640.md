# <a name="instance-of-restricted-type-typename-cannot-be-used-in-a-lambda-expression"></a>受限制的類型 '\<typename>' 之執行個體不能用於 Lambda 運算式

受限制之類型的執行個體 (例如 <xref:System.ArgIterator>、 <xref:System.RuntimeArgumentHandle>和 <xref:System.TypedReference>) 不能用於 Lambda 運算式或查詢運算式。  
  
 **錯誤 ID︰** BC36640
