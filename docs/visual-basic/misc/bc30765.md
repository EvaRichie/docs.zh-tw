# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Visual Basic 解決方案中的所有專案都必須以相同的平台為目標，但您想要加入的專案卻不是以解決方案指定的平台為目標

Visual Basic 方案中的所有專案都必須以相同的平臺為目標，但您要新增的專案會以非解決方案所指定的平臺為目標。  
  
 **錯誤 ID：** BC30765  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 只加入與解決方案指定平台同目標的專案。
