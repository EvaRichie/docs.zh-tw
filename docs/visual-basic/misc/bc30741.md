# <a name="typename1-cannot-be-converted-to-typename2"></a>' \<typename1> ' 無法轉換成 ' \<typename2> '

已嘗試在兩個不相容的類型之間進行類型轉換。  
  
 **錯誤 ID︰** BC30741  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確認正在轉換的類型。
