# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-to-specifier1-type1-through-specifier2-type2"></a>' \<name> ' 無法將其正在執行之事件的基礎委派類型 ' ' 公開 \<delegatetype> 給 \<specifier1> ' \<type1> ' 至 \<specifier2> ' \<type2> '

這個程式碼會在公用類別外部公開私用類型。  
  
 **錯誤 ID︰** BC30914  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 宣告 `Public` 類型或使用另一種類型。
