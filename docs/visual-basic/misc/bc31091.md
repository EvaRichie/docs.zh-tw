# <a name="import-of-type-typename-from-assembly-or-module-name-failed"></a>從組件或模組 '\<name>' 匯入類型 '\<typename>' 失敗

請確定所有參考的程式庫彼此相容。  
  
 **錯誤 ID︰** BC31091  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確認包含參考類型的檔案未損毀。
