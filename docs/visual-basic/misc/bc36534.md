# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>運算式無法轉換為運算式樹狀架構

大多數的運算式都可以轉換為運算式樹狀架構，但有些不能。 這個錯誤表示嘗試轉換無法轉換的運算式。  
  
 **錯誤 ID︰** BC36534
