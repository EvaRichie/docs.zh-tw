# <a name="error-structurename1-contains-structurename2"></a>\<error>： ' \<structurename1> ' 包含 ' \<structurename2> '

結構將其本身指定為它自己的其中一個成員時，就會發生循環結構定義。  
  
 **錯誤 ID︰** BC30293  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 變更結構成員的名稱。
