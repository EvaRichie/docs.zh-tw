# <a name="argument-argumentname-must-be-within-the-range-1-to-255"></a>引數 '\<argumentname>' 的範圍必須介於 1 到 255 之間
引數無效，因為它超出 0 到 255 的範圍。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請檢查運算式中的引數拼法。 拼錯的變數名稱可能會隱含地建立初始化為零的數值變數。  
  
2. 請檢查運算式中先前的變數作業，特別是那些作為引數從其他程序傳遞至程序的變數。
