## <a name="example"></a><span data-ttu-id="b2a34-109">範例</span><span class="sxs-lookup"><span data-stu-id="b2a34-109">Example</span></span>  

 <span data-ttu-id="b2a34-110">下列類別包含需要整數引數的預設屬性 `defaultProp`。</span><span class="sxs-lookup"><span data-stu-id="b2a34-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```vb  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```

```vb

```
