# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>屬性的存取必須是指定值給屬性或使用屬性值

您已嘗試存取屬性，但未指派或使用它的值。
  
 **錯誤 ID︰** BC30545  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將值指派給屬性。  
  
     \- 或 -  
  
- 使用屬性的值。
