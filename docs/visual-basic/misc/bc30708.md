# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>[即時運算] 視窗中的迴圈陳述式無效

`Loop`**在 [即時**運算] 視窗中不能使用語句。  
  
 **錯誤 ID︰** BC30708  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從偵錯程式碼中移除 `Loop` 陳述式。
