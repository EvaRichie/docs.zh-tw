# <a name="end-region-must-be-preceded-by-a-matching-region"></a>'#End Region' 之前必須搭配相對應的 '#Region'

`#Region`使用 Visual Studio Code 編輯器的大綱功能時，您可以使用來指定要展開或折迭的程式碼區塊。 `#Region` 陳述式的開頭和結尾必須在相同的程式碼區塊中。  
  
 **錯誤 ID：** BC30680  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請在對應 `#Region``#End Region` 。
