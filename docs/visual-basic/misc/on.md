# <a name="on-visual-basic"></a>On (Visual Basic)

引進對執行階段錯誤的回應，或開啟編譯器選項。  
  
 在 [查詢] 中，識別所聯結集合的索引鍵。

 `On`關鍵字會在這些內容中使用：

 [Join 子句](../language-reference/queries/join-clause.md)
