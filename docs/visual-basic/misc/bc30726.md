# <a name="with-contexts-and-statements-are-not-valid-in-debug-windows"></a>偵錯視窗中的 'With' 內容和陳述式無效

`With` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰** BC30726  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從偵錯程式碼中移除 `With` 陳述式。
