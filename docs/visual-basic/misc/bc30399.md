# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>無法以方法 '\<methodname>' 使用 'MyBase'，因為它被宣告為 'MustOverride'

您嘗試搭配使用 `MyBase` 與已宣告 `MustOverride`的方法。  
  
 **錯誤 ID︰** BC30399  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除 `MustOverride` 宣告。
