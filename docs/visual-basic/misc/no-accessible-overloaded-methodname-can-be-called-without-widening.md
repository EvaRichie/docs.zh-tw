# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-widening-conversion-list"></a>使用不具擴展轉換的這些引數，無法呼叫可存取的多載 '\<methodname>'：\<list>

已呼叫多載方法，但沒有任何方法可在未擴展轉換的情況下，符合所提供的引數清單。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請指定 `Option Strict Off`。  
  
- 變更引數，以符合多載方法的其中一個特徵標記。
