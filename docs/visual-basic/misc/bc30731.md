# <a name="addressof-expressions-are-not-valid-in-debug-windows"></a>'AddressOf' 運算式在偵錯視窗中無效

`AddressOf` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰** BC30731  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請從偵錯程式碼中移除 `AddressOf` 陳述式。
