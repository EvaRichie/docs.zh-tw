# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="0d738-102">XML 常值不可出現在這裡，除非加上括弧</span><span class="sxs-lookup"><span data-stu-id="0d738-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>

<span data-ttu-id="0d738-103">XML 常值宣告用在位置中對 Visual Basic 編譯器不明確的運算式中。</span><span class="sxs-lookup"><span data-stu-id="0d738-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="0d738-104">也就是說，Visual Basic 編譯器無法判斷小於字元 ( # A0) 是否要做為比較運算子或 XML 常值的開頭。</span><span class="sxs-lookup"><span data-stu-id="0d738-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="0d738-105">下列程式碼提供一個範例：</span><span class="sxs-lookup"><span data-stu-id="0d738-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="0d738-106">**錯誤 ID：** BC31198</span><span class="sxs-lookup"><span data-stu-id="0d738-106">**Error ID:** BC31198</span></span>
