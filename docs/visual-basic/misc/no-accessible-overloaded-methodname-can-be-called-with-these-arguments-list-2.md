# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-narrowing-conversion-list"></a>無法使用這些引數呼叫可存取的多載 ' \<methodname> '，而不需要縮小轉換：\<list>
已呼叫多載方法，但沒有方法可在未縮小轉換的情況下，符合所提供的引數清單。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 指定 `Option Strict Off`。
  
2. 變更引數，以符合多載方法的簽章。
