# <a name="property-parameters-cannot-have-the-name-value"></a>屬性參數不可以有名稱 'Value'

`Property` 陳述式中的參數名為 `Value`。 這個名稱在屬性定義內保留給指派給 `Set` 程序中之屬性的新值。  
  
 **錯誤 ID：** BC32201  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請變更 `Property` 陳述式中的參數名稱。
