# <a name="reference-required-to-module-modulename-containing-the-implemented-interface-interfacename"></a>需要模組 ' \<modulename> ' 的參考，其中包含實介面 ' \<interfacename> '

需要模組 ' \<modulename> ' 的參考，其中包含實介面 ' ' \<interfacename> 。 請在專案中加入一個參考。  
  
 此介面是在專案中未直接參考的模組中所定義。 Visual Basic 編譯器需要參考，以避免當介面在多個模組中定義時，不明確。  
  
 **錯誤識別碼：** BC30010  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請在您的專案參考中包含未參考之模組的名稱。
