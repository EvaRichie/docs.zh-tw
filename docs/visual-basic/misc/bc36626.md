# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>無法將 Lambda 運算式轉換為 '\<typename>'，因為類型 '\<typename>' 宣告為 'MustInherit' 而且無法建立
陳述式會嘗試將 Lambda 運算式轉換成只能是基底類別且不能用來建立執行個體的類型。  
  
 **錯誤 ID︰** BC36626
