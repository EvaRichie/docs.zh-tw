# <a name="option-optionname-requires-argument"></a>選項 \<optionname> 需要 \<argument>

已省略必要的引數。  
  
 **錯誤 ID︰** BC2006  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請提供遺漏的引數。
