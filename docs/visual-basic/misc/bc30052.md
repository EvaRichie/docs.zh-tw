# <a name="array-exceeds-the-limit-of-32-dimensions"></a>陣列超過 32 個維度的限制

陣列不可以有 32 個以上的維度。  
  
 **錯誤 ID︰** BC30052  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 減少陣列宣告中的維度數目。
