# <a name="xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>在相同的 XML 註解區塊中，XML 註解標記 '\<tag>' 以相同的屬性出現多次

\<tag>在相同的 xml 批註區塊中，具有相同屬性的 xml 註解標記會出現一次以上。 將會忽略 XML 註解。  
  
 XML 註解包含多個具有相同屬性的標記。  
  
 **錯誤 ID︰** BC42305  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除重複的標記。
