# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>方法 '\<methodname>' 不可以宣告為 'Partial'，因為只有一個方法 '\<methodname>' 可以標記為 'Partial'
您不能有兩個部分方法宣告具有相同名稱和簽章。  
  
 **錯誤 ID︰** BC31433  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請變更其中一個部分方法宣告的名稱或簽章。
