# <a name="operator-is-not-valid-for-name1-and-name2"></a>運算子對 ' \<name1> ' 和 ' ' 而言無效 \<name2>

運算子不適用於所操作的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確定運算子適用於所操作的類型。
