# <a name="division-by-zero-visual-basic-run-time-error"></a>除數為零 (Visual Basic 執行階段錯誤)
當做除數的運算式其值為零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請檢查運算式中的變數拼法。 拼錯的變數名稱可能會隱含地建立初始化為零的數值變數。  
  
2. 請檢查運算式中先前的變數作業，特別是那些作為引數從其他程序傳遞至程序的變數。
