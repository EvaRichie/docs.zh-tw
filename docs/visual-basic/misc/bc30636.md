# <a name="-expected"></a>必須是 ' > '

所指定程式碼的語法不正確。 這個錯誤通常表示遺漏屬性或 XML 項目常值的右角括弧。  
  
 **錯誤 ID︰** BC30636  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查所使用程式碼的語法。
