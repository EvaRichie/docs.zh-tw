# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>參考的物件 '\<objectname>' 具有 'Nothing' 值

參考的物件尚未設定，或已設定為 `Nothing`值。  
  
 **錯誤 ID︰** BC30705  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確認物件名稱拼字正確，且已設定該物件。
