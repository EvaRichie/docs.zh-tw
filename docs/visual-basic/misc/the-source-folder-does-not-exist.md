# <a name="the-source-folder-does-not-exist"></a>來源資料夾不存在
您嘗試執行參考不存在之來源資料夾的作業。 這可能是提供不正確檔案路徑的結果。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查您使用正確的路徑 (特別是使用相對路徑時)。
