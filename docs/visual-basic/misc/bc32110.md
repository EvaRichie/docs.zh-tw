# <a name="constraint-constraint1-conflicts-with-the-indirect-constraint-constraint2-obtained-from-the-type-parameter-constraint-typeparameter1"></a>條件約束 ' \<constraint1> ' 與 \<constraint2> 從類型參數條件約束 ' ' 取得的間接條件約束 ' ' 衝突 \<typeparameter1>

宣告的泛型類型因直接和間接條件約束組合而具有衝突的條件約束。  
  
 下列陳述式可能會產生這個錯誤。  
  
 `Public Class testClass(Of t1 As {Structure, t2}, t2 As Class)`  
  
 直接條件約束 `Structure` 和間接條件約束 `Class` 導致類型參數 `t1`的衝突，因為 `Structure` 條件約束需要對應的類型引數是實值類型，而 `Class` 需要它是參考類型。  
  
 **錯誤 ID︰** BC32110  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請變更類型參數條件約束，以避免衝突的條件約束。
