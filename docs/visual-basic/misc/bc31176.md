# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>XML 常值中不允許 'character' 字元 (0x\<hexadecimal>)

已在 XML 常值中使用不支援的字元。  
  
 **錯誤 ID︰** BC31176  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除不支援的字元。
