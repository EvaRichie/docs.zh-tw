# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>\<qualifiedmethod>無法在此內容中存取 ' '，因為它是 ' \<modifier> '
您嘗試存取因已套用修飾詞而無法在此內容中存取的方法。  
  
 **錯誤 ID︰** BC30390  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除修飾詞。
