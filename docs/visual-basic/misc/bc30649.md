# <a name="typename-is-an-unsupported-type"></a>' \<typename> ' 是不支援的類型

指定的類型無法以 Visual Studio 表示。  
  
 **錯誤 ID：** BC30649  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用支援的類型。  
  
- 開發新的類型。
