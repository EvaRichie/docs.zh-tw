# <a name="date-constant-is-not-valid"></a>無效的日期常數

所宣告的日期常數具有無效日期。  
  
 **錯誤 ID︰** BC31085  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請更正常數中所使用的日期。
