# <a name="unable-to-open-file-filename-error"></a>無法開啟檔案 ' \<filename> '： \<error>

開啟這個專案中的檔案時發生錯誤。  
  
 **錯誤 ID︰** BC31027  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 確認檔案存在，而且未遭其他應用程式損毀或鎖定。
