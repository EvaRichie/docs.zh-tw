# <a name="object-doesnt-support-current-locale-setting-visual-basic"></a>物件不支援目前的地區設定 (Visual Basic)
您嘗試存取不支援目前專案地區設定的物件。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查物件支援哪些地區設定。
