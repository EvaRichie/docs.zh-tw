# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>未定義類型 'type1' 和 'type2' 的運算子 'operator'

未定義類型 'type1' 和 'type2' 的運算子 'operator'。 您可以使用 ' Value ' 屬性來取得 ' ' 第一個元素的字串值 \<parentElement> 。  
  
 二元或 `Boolean` 比較中已使用 XML 常值。  
  
 **錯誤 ID︰** BC31195  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用 XML 常值的 `Value` 屬性來進行二元或 `Boolean` 比較。
