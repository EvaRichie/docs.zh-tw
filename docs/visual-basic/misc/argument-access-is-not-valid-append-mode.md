# <a name="argument-access-is-not-valid-valid-values-for-append-mode-are-openaccesswrite-and-openaccessdefault"></a>引數 'Access' 無效。 Append 模式的有效值是 'OpenAccess.Write' 和 'OpenAccess.Default'

存取值在 Append 模式中無效。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 將此值變更為 `OpenAccess.Write` 或 `OpenAccess.Default`。
