# <a name="declare-statements-in-a-module-cannot-be-declared-specifier"></a>模組中的 Declare 語句不可以宣告為 ' \<specifier> '
`Declare` 宣告的規範在 `Module` 宣告內無效。 模組絕不會執行個體化、不支援繼承，且不能實作介面。  
  
 **錯誤 ID︰** BC30786  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除規範。
