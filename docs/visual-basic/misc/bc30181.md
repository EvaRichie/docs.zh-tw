# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>規範只在宣告的開頭有效

指定存取範圍和其他指定的關鍵字必須出現在 `Dim` 或 `Function`等宣告關鍵字之前。  
  
 **錯誤 ID︰** BC30181  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在屬性清單 (如果有) 和宣告關鍵字之間移動所有規範。
