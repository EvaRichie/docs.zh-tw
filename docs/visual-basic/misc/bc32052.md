 <span data-ttu-id="55baa-104">`Is` 會比較兩個參考類型，以判斷它們是否指向記憶體中的相同物件執行個體。</span><span class="sxs-lookup"><span data-stu-id="55baa-104">`Is` compares two reference types to determine whether they point to the same object instance in memory.</span></span> <span data-ttu-id="55baa-105">除非另一個運算元是 [Nothing](../language-reference/nothing.md)，否則它無法採用不是參考型別的運算元。</span><span class="sxs-lookup"><span data-stu-id="55baa-105">It cannot take an operand that is not a reference type unless the other operand is [Nothing](../language-reference/nothing.md).</span></span>
