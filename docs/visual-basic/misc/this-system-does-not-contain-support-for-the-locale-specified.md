# <a name="this-system-does-not-contain-support-for-the-locale-specified"></a>此系統沒有包含指定地區設定的支援

您嘗試使用 `VbStrConv` 列舉，但您的系統不支援指定的地區設定。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請升級電腦以支援地區設定，或在支援該地區設定的電腦上執行應用程式。
