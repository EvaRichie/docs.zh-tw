# <a name="custom-event-must-end-with-a-matching-end-event"></a>'Custom Event' 結尾必須是對應的 'End Event'
`Custom Event` 宣告的結尾必須是 `End Event` 陳述式。  
  
 **錯誤 ID：** BC31114  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查 `Custom Event` 宣告的結尾是否為 `End Event` 陳述式。
