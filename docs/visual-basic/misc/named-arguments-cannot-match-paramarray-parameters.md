# <a name="named-arguments-cannot-match-paramarray-parameters"></a>具名引數與 ParamArray 參數不相符

參數陣列必須以傳值方式傳遞。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 傳遞引數 `ByVal`，而不是透過命名引數。
