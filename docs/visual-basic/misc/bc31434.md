<span data-ttu-id="b6e4e-103">方法 ' \<methodname1> ' 無法實作為部分方法 ' \<methodname2> '，因為 ' \<methodname3> ' 已經執行。</span><span class="sxs-lookup"><span data-stu-id="b6e4e-103">Method '\<methodname1>' cannot implement partial method '\<methodname2>' because '\<methodname3>' already implements it.</span></span> <span data-ttu-id="b6e4e-104">只有一個方法可實作部分方法。</span><span class="sxs-lookup"><span data-stu-id="b6e4e-104">Only one method can implement a partial method.</span></span>  
  
 <span data-ttu-id="b6e4e-105">您不能同時有兩個部分方法實作同一個部分方法宣告。</span><span class="sxs-lookup"><span data-stu-id="b6e4e-105">You cannot have two partial methods that implement the same partial method declaration.</span></span> <span data-ttu-id="b6e4e-106">下列程式碼會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="b6e4e-106">The following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub ValueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' First implementation of the partial method.  
    Private Sub ValueChanged()  
        MsgBox(Value was changed to " & Me.Quantity)  
    End Sub  
  
    ' Second implementation of the partial method causes this error.  
    'Private Sub ValueChanged()  
    '    Console.WriteLine("Quantity was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```
