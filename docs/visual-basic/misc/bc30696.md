# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>' \<membername1> ' 和 ' ' 無法互相多載， \<membername2> 因為它們的差異只在於選擇性參數的類型不同
多載成員的一或多個必要參數必須具有不同的資料類型。  
  
 **錯誤 ID：** BC30696  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請至少將一個必要的參數定義為不同的資料類型。
