# <a name="cannot-convert-start-value-of-type-type1-and-step-value-of-type-type2-to-a-common-type"></a>無法將型別 '\<type1>' 的起始值和型別 '\<type2>' 的間距值轉換為一般型別

指定的 `For...Next` 控制項值具有不同的類型，因此無效，因為它們無法轉換成一般類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請變更一個或多個值的類型，以將所有的值轉換成一般類型。
