# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>衍生自 COM 類別的 Managed 類別無法以晚期繫結的方式呼叫。

您嘗試對衍生自 COM 類別的 Managed 類別進行晚期繫結呼叫；不支援這類呼叫。

## <a name="to-correct-the-problem"></a>更正這個問題

請以早期繫結方式進行呼叫。
