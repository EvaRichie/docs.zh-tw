# <a name="argument-not-specified-for-parameter-parametername-of-methodname"></a>\<parametername>' ' 的參數 ' ' 未指定自 \<methodname> 變數

尚未提供必要參數的引數。  
  
 **錯誤 ID︰** BC30455  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請為指定的參數提供引數。
