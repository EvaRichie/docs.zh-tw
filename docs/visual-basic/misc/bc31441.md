# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a><span data-ttu-id="29c69-102">方法 ' \<methodname1> ' 必須宣告為 ' Private '，才能執行部分方法 ' \<methodname2> '</span><span class="sxs-lookup"><span data-stu-id="29c69-102">Method '\<methodname1>' must be declared 'Private' in order to implement partial method '\<methodname2>'</span></span>

<span data-ttu-id="29c69-103">部分方法的實作必須宣告 `Private`。</span><span class="sxs-lookup"><span data-stu-id="29c69-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="29c69-104">例如，下列程式碼會造成這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="29c69-104">For example, the following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,
    ' causes this error.
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 <span data-ttu-id="29c69-105">**錯誤 ID︰** BC31441</span><span class="sxs-lookup"><span data-stu-id="29c69-105">**Error ID:** BC31441</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="29c69-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="29c69-106">To correct this error</span></span>  
  
1. <span data-ttu-id="29c69-107">請在部分方法的實作中使用存取修飾詞 `Private` ，如下列範例所示。</span><span class="sxs-lookup"><span data-stu-id="29c69-107">Use the access modifier `Private` in the implementation of the partial method, as shown in the following example.</span></span>  
  
```vb  
Private Sub valueChanged()  
    MsgBox("Value was changed to " & Me.Quantity)  
End Sub  
```
