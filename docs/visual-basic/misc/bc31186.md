# <a name="-is-not-allowed"></a>不允許 ':'
不允許 ':'。 XML 限定名稱不能用在此內容中。  
  
 已在無效位置指定具有限定名稱的 XML 常值 (例如，處理指示常值中的目標)。  
  
 **錯誤 ID︰** BC31186  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除具有限定名稱的 XML 常值，或將它移至有效位置。  
  
     如需 XML 名稱中有效字元的相關資訊，請參閱 [可延伸標記語言 (XML) 1.0](https://www.w3.org/TR/xml)。
