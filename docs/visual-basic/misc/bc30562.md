# <a name="name-is-ambiguous-between-declarations-in-modules-modulename1-and-modulename2"></a>' \<name> ' 在模組 ' \<modulename1> ' 和 ' ' 中的宣告之間不明確 \<modulename2>

您提供的名稱模稜兩可，因此與另一個名稱發生衝突。 Visual Basic 編譯器沒有任何衝突解決規則;您必須自行區分名稱。  
  
 **錯誤識別碼：** BC30562  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 完整限定名稱。
