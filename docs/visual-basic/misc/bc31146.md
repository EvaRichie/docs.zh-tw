# <a name="xml-name-expected"></a>必須是 XML 名稱

已不具名指定 XML 常值或 XML 軸屬性。  
  
 **錯誤識別碼：** BC31146  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 加入 XML 常值或 XML 軸屬性的名稱。
