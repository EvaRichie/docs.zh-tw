 <span data-ttu-id="9e0ec-104">類型參數的條件約束清單可以指定傳遞至該類型參數的類型引數必須是實值類型 (使用 `Structure` 條件約束)，或必須是參考類型 (使用 [Class (Visual Basic)](../language-reference/statements/class-statement.md) 條件約束)。</span><span class="sxs-lookup"><span data-stu-id="9e0ec-104">A constraint list on a type parameter can specify that the type argument passed to that type parameter must be a value type (with the `Structure` constraint) or must be a reference type (with the [Class (Visual Basic)](../language-reference/statements/class-statement.md) constraint).</span></span> <span data-ttu-id="9e0ec-105">您無法針對相同的類型參數同時指定兩個條件約束，並且無法多次指定其中一個。</span><span class="sxs-lookup"><span data-stu-id="9e0ec-105">You cannot specify both constraints on the same type parameter, and you cannot specify either one more than once.</span></span>  
  
 <span data-ttu-id="9e0ec-106">`New` 條件約束指定類型引數必須公開建立程式碼可以存取的無參數建構函式。</span><span class="sxs-lookup"><span data-stu-id="9e0ec-106">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="9e0ec-107">不過，結構不能有非共用的無參數建構函式。</span><span class="sxs-lookup"><span data-stu-id="9e0ec-107">However, a structure cannot have a nonshared parameterless constructor.</span></span> <span data-ttu-id="9e0ec-108">因此， `New` 和 `Structure` 條件約束會發生衝突。</span><span class="sxs-lookup"><span data-stu-id="9e0ec-108">Therefore, the `New` and `Structure` constraints are in conflict.</span></span>
