# <a name="cannot-determine-array-type-because-it-is-nothing"></a>無法判斷陣列的類型，因為陣列為 Nothing
無法判斷您陣列的類型，因為陣列設定為值 `Nothing`。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 針對陣列，提供 `Nothing` 以外的值。
