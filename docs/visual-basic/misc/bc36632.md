# <a name="nullable-parameters-must-specify-a-type"></a>可為 Null 的參數必須指定類型
參數包含可為 Null 的類型修飾詞 (?)，但未指定類型。  
  
 **錯誤 ID︰** BC36632  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 宣告可為 null 的實值型別參數時，請指定型別。
