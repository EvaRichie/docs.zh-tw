# <a name="byval-and-byref-cannot-be-combined"></a>無法結合 'ByVal' 和 'ByRef'

`ByVal` 與 `ByRef` 關鍵字互斥。  
  
 **錯誤 ID︰** BC30641  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請指定 `ByVal` 或 `ByRef`，但兩者不能同時指定。
