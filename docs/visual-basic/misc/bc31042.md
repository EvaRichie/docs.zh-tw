# <a name="sub-new-cannot-implement-interface-members"></a>'Sub New' 無法實作介面成員
`Sub New` 是建構函式，而且無法實作成員。  
  
 **錯誤 ID︰** BC31042  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 從 `Implements` 程序中移除 `Sub New` 陳述式。
