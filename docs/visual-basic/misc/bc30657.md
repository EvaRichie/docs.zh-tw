# <a name="procedurename-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>' \<procedurename> ' 具有不支援的傳回類型，或不支援的參數類型

以參數或無法以 Visual Studio 表示的傳回型別宣告程式。  
  
 **錯誤 ID︰** BC30657  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將傳回類型或參數定義為支援的類型。  
  
- 開發可描述這個程序所使用資料的新類型。
