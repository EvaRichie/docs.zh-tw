# <a name="locale-id-name-is-not-supported-on-this-system"></a>在此系統上不支援地區設定識別碼 '\<name>'

您嘗試使用 `VbStrConv`，但您的系統不支援指定的地區設定。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請升級電腦以支援地區設定，或在支援指定地區設定的電腦上執行應用程式。
