<span data-ttu-id="8acc2-103">當方法與委派之間無法進行必要的轉換時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="8acc2-103">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="8acc2-104">錯誤的原因可能在於參數之間的轉換，或在於傳回值中的轉換 (當方法和委派為函式時)。</span><span class="sxs-lookup"><span data-stu-id="8acc2-104">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>  
  
 <span data-ttu-id="8acc2-105">下列程式碼描述失敗的轉換。</span><span class="sxs-lookup"><span data-stu-id="8acc2-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="8acc2-106">委派為 `FunDel`。</span><span class="sxs-lookup"><span data-stu-id="8acc2-106">The delegate is `FunDel`.</span></span>  
  
```vb  
Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer  
```  
  
 <span data-ttu-id="8acc2-107">下列各個函式與 `FunDel` 不同，而且其差異會造成這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="8acc2-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>

End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="8acc2-108">下列每個指派陳述式都會造成此錯誤。</span><span class="sxs-lookup"><span data-stu-id="8acc2-108">Each of the following assignment statements causes the error.</span></span>

    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2

```
