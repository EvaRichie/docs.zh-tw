# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>' \<argumentname> ' 不是 ' ' 的參數 \<methoddeclaration>
程序呼叫可提供程序未定義的其他引數。  
  
 **錯誤 ID︰** BC30272  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從引數清單中移除多餘的引數。
