# <a name="class-classname-cannot-be-indexed-because-it-has-no-default-property"></a>無法對類別 '\<classname>' 進行索引，因為它沒有預設屬性

您無法提供類別的預設屬性。  
  
 **錯誤 ID︰** BC30367  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 指定屬性作為預設屬性，方法是使用 `Default` 關鍵字來開始它的宣告。
