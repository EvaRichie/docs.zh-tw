# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>引數 '\<argumentname>' 的長度必須大於零
引數的值等於或小於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請提供大於零的整數。
