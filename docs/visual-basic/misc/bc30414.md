# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>類型 '\<type1>' 的值無法轉換成 '\<type2>'，因為陣列類型的維度數目不同

您嘗試將某個陣列轉換成另一個具有不同維度數目的陣列。  
  
 **錯誤 ID：** BC30414  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請變更其中一個或兩個陣列，讓維度的數目相符。
