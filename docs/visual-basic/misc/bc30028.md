# <a name="else-must-be-preceded-by-a-matching-if-or-elseif"></a><span data-ttu-id="c9e90-102">'#Else' 之前必須搭配相對應的 '#If' 或 '#ElseIf'</span><span class="sxs-lookup"><span data-stu-id="c9e90-102">'#Else' must be preceded by a matching '#If' or '#ElseIf'</span></span>

<span data-ttu-id="c9e90-103">`#Else` 是條件式編譯指示詞。</span><span class="sxs-lookup"><span data-stu-id="c9e90-103">`#Else` is a conditional compilation directive.</span></span> <span data-ttu-id="c9e90-104">`#Else` 指示詞的前面沒有對應的 `#If` 或 `#ElseIf` 指示詞。</span><span class="sxs-lookup"><span data-stu-id="c9e90-104">An `#Else` directive is not preceded by a corresponding `#If` or `#ElseIf` directive.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="c9e90-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="c9e90-106">To correct this error</span></span>
