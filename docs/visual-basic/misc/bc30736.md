# <a name="expression-cannot-be-evaluated-at-this-time"></a>此時無法評估運算式
無法在目前的內容中評估指定的運算式。  
  
 **錯誤 ID︰** BC30736  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請變更運算式的語法。
