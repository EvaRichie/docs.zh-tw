# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-delegatename"></a><span data-ttu-id="640bf-102">Option Strict 為 On 不允許縮小 lambda 運算式和委派 ' ' 之間的隱含類型轉換 \<delegatename></span><span class="sxs-lookup"><span data-stu-id="640bf-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate '\<delegatename>'</span></span>

<span data-ttu-id="640bf-103">當 `Option Strict` 為 On 時，您不能在委派中的參數與指派給該委派類型之變數的 Lambda 運算式的對應參數之間，進行資料類型的縮小轉換。</span><span class="sxs-lookup"><span data-stu-id="640bf-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="640bf-104">例如，在下列程式碼中，委派 `Del` 具有一個類型為 `Integer`的參數。</span><span class="sxs-lookup"><span data-stu-id="640bf-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="640bf-105">因此，指派給類型 `Del` 之變數的任何 Lambda 運算式的對應參數可以是 `Integer` 或可從 `Integer`擴展轉換的任何資料類型。</span><span class="sxs-lookup"><span data-stu-id="640bf-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="640bf-106">**錯誤 ID︰** BC36662</span><span class="sxs-lookup"><span data-stu-id="640bf-106">**Error ID:** BC36662</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="640bf-107">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="640bf-107">To correct this error</span></span>  
  
- <span data-ttu-id="640bf-108">變更委派或 Lambda 運算式中之參數的資料類型，以確保存在擴展關聯性。</span><span class="sxs-lookup"><span data-stu-id="640bf-108">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>  
  
- <span data-ttu-id="640bf-109">不要指定 Lambda 運算式中的參數資料類型。</span><span class="sxs-lookup"><span data-stu-id="640bf-109">Do not specify parameter data types in the lambda expression.</span></span> <span data-ttu-id="640bf-110">類型將從委派中的對應參數進行推斷。</span><span class="sxs-lookup"><span data-stu-id="640bf-110">Types will be inferred from the corresponding parameters in the delegate.</span></span>  
  
```vb  
Dim example4 As Del = Function(n) "Valid"  
```
