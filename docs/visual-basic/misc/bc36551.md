## <a name="example"></a><span data-ttu-id="4e982-108">範例</span><span class="sxs-lookup"><span data-stu-id="4e982-108">Example</span></span>  
 <span data-ttu-id="4e982-109">下列範例透過加入 `String` 方法來擴充 `Print` 類別。</span><span class="sxs-lookup"><span data-stu-id="4e982-109">The following example extends the `String` class, adding a `Print` method.</span></span>  
  
```vb

Imports System.Runtime.CompilerServices  
Namespace StringUtility  
    <Extension()> _  
    Module StringExtensions  
        <Extension()> _  
        Public Sub Print (ByVal str As String)  
            Console.WriteLine(str)  
        End Sub  
    End Module  
End Namespace  
```
