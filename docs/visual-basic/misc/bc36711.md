# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>符合參數 ' ' 的引數會 \<parametername> 縮小為 ' \<typename> '
您已呼叫多載方法，但編譯器找不到可以呼叫且不用縮小轉換的方法。 *「縮小轉換」* (narrowing conversion) 會將值變更為資料類型，而可能無法保留一些可能的值。  
  
 **錯誤 ID︰** BC36711
