# <a name="typename-cannot-be-used-as-an-attribute-because-it-has-mustoverride-methods-that-have-not-been-overridden"></a>' \<typename> ' 無法當做屬性使用，因為它具有尚未覆寫的 ' MustOverride ' 方法

無法將具有 `MustOverride` 方法的類別當做屬性使用。  
  
 屬性類別的`MustOverride` 成員只能透過覆寫這類成員的衍生類別使用。  
  
 **錯誤 ID︰** BC31507  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 從屬性類別成員移除 `MustOverride` 修飾詞。  
  
2. 在衍生類別中實作 `MustOverride` 成員，並使用該類別作為屬性。
