# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>已經自動新增預設組件參考 '\<reference>'；已忽略

已嘗試加入預設組件參考，但組件已經加入，因此忽略該嘗試。  
  
 **錯誤 ID︰** BC2024  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除多餘的程式碼。
