# <a name="maximum-number-of-warnings-has-been-exceeded"></a>已經超過警告最大數目
編譯警告數目上限為 100。 編譯器會施加此限制，以減少錯誤和警告報告的大小。  
  
 根據預設，這個訊息是一個警告。 如需隱藏警告或將警告視為錯誤的詳細資訊，請參閱 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)。  
  
 **錯誤 ID︰** BC42206  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢閱此編譯所產生的警告並盡可能全數解決。 如果原始程式碼包含超過 100 個警告，您可能會在下次編譯中看到更多警告。
