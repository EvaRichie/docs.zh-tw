# <a name="conversions-from-typename1-to-typename2-must-be-explicit"></a>從 '\<typename1>' 至 '\<typename2>' 的轉換必須是明確的

無法以隱含方式完成這項轉換。  
  
 **錯誤 ID：** BC30664  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用 `CType`或其中一個特定轉換函式，例如 `CInt` 或 `CDec`。
