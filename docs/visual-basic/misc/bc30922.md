# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="1e905-102">' \<derivedtypename> ' 無法繼承自 \<type> ' '， \<constructedbasetypename> 因為它會在元件外展開類型 ' ' 的存取權 \<internaltypename></span><span class="sxs-lookup"><span data-stu-id="1e905-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>

<span data-ttu-id="1e905-103">衍生的類別或介面嘗試使用受限類型作為基底類別或介面的類型引數，來展開受限類型的存取層級。</span><span class="sxs-lookup"><span data-stu-id="1e905-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="1e905-104">下列程式碼可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="1e905-104">The following code can generate this error.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="1e905-105">在組件外的程式碼不允許存取 `restrictedStructure`。</span><span class="sxs-lookup"><span data-stu-id="1e905-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="1e905-106">不過，可以從任何可參考 `derivedClass` 的程式碼存取它。</span><span class="sxs-lookup"><span data-stu-id="1e905-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="1e905-107">因此， `derivedClass` 無法繼承 `baseClass` ，如果它使用 `restrictedStructure` 作為類型引數的話，因為那可能將 `restrictedStructure` 公開給任何組件中的程式碼。</span><span class="sxs-lookup"><span data-stu-id="1e905-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="1e905-108">**錯誤 ID︰** BC30922</span><span class="sxs-lookup"><span data-stu-id="1e905-108">**Error ID:** BC30922</span></span>
