# <a name="events-cannot-have-a-return-type"></a>事件不可以有傳回類型
事件可以接受引數，但不能直接傳回值。  
  
 **錯誤 ID︰** BC30032  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從 `Event` 陳述式中移除傳回類型。
