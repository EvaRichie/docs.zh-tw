# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>' \<specifier> ' 在結構宣告中無效

您使用了對 `Dim` 宣告無效的指定名稱，例如 `Override`或 `Structure` 。  
  
 **錯誤識別碼：** BC30395  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除規範。
