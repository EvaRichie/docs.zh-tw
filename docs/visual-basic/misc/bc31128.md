# <a name="removehandler-is-already-declared"></a>'RemoveHandler' 已宣告
多個 `RemoveHandler` 宣告出現在自訂事件宣告中。 `RemoveHandler` 宣告會宣告用來移除事件處理常式的程序。  
  
 **錯誤 ID︰** BC31128  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除多餘的 `RemoveHandler` 陳述式。
