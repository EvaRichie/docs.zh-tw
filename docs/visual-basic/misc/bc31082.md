# <a name="variablename-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-catch-variable"></a>' \<variablename> ' 不是區域變數或參數，所以不能當做 ' Catch ' 變數使用

`Try...Catch...Finally` 陳述式中的變數必須在本機宣告，或為目前程序的參數。  
  
 **錯誤 ID︰** BC31082  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 宣告 `Try...Catch...Finally` 陳述式的區域變數或參數。
