# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>不再支援 'Let' 和 'Set' 指派陳述式

`Let` 和 `Set` 不可以用於指派中。  
  
 **錯誤 ID︰** BC30807  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 指派陳述式中省略 `Let` 和 `Set` ；例如：  
  
     `X = 2`
