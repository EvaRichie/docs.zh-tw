# <a name="expected-closing--for-xml-processor-instruction"></a>XML 處理器指令必須以 '？ > ' 結尾
XML 處理指示常值遺漏必要的結尾 `?>` 字元。  
  
 **錯誤 ID︰** BC31160  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將必要的結尾 `?>` 字元加入 XML 處理指示常值的結尾。
