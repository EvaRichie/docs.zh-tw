# <a name="property-propertyname-is-writeonly"></a>屬性 '\<propertyname>' 是 WriteOnly

您已嘗試從宣告為 `WriteOnly`的屬性擷取值。  
  
 **錯誤 ID︰** BC30524  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除屬性宣告的 `WriteOnly` 指定名稱。
