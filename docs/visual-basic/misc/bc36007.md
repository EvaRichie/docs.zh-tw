# <a name="end-using-must-be-preceded-by-a-matching-using"></a>'End Using' 之前必須搭配相對應的 'Using'
`End Using` 陳述式的前面沒有配對的 `Using` 宣告。  
  
 **錯誤 ID︰** BC36007  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 若 `End Using` 陳述式為多餘，請予以移除。  
  
- 提供遺漏的[Using 語句](../language-reference/statements/using-statement.md)（如果遺漏的話）。  
  
- 請將 `End Using` 陳述式移至程式碼中的適當位置。
