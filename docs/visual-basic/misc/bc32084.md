# <a name="type-parameter-typeparametername1-must-have-either-a-new-constraint-or-a-structure-constraint-to-satisfy-the-new-constraint-for-type-parameter-typeparametername2"></a><span data-ttu-id="903da-102">類型參數 ' \<typeparametername1> ' 必須有 ' new ' 條件約束或 ' Structure ' 條件約束，才能滿足類型參數 ' ' 的 ' new ' 條件 \<typeparametername2> 約束</span><span class="sxs-lookup"><span data-stu-id="903da-102">Type parameter '\<typeparametername1>' must have either a 'New' constraint or a 'Structure' constraint to satisfy the 'New' constraint for type parameter '\<typeparametername2>'</span></span>

<span data-ttu-id="903da-103">陳述式會建構泛型類型，而泛型類型傳遞不一定要滿足 `New` 條件約束的類型參數。</span><span class="sxs-lookup"><span data-stu-id="903da-103">A statement constructs a generic type passing a type parameter that is not constrained to satisfy a `New` constraint.</span></span>

<span data-ttu-id="903da-104">`New` 條件約束表示提供給該類型參數的類型引數必須公開從中建立物件之程式碼可存取的無參數建構函式。</span><span class="sxs-lookup"><span data-stu-id="903da-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="903da-105">所有實值類型都有無參數建構函式，但並非所有參考類型都有無參數建構函式。</span><span class="sxs-lookup"><span data-stu-id="903da-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="903da-106">因此， `Structure` 條件約束滿足 `New` 條件約束，但 `Class` 條件約束 (或類別或介面名稱) 則否。</span><span class="sxs-lookup"><span data-stu-id="903da-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>

<span data-ttu-id="903da-107">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="903da-107">The following statements can generate this error.</span></span>

```vb
Public Class c1(Of t As New)
End Class
Public Class c2(Of u)
    Public testObject As New c1(Of u)
End Class
```

<span data-ttu-id="903da-108">類別 `c2` 從 `c1`建立物件時，類型參數 `u` 未滿足類型參數 `New` 的 `t`條件約束。</span><span class="sxs-lookup"><span data-stu-id="903da-108">When class `c2` creates an object from `c1`, type parameter `u` does not satisfy the `New` constraint on type parameter `t`.</span></span>

<span data-ttu-id="903da-109">**錯誤 ID︰** BC32084</span><span class="sxs-lookup"><span data-stu-id="903da-109">**Error ID:** BC32084</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="903da-110">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="903da-110">To correct this error</span></span>

- <span data-ttu-id="903da-111">如果要傳遞給泛型類型的類型參數可以滿足 `Structure` 或 `New` 條件約束，則請將適當的條件約束加入其定義中。</span><span class="sxs-lookup"><span data-stu-id="903da-111">If the type parameter to be passed to the generic type can satisfy the `Structure` or `New` constraint, then add the appropriate constraint to its definition.</span></span>


- <span data-ttu-id="903da-112">如果類型參數無法滿足 `Structure` 或 `New` 條件約束，則請不要將它傳遞給泛型類型。</span><span class="sxs-lookup"><span data-stu-id="903da-112">If the type parameter cannot satisfy the `Structure` or `New` constraint, then do not pass it to the generic type.</span></span> <span data-ttu-id="903da-113">您必須傳遞其他項目作為類型引數。</span><span class="sxs-lookup"><span data-stu-id="903da-113">You must pass something else as the type argument.</span></span>
