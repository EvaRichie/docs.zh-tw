# <a name="interface-must-end-with-a-matching-end-interface"></a>'Interface' 之後必須搭配相對應的 'End Interface'

`Interface` 陳述式沒有相對應的 `End Interface` 陳述式。 您必須使用 `End Interface` 陳述式來結束介面定義。  
  
 **錯誤 ID︰** BC30253  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確認已正確地格式化 `Interface` 成員。  
  
2. 將 `End Interface` 陳述式加到介面定義結尾。
