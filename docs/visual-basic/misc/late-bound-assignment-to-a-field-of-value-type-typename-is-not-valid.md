# <a name="late-bound-assignment-to-a-field-of-value-type-typename-is-not-valid-when-name-is-the-result-of-a-late-bound-expression"></a>晚期繫結運算式的運算結果為 '\<name>' 時，將晚期繫結指派至實值型別 '\<typename>' 的欄位是無效的

您嘗試使用無效的晚期繫結指派。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將指派設定為早期繫結。
