# <a name="module-modulename-cannot-be-used-as-a-type"></a>模組 '\<modulename>' 無法當做類型使用

您已嘗試使用模組作為類型。 無法使用模組，但可以當成 `Get` 類型使用。  
  
 **錯誤 ID︰** BC30371  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將模組取代為可當成有效類型使用的某個項目。
