# <a name="method1-and-method2-cannot-overload-each-other-because-they-differ-only-by-return-types"></a>' ' \<method1> 和 ' \<method2> ' 無法互相多載，因為它們的差異只在於傳回類型不同

您已嘗試將某個方法多載另一個方法，後者只有傳回類型和前者不同。 在多載中，您必須藉引數清單區分任何兩個版本的方法；除了引數清單，沒有任何其他項目可以區別方法。  
  
 **錯誤識別碼：** BC30301  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定方法是以其引數清單區分。
