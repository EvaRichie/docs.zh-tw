# <a name="member-is-already-declared-in-this-structure"></a>' \<member> ' 已在這個結構中宣告

某個列舉成員已在這個結構中多次宣告。  
  
 **錯誤 ID︰** BC31421  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 移除多餘的宣告。
