# <a name="nothing-cannot-be-evaluated"></a>無法評估 'Nothing'
已在偵錯視窗中使用 `Nothing` 關鍵字。  
  
 **錯誤 ID︰** BC30701  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請不要在偵錯時使用 `Nothing` 。
