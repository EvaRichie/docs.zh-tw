# <a name="cast-from-type-typename1-to-type-typename2-is-not-valid"></a>從類型 '\<typename1>' 到類型 '\<typename2>' 的轉換無效
嘗試將變數從一種類型轉換成另一種類型不正確。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請檢查轉換中的類型，並確定您嘗試轉換成有意義的類型。
