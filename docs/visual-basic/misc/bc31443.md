# <a name="name-of-type-parameter-typeparametername1-does-not-match-typeparametername2-the-corresponding-type-parameter-defined-on-the-partial-method-declaration-methodname"></a>類型參數 ' ' 的名稱 \<typeparametername1> 不符合 ' ' \<typeparametername2> ，在部分方法宣告 ' ' 上定義的對應類型參數 \<methodname>

在包含一個或多個類型參數的部分方法中，方法宣告與方法實作中的類型參數名稱必須相同。  
  
 例如，下列宣告和實作會造成這個錯誤。  
  
```vb  
' Definition of the partial method signature with type parameter T.  
Partial Private Sub OnNameChanged(Of T)()  
End Sub  
```  
  
```vb  
'' Implementation of the partial method with type parameter N.  
'Private Sub OnNameChanged(Of N)()  
'    Console.WriteLine("Name was changed to " & Me.Name)  
'End Sub  
```  
  
 **錯誤 ID︰** BC31443  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 檢查類型參數，以判斷不相符的位置。 視需要變更名稱，將它們設為相同。
