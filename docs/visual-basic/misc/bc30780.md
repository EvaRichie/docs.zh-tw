# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'Continue' 陳述式無效

' Continue ' 語句只能用於原始程式碼中。  
  
 **錯誤 ID︰** BC30780  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 從偵錯工具程式碼中移除 ' Continue ' 語句。
