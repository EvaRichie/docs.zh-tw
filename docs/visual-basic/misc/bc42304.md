# <a name="xml-documentation-parse-error-error"></a>XML 文件剖析錯誤：\<error>
這個錯誤可能會出現下列訊息。 如需詳細資訊，請參閱個別主題：  
  
- [在相同的 XML 註解區塊中，XML 註解標記 '\<tag>' 以相同的屬性出現多次](bc42305.md)  
  
- ['\<element>' 語言項目上不可使用 XML 註解標記 '\<tag>'](bc42306.md)  
  
- [XML 註解參數 '\<parameter>' 與對應的 '\<keyword>' 陳述式上之參數不相符](bc42307.md)  
  
- [XML 註解參數必須要有 'name' 屬性](bc42308.md)  
  
- [XML 註解的標記具有無法解析的 'cref' 屬性 '\<attribute>'](bc42309.md)  
  
- [XML 註解標記 'include' 必須有 '\<attribute>' 屬性](bc42310.md)  
  
 **錯誤 ID︰** BC42304
