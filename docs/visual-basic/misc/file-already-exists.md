# <a name="file-already-exists"></a>檔案已經存在

如果將新的檔案命名為現有檔案的名稱，或使用 [另存新檔] **** 命令將目前載入的專案儲存為已存在的名稱，則會發生這個錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查名稱，並將它變更為不會衝突的名稱，或刪除預先存在的檔案或專案 (不再需要時)。
