# <a name="property-propertyname-is-readonly"></a>屬性 '\<propertyname>' 是 'ReadOnly'

您已嘗試指派值給宣告為 `ReadOnly`的屬性。  
  
 **錯誤識別碼：** BC30526  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除屬性宣告的 `ReadOnly` 指定名稱。
