# <a name="properties-in-a-module-cannot-be-declared-specifier"></a>模組中的屬性不可以宣告為 ' \<specifier> '

您已嘗試在模組的屬性上使用規範 (例如 `MustOverride`、 `NotOverridable`或 `Overridable`)。  
  
 **錯誤 ID︰** BC30503  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除規範。
