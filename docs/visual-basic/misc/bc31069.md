# <a name="enum-in-an-interface-cannot-be-declared-specifier"></a>介面中的列舉不可以宣告為 ' \<specifier> '
指定的無效修飾詞具有在介面中宣告的列舉。  
  
 **錯誤 ID︰** BC31069  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除修飾詞 (例如 `Public`、 `Private`、 `Shared`、 `Friend`、 `Protected`、 `Protected Friend`或 `Overridable`)。
