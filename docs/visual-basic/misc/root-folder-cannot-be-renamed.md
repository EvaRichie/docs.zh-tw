# <a name="root-folder-cannot-be-renamed"></a>無法重新命名根資料夾

已嘗試重新命名根目錄。 這可能是檔案路徑剖析錯誤的結果。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查並確定檔案路徑是否正確。
