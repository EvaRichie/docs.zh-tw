# <a name="constants-must-have-a-value"></a>必須有常數值

您已宣告常數但未指派其值。  
  
 **錯誤識別碼︰** BC30438  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 為常數指派一個值。
