# <a name="get-is-already-declared"></a>'Get' 已宣告

`Get` 陳述式在程式碼區塊中出現一次以上。 `Get` 陳述式會宣告用來擷取屬性值的 `Get` 屬性程序。  
  
 **錯誤 ID：** BC30443  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除多餘的 `Get` 陳述式。
