<span data-ttu-id="49d53-103">泛型類型和當成類型引數的 `MustInherit` 類別一起叫用，而對應的類型參數和 `New` 條件約束一起宣告。</span><span class="sxs-lookup"><span data-stu-id="49d53-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="49d53-104">`New` 條件約束要求在對應的類型引數中傳遞的類型必須支援建立物件。</span><span class="sxs-lookup"><span data-stu-id="49d53-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="49d53-105">不過， *「抽象」* (abstract) 類別，也就是宣告為 `MustInherit`的類別，不會公開任何建構函式，因為您無法從它建立任何物件。</span><span class="sxs-lookup"><span data-stu-id="49d53-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="49d53-106">**錯誤識別碼：** BC32082</span><span class="sxs-lookup"><span data-stu-id="49d53-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="49d53-107">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="49d53-107">To correct this error</span></span>  
  
1. <span data-ttu-id="49d53-108">如果用於類型引數的類別不必是抽象的，則從其宣告中移除 `MustInherit` 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="49d53-108">If the class used in the type argument does not need to be abstract, then remove the `MustInherit` keyword from its declaration.</span></span>  
  
2. <span data-ttu-id="49d53-109">如果用於類型引數的類別必須是抽象的，但不必用來建構泛型類型，則在類型引數中傳遞不同的類別。</span><span class="sxs-lookup"><span data-stu-id="49d53-109">If the class used in the type argument needs to be abstract but does not need to be used to construct the generic type, then pass a different class in the type argument.</span></span>
