# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>無法同時在變數及其類型上指定陣列修飾詞

陣列修飾詞存在於變數和其類型上，這表示陣列的陣列。  
  
 **錯誤 ID︰** BC31087  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從類型規範中移除陣列修飾詞。
