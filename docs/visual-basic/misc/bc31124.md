# <a name="end-addhandler-must-be-preceded-by-a-matching-addhandler-declaration"></a>'End AddHandler' 之前必須搭配相對應的 'AddHandler' 宣告
`End AddHandler` 陳述式沒有相對應的 `AddHandler` 陳述式。 `End AddHandler` 之前必須搭配相對應的 `AddHandler` 陳述式。  
  
 **錯誤 ID︰** BC31124  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查前面的 `AddHandler` 陳述式有效而且拼寫正確。
