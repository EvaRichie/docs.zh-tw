# <a name="reference-required-to-module-modulename-containing-the-base-class-classname"></a>需要模組 ' ' （ \<modulename> 包含基類 ' '）的 \<classname> 參考

需要模組 ' ' （ \<modulename> 包含基類 ' '）的參考 \<classname> 。 請在專案中加入一個參考。  
  
 此類別是在專案中未直接參考的模組中所定義。 Visual Basic 編譯器需要參考，以避免當類別在多個模組中定義時不明確。  
  
 **錯誤 ID︰** BC30008  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請在您的專案參考中包含未參考之模組的名稱。
