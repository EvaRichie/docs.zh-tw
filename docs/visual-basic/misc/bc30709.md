# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>[即時運算] 視窗中的變數宣告陳述式無效

在 [即時運算] **** 視窗中不能使用變數宣告。  
  
 **錯誤 ID︰** BC30709  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請不要在 [即時運算] **** 視窗中宣告變數。
