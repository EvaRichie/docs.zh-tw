# <a name="source-file-filename-specified-multiple-times"></a>已經多次指定原始程式檔 \<filename>

 **錯誤 ID︰** BC2002  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除多餘的檔案規範。
