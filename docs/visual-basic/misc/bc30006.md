# <a name="reference-required-to-module-modulename-containing-the-definition-for-event-eventname"></a>需要模組 ' \<modulename> ' 的參考，其中包含事件 ' ' 的定義 \<eventname>

需要模組 ' <`modulename`> ' 的參考，其中包含事件 ' <`eventname`> ' 的定義。 請在專案中加入一個參考。  
  
 此事件是在專案中未直接參考的模組中所定義。 Visual Basic 編譯器需要參考，以避免事件在多個模組中定義時的不明確。  
  
 **錯誤 ID︰** BC30006  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請在您的專案參考中包含未參考之模組的名稱。
