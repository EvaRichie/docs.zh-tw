# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>因為沒有可利用這些引數呼叫之可存取的 '\<method>'，所以多載解析失敗：\<error>
您已呼叫多載的方法，但編譯器找不到具有相符引數清單的方法。  
  
 **錯誤 ID︰** BC30518  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查您提供的引數清單。
