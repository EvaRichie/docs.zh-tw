# <a name="stop-statement-encountered"></a>出現 Stop 陳述式

遇到暫停執行的 `Stop` 陳述式。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請移除 `Stop` 陳述式。
