# <a name="catch-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'Catch' 陳述式無效

`Catch` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID︰** BC30715  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從偵錯程式碼中移除 `Catch` 陳述式。
