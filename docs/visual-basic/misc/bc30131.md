# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>無法將安全屬性 '\<attributename>' 套用至模組

Common Language Runtime 不接受模組上的安全屬性。

**錯誤 ID︰** BC30131

## <a name="to-correct-this-error"></a>更正這個錯誤

- 請從宣告中移除屬性。
