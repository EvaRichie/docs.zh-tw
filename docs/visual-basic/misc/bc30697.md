# <a name="membername1-cannot-override-membername2-because-they-differ-by-the-types-of-optional-parameters"></a>' \<membername1> ' 無法覆寫 ' \<membername2> '，因為它們的選擇性參數類型不同
多載成員的一或多個必要參數必須具有不同的資料類型。  
  
 **錯誤 ID︰** BC30697  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請至少將一個必要的參數定義為不同的資料類型。
