# <a name="argument-argumentname-must-be-greater-than-zero"></a>引數 '\<argumentname>' 必須大於零
引數無效，因為它小於或等於零。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請檢查運算式中的引數拼法。 拼錯的變數名稱可能會隱含地建立初始化為零的數值變數。  
  
2. 請檢查運算式中先前的變數作業，特別是那些作為引數從其他程序傳遞至程序的變數。
