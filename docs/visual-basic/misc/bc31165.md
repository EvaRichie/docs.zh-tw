# <a name="expected-beginning--for-an-xml-tag"></a>XML 標記的開頭必須 \< 是 ' '
XML 常值的 XML 標記缺少必要的開頭 '<' 字元。  
  
 **錯誤 ID︰** BC31165  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請在 XML 常值的 XML 標記開頭加入 '<' 字元。
