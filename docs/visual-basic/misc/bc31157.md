# <a name="xml-attribute-attribute1-must-appear-before-xml-attribute-attribute2"></a>XML 屬性 'attribute1' 必須出現在 XML 屬性 'attribute2' 之前

XML 文件常值中的屬性指定順序無效。 XML 文件常值的有效屬性順序是以 XML 1.0 規格為基礎。 例如，XML 文件常值的 `encoding` 屬性必須緊接在 `version` 屬性之後。  
  
 **錯誤 ID︰** BC31157  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請使用 XML 1.0 規格中指定的指導方針，來更新 XML 文件常值的屬性順序。
