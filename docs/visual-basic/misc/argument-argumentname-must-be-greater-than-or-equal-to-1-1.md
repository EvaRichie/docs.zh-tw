# <a name="argument-argumentname-must-be-greater-than-or-equal-to--1"></a>引數 ' \<argumentname> ' 必須大於或等於-1
引數的值小於 -1。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確定引數的值落在有效範圍內。  
  
2. 如果透過計算來提供引數，請檢查計算所處理的值。
