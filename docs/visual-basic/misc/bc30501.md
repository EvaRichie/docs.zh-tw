# <a name="shared-cannot-be-combined-with-specifier-on-a-method-declaration"></a>'Shared' 無法在方法宣告上和 '\<specifier>' 結合

您嘗試合併 `Shared` 修飾詞與規範，例如 `Overridable`、 `NotOverridable`或 `MustOverride`，這種合併在方法宣告中無效。  
  
 **錯誤 ID︰** BC30501  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除規範。
