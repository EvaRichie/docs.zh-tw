# <a name="error-setting-assembly-manifest-option-error-message"></a>設定組件資訊清單選項時發生錯誤：\<error message>

Visual Basic 編譯器會呼叫元件連結器 ( # A0，也稱為 Alink) 來產生具有資訊清單的元件。 尋找用來簽署組件的金鑰檔或金鑰名稱時，連結器回報錯誤。  
  
 **錯誤 ID︰** BC30139  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 檢查引號的錯誤訊息，並查閱 [Al.exe (元件連結器) ](../../framework/tools/al-exe-assembly-linker.md) 上所列出的錯誤，以取得進一步的說明和建議。  
  
2. 如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
