# <a name="for-loop-not-initialized-visual-basic"></a>For 迴圈未初始化 (Visual Basic)
您已跳入 `For...Next` 迴圈的中間。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請移除跳躍到迴圈。
