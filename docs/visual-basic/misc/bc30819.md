# <a name="mathfunction1-is-not-declared"></a>' \<mathfunction1> ' 未宣告

未宣告 '\<mathfunction1>'。 函式已移至 system.string 類別，現在稱為 ' \<mathfunction2> '。  
  
 先前版本中 Visual Basic 內建的數個函式已移至 <xref:System.Math?displayProperty=nameWithType> 命名空間。 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰** BC30819  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請使用 <xref:System.Math?displayProperty=nameWithType>中宣告的方法。
