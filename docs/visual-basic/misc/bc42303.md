# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>XML 註解不得出現在方法或屬性中

XML 註解不得出現在方法或屬性中。 將會忽略 XML 註解。 XML 註解區塊不得出現在方法或屬性中。  
  
 **錯誤 ID︰** BC42303  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 移除註解區塊，或將其移至更適當的位置。
