# <a name="expected-closing--for-embedded-expression"></a>內嵌運算式必須以 '% > ' 結尾

XML 常值中的內嵌運算式遺漏必要的結尾 `%>` 字元。  
  
 **錯誤 ID︰** BC31159  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將必要的結尾 `%>` 字元加入內嵌運算式的結尾。
