# <a name="exit-operator-is-not-valid-use-return-to-exit-an-operator"></a>'Exit Operator' 無效。 請使用 'Return' 結束運算子

`Exit Operator` 程序中出現 `Operator` 陳述式。  
  
 您必須使用 [Return 語句](../language-reference/statements/return-statement.md) 從程式傳回 `Operator` 。 [Exit 語句](../language-reference/statements/exit-statement.md)不接受 `Operator` 關鍵字，而且語句不會將 `End Operator` 控制權傳回給呼叫程式碼。  
  
 **錯誤 ID︰** BC33008  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 以 `Exit Operator` 陳述式取代 `Return` 陳述式。
