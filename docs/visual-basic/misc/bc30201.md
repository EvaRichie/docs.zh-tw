# <a name="expression-expected"></a>必須有運算式

運算子 (例如 `+` 或 `=` ) 需要其右運算元有一個運算式，但未出現可辨識的運算式。  
  
 **錯誤 ID︰** BC30201  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 針對運算子的右運算元，提供有效的運算式。
