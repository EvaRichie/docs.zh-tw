# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>不再支援 'Variant' 類型。請使用 'Object' 類型代替

`Object` 資料類型取代 `Variant`。  
  
 **錯誤 ID︰** BC30804  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將項目宣告為 `Object`。
