# <a name="file-specified-by-filename-does-not-use-the-encoding-specified-by-fileencoding"></a>FileName 指定的檔案未使用 FileEncoding 指定的編碼方式

已嘗試讀取自檔案，但該檔案未使用指定的編碼方式。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確定您提供的檔案編碼方式正確。
