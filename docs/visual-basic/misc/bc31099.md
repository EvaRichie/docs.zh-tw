# <a name="property-accessors-cannot-be-declared-keyword"></a>屬性存取子不可宣告為 ' \<keyword> '
`Get` 或 `Set` 程序宣告中包含對屬性程序無效的關鍵字。  
  
 [Get 語句](../language-reference/statements/get-statement.md)和[Set 語句](../language-reference/statements/set-statement.md)只允許存取修飾詞（ `Public` 、 `Protected` 、 `Friend` 、 `Protected Friend` 、 `Private` ）。  
  
 **錯誤 ID︰** BC31099  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從 `Get` 或 `Set` 陳述式中移除無效的關鍵字。
