# <a name="property-propertyname-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-name"></a>屬性 ' \<propertyname> ' 必須宣告為 ' Default '，因為它會覆寫基底 ' ' 上的預設屬性 \<name>

程式碼中的 `Property` 陳述式會覆寫基底類別、結構或介面上的 `Default` 屬性，因此必須宣告為 `Default`。  
  
 **錯誤 ID︰** BC30360  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 宣告 `Default`屬性。
