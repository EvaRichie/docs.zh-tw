# <a name="argument-argumentname-cannot-be-converted-to-a-numeric-value"></a>無法將引數 '\<argumentname>' 轉換為數值
已嘗試將變數，例如字串，轉換為數值。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請尋找有意義的轉換。
