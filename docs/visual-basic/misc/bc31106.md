# <a name="property-accessors-cannot-be-declared-accessmodifier-in-a-notoverridable-property"></a><span data-ttu-id="8ac99-102">屬性存取子在 'NotOverridable' 屬性中不可以宣告為 '\<accessmodifier>'</span><span class="sxs-lookup"><span data-stu-id="8ac99-102">Property accessors cannot be declared '\<accessmodifier>' in a 'NotOverridable' property</span></span>

<span data-ttu-id="8ac99-103">屬性中的 [Get 語句](../language-reference/statements/get-statement.md) 或 [Set 語句](../language-reference/statements/set-statement.md) `NotOverridable` 包含 `Private` 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="8ac99-103">A [Get Statement](../language-reference/statements/get-statement.md) or [Set Statement](../language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="8ac99-104">下列幾行推論說明無法在 `NotOverridable` 中搭配使用 `Private` 和 [Private](../language-reference/statements/property-statement.md)的原因：</span><span class="sxs-lookup"><span data-stu-id="8ac99-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../language-reference/statements/property-statement.md):</span></span>  
  
1. <span data-ttu-id="8ac99-105">未覆寫基底類別屬性或程序的屬性或程序具有預設值 [NotOverridable](../language-reference/modifiers/notoverridable.md)。</span><span class="sxs-lookup"><span data-stu-id="8ac99-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../language-reference/modifiers/notoverridable.md).</span></span>  
  
2. <span data-ttu-id="8ac99-106">但是，覆寫基底類別屬性或程序之衍生類別中的屬性或程序具有預設值 [Overridable](../language-reference/modifiers/overridable.md)。</span><span class="sxs-lookup"><span data-stu-id="8ac99-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="8ac99-107">若要終止覆寫的階層架構，您可以將它宣告為 `NotOverridable`。</span><span class="sxs-lookup"><span data-stu-id="8ac99-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="8ac99-108">這是您可以使用 `NotOverridable`的唯一內容。</span><span class="sxs-lookup"><span data-stu-id="8ac99-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="8ac99-109">也就是說，您只能搭配 `NotOverridable` 使用 [NotOverridable](../language-reference/modifiers/overrides.md)。</span><span class="sxs-lookup"><span data-stu-id="8ac99-109">That is, you can use `NotOverridable` only in combination with [Overrides](../language-reference/modifiers/overrides.md).</span></span>  
  
3. <span data-ttu-id="8ac99-110">如果基底類別屬性或程序宣告為 [Private](../language-reference/modifiers/private.md)，衍生的類別就不能覆寫該屬性或程序，因為無法存取它。</span><span class="sxs-lookup"><span data-stu-id="8ac99-110">If a base class property or procedure is declared [Private](../language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="8ac99-111">因此，您無法搭配 `Private` 使用 `Overridable`。</span><span class="sxs-lookup"><span data-stu-id="8ac99-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4. <span data-ttu-id="8ac99-112">若要覆寫屬性或程序，覆寫屬性或程序不僅必須具有完全相同的簽章，也必須具有相同的存取層級。</span><span class="sxs-lookup"><span data-stu-id="8ac99-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="8ac99-113">這表示覆寫屬性或程序無法指定 `Private`，因為可覆寫的屬性或程序無法指定 `Private`。</span><span class="sxs-lookup"><span data-stu-id="8ac99-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5. <span data-ttu-id="8ac99-114">由於您只能在覆寫屬性或程序上指定 `NotOverridable` ，因此無法搭配 `Private`使用。</span><span class="sxs-lookup"><span data-stu-id="8ac99-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="8ac99-115">同理可證，覆寫屬性的個別屬性程序 (`Get` 和 `Set`) 不可為 `Private`。</span><span class="sxs-lookup"><span data-stu-id="8ac99-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>  
  
 <span data-ttu-id="8ac99-116">**錯誤 ID︰** BC31106</span><span class="sxs-lookup"><span data-stu-id="8ac99-116">**Error ID:** BC31106</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8ac99-117">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="8ac99-117">To correct this error</span></span>  
  
- <span data-ttu-id="8ac99-118">從 `Private` 或 `Get` 陳述式中移除 `Set` 關鍵字，或從 `Overrides` 陳述式中移除 `NotOverridable` 和 `Property` 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="8ac99-118">Remove the `Private` keyword from the `Get` or `Set` statement, or remove the `Overrides` and `NotOverridable` keywords from the `Property` statement.</span></span>
