# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>方法宣告陳述式必須是邏輯程式敘述行中的第一個陳述式
`Function` 或 `Sub` 陳述式接在冒號 (:) 陳述式分隔符號的後面。 `Function` 或 `Sub` 必須是其原始程式碼行唯一的陳述式。  
  
 **錯誤識別碼：** BC32009  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將多個陳述式分成不同的行。
