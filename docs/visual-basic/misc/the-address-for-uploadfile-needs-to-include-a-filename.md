# <a name="the-address-for-uploadfile-needs-to-include-a-filename"></a>UploadFile 的位址必須包含檔名
使用 `UploadFile` 方法上傳檔案時，您必須在目的地路徑中提供檔名。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請提供檔名。
