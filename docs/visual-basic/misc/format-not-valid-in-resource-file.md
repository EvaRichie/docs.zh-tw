# <a name="format-not-valid-in-resource-file"></a>資源檔中的格式無效

您專案中的資源檔未遵循 Windows 資源檔的標準格式，或資源檔與 Visual Basic 專案之間的不相符。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請重新建立資源來源和資源檔，並遵循 Windows 軟體開發套件中所述的語法。  
  
2. 請從專案移除無效的資源檔。
