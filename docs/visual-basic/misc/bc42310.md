# <a name="xml-comment-tag-include-must-have-a-attribute-attribute"></a>XML 註解標記 'include' 必須有 '\<attribute>' 屬性

XML 註解標記 ' include ' 必須有 \<attribute> 屬性。 將會忽略 XML 註解。  
  
 傳遞給 `file` Include `path` 標記的 `<`或`>` 屬性遺漏或不完整。  
  
 **錯誤 ID︰** BC42310  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 新增或完成標記的必要屬性 \<include> 。
