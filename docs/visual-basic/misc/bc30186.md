# <a name="using-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'Using' 陳述式無效
`Using` 陳述式只能用於原始程式碼中。  
  
 **錯誤 ID：** BC30186  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 從 [即時運算] `Using`**視窗移除** 陳述式。
