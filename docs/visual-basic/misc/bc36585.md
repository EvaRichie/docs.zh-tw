# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>' \<elementname> ' 不是 \<methodname> ' ' 中定義之擴充方法 ' ' 的 \<typename> 參數
程序呼叫可提供擴充方法未定義的其他引數。  
  
 **錯誤 ID︰** BC36585  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從引數清單中移除多餘的引數。
