# <a name="if-block-must-end-with-a-matching-end-if"></a>'#If' 區塊之後必須搭配相對應的 '#End If'

`#If` 是條件式編譯指示詞。 `#If` 區塊不是透過 `#End If` 指示詞所終止。  
  
 **錯誤 ID︰** BC30012  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將 `#End If` 指示詞加入條件式編譯區塊的結尾。
