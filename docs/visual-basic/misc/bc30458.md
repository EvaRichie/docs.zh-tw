# <a name="types-are-not-available-in-this-context"></a>在此內容中無法使用類型
您嘗試參考 `#If` 指示詞內的類型。  
  
 **錯誤 ID：** BC30458  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 移除參考。
