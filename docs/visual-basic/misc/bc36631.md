# <a name="join-expected"></a><span data-ttu-id="1d041-102">必須是 'Join'</span><span class="sxs-lookup"><span data-stu-id="1d041-102">'Join' expected</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="1d041-105">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="1d041-105">To correct this error</span></span>
