# <a name="type-typename-cannot-be-used-as-an-attribute-because-its-container-containername-is-not-declared-public"></a>無法將類型 '\<typename>' 當做屬性使用，因為其容器 '\<containername>' 未宣告為 'Public'

未使用 `Public` 修飾詞來宣告在其中定義這個屬性的類別或模組。 未指定存取修飾詞的類別和模組預設會宣告為 `Friend` 。  
  
 **錯誤 ID：** BC31517  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 將 `Public` 修飾詞加入在其中定義這個屬性的類別或模組。
