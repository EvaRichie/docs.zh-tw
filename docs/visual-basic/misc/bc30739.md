# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>即時運算視窗中的 'RaiseEvent' 陳述式無效
`RaiseEvent`**[即時**運算] 視窗中不能使用語句。  
  
 **錯誤 ID︰** BC30739  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 在 [程式碼編輯器] `RaiseEvent`**中，透過程式碼測試**陳述式。
