# <a name="typename-is-a-type-in-containername-and-cannot-be-used-as-an-expression"></a>' ' \<typename> 在 ' ' 中為類型 \<containername> ，無法當做運算式使用

正在使用的名稱之一已經宣告為類型名稱，不能用在運算式中。  
  
 **錯誤 ID：** BC30691  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 成員的唯一名稱。  
  
- 成員名稱請使用方括弧括住。
