# <a name="variable-variablename-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>為變數 '\<variablename>' 指派值之前，已使用該變數 (Visual Basic 錯誤)

為變數指派值之前，已使用該變數，導致 <xref:System.NullReferenceException>  
  
 **錯誤 ID︰** BC42109  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定已為變數指派值。
