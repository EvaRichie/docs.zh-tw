# <a name="definition-of-method-method-is-not-accessible-in-this-context"></a>在此內容中無法存取方法 \<method> 的定義

不存在的彙總函式已隨附於 `Into` 運算子。  
  
 **錯誤 ID︰** BC36594  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查彙總函式名稱的拼字，並進行任何必要的更正。  
  
- 請將方法以自訂彙總函式 (擴充方法) 加入。 如需有關建立自訂彙總函式的詳細資訊，請參閱 [Aggregate 子句](../language-reference/queries/aggregate-clause.md)。
