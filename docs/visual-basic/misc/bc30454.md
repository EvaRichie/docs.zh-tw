# <a name="expression-is-not-a-method"></a>運算式不是方法

方法呼叫的目標不是方法。  
  
 **錯誤 ID︰** BC30454  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請確定您未拼錯方法名稱。
