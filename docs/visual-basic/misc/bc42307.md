# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>XML 註解參數 '\<parameter>' 與對應的 '\<keyword>' 陳述式上之參數不相符
XML 批註參數 \<parameter> 與對應語句上的參數不相符 \<keyword> 。 將會忽略 XML 註解。  
  
 已針對方法的部分參數，但不是全部參數，指定方法的 XML 註解標記。  
  
 **錯誤識別碼：** BC42307  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請為方法中的所有參數提供 XML 註解標記。
