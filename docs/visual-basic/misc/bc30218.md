# <a name="lib-expected"></a>必須有 'Lib'
`Declare` 陳述式指定程式庫名稱，而不需要使用 `Lib` 關鍵字。  
  
 **錯誤 ID︰** BC30218  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請在程式庫名稱前面加入 `Lib` 關鍵字。
