# <a name="typename-values-cannot-be-converted-to-char"></a>' \<typename> ' 值無法轉換成 ' Char '

' \<typename> ' 值無法轉換為 Char。 使用 'Microsoft.VisualBasic.ChrW' 將數字值解譯為 Unicode 字元，或先將它轉換為 'String' 以產生數字。  
  
 運算式嘗試將資料類型轉換成 `String` 或 `Object` 以外的 `Char`。  
  
 **錯誤識別碼：** BC32007  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用 `ChrW` 函式將數值轉換成 Unicode 字元，或先將值轉換成 `String` 再轉換成 `Char`。
