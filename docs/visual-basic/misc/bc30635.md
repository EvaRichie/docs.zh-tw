# <a name="readonly-and-writeonly-cannot-be-combined"></a>無法結合 'ReadOnly' 和 'WriteOnly'

`ReadOnly` 與 `WriteOnly` 修飾詞互斥。  
  
 **錯誤 ID︰** BC30635  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請指定 `ReadOnly` 或 `WriteOnly`，但兩者不能同時指定。
