# <a name="unable-to-open-module-file-filename-error"></a>無法開啟模組檔 ' \<filename> '： \<error>
無法開啟包含此專案之模組的檔案。  
  
 **錯誤 ID︰** BC31007  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確認檔案位於指定的位置，且未被其他應用程式鎖定。
