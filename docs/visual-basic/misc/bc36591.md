# <a name="too-many-type-arguments-to-extension-method-methodname-defined-in-typename"></a>\<methodName>' ' 中定義之擴充方法 ' ' 的類型引數太多 \<typeName>
用來叫用泛型擴充方法的類型引數數目超過類型參數數目。  
  
 當您叫用泛型方法時，您必須針對該方法所定義的每個類型參數提供一個類型引數。  
  
 **錯誤 ID︰** BC36591  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從您的類型引數清單中移除類型引數，讓您叫用之泛型方法所定義的每個類型參數各有一個對應的類型引數。
