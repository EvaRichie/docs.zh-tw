# <a name="unable-to-sign-assembly-error-message"></a>無法簽署組件：\<error message>

Visual Basic 編譯器會呼叫元件連結器 ( # A0，也稱為 Alink) 來產生具有資訊清單的元件。 連結器在建立組件後進行簽署時回報了錯誤。 這可能是因為金鑰無效。  
  
 **錯誤 ID︰** BC30146  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 查看引號的錯誤訊息，並參閱  [Al.exe (元件連結) 器 ](../../framework/tools/al-exe-assembly-linker.md) 的主題，以取得進一步的說明和建議。  
  
2. 如果錯誤持續發生，請收集情況的相關資訊，並通知 Microsoft 產品支援服務。
