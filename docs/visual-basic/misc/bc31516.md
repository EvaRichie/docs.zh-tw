# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>無法將型別 '\<typename>' 當做屬性使用，因為它未宣告為 'Public'

屬性必須是公用的。  
  
 **錯誤 ID︰** BC31516  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請將 `Public` 修飾詞加入屬性類別。
