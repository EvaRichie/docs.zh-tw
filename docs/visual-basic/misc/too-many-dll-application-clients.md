# <a name="too-many-dll-application-clients"></a>太多 DLL 應用程式用戶端

Visual Basic 的動態連結程式庫 (DLL) 只能容納有限數目的主應用程式存取。 您的應用程式和其他 Visual Basic 主機的應用程式 (其中一些可由您的應用程式存取，) 會同時嘗試存取 Visual Basic DLL。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 減少存取 Visual Basic 的開啟應用程式數目。
