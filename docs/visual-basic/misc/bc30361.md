# <a name="property1-and-property2-cannot-overload-each-other-because-only-one-is-declared-default"></a>' ' \<property1> 和 ' \<property2> ' 無法互相多載，因為只有一個宣告為 ' Default '

如果屬性指定 `Default`，該名稱上多載的所有屬性也必須指定 `Default`。  
  
 **錯誤 ID︰** BC30361  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確定將所有多載的屬性宣告為 `Default`。
