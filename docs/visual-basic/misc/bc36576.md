<span data-ttu-id="077d4-103">在未指派給屬性名稱的情況下，無法推斷屬性名稱的值已包括在匿名類型宣告的初始設定式清單中。</span><span class="sxs-lookup"><span data-stu-id="077d4-103">A value from which a property name cannot be inferred has been included in the initializer list of an anonymous type declaration without being assigned to a property name.</span></span>
