# <a name="interface-interfacename-can-be-implemented-only-once-by-this-type"></a>介面 '\<interfacename>' 只能由此類型實作一次
指定的介面已經透過這個類型進行實作，因此無法重新進行實作。  
  
 **錯誤 ID︰** BC31033  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 只能使用這個類型實作這個介面一次。
