# <a name="unrecognized-option-optionname-ignored"></a>無法辨識的選項 \<optionname> ; 已忽略

已從命令列指定無效的選項。  
  
 **錯誤 ID︰** BC2007  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查指定的選項，來找出無效的選項。
