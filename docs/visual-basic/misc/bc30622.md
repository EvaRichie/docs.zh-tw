# <a name="end-module-must-be-preceded-by-a-matching-module"></a>'End Module' 之前必須搭配相對應的 'Module'
模組區塊會以 `Module` 陳述式開頭並以 `End Module` 建構結尾。  
  
 **錯誤 ID：** BC30622  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確定模組區塊以 `Module` 陳述式開頭。
