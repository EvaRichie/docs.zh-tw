# <a name="event-name-length-cannot-exceed-1011-characters"></a>事件名稱長度不能超過 1011 個字元
使用 `Event` 陳述式宣告的每個事件的名稱或識別項限制為 1011 個字元。  
  
 **錯誤 ID：** BC32204  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 減少識別項的長度。
