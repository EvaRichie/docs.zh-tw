# <a name="argument-argumentname-cannot-be-converted-to-type-typename"></a>引數 ' \<argumentname> ' 無法轉換成類型 ' \<typename> '

轉換會嘗試將引數變更成無效的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將類型變更為有效的類型。
