# <a name="operator-operator-must-have-a-second-parameter-of-type-integer"></a>運算子 '\<operator>' 的第二個參數類型必須為 'Integer'
位元移位運算子和不是 `Integer`的第二個類型參數一起宣告。  
  
 當您在運算式中使用右移 (`>>`) 或左移 (`<<`) 運算子時，您可以在第二個運算元中指定移位量。 對於這個運算元，Visual Basic 可讓您提供任何可擴大至的資料類型 `Integer` 。 不過，第二個運算元的定義嚴定為 `Integer`。 如果用類別或結構上的位元移位運算子定義該類別或結構，第二個運算元的定義必須指定 `Integer` 。  
  
 **錯誤識別碼：** BC33041  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 變更位元移位運算子的定義以傳回 `Integer` 值。
