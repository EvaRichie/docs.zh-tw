# <a name="addhandler-declaration-must-end-with-a-matching-end-addhandler"></a>'AddHandler' 宣告之後必須搭配相對應的 'End AddHandler'
`AddHandler` 宣告的結尾必須是 `End AddHandler` 陳述式。  
  
 **錯誤 ID︰** BC31115  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查 `AddHandler` 宣告的結尾是否為 `End AddHandler` 陳述式。
