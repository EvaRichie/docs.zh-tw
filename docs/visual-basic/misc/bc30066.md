# <a name="exit-property-is-not-valid-in-a-function-or-sub"></a>Function 或 Sub 中的 'Exit Property' 無效

`Exit Property` 出現在 `Function` 程序或 `Sub` 程序。 `Exit` 陳述式必須符合它發生所在的區塊。  
  
 **錯誤 ID︰** BC30066  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請視需要將 `Exit Property` 取代為 `Exit Function` 或 `Exit Sub` 陳述式。
