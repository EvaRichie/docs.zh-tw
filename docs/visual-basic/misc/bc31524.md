# <a name="systemruntimeinteropservicesdllimportattribute-cannot-be-applied-to-a-get-or-set"></a>'System.Runtime.InteropServices.DllImportAttribute' 無法套用至 'Get' 或 'Set'
`DllImportAttribute` 屬性已套用至 `Get` 或 `Set` 屬性程序。  
  
 **錯誤 ID︰** BC31524  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請從 `DllImportAttribute` 和 `Get` 屬性程序中移除 `Set` 。
