# <a name="operator-operatorname-is-not-defined-for-type-typename"></a>\<operatorname>未定義類型 ' ' 的運算子 ' ' \<typename>

您嘗試在未定義它的類型 (例如`+`) 上使用一元運算子 ( `-`、 `Not`或 `String`)。  
  
 **錯誤 ID︰** BC30487  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請將一元運算子取代為運算式的適當運算子。
