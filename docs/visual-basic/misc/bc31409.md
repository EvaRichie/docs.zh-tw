# <a name="method-methodname1-must-be-declared-overloads-because-another-methodname2-is-declared-overloads"></a>\<method>' \<methodname1> ' 必須宣告為 ' overloads '，因為另一個 ' \<methodname2> ' 已宣告為 ' overloads '
使用 `Overloads` 修飾詞宣告同名的另一個屬性或方法。  
  
 **錯誤 ID︰** BC31409  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 將 `Overloads` 修飾詞加入屬性或方法。  
  
2. 從同名的所有屬性或方法中移除 `Overloads` 修飾詞。
