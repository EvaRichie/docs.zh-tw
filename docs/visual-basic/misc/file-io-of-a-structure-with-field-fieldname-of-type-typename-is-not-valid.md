# <a name="file-io-of-a-structure-with-field-fieldname-of-type-typename-is-not-valid"></a>具有類型 ' ' 之欄位 ' ' 的結構的檔 i/o 無效 \<fieldname> \<typename>

您嘗試寫入檔案的結構包含不支援的類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查結構以決定不受支援類型的發生位置，然後將它變更為有效的類型。
