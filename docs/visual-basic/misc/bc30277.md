# <a name="type-character-charactername-does-not-match-declared-data-type-type"></a>類型字元 ' \<charactername> ' 不符合宣告的資料類型 ' \<type> '
將 `K$ = 10` 宣告為 `K` 時，變數是使用某種資料類型宣告，但參考時卻使用代表不相容資料類型的類型字元，例如 `Integer`。  
  
 **錯誤 ID：** BC30277  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 變更宣告變數的資料類型，或變更或移除參考中的類型字元。
