# <a name="end-raiseevent-must-be-preceded-by-a-matching-raiseevent-declaration"></a>'End RaiseEvent' 之前必須搭配相對應的 'RaiseEvent' 宣告
`End RaiseEvent` 陳述式沒有相對應的 `RaiseEvent` 陳述式。 `End RaiseEvent` 之前必須搭配相對應的 `RaiseEvent` 陳述式。  
  
 **錯誤 ID︰** BC31126  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請檢查前面的 `RaiseEvent` 陳述式是否有效且拼寫正確。
