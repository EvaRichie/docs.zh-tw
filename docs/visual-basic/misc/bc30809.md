# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>不再支援 'Wend' 陳述式；請改用 'End While' 陳述式。
`End While` 建構會取代 `Wend` 關鍵字。  
  
 **錯誤 ID︰** BC30809  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用 `End While` 結束 `While` 區塊。
