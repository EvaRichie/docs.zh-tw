# <a name="type-character-character-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>類型字元 '\<character>' 不可以使用在明確類型的宣告中

明確指定類型時使用了類型字元，例如 `$`。  
  
 **錯誤 ID︰** BC30302  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請從宣告移除類型字元，或移除 `As <Type>` 子句。
