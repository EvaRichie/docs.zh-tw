# <a name="cast-from-string-quotstringquot-to-type-typename-is-not-valid"></a>從字串轉換 &quot; \<string> &quot; 為類型 ' \<typename> ' 無效
嘗試將字串變數轉換成另一種類型不正確。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請檢查轉換中的類型，並確定您嘗試轉換成有意義的類型。
