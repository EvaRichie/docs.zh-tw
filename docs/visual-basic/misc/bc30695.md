# <a name="typename-membername-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>\<typename>' \<membername> ' 必須宣告為 ' shadows '，因為具有這個名稱的另一個成員已宣告為 ' shadows '
這個屬性或程序的名稱與另一個以 `Shadows` 修飾詞標記的屬性或程序相同。  
  
 **錯誤 ID︰** BC30695  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請使用 `Shadows` 修飾詞。
