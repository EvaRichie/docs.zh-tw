# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>\<methodname>已在介面 ' ' 中宣告方法 ' \<interfacename> '

您已第二次宣告介面方法，該介面方法重複。  
  
 **錯誤 ID︰** BC30377  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除重複的宣告。
