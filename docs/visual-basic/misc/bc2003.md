# <a name="response-file-filename-included-multiple-times"></a>回應檔 '\<filename>' 已經加入多次

已多次指定回應檔。 回應檔包含編譯器選項以及要編譯的原始程式碼檔。  
  
 **錯誤 ID︰** BC2003  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除多餘的回應檔規範。
