# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>' \<modulename> ' 是一個模組，無法參考為元件

嘗試將模組當做組件來使用。  
  
 **錯誤 ID︰** BC31076  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 變更組件的參考。
