# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="6de32-102">匿名類型成員或屬性 '\<propertyname>' 已宣告</span><span class="sxs-lookup"><span data-stu-id="6de32-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="6de32-103">屬性名稱只能在匿名類型的宣告中建立一次。</span><span class="sxs-lookup"><span data-stu-id="6de32-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="6de32-104">例如，下列宣告無效：</span><span class="sxs-lookup"><span data-stu-id="6de32-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="6de32-105">**錯誤 ID︰** BC36547</span><span class="sxs-lookup"><span data-stu-id="6de32-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6de32-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="6de32-106">To correct this error</span></span>  
  
- <span data-ttu-id="6de32-107">請為其中一個屬性選擇不同的名稱。</span><span class="sxs-lookup"><span data-stu-id="6de32-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.

```  
  
- <span data-ttu-id="6de32-108">請為您正在推斷名稱和值的變數或屬性名稱提供新名稱。</span><span class="sxs-lookup"><span data-stu-id="6de32-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.

```
