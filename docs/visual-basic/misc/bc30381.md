# <a name="finally-can-only-appear-once-in-a-try-statement"></a>'Finally' 只能在 'Try' 陳述式中出現一次
`Finally` 用來完成 `Try...Catch...Finally` 區塊，因此它只能出現在區塊結尾一次。  
  
 **錯誤 ID︰** BC30381  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請找到並移除多餘的 `Finally`。
