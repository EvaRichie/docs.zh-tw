<span data-ttu-id="7954c-103">`For` 迴圈指定類型的計數器變數，而這個變數未定義 `>=` 或具有其專屬類型之參數的 `<=` 運算子。</span><span class="sxs-lookup"><span data-stu-id="7954c-103">A `For` loop specifies a counter variable of a type that does not define the `>=` or `<=` operator with parameters of its own type.</span></span>  
  
 <span data-ttu-id="7954c-104">此計數器變數的類型必須支援可比較其包含類型的大於或等於 (`>=`) 和小於或等於 (`<=`) 運算子。</span><span class="sxs-lookup"><span data-stu-id="7954c-104">The counter variable must be of a type that supports greater-than-or-equal (`>=`) and less-than-or-equal (`<=`) operators that compare their containing type.</span></span> <span data-ttu-id="7954c-105">這表示這兩個運算元都必須與計數器變數同一類型。</span><span class="sxs-lookup"><span data-stu-id="7954c-105">This means both of the operands must be of the type of the counter variable.</span></span>  
  
 <span data-ttu-id="7954c-106">如果計數器變數使用了數值資料類型，則包含的類型也會支援 `>=` 和 `<=` 運算子。</span><span class="sxs-lookup"><span data-stu-id="7954c-106">If you use a numeric data type for the counter variable, the `>=` and `<=` operators are supported on the containing type.</span></span> <span data-ttu-id="7954c-107">如果您使用使用者定義的類別或結構，就必須使用您自己的類別或結構類型的運算元來定義這兩個運算子。</span><span class="sxs-lookup"><span data-stu-id="7954c-107">If you use a user-defined class or structure, you must define both operators with operands of the type of your class or structure.</span></span>  
  
 <span data-ttu-id="7954c-108">**錯誤 ID︰** BC33040</span><span class="sxs-lookup"><span data-stu-id="7954c-108">**Error ID:** BC33040</span></span>
