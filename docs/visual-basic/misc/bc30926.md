# <a name="mustinherit-cannot-be-specified-for-partial-type-partialtypename-because-it-cannot-be-combined-with-notinheritable-specified-for-one-of-its-other-partial-types"></a><span data-ttu-id="0daa2-102">不可為部分類型 '\<partialtypename>' 指定 'MustInherit'，因為其無法與針對某個其他部分類型所指定的 'NotInheritable' 相結合</span><span class="sxs-lookup"><span data-stu-id="0daa2-102">'MustInherit' cannot be specified for partial type '\<partialtypename>' because it cannot be combined with 'NotInheritable' specified for one of its other partial types</span></span>
<span data-ttu-id="0daa2-103">類別是在多個部分宣告中定義，其中一個指定 `MustInherit` ，另一個指定 `NotInheritable`。</span><span class="sxs-lookup"><span data-stu-id="0daa2-103">A class is defined in multiple partial declarations, one of which specifies `MustInherit` and another of which specifies `NotInheritable`.</span></span>  
  
 <span data-ttu-id="0daa2-104">當您分割數個部分宣告中的類別定義時，編譯器會將類別視為其所有部分宣告的聯集。</span><span class="sxs-lookup"><span data-stu-id="0daa2-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="0daa2-105">這不只適用於成員，同時也適用於實作、繼承和存取層級。</span><span class="sxs-lookup"><span data-stu-id="0daa2-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="0daa2-106">類別不能同時為 *「抽象」* (abstract) 和 *「密封」*(sealed)，亦即它不能同時要求和禁止繼承。</span><span class="sxs-lookup"><span data-stu-id="0daa2-106">A class cannot be both *abstract* and *sealed*, meaning it cannot both require and forbid inheritance.</span></span> <span data-ttu-id="0daa2-107">因此，同一類別無法同時指定 `MustInherit` 和 `NotInheritable` 。</span><span class="sxs-lookup"><span data-stu-id="0daa2-107">Therefore you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>
