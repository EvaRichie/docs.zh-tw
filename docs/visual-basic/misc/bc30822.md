# <a name="nullconstant-is-not-declared"></a>' \<nullconstant> ' 未宣告
未宣告 '\<nullconstant>'。 不再支援 Null 常數。請使用 System.DBNull 代替。  
  
 陳述式使用 Visual Basic 不再支援的 `Null` 關鍵字。  
  
 **錯誤 ID︰** BC30822  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 使用 <xref:System.DBNull> 取代 `Null`。 下列範例示範此作業。  
  
    ```vb  
    Sub TestDBNull()  
        Dim t As DataTable  
        ' Assume the DataGrid is bound to a DataTable.  
        t = CType(DataGrid1.DataSource, DataTable)  
        Dim r As DataRow  
        r = t.Rows(datagrid1.CurrentCell.RowNumber)  
        r.BeginEdit  
        r(1) = System.DBNull.Value ' Assign DBNull to the record.  
        r.EndEdit  
        r.AcceptChanges  
        If r.IsNull(1) Then  
            MsgBox("")  
        End If  
    End Sub  
    ```  
  
2. 當您使用物件變數時，請為指派和比較使用「[無](../language-reference/nothing.md)」關鍵字。 下列範例示範此作業。  
  
    ```vb  
    Sub TestNothing()  
        Dim cls As Object  
        ' cls is Nothing if it has not been assigned using the New keyword.  
        If (cls Is Nothing) Then  
            MsgBox("cls is Nothing")  
        End If  
        cls = Nothing ' Assign Nothing to the class variable cls.  
    End Sub  
    ```
