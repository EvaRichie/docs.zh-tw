# <a name="type-statements-are-no-longer-supported-use-structure-statements-instead"></a>不再支援 'Type' 陳述式。請使用 'Structure' 陳述式代替

您無法再使用 `Type` 關鍵字來定義使用者定義類型。  
  
 **錯誤 ID︰** BC30802  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 使用 `Structure` 陳述式來定義新的類型。
