# <a name="type1-typename-and-type2-typename-conflict-in-namespace-namespacename"></a>\<type1>' ' 中的 ' \<typename> ' 和 \<type2> ' \<typename> ' 衝突 \<namespace> \<namespacename>

兩個宣告的項目 (例如類別和模組) 在相同的命名空間中同名。  
  
 **錯誤 ID：** BC30179  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 將其中一個項目的宣告移至不同的命名空間。  
  
- 變更其中一個項目的名稱。
