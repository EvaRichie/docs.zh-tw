# <a name="extension-methods-must-declare-at-least-one-parameter"></a><span data-ttu-id="057fd-102">擴充方法必須宣告至少一個參數</span><span class="sxs-lookup"><span data-stu-id="057fd-102">Extension methods must declare at least one parameter</span></span>

<span data-ttu-id="057fd-103">擴充方法必須宣告至少一個參數。</span><span class="sxs-lookup"><span data-stu-id="057fd-103">Extension methods must declare at least one parameter.</span></span> <span data-ttu-id="057fd-104">第一個參數會指定要擴充的類型。</span><span class="sxs-lookup"><span data-stu-id="057fd-104">The first parameter specifies which type to extend.</span></span>  
  
 <span data-ttu-id="057fd-105">不含參數的擴充方法無效，因為第一個參數指定方法所擴充的資料類型。</span><span class="sxs-lookup"><span data-stu-id="057fd-105">An extension method without parameters is not valid because the first parameter specifies which data type the method extends.</span></span> <span data-ttu-id="057fd-106">第一個參數會繫結至叫用方法的資料類型執行個體。</span><span class="sxs-lookup"><span data-stu-id="057fd-106">The first parameter is bound to the instance of the data type that invokes the method.</span></span>  
  
 <span data-ttu-id="057fd-107">**錯誤 ID︰** BC36552</span><span class="sxs-lookup"><span data-stu-id="057fd-107">**Error ID:** BC36552</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="057fd-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="057fd-108">To correct this error</span></span>  
  
- <span data-ttu-id="057fd-109">加入方法所擴充之類型的參數。</span><span class="sxs-lookup"><span data-stu-id="057fd-109">Add a parameter of the type that your method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="057fd-110">範例</span><span class="sxs-lookup"><span data-stu-id="057fd-110">Example</span></span>  

 <span data-ttu-id="057fd-111">下列範例中的第一個參數表示 `Print` 方法可擴充 `String` 資料類型。</span><span class="sxs-lookup"><span data-stu-id="057fd-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```vb  
<Extension()> _  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="057fd-112">如下所示呼叫擴充方法時， `str` 方法中的參數會繫結至 `greeting`，也就是呼叫 `String` 的 `Print`執行個體。</span><span class="sxs-lookup"><span data-stu-id="057fd-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="057fd-113">編譯器將使用 `greeting` 作為擴充方法 `Print`的引數。</span><span class="sxs-lookup"><span data-stu-id="057fd-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>  
  
```vb  
Dim greeting As String = "Hello"  
greeting.Print()  
```
