# <a name="reference-required-to-assembly-assemblyname-containing-the-type-membername"></a>需要元件 ' ' （ \<assemblyname> 包含類型 ' \<membername> '）的參考

需要元件 ' ' （ \<assemblyname> 包含類型 ' '）的參考 \<membername> 。 請在專案中加入一個參考。  
  
 專案所含的類型取決於未包含在專案參考之組件的另一個類型。  
  
 **錯誤 ID：** BC30652  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請在專案中加入相依類型的專案參考。
