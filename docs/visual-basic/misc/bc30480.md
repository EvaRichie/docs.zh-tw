# <a name="shared-sub-new-cannot-be-declared-specifier"></a>共用的 ' Sub New ' 不可以宣告為 ' \<specifier> '
您已嘗試使用 `Shared Sub New` 宣告中的規範，這是無效的作業。  
  
 **錯誤 ID︰** BC30480  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除規範。
