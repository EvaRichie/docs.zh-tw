# <a name="argument-argumentname1-must-be-less-than-or-equal-the-length-of-argument-argumentname2"></a>引數 ' \<argumentname1> ' 必須小於或等於引數 ' ' 的長度 \<argumentname2>

提供的引數長度超過它正在取代的引數長度。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請讓第一個引數的長度小於或等於第二個引數的長度。
