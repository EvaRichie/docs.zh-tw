# <a name="type-or-with-expected"></a><span data-ttu-id="27cee-102">必須是類型或 'With'</span><span class="sxs-lookup"><span data-stu-id="27cee-102">Type or 'With' expected</span></span>
<span data-ttu-id="27cee-103">當您宣告類別的執行個體時， `New` 關鍵字後面必須接著類型名稱或 `With`。</span><span class="sxs-lookup"><span data-stu-id="27cee-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="27cee-104">例如，下列陳述式每個都宣告 `client` 為 `Customer` 類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="27cee-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="27cee-105">`Customer` 後面是類型名稱 `New`。</span><span class="sxs-lookup"><span data-stu-id="27cee-105">The type name `Customer` follows `New`.</span></span>  
  
```vb  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="27cee-106">從 Visual Basic 2008 開始，您可以將物件宣告為匿名型別的實例，在這種情況下，您不會指定資料類型。</span><span class="sxs-lookup"><span data-stu-id="27cee-106">Beginning with Visual Basic 2008, you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="27cee-107">在匿名類型宣告中，關鍵字 `With` 跟隨在 `New`的後面。</span><span class="sxs-lookup"><span data-stu-id="27cee-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```vb  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```
