# <a name="lsetstmt-is-not-declared"></a>' \<lsetstmt> ' 未宣告

未宣告 '\<lsetstmt>'。 不再支援  LSet 陳述式。請使用 Microsoft.VisualBasic.LSet 代替。  
  
 先前版本中 Visual Basic 內建的數個函式已移至 <xref:Microsoft.VisualBasic?displayProperty=nameWithType> 命名空間。 因此其功能可更廣泛地供所有程式設計語言使用。  
  
 **錯誤 ID︰** BC30820  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 改為在 `LSet` 命名空間中使用 `Microsoft.VisualBasic` 函式。
