# <a name="public-member-membername-on-type-typename-not-found"></a>在類型 '\<typename>' 上找不到 Public 成員 '\<membername>'

找不到您嘗試存取的成員。 存取晚期繫結物件的非公用成員時，就會發生這個錯誤。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 如果成員存在，請確定它已宣告為 `Public`。
