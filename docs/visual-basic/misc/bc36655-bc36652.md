# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments-because-more-than-one-type-is-possible"></a><span data-ttu-id="70d02-102">因為可能的類型不只一種，所以無法從這些引數推斷定義在 '\<typename>' 中擴充方法 '\<methodname>' 內類型參數的資料類型</span><span class="sxs-lookup"><span data-stu-id="70d02-102">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible</span></span>

<span data-ttu-id="70d02-103">無法從這些引數推斷在 ' ' 中定義之擴充方法 ' ' 內類型參數的資料類型 \<methodname> \<typename> ，因為可能會有一個以上的類型。</span><span class="sxs-lookup"><span data-stu-id="70d02-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="70d02-104">明確指定資料類型或許可以更正這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="70d02-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="70d02-105">嘗試使用類型推斷來判斷泛型擴充方法呼叫中之類型參數的類型。</span><span class="sxs-lookup"><span data-stu-id="70d02-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="70d02-106">編譯器發現一或多個類型參數的多個可能資料類型，並報告這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="70d02-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="70d02-107">當無法指定引數時 (例如，針對查詢運算式中的查詢運算子)，會顯示不含第二句的錯誤訊息。</span><span class="sxs-lookup"><span data-stu-id="70d02-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="70d02-108">下列程式碼示範這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="70d02-108">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Imports System.Runtime.CompilerServices
Module Module1
    Sub Main()

        Dim caller As New Class1
        '' Not valid.
        'caller.targetExtension(1, "2")

    End Sub

    <Extension()> _
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```

<span data-ttu-id="70d02-109">**錯誤識別碼：** BC36655 （在 LINQ 查詢中）和 BC36652 （在查詢之外）</span><span class="sxs-lookup"><span data-stu-id="70d02-109">**Error ID:** BC36655 (within LINQ queries) and BC36652 (outside queries)</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="70d02-110">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="70d02-110">To correct this error</span></span>

<span data-ttu-id="70d02-111">如果錯誤出現在查詢外，請嘗試明確指定類型參數的資料類型：</span><span class="sxs-lookup"><span data-stu-id="70d02-111">If the error appears outside of a query, try specifying the data type of the type parameter or parameters explicitly:</span></span>

