# <a name="xml-comment-type-parameter-parameter-does-not-match-a-type-parameter-on-the-corresponding-keyword-statement"></a>XML 註解類型參數 \<parameter> 與對應的 \<keyword> 陳述式上之類型參數不相符
XML 批註參數 \<parameter> 與對應語句上的類型參數不相符 \<keyword> 。 將會忽略 XML 註解。  
  
 已針對方法的部分類別參數，但不是全部類別參數，指定方法的 XML 註解標記。  
  
 **錯誤 ID︰** BC42317  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請為方法中的所有類型參數提供 XML 註解標記。
