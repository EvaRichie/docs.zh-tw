# <a name="declare-statements-in-a-structure-cannot-be-declared-specifier"></a>結構中的 ' Declare ' 語句不可以宣告為 ' \<specifier> '
`Declare` 宣告的規範在 `Structure` 宣告內無效。  
  
 **錯誤 ID︰** BC30791  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除規範。
