# <a name="filename-cannot-be-referenced-because-it-is-not-an-assembly"></a>\<filename>無法參考 ' '，因為它不是元件

已對不是 .NET 組件的檔案進行參考。  
  
 **錯誤 ID︰** BC31519  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確認組件的名稱和位置。  
  
2. 請確認組件未損毀。
