# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-for-some-type-arguments"></a><span data-ttu-id="9bd61-102">無法繼承介面 '\<interfacename1>'，因為它所繼承的介面 '\<interfacename2>' 可能與某些類型引數之介面 '\<interfacename3>' 相同</span><span class="sxs-lookup"><span data-stu-id="9bd61-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' for some type arguments</span></span>
<span data-ttu-id="9bd61-103">泛型介面繼承自兩個以上的泛型介面，而且兩個繼承的類型引數之特定值可能發生衝突。</span><span class="sxs-lookup"><span data-stu-id="9bd61-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="9bd61-104">下列陳述式可能會產生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="9bd61-104">The following statements can generate this error.</span></span>  
  
```vb  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="9bd61-105">如果建構或實作 `derivedInterface` 時針對 `t1` 和 `t2`提供相同的類型，其一定會繼承含相同類型引數之 `interfaceX` 的兩個版本。</span><span class="sxs-lookup"><span data-stu-id="9bd61-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="9bd61-106">這樣會造成無法確定存取哪一個版本的情況。</span><span class="sxs-lookup"><span data-stu-id="9bd61-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="9bd61-107">**錯誤 ID：** BC32121</span><span class="sxs-lookup"><span data-stu-id="9bd61-107">**Error ID:** BC32121</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9bd61-108">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="9bd61-108">To correct this error</span></span>  
  
- <span data-ttu-id="9bd61-109">變更其中一個提供給衍生介面的類型引數，使其沒有衝突。</span><span class="sxs-lookup"><span data-stu-id="9bd61-109">Change one of the type arguments supplied to the derived interface so that there is no conflict.</span></span>  
  
     <span data-ttu-id="9bd61-110">-或-</span><span class="sxs-lookup"><span data-stu-id="9bd61-110">-or-</span></span>
