# <a name="character-character-0xhexadecimal-is-not-allowed-at-the-beginning-of-an-xml-name"></a>XML 名稱的開頭不可為字元 'character' (0x\<hexadecimal>)

XML 常值或 XML 軸屬性名稱開頭包含不支援的字元。  
  
 **錯誤 ID︰** BC31169  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除 XML 常值或 XML 軸屬性名稱開頭中不支援的字元。
