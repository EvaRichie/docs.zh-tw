# <a name="path-pathname-not-found"></a>找不到路徑 '\<pathname>'

作業系統在進行檔案存取或磁碟存取作業時找不到指定的路徑。 檔案的路徑必須包含磁碟機指定，以及必須周遊以找出檔案的目錄和子目錄。 路徑可為相對或絕對路徑。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請確定路徑中包含必要的項目。
