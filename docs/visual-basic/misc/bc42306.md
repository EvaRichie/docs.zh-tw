# <a name="xml-comment-tag-tag-is-not-permitted-on-a-element-language-element"></a>'\<element>' 語言項目上不可使用 XML 註解標記 '\<tag>'
\<tag>' ' 元素上不允許 XML 註解標記 \<element> 。 將會忽略 XML 註解。  
  
 語言項目不支援套用它的 XML 註解標記。  
  
 **錯誤 ID︰** BC42306  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 請移除不支援的 XML 註解標記。
