<span data-ttu-id="35369-103">如果變數用於在宣告中建立其初始值，則編譯器無法推斷變數的資料類型。</span><span class="sxs-lookup"><span data-stu-id="35369-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>

