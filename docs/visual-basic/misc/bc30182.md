# <a name="type-expected"></a>預期的類型
發現無法辨識為類型名稱的原始程式碼字元。 這個內容呼叫基本資料類型或代表類型的已宣告項目名稱。  
  
 **錯誤 ID︰** BC30182  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
- 確認已定義類型。  
  
- 請檢查類型名稱拼寫正確。
