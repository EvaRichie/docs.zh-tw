 示範如何撥接連接到電腦序列埠的數據機。  
  
 [作法：將字串傳送至序列埠](how-to-send-strings-to-serial-ports.md)  
 示範如何將字串傳送至電腦的序列埠。  
  
 [作法：接收來自序列埠的字串](how-to-receive-strings-from-serial-ports.md)  
 示範如何接收來自電腦序列埠的字串。
