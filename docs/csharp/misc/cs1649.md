# <a name="compiler-error-cs1649"></a><span data-ttu-id="4f7dc-103">編譯器錯誤 CS1649</span><span class="sxs-lookup"><span data-stu-id="4f7dc-103">Compiler Error CS1649</span></span>

<span data-ttu-id="4f7dc-104">無法傳遞 ref 或 out 給唯讀欄位 'identifier' 的成員 (除非是在建構函式中)</span><span class="sxs-lookup"><span data-stu-id="4f7dc-104">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="4f7dc-105">如果您將變數傳遞給某個函式，該函式是以 `readonly` 或 `ref` 引數作為 `out` 欄位的成員，就會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="4f7dc-105">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="4f7dc-106">由於該函式可修改 `ref` 和 `out` 參數，因此不允許這種情況。</span><span class="sxs-lookup"><span data-stu-id="4f7dc-106">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="4f7dc-107">若要解決此錯誤，請移除欄位的 `readonly` 關鍵字，或不要將 `readonly` 欄位的成員傳遞給函式。</span><span class="sxs-lookup"><span data-stu-id="4f7dc-107">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="4f7dc-108">例如，您可以嘗試建立可修改的暫存變數，並將其當作 `ref` 引數傳遞，如下列範例所示。</span><span class="sxs-lookup"><span data-stu-id="4f7dc-108">For example, you might try creating a temporary variable which can be modified and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4f7dc-109">範例</span><span class="sxs-lookup"><span data-stu-id="4f7dc-109">Example</span></span>  

 <span data-ttu-id="4f7dc-110">下列範例會產生 CS1649：</span><span class="sxs-lookup"><span data-stu-id="4f7dc-110">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
