# <a name="compiler-error-cs0156"></a><span data-ttu-id="38ed9-103">編譯器錯誤 CS0156</span><span class="sxs-lookup"><span data-stu-id="38ed9-103">Compiler Error CS0156</span></span>
<span data-ttu-id="38ed9-104">最內層 catch 子句中巢狀 finally 子句不允許沒有引數的 throw 陳述式</span><span class="sxs-lookup"><span data-stu-id="38ed9-104">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="38ed9-105">沒有任何參數的 [throw](../language-reference/keywords/throw.md) 陳述式只會出現在不接受任何參數的 **catch** 子句中。</span><span class="sxs-lookup"><span data-stu-id="38ed9-105">A [throw](../language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="38ed9-106">如需詳細資訊，請參閱 [例外狀況和例外狀況處理](../programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="38ed9-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="38ed9-107">下列範例會產生 CS0156：</span><span class="sxs-lookup"><span data-stu-id="38ed9-107">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
