# <a name="compiler-error-cs1715"></a><span data-ttu-id="133b2-102">編譯器錯誤 CS1715</span><span class="sxs-lookup"><span data-stu-id="133b2-102">Compiler Error CS1715</span></span>

<span data-ttu-id="133b2-103">'Type1' : 類型必須是 'Type2' 以符合覆寫的成員 'MemberName'</span><span class="sxs-lookup"><span data-stu-id="133b2-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="133b2-104">這個錯誤與 [Compiler Error CS0508](./cs0508.md)相同，只不過 CS0508 現在只適用於具有傳回類型的方法，而 CS1715 則適用於只有 'types' 沒有 'return types' 的屬性與索引子。</span><span class="sxs-lookup"><span data-stu-id="133b2-104">This error is the same as [Compiler Error CS0508](./cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="133b2-105">範例</span><span class="sxs-lookup"><span data-stu-id="133b2-105">Example</span></span>  

 <span data-ttu-id="133b2-106">下列程式碼會產生 CS1715。</span><span class="sxs-lookup"><span data-stu-id="133b2-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
