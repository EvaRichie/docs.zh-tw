# <a name="compiler-error-cs0202"></a><span data-ttu-id="f6b06-103">編譯器錯誤 CS0202</span><span class="sxs-lookup"><span data-stu-id="f6b06-103">Compiler Error CS0202</span></span>
<span data-ttu-id="f6b06-104">foreach 要求 'type.GetEnumerator()' 的傳回類型 'type' 必須要有適合的公用 MoveNext 方法以及公用 Current 屬性</span><span class="sxs-lookup"><span data-stu-id="f6b06-104">foreach requires that the return type 'type' of 'type.GetEnumerator()' must have a suitable public MoveNext method and public Current property</span></span>  
  
 <span data-ttu-id="f6b06-105"><xref:System.Collections.IEnumerable.GetEnumerator%2A> 函式，用來啟用 foreach 陳述式的使用，無法傳回指標或陣列；它必須傳回可作為列舉值之類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="f6b06-105">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="f6b06-106">作為列舉值的適當需求包括公用 Current 屬性和公用 MoveNext 方法。</span><span class="sxs-lookup"><span data-stu-id="f6b06-106">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="f6b06-107">在 C# 2.0 中，編譯器會自動為您產生 Current 和 MoveNext。</span><span class="sxs-lookup"><span data-stu-id="f6b06-107">In C# 2.0, the compiler will automatically generate Current and MoveNext for you.</span></span> <span data-ttu-id="f6b06-108">如需詳細資訊，請參閱 [泛型介面](../programming-guide/generics/generic-interfaces.md)中的程式碼範例。</span><span class="sxs-lookup"><span data-stu-id="f6b06-108">For more information, see the code example in [Generic Interfaces](../programming-guide/generics/generic-interfaces.md).</span></span>  
  
 <span data-ttu-id="f6b06-109">下列範例會產生 CS0202：</span><span class="sxs-lookup"><span data-stu-id="f6b06-109">The following sample generates CS0202:</span></span>  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
