<span data-ttu-id="1b5e4-104">'member' 無法同時為 extern 並宣告為主體</span><span class="sxs-lookup"><span data-stu-id="1b5e4-104">'member' cannot be extern and declare a body</span></span>  
  
 <span data-ttu-id="1b5e4-105">類別成員標記為 [extern](../language-reference/keywords/extern.md)時，表示成員的定義位於另一個檔案中。</span><span class="sxs-lookup"><span data-stu-id="1b5e4-105">When a class member is marked [extern](../language-reference/keywords/extern.md), it means that the member's definition is located in another file.</span></span> <span data-ttu-id="1b5e4-106">因此，標記為 **extern** 的類別成員不能定義於類別中。</span><span class="sxs-lookup"><span data-stu-id="1b5e4-106">Therefore, a class member marked as **extern** cannot be defined in the class.</span></span> <span data-ttu-id="1b5e4-107">請移除 `extern` 關鍵字，或刪除定義。</span><span class="sxs-lookup"><span data-stu-id="1b5e4-107">Either remove the `extern` keyword or delete the definition.</span></span> <span data-ttu-id="1b5e4-108">如需詳細資訊，請參閱[方法](../programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="1b5e4-108">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="1b5e4-109">下列範例會產生 CS0179：</span><span class="sxs-lookup"><span data-stu-id="1b5e4-109">The following sample generates CS0179:</span></span>

```csharp  
   // public extern int ExternMethod(int aa);

   {  
   }  
}  
```
