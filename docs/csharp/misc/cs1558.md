# <a name="compiler-error-cs1558"></a><span data-ttu-id="44506-103">編譯器錯誤 CS1558</span><span class="sxs-lookup"><span data-stu-id="44506-103">Compiler Error CS1558</span></span>
<span data-ttu-id="44506-104">'class' 沒有合適的靜態 Main 方法</span><span class="sxs-lookup"><span data-stu-id="44506-104">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="44506-105">[-Main](../language-reference/compiler-options/main-compiler-option.md)編譯器選項指定了要在其中尋找**main**方法的類別。</span><span class="sxs-lookup"><span data-stu-id="44506-105">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="44506-106">不過， [Main](../programming-guide/main-and-command-args/index.md) 方法未正確定義。</span><span class="sxs-lookup"><span data-stu-id="44506-106">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="44506-107">下列範例會產生 CS1558，因為傳回類型無效。</span><span class="sxs-lookup"><span data-stu-id="44506-107">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
