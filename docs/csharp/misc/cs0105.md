# <a name="compiler-warning-level-3-cs0105"></a>編譯器警告 (層級 3) CS0105

'namespace' 的 using 指示詞之前在此命名空間出現過  
  
 [namespace](../language-reference/keywords/namespace.md)，應該只宣告一次，但卻宣告了一次以上，因此請移除所有重複的命名空間宣告。  
  
 下列範例會產生 CS0105：  
  
```csharp  
// CS0105.cs  
// compile with: /W:3  
using System;  
using System;   // CS0105  
  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
```
