# <a name="compiler-error-cs0534"></a><span data-ttu-id="1c946-103">編譯器錯誤 CS0534</span><span class="sxs-lookup"><span data-stu-id="1c946-103">Compiler Error CS0534</span></span>
<span data-ttu-id="1c946-104">'function1' 未實作繼承的抽象成員 'function2'</span><span class="sxs-lookup"><span data-stu-id="1c946-104">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="1c946-105">需要有類別，才能實作基底類別中的所有 [abstract](../language-reference/keywords/abstract.md) 成員，除非該類別也是 abstract。</span><span class="sxs-lookup"><span data-stu-id="1c946-105">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="1c946-106">下列範例會產生 CS0534：</span><span class="sxs-lookup"><span data-stu-id="1c946-106">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
