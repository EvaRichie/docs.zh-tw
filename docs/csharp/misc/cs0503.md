 <span data-ttu-id="07d3b-105">不需要將成員方法標記為 [abstract](../language-reference/keywords/abstract.md) 和 [virtual](../language-reference/keywords/virtual.md) ，因為 **abstract** 即表示 **virtual**。</span><span class="sxs-lookup"><span data-stu-id="07d3b-105">It is redundant to mark a member method as both [abstract](../language-reference/keywords/abstract.md) and [virtual](../language-reference/keywords/virtual.md) because **abstract** implies **virtual**.</span></span>  
  
 <span data-ttu-id="07d3b-106">下列範例會產生 CS0503：</span><span class="sxs-lookup"><span data-stu-id="07d3b-106">The following sample generates CS0503:</span></span>

      abstract virtual public void f();   // CS0503
