<span data-ttu-id="26d6b-104">屬性存取子已定義</span><span class="sxs-lookup"><span data-stu-id="26d6b-104">Property accessor already defined</span></span>  
  
 <span data-ttu-id="26d6b-105">當您宣告 [屬性](../programming-guide/classes-and-structs/using-properties.md)時，您也必須宣告其存取子方法。</span><span class="sxs-lookup"><span data-stu-id="26d6b-105">When you declare a [property](../programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="26d6b-106">不過，屬性不能有多個 `get` 存取子方法或多個 `set` 存取子方法。</span><span class="sxs-lookup"><span data-stu-id="26d6b-106">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="26d6b-107">範例</span><span class="sxs-lookup"><span data-stu-id="26d6b-107">Example</span></span>  

 <span data-ttu-id="26d6b-108">下列範例會產生 CS1007：</span><span class="sxs-lookup"><span data-stu-id="26d6b-108">The following sample generates CS1007:</span></span>  
  
```csharp

{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
