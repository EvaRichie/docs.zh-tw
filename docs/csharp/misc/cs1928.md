# <a name="compiler-error-cs1928"></a><span data-ttu-id="45207-102">編譯器錯誤 CS1928</span><span class="sxs-lookup"><span data-stu-id="45207-102">Compiler Error CS1928</span></span>

<span data-ttu-id="45207-103">'Type' 不包含 'method' 的定義，而且最佳的擴充方法多載 'method' 有一些無效的引數。</span><span class="sxs-lookup"><span data-stu-id="45207-103">'Type' does not contain a definition for 'method' and the best extension method overload 'method' has some invalid arguments.</span></span>  
  
 <span data-ttu-id="45207-104">編譯器找不到具有所呼叫之方法名稱的類別成員時，會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="45207-104">This error is produced when the compiler cannot find a class member with the name of the method you have called.</span></span> <span data-ttu-id="45207-105">它可以找到具有該名稱的擴充方法，但未包含符合透過方法呼叫傳入之類型的簽章。</span><span class="sxs-lookup"><span data-stu-id="45207-105">It can find an extension method with that name, but not with a signature that matches the types you passed in with your method call.</span></span>

## <a name="example"></a><span data-ttu-id="45207-108">範例</span><span class="sxs-lookup"><span data-stu-id="45207-108">Example</span></span>  

 <span data-ttu-id="45207-109">下列程式碼會產生 CS1928：</span><span class="sxs-lookup"><span data-stu-id="45207-109">The following code generates CS1928:</span></span>  
  
```csharp

class Test  
{  
    static void Main()  
    {  
        Test t = new Test();  
        t.M("hi"); // CS1928  
    }  
}

{  
    public static void M(this Test t, int i)  
    {  
    }  
}  
```
