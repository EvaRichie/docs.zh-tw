# <a name="compiler-error-cs1672"></a>編譯器錯誤 CS1672

/platform 的 'option' 選項無效; 必須是 anycpu、x86、Itanium 或 x64  
  
 這些選項會指定處理器類型；請使用其中一個列出的格式。
