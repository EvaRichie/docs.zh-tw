# <a name="compiler-error-cs0193"></a><span data-ttu-id="6d312-103">編譯器錯誤 CS0193</span><span class="sxs-lookup"><span data-stu-id="6d312-103">Compiler Error CS0193</span></span>

<span data-ttu-id="6d312-104">\*或-> 運算子必須套用至指標</span><span class="sxs-lookup"><span data-stu-id="6d312-104">The \* or -> operator must be applied to a pointer</span></span>  
  
<span data-ttu-id="6d312-105">\*或-> 運算子搭配指標類型使用。</span><span class="sxs-lookup"><span data-stu-id="6d312-105">The \* or -> operator was used with a nonpointer type.</span></span> <span data-ttu-id="6d312-106">如需詳細資訊，請參閱[指標類型](../programming-guide/unsafe-code-pointers/pointer-types.md)。</span><span class="sxs-lookup"><span data-stu-id="6d312-106">For more information, see [Pointer types](../programming-guide/unsafe-code-pointers/pointer-types.md).</span></span>  
  
 <span data-ttu-id="6d312-107">下列範例會產生 CS0193：</span><span class="sxs-lookup"><span data-stu-id="6d312-107">The following sample generates CS0193:</span></span>  
  
```csharp  
// CS0193.cs  
using System;  
  
public struct Age  
{  
   public int AgeYears;  
   public int AgeMonths;  
   public int AgeDays;  
}  
  
public class MyClass  
{  
   public static void SetAge(ref Age anAge, int years, int months, int days)  
   {  
      anAge->Months = 3;   // CS0193, anAge is not a pointer  
      // try the following line instead  
      // anAge.AgeMonths = 3;  
   }  
  
   public static void Main()  
   {  
      Age MyAge = new Age();  
      Console.WriteLine(MyAge.AgeMonths);  
      SetAge(ref MyAge, 22, 4, 15);  
      Console.WriteLine(MyAge.AgeMonths);  
   }  
}  
```
