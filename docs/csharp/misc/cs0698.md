# <a name="compiler-error-cs0698"></a>編譯器錯誤 CS0698
泛型類型不能衍生自 'class'，因為它是屬性類別  
  
 任何衍生自屬性類別的類別皆必須為屬性。 屬性不可以是泛型類型。  
  
 下列範例會產生 CS0698：  
  
```csharp  
// CS0698.cs  
class C<T> : System.Attribute  // CS0698  
{  
}  
```
