# <a name="compiler-error-cs1604"></a>編譯器錯誤 CS1604

無法指派給 'variable'，因為它是唯讀的  
  
 已對唯讀變數進行指派。 若要避免這個錯誤，請不要指派或嘗試修改這個內容中的這個變數。
