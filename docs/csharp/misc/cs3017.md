# <a name="compiler-warning-level-1-cs3017"></a>編譯器警告 (層級 1) CS3017

在模組上指定的 CLSCompliant 屬性不能與組件上的 CLSCompliant 屬性不同  
  
 如果您所擁有之組件的 CLSCompliant 屬性與模組的 CLSCompliant 屬性衝突，就會發生這個警告。 符合 CLS 標準的組件不能包含不符合 CLS 標準的模組。 若要解決這個警告，請確定組件和模組的 CLSCompliant 屬性同時為 true 或 false，或是移除其中一個屬性。 如需 CLS 合規性的詳細資訊，請參閱 [語言獨立性和與語言無關的元件](../../standard/language-independence.md)。
  
## <a name="example"></a>範例  

 下列範例會產生 CS3017：  
  
```csharp  
// CS3017.cs  
// compile with: /target:module  
  
using System;  
  
[module: CLSCompliant(true)]  
[assembly: CLSCompliant(false)]  // CS3017  
// Try this line instead:  
// [assembly: CLSCompliant(true)]  
class C  
{  
    static void Main() {}  
}  
```
