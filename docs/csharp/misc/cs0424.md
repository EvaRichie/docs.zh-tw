<span data-ttu-id="caf50-104">'class': 具有 ComImport 屬性的類別不能指定基底類別</span><span class="sxs-lookup"><span data-stu-id="caf50-104">'class': a class with the ComImport attribute cannot specify a base class</span></span>  
  
 <span data-ttu-id="caf50-105">指定 <xref:System.Runtime.InteropServices.ComImportAttribute> 屬性表示是要從 COM 模組匯入類別的實作。</span><span class="sxs-lookup"><span data-stu-id="caf50-105">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="caf50-106">不允許將繼承自基底類別的其他方法或欄位加入 COM 模組中所定義的實作。</span><span class="sxs-lookup"><span data-stu-id="caf50-106">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>  
  
 <span data-ttu-id="caf50-107">下列範例會產生 CS0424：</span><span class="sxs-lookup"><span data-stu-id="caf50-107">The following sample generates CS0424:</span></span>

