## <a name="example"></a><span data-ttu-id="f6bdb-109">範例</span><span class="sxs-lookup"><span data-stu-id="f6bdb-109">Example</span></span>  

 <span data-ttu-id="f6bdb-110">下列程式碼會產生 CS1914，因為初始設定式嘗試初始化 `TestClass.Number` 欄位，它是 `static`：</span><span class="sxs-lookup"><span data-stu-id="f6bdb-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
