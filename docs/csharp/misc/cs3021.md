<span data-ttu-id="63171-103">'type' 不需要 CLSCompliant 屬性，因為組件沒有 CLSCompliant 屬性</span><span class="sxs-lookup"><span data-stu-id="63171-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="63171-104">如果 `[CLSCompliant(false)]` 出現在元件層級 CLSCompliant 屬性未設定為 true (也就是) 行的元件中，則會發生這個警告 `[assembly: CLSCompliant(true)]` 。</span><span class="sxs-lookup"><span data-stu-id="63171-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="63171-105">因為組件未將它自己宣告為符合 CLS 標準，所以組件內的任何項目都不需要將它自己宣告為不符合標準，因為它是假設為不符合標準。</span><span class="sxs-lookup"><span data-stu-id="63171-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="63171-106">如需 CLS 合規性的詳細資訊，請參閱 [語言獨立性和與語言無關的元件](../../standard/language-independence.md)。</span><span class="sxs-lookup"><span data-stu-id="63171-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>

## <a name="example"></a><span data-ttu-id="63171-108">範例</span><span class="sxs-lookup"><span data-stu-id="63171-108">Example</span></span>  

 <span data-ttu-id="63171-109">下列範例會產生 CS3021：</span><span class="sxs-lookup"><span data-stu-id="63171-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
