# <a name="compiler-error-cs1562"></a>編譯器錯誤 CS1562

沒有來源的輸出必須有指定的 /out 選項  
  
 此編譯可建立輸出檔，但沒有作為輸入的原始程式碼檔，以從中隱含輸出檔的名稱。 例如，您可能嘗試編譯中繼資料或資源專用檔。  
  
 使用 [-out](../language-reference/compiler-options/out-compiler-option.md) 編譯器選項來指定輸出檔的名稱。
