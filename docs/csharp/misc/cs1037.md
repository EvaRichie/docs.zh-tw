# <a name="compiler-error-cs1037"></a>編譯器錯誤 CS1037

必須是可多載的運算子  
  
 使用 [-doc](../language-reference/compiler-options/doc-compiler-option.md)指定批註時，編譯器會遇到不正確連結。
