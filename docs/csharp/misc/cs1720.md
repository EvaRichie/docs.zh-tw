# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="ccf2d-102">編譯器警告 (層級 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="ccf2d-102">Compiler Warning (level 1) CS1720</span></span>

<span data-ttu-id="ccf2d-103">運算式一律會造成 System.NullReferenceException，因為 'generic type' 的預設值是 null</span><span class="sxs-lookup"><span data-stu-id="ccf2d-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="ccf2d-104">如果您撰寫的運算式涉及的泛型類型變數預設值為參考類型 (例如類別)，則會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="ccf2d-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="ccf2d-105">以下列運算式為例：</span><span class="sxs-lookup"><span data-stu-id="ccf2d-105">Consider the following expression:</span></span>  
  
```csharp  
default(T).ToString()  
```  
  
 <span data-ttu-id="ccf2d-106">由於 `T` 是參考類型，其預設值為 null，因此嘗試對其套用 <xref:System.Object.ToString%2A> 方法時會擲回 <xref:System.NullReferenceException>。</span><span class="sxs-lookup"><span data-stu-id="ccf2d-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ccf2d-107">範例</span><span class="sxs-lookup"><span data-stu-id="ccf2d-107">Example</span></span>  

 <span data-ttu-id="ccf2d-108">針對類型 `T` 的類別參考條件約束可確保 `T` 是參考類型。</span><span class="sxs-lookup"><span data-stu-id="ccf2d-108">The class reference constraint on type `T` ensures that `T` is a reference type.</span></span>  
  
 <span data-ttu-id="ccf2d-109">下列範例會產生 CS1720：</span><span class="sxs-lookup"><span data-stu-id="ccf2d-109">The following sample generates CS1720.</span></span>  
  
```csharp  
// CS1720.cs  
using System;  
public class Tester
{  
    public static void GenericClass<T>(T t1) where T : class
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
