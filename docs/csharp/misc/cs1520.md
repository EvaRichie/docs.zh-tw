# <a name="compiler-error-cs1520"></a><span data-ttu-id="25560-103">編譯器錯誤 CS1520</span><span class="sxs-lookup"><span data-stu-id="25560-103">Compiler Error CS1520</span></span>
<span data-ttu-id="25560-104">方法必須要有傳回類型</span><span class="sxs-lookup"><span data-stu-id="25560-104">Method must have a return type</span></span>  
  
 <span data-ttu-id="25560-105">在類別、結構或介面中宣告的方法必須有明確的傳回類型。</span><span class="sxs-lookup"><span data-stu-id="25560-105">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="25560-106">在下列範例中，Square 方法具有 [字串](../language-reference/builtin-types/reference-types.md)的傳回值：</span><span class="sxs-lookup"><span data-stu-id="25560-106">In the following example, the Square method has a return value of [string](../language-reference/builtin-types/reference-types.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="25560-107">下列範例會產生 CS1520：</span><span class="sxs-lookup"><span data-stu-id="25560-107">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="25560-108">或者，當建構函式名稱的大小寫與類別或結構宣告的大小寫不同時，可能會發生此錯誤，如下列範例所示。</span><span class="sxs-lookup"><span data-stu-id="25560-108">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="25560-109">因為名稱不完全與類別名稱相同，編譯器會將它解譯為一般方法，而不是建構函式，因此產生錯誤：</span><span class="sxs-lookup"><span data-stu-id="25560-109">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
