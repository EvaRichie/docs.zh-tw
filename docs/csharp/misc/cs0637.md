# <a name="compiler-error-cs0637"></a><span data-ttu-id="51d94-103">編譯器錯誤 CS0637</span><span class="sxs-lookup"><span data-stu-id="51d94-103">Compiler Error CS0637</span></span>

<span data-ttu-id="51d94-104">Static 或 const 欄位不能有 FieldOffset 屬性。</span><span class="sxs-lookup"><span data-stu-id="51d94-104">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="51d94-105">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 屬性不能用於標上 [static](../language-reference/keywords/static.md) 或 [const](../language-reference/keywords/const.md)的欄位。</span><span class="sxs-lookup"><span data-stu-id="51d94-105">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../language-reference/keywords/static.md) or [const](../language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="51d94-106">下列範例會產生 CS0637：</span><span class="sxs-lookup"><span data-stu-id="51d94-106">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
