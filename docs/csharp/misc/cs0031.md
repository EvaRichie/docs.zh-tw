# <a name="compiler-error-cs0031"></a><span data-ttu-id="55f63-103">編譯器錯誤 CS0031</span><span class="sxs-lookup"><span data-stu-id="55f63-103">Compiler Error CS0031</span></span>

<span data-ttu-id="55f63-104">無法將常數值 'value' 轉換為 'type'</span><span class="sxs-lookup"><span data-stu-id="55f63-104">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="55f63-105">嘗試將值指派給其類型無法儲存值的變數。</span><span class="sxs-lookup"><span data-stu-id="55f63-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="55f63-106">如需詳細資訊，請參閱[類型](../programming-guide/types/index.md)。</span><span class="sxs-lookup"><span data-stu-id="55f63-106">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="55f63-107">下列範例會在已核對及未核對的內容中產生 CS0031：</span><span class="sxs-lookup"><span data-stu-id="55f63-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
