## <a name="example"></a><span data-ttu-id="72999-106">範例</span><span class="sxs-lookup"><span data-stu-id="72999-106">Example</span></span>  

 <span data-ttu-id="72999-107">下列範例會產生 CS0472。</span><span class="sxs-lookup"><span data-stu-id="72999-107">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;
        return counter;  
    }  
}  
```
