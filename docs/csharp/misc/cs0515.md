# <a name="compiler-error-cs0515"></a>編譯器錯誤 CS0515
'function': 在靜態建構函式中不允許有存取修飾詞  
  
 靜態建構函式不能有 [存取修飾詞](../language-reference/keywords/index.md)。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0515：  
  
```csharp  
// CS0515.cs  
public class Clx  
{  
    public static void Main()  
    {  
    }  
}  
  
public class Clz  
{  
    public static Clz()   // CS0515, remove public keyword  
    {  
    }  
}  
```
