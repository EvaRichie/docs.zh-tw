## <a name="example"></a><span data-ttu-id="4e2e7-112">範例</span><span class="sxs-lookup"><span data-stu-id="4e2e7-112">Example</span></span>

<span data-ttu-id="4e2e7-113">下列程式碼會產生 CS0473：</span><span class="sxs-lookup"><span data-stu-id="4e2e7-113">The following code generates CS0473:</span></span>

```csharp
public interface ITest<T>
{
    int TestMethod(int i);
    int TestMethod(T i);
}

public class ImplementingClass : ITest<int>
{
    int ITest<int>.TestMethod(int i) // CS0473
    {
        return i + 1;
    }

    public int TestMethod(int i)
    {
        return i - 1;
    }
}

class T
{
    static int Main()
    {
        ImplementingClass a = new ImplementingClass();
        if (a.TestMethod(0) != -1)
            return -1;

        ITest<int> i_a = a;
        System.Console.WriteLine(i_a.TestMethod(0).ToString());
        if (i_a.TestMethod(0) != 1)
            return -1;

        return 0;
    }
}
```
