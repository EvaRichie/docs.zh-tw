# <a name="compiler-error-cs0738"></a><span data-ttu-id="8076e-103">編譯器錯誤 CS0738</span><span class="sxs-lookup"><span data-stu-id="8076e-103">Compiler Error CS0738</span></span>

<span data-ttu-id="8076e-104">'type name' 未實作介面成員 'member name'。</span><span class="sxs-lookup"><span data-stu-id="8076e-104">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="8076e-105">'method name' 無法實作 'interface member'，因為它沒有符合的傳回類型 'type name'。</span><span class="sxs-lookup"><span data-stu-id="8076e-105">'method name' cannot implement 'interface member' because it does not have the matching return type of ' type name'.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="8076e-107">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="8076e-107">To correct this error</span></span>  
  
1. <span data-ttu-id="8076e-108">請變更方法的傳回類型，使其符合介面成員的傳回類型。</span><span class="sxs-lookup"><span data-stu-id="8076e-108">Change the return type of the method to match that of the interface member.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8076e-109">範例</span><span class="sxs-lookup"><span data-stu-id="8076e-109">Example</span></span>  

 <span data-ttu-id="8076e-110">因為類別方法傳回 `void` ，而且同名的介面成員傳回 `int`，所以下列程式碼會產生 CS0738：</span><span class="sxs-lookup"><span data-stu-id="8076e-110">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>  
  
```csharp  
using System;  
  
interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
