# <a name="compiler-error-cs1529"></a><span data-ttu-id="915de-103">編譯器錯誤 CS1529</span><span class="sxs-lookup"><span data-stu-id="915de-103">Compiler Error CS1529</span></span>

 <span data-ttu-id="915de-105">[using](../language-reference/keywords/using.md) 子句必須在命名空間中第一個出現。</span><span class="sxs-lookup"><span data-stu-id="915de-105">A [using](../language-reference/keywords/using.md) clause must appear first in a namespace.</span></span>  
  
## <a name="example"></a><span data-ttu-id="915de-106">範例</span><span class="sxs-lookup"><span data-stu-id="915de-106">Example</span></span>  
 <span data-ttu-id="915de-107">下列範例會產生 CS1529：</span><span class="sxs-lookup"><span data-stu-id="915de-107">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
