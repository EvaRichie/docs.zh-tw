# <a name="compiler-error-cs0514"></a><span data-ttu-id="2c143-103">編譯器錯誤 CS0514</span><span class="sxs-lookup"><span data-stu-id="2c143-103">Compiler Error CS0514</span></span>

<span data-ttu-id="2c143-104">'constructor'：靜態建構函式不能有明確的 'this' 或 'base' 建構函式呼叫</span><span class="sxs-lookup"><span data-stu-id="2c143-104">'constructor' : static constructor cannot have an explicit 'this' or 'base' constructor call</span></span>  
  
 <span data-ttu-id="2c143-105">不允許在靜態建構函式中呼叫 `this` ，因為在建立任何類別執行個體之前會自動呼叫靜態建構函式。</span><span class="sxs-lookup"><span data-stu-id="2c143-105">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="2c143-106">此外，靜態建構函式不會被繼承，也不能直接呼叫。</span><span class="sxs-lookup"><span data-stu-id="2c143-106">Also, static constructors are not inherited, and cannot be called directly.</span></span>  
  
 <span data-ttu-id="2c143-107">如需詳細資訊，請參閱 [此](../language-reference/keywords/this.md) 和 [基底](../language-reference/keywords/base.md)。</span><span class="sxs-lookup"><span data-stu-id="2c143-107">For more information, see [this](../language-reference/keywords/this.md) and [base](../language-reference/keywords/base.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="2c143-108">範例</span><span class="sxs-lookup"><span data-stu-id="2c143-108">Example</span></span>  

 <span data-ttu-id="2c143-109">下例會產生 CS0514：</span><span class="sxs-lookup"><span data-stu-id="2c143-109">The following example generates CS0514:</span></span>  
  
```csharp  
// CS0514.cs  
class A  
{  
    static A() : base(0) // CS0514  
    {  
    }  
  
    public A(object o)  
    {  
    }  
}  
  
class B  
{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
