## <a name="example"></a><span data-ttu-id="5f9f3-109">範例</span><span class="sxs-lookup"><span data-stu-id="5f9f3-109">Example</span></span>  

 <span data-ttu-id="5f9f3-110">下列程式碼會產生 CS0314：</span><span class="sxs-lookup"><span data-stu-id="5f9f3-110">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
