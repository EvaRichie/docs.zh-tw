# <a name="compiler-error-cs0216"></a><span data-ttu-id="dc5f7-103">編譯器錯誤 CS0216</span><span class="sxs-lookup"><span data-stu-id="dc5f7-103">Compiler Error CS0216</span></span>
<span data-ttu-id="dc5f7-104">運算子 'operator' 的配對運算子 'missing_operator' 也需要定義</span><span class="sxs-lookup"><span data-stu-id="dc5f7-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="dc5f7-105">使用者定義的 [true](../language-reference/operators/true-false-operators.md) 運算子需要使用者定義的 [false](../language-reference/operators/true-false-operators.md) 運算子，反之亦然。</span><span class="sxs-lookup"><span data-stu-id="dc5f7-105">A user-defined [true](../language-reference/operators/true-false-operators.md) operator requires a user-defined [false](../language-reference/operators/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="dc5f7-106">下列範例會產生 CS0216：</span><span class="sxs-lookup"><span data-stu-id="dc5f7-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
