<span data-ttu-id="6c1c9-104">運算子 'operator' 的配對運算子 'missing_operator' 也需要定義</span><span class="sxs-lookup"><span data-stu-id="6c1c9-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="6c1c9-105">使用者定義 [==](../language-reference/operators/equality-operators#equality-operator-) 運算子需要使用者定義的 [！ =](../language-reference/operators/inequality-operators#inequality-operator-) 運算子，反之亦然。</span><span class="sxs-lookup"><span data-stu-id="6c1c9-105">A user-defined [==](../language-reference/operators/equality-operators#equality-operator-) operator requires a user-defined [!=](../language-reference/operators/inequality-operators#inequality-operator-) operator, and vice versa.</span></span>  
 <span data-ttu-id="6c1c9-106">同樣也適用于使用者定義的 [true](../language-reference/operators/true-false-operators.md) 運算子和使用者定義的 [false](../language-reference/operators/true-false-operators.md) 運算子。</span><span class="sxs-lookup"><span data-stu-id="6c1c9-106">The same applies also to a user-defined [true](../language-reference/operators/true-false-operators.md) operator and a user-defined [false](../language-reference/operators/true-false-operators.md) operator.</span></span>  
  
 <span data-ttu-id="6c1c9-107">下列範例會產生 CS0216：</span><span class="sxs-lookup"><span data-stu-id="6c1c9-107">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator == (MyClass MyIntLeft, MyClass MyIntRight)   // CS0216  
   {  
      return MyIntLeft == MyIntRight;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator != (MyClass MyIntLeft, MyClass MyIntRight)  
   {  
      return MyIntLeft != MyIntRight;  
   }  
   */  
  
   public override bool Equals (object obj)  
   {  
      return base.Equals (obj);  
   }  
  
   public override int GetHashCode()  
   {  
      return base.GetHashCode();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
