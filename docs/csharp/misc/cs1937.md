## <a name="example"></a><span data-ttu-id="dab82-110">範例</span><span class="sxs-lookup"><span data-stu-id="dab82-110">Example</span></span>  

 <span data-ttu-id="dab82-111">下列範例會產生 CS1937：</span><span class="sxs-lookup"><span data-stu-id="dab82-111">The following example generates CS1937.</span></span>  
  
```csharp  
// cs1937.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] sourceA = { 1, 2, 3, 4, 5 };  
        int[] sourceB = { 3, 4, 5, 6, 7 };  
  
        var query = from a in sourceA  
                    join b in sourceB on b equals a // CS1937  
                    // Try the following line instead.  
                    //join b in sourceB on a equals b  
                    select new { a, b };  
    }  
}  
```
