## <a name="example"></a><span data-ttu-id="0015f-106">範例</span><span class="sxs-lookup"><span data-stu-id="0015f-106">Example</span></span>  

 <span data-ttu-id="0015f-107">下列範例會產生 CS0730。</span><span class="sxs-lookup"><span data-stu-id="0015f-107">The following sample generates CS0730.</span></span> <span data-ttu-id="0015f-108">它是由兩個原始程式檔所組成。</span><span class="sxs-lookup"><span data-stu-id="0015f-108">It consists of two source files.</span></span> <span data-ttu-id="0015f-109">會先編譯程式庫檔案 `CS0730a.cs`，接著編譯參考程式庫檔案的檔案 `CS0730.cs`</span><span class="sxs-lookup"><span data-stu-id="0015f-109">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
