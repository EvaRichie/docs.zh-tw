# <a name="compiler-error-cs1900"></a>編譯器錯誤 CS1900

警告層級必須介於範圍 0 到 4 之間  
  
 [-警告](../language-reference/compiler-options/warn-compiler-option.md)編譯器選項只能採用五個可能值的其中一個 (0、1、2、3或 4) 。 任何其他傳遞給 **/warn** 的值都會導致 CS1900。  
  
 下列範例會產生 CS1900：  
  
```csharp  
// CS1900.cs  
// compile with: /W:5  
// CS1900 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
