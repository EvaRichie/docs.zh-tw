# <a name="compiler-error-cs0308"></a><span data-ttu-id="b07a4-103">編譯器錯誤 CS0308</span><span class="sxs-lookup"><span data-stu-id="b07a4-103">Compiler Error CS0308</span></span>
<span data-ttu-id="b07a4-104">非泛型類型或方法 'identifier' 不能配合類型引數使用。</span><span class="sxs-lookup"><span data-stu-id="b07a4-104">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="b07a4-105">方法或類型不是泛型，但搭配類型引數使用。</span><span class="sxs-lookup"><span data-stu-id="b07a4-105">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="b07a4-106">若要避免這個錯誤，請移除角括弧和類型引數，或是將方法或類型重新宣告為泛型方法或類型。</span><span class="sxs-lookup"><span data-stu-id="b07a4-106">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="b07a4-107">下列範例會產生 CS0308：</span><span class="sxs-lookup"><span data-stu-id="b07a4-107">The following example generates CS0308:</span></span>  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="b07a4-108">下列範例也會產生 CS0308。</span><span class="sxs-lookup"><span data-stu-id="b07a4-108">The following example also generates CS0308.</span></span> <span data-ttu-id="b07a4-109">若要解決此錯誤，請使用指示詞 "using System.Collections.Generic"。</span><span class="sxs-lookup"><span data-stu-id="b07a4-109">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
