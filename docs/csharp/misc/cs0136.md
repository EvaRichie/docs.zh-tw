## <a name="example"></a><span data-ttu-id="a590b-107">範例</span><span class="sxs-lookup"><span data-stu-id="a590b-107">Example</span></span>

```csharp

   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }
```  
