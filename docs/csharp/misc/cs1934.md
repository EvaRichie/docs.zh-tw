## <a name="example"></a><span data-ttu-id="19713-110">範例</span><span class="sxs-lookup"><span data-stu-id="19713-110">Example</span></span>  

 <span data-ttu-id="19713-111">下列範例會示範產生 CS1934的一種方法：</span><span class="sxs-lookup"><span data-stu-id="19713-111">The following example shows one way to produce CS1934:</span></span>  
  
```csharp

using System.Collections;

{  
    public static void Main()  
    {
```  
