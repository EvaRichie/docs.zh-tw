## <a name="example"></a><span data-ttu-id="1b4d5-108">範例</span><span class="sxs-lookup"><span data-stu-id="1b4d5-108">Example</span></span>  
 <span data-ttu-id="1b4d5-109">下列程式碼會產生 CS0754：</span><span class="sxs-lookup"><span data-stu-id="1b4d5-109">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
