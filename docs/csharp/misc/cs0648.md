# <a name="compiler-error-cs0648"></a>編譯器錯誤 CS0648
此語言不支援類型 'type'  
  
 可能是從 C++ 的另一種語言產生的中繼資料，包含未標記為Managed 的類型。 此類型的相關資訊會包含在中繼資料，但是類型不適用於以 C# 撰寫的程式。
