# <a name="compiler-error-cs1608"></a>編譯器錯誤 CS1608

C# 類型上不允許出現 Required 屬性  
  
 以 C# 程式碼定義的類型上不允許<xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> 。
