# <a name="compiler-error-cs0682"></a>編譯器錯誤 CS0682
'type1' 不可實作 'type2'，因為此語言不支援它  
  
 如果您嘗試實作以另一種語言撰寫的介面，而且編譯器不支援該介面，則會發生這個錯誤。
