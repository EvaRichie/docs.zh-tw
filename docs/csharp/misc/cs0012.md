# <a name="compiler-error-cs0012"></a>編譯器錯誤 CS0012
類型 'type' 定義在未被參考的組件中。 您必須加入對組件 'assembly' 的參考。  
  
 找不到參考類型的定義。 如果必要的 DLL 檔案未包含在編譯中，就會發生此狀況。 如需詳細資訊，請參閱 [加入參考對話方塊](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) 和 [-參考 (c # 編譯器選項) ](../language-reference/compiler-options/reference-compiler-option.md)。  
  
 下列編譯序列將會導致 CS0012：  
  
```csharp  
// cs0012a.cs  
// compile with: /target:library  
public class A {}  
```  
  
 然後：  
  
```csharp  
// cs0012b.cs  
// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 然後：  
  
```csharp  
// cs0012c.cs  
// compile with: /reference:cs0012b.dll  
class C  
{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```  
  
 您可以使用 `/reference:cs0012b.dll;cs0012a.dll`編譯來解決這個 CS0012，或在 Visual Studio 中使用 [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) 來加入 cs0012b.dll 和 cs0012a.dll 的參考。
