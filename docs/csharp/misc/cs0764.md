 <span data-ttu-id="45f89-105">部分方法是由定義宣告 (特徵標記) 和選擇性的實作宣告 (主體) 所組成。</span><span class="sxs-lookup"><span data-stu-id="45f89-105">A partial method consists of a defining declaration (signature) and an optional implementing declaration (body).</span></span> <span data-ttu-id="45f89-106">如果定義宣告有 `unsafe` 修飾詞，實作宣告就必須也有。</span><span class="sxs-lookup"><span data-stu-id="45f89-106">If the defining declaration has the `unsafe` modifier, the implementing declaration must also have it.</span></span> <span data-ttu-id="45f89-107">反之，如果實作宣告有 `unsafe` 修飾詞，定義宣告也必須有。</span><span class="sxs-lookup"><span data-stu-id="45f89-107">Conversely, if the implementing declaration has the `unsafe` modifier, the defining declaration must also.</span></span>
