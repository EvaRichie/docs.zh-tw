## <a name="example"></a><span data-ttu-id="dd949-106">範例</span><span class="sxs-lookup"><span data-stu-id="dd949-106">Example</span></span>  

 <span data-ttu-id="dd949-107">下列範例會產生 C3015。</span><span class="sxs-lookup"><span data-stu-id="dd949-107">The following sample generates C3015.</span></span>  
  
```csharp  
// CS3015.cs  
// compile with: /target:library

[assembly:CLSCompliant(true)]  
public class MyAttribute : Attribute  
{  
   public MyAttribute(int[] ai) {}   // CS3015  
   // try the following line instead  
   // public MyAttribute(int ai) {}  
}  
```
