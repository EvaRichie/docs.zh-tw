# <a name="compiler-error-cs0535"></a>編譯器錯誤 CS0535
'class' 未實作介面成員 'member'  
  
 [類別](../language-reference/keywords/class.md) 衍生自 [介面](../language-reference/keywords/interface.md)，但是類別未實作一個或多個介面的成員。 類別必須實作從中衍生或宣告為 `abstract`之介面的所有成員。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0535。  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0535。  
  
```csharp  
// CS0535_b.cs  
using System;  
class C : IDisposable {}   // CS0535  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
