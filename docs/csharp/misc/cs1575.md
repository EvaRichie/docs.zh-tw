# <a name="compiler-error-cs1575"></a><span data-ttu-id="4abd1-103">編譯器錯誤 CS1575</span><span class="sxs-lookup"><span data-stu-id="4abd1-103">Compiler Error CS1575</span></span>

<span data-ttu-id="4abd1-104">stackalloc 運算式在類型之後需要有 []</span><span class="sxs-lookup"><span data-stu-id="4abd1-104">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="4abd1-105">要求的配置大小與 [stackalloc](../language-reference/operators/stackalloc.md)，必須在方括號中指定。</span><span class="sxs-lookup"><span data-stu-id="4abd1-105">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="4abd1-106">下例會產生 CS1575：</span><span class="sxs-lookup"><span data-stu-id="4abd1-106">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
