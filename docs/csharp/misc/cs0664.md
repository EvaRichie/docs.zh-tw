# <a name="compiler-error-cs0664"></a><span data-ttu-id="2ad78-103">編譯器錯誤 CS0664</span><span class="sxs-lookup"><span data-stu-id="2ad78-103">Compiler Error CS0664</span></span>

<span data-ttu-id="2ad78-104">不能隱含將類型 double 的常值轉換為類型 'type'; 請使用 'suffix' 後置字元來建立此類型的常值</span><span class="sxs-lookup"><span data-stu-id="2ad78-104">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="2ad78-105">無法完成指派；請使用後置字元來更正指令。</span><span class="sxs-lookup"><span data-stu-id="2ad78-105">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="2ad78-106">每個類型的文件可識別類型的對應後置字元。</span><span class="sxs-lookup"><span data-stu-id="2ad78-106">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="2ad78-107">如需轉換的詳細資訊，請參閱轉換 [和類型轉換](../programming-guide/types/casting-and-type-conversions.md)。</span><span class="sxs-lookup"><span data-stu-id="2ad78-107">For more information on conversions, see [Casting and Type Conversions](../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="2ad78-108">下列範例會產生 CS0664：</span><span class="sxs-lookup"><span data-stu-id="2ad78-108">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
