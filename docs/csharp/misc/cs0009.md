# <a name="compiler-error-cs0009"></a>編譯器錯誤 CS0009

無法開啟中繼資料檔 'file' — 'description'  
  
 以 [-reference](../language-reference/compiler-options/reference-compiler-option.md) 編譯器選項指定的檔案未包含有效的中繼資料。
