# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="b4f75-103">編譯器警告 (層級 3) CS0168</span><span class="sxs-lookup"><span data-stu-id="b4f75-103">Compiler Warning (level 3) CS0168</span></span>

<span data-ttu-id="b4f75-104">變數 ' var ' 已宣告但從未使用過</span><span class="sxs-lookup"><span data-stu-id="b4f75-104">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="b4f75-105">當您宣告變數但不使用它時，編譯器會發出層級三警告。</span><span class="sxs-lookup"><span data-stu-id="b4f75-105">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="b4f75-106">下列範例會產生 CS0168 警告：</span><span class="sxs-lookup"><span data-stu-id="b4f75-106">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
