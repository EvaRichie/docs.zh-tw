# <a name="compiler-error-cs1518"></a>編譯器錯誤 CS1518
必須是類別、委派、列舉、介面或結構  
  
 在 [命名空間](../language-reference/keywords/namespace.md)中，找到不支援的宣告。 在命名空間內，編譯器只會接受類別、結構、列舉、介面、命名空間和委派。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS1518：  
  
```csharp  
// CS1518.cs  
namespace x  
{  
   sealed class c1 {};      // OK  
   namespace f2 {};         // OK  
   sealed f3 {};            // CS1518  
}  
```
