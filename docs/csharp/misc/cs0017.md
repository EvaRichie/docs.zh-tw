<span data-ttu-id="d3ddd-104">程式 'output file name' 有一個以上的已定義進入點。</span><span class="sxs-lookup"><span data-stu-id="d3ddd-104">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="d3ddd-105">請以 /main 編譯以指定包含進入點的類型。</span><span class="sxs-lookup"><span data-stu-id="d3ddd-105">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="d3ddd-106">程式只能有一個 [Main](../programming-guide/main-and-command-args/index.md) 方法。</span><span class="sxs-lookup"><span data-stu-id="d3ddd-106">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="d3ddd-107">若要解決這個錯誤，您可以刪除程式碼中的所有主要方法（除了其中一個），或使用 [-main](../language-reference/compiler-options/main-compiler-option.md) 編譯器選項來指定您要使用的主要方法。</span><span class="sxs-lookup"><span data-stu-id="d3ddd-107">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="d3ddd-108">下列範例會產生 CS0017：</span><span class="sxs-lookup"><span data-stu-id="d3ddd-108">The following sample generates CS0017:</span></span>

```csharp  
{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
