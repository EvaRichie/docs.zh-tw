# <a name="compiler-error-cs0662"></a><span data-ttu-id="5aa84-103">編譯器錯誤 CS0662</span><span class="sxs-lookup"><span data-stu-id="5aa84-103">Compiler Error CS0662</span></span>

<span data-ttu-id="5aa84-104">'method' 不能在 ref 參數上只指定 Out 屬性。</span><span class="sxs-lookup"><span data-stu-id="5aa84-104">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="5aa84-105">請同時使用 In 與 Out 屬性，或都不使用。</span><span class="sxs-lookup"><span data-stu-id="5aa84-105">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="5aa84-106">介面方法所包含的參數，使用了僅指定 [Out](../language-reference/keywords/ref.md) 屬性的 [ref](xref:System.Runtime.InteropServices.OutAttribute) 。</span><span class="sxs-lookup"><span data-stu-id="5aa84-106">An interface method has a parameter that uses [ref](../language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="5aa84-107">使用 `ref` Out **屬性的** 參數也必須使用 [In](xref:System.Runtime.InteropServices.InAttribute) 屬性。</span><span class="sxs-lookup"><span data-stu-id="5aa84-107">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="5aa84-108">下列範例會產生 CS0662：</span><span class="sxs-lookup"><span data-stu-id="5aa84-108">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
