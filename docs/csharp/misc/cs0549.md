# <a name="compiler-error-cs0549"></a><span data-ttu-id="b973e-103">編譯器錯誤 CS0549</span><span class="sxs-lookup"><span data-stu-id="b973e-103">Compiler Error CS0549</span></span>

<span data-ttu-id="b973e-104">'function' 是密封類別 'class' 的新虛擬成員</span><span class="sxs-lookup"><span data-stu-id="b973e-104">'function' is a new virtual member in sealed class 'class'</span></span>

 <span data-ttu-id="b973e-107">下列範例會產生 CS0549：</span><span class="sxs-lookup"><span data-stu-id="b973e-107">The following sample generates CS0549:</span></span>

```csharp  
}  
```
