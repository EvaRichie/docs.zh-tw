# <a name="compiler-warning-level-2-cs1710"></a><span data-ttu-id="55d85-102">編譯器警告 (層級 2) CS1710</span><span class="sxs-lookup"><span data-stu-id="55d85-102">Compiler Warning (level 2) CS1710</span></span>

 <span data-ttu-id="55d85-104">泛型類型的文件包括類型參數的重複標記。</span><span class="sxs-lookup"><span data-stu-id="55d85-104">The documentation of a generic type includes a duplicate tag for the type parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="55d85-105">範例</span><span class="sxs-lookup"><span data-stu-id="55d85-105">Example</span></span>  

 <span data-ttu-id="55d85-106">下列程式碼會導致出現警告 CS1710。</span><span class="sxs-lookup"><span data-stu-id="55d85-106">The following code will cause warning CS1710 to appear.</span></span>  
  
```csharp  
// CS1710.cs  
// compile with: /doc:cs1710.xml  
// To resolve this warning, delete one of the duplicate <typeparam>'s.  
using System;  
class Stack<ItemType>  
{  
}  
  
/// <typeparam name="MyType">can be an int</typeparam>  
/// <typeparam name="MyType">can be an int</typeparam>  
class MyStackWrapper<MyType>  
{  
    // Open constructed type Stack<MyType>.  
    Stack<MyType> stack;  
    public MyStackWrapper(Stack<MyType> s)  
    {  
        stack = s;  
    }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        // Closed constructed type Stack<int>.  
        Stack<int> stackInt = new Stack<int>();  
        MyStackWrapper<int> MyStackWrapperInt =  
            new MyStackWrapper<int>(stackInt);  
    }  
}  
```
