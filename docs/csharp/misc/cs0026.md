## <a name="example"></a><span data-ttu-id="76f25-108">範例</span><span class="sxs-lookup"><span data-stu-id="76f25-108">Example</span></span>  

 <span data-ttu-id="76f25-109">下列範例會產生 CS0026：</span><span class="sxs-lookup"><span data-stu-id="76f25-109">The following example generates CS0026:</span></span>
