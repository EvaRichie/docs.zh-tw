<span data-ttu-id="17abb-103">因為 'type' 是不符合 CLS 標準之類型 'type' 的成員，所以不可標記為符合 CLS 標準</span><span class="sxs-lookup"><span data-stu-id="17abb-103">'type' cannot be marked as CLS-Compliant because it is a member of non CLS-compliant type 'type'</span></span>

## <a name="example"></a><span data-ttu-id="17abb-108">範例</span><span class="sxs-lookup"><span data-stu-id="17abb-108">Example</span></span>  

 <span data-ttu-id="17abb-109">下列範例會產生 CS3018。</span><span class="sxs-lookup"><span data-stu-id="17abb-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
