# <a name="compiler-error-cs1536"></a>編譯器錯誤 CS1536

參數類型 'void' 無效  
  
 不必指定 void 指標以外的 [void](../language-reference/builtin-types/void.md) 參數，或此動作無效。  
  
 下列範例會產生 CS1536：  
  
```csharp  
// CS1536.cs  
class a  
{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
