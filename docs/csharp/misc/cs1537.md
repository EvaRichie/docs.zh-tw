# <a name="compiler-error-cs1537"></a>編譯器錯誤 CS1537
所使用的別名 'alias' 之前曾出現於此命名空間  
  
 您重複定義某個符號作為命名空間的別名。 一個符號只能定義一次。  
  
 下列範例會產生 CS1537：  
  
```csharp  
// CS1537.cs  
namespace x  
{  
   using System;  
   using Object = System.Object;  
   using Object = System.Object;   // CS1537, delete this line to resolve  
   using System = System;  
}  
```
