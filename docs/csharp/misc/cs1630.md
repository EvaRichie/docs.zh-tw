# <a name="compiler-error-cs1630"></a>編譯器錯誤 CS1630
無效的 /errorreport 選項 'option'; 必須是 prompt、send、queue 或 none  
  
 命令列選項 [-errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) 後面必須接著 **prompt**、 **send**、 **queue**或 **none**，以指定發生內部編譯器錯誤時所要採取的動作。
