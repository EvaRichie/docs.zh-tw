# <a name="compiler-error-cs0761"></a><span data-ttu-id="2ed2e-103">編譯器錯誤 CS0761</span><span class="sxs-lookup"><span data-stu-id="2ed2e-103">Compiler Error CS0761</span></span>
<span data-ttu-id="2ed2e-104">' Method ' 的部分方法宣告 \<T> 有不一致的類型參數條件約束。</span><span class="sxs-lookup"><span data-stu-id="2ed2e-104">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="2ed2e-105">如果部分方法具有實作，泛型類型條件約束必須與方法簽章上所定義的條件約束完全相同。</span><span class="sxs-lookup"><span data-stu-id="2ed2e-105">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2ed2e-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="2ed2e-106">To correct this error</span></span>  
  
1. <span data-ttu-id="2ed2e-107">使部分方法之每部分的泛型類型條件約束皆相同。</span><span class="sxs-lookup"><span data-stu-id="2ed2e-107">Make the generic type constraints identical on each part of the partial method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2ed2e-108">範例</span><span class="sxs-lookup"><span data-stu-id="2ed2e-108">Example</span></span>  
 <span data-ttu-id="2ed2e-109">下列程式碼會產生 CS0761：</span><span class="sxs-lookup"><span data-stu-id="2ed2e-109">The following code generates CS0761:</span></span>

```csharp  
    partial void Part<T>() where T : class;

    {  
        return 1;  
    }  
}  
```
