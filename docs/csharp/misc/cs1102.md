 <span data-ttu-id="d07ce-105">當 `this` 關鍵字修改靜態方法的第一個參數時，它會通知編譯器，該方法是一個擴充方法。</span><span class="sxs-lookup"><span data-stu-id="d07ce-105">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="d07ce-106">擴充方法的第一個參數上不需要也不允許其他修飾詞。</span><span class="sxs-lookup"><span data-stu-id="d07ce-106">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>

1. <span data-ttu-id="d07ce-108">請從第一個參數移除未經授權的修飾詞。</span><span class="sxs-lookup"><span data-stu-id="d07ce-108">Remove the unauthorized modifiers from the first parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d07ce-109">範例</span><span class="sxs-lookup"><span data-stu-id="d07ce-109">Example</span></span>  

 <span data-ttu-id="d07ce-110">下列範例會產生 CS1102：</span><span class="sxs-lookup"><span data-stu-id="d07ce-110">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
