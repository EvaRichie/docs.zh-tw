# <a name="compiler-error-cs0765"></a><span data-ttu-id="68c8e-103">編譯器錯誤 CS0765</span><span class="sxs-lookup"><span data-stu-id="68c8e-103">Compiler Error CS0765</span></span>
<span data-ttu-id="68c8e-104">在運算式樹狀結構中，不可使用只具有定義宣告或已移除條件式方法的部分方法。</span><span class="sxs-lookup"><span data-stu-id="68c8e-104">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="68c8e-105">雖然對移除的部分方法的呼叫是運算式，卻不是運算式樹狀結構接受的運算式。</span><span class="sxs-lookup"><span data-stu-id="68c8e-105">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="68c8e-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="68c8e-106">To correct this error</span></span>  
  
1. <span data-ttu-id="68c8e-107">加入部分方法的實作宣告，或移除造成編譯要排除之條件式方法的程式碼。</span><span class="sxs-lookup"><span data-stu-id="68c8e-107">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="68c8e-108">範例</span><span class="sxs-lookup"><span data-stu-id="68c8e-108">Example</span></span>  
 <span data-ttu-id="68c8e-109">下列程式碼在兩個位置會產生 CS0765：</span><span class="sxs-lookup"><span data-stu-id="68c8e-109">The following code generates CS0765 in two locations:</span></span>  
  
```csharp  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
