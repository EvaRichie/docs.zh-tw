<span data-ttu-id="8b68b-104">可能誤用了參考比較; 若要進行數值比較，請將右側轉型為類型 'type'</span><span class="sxs-lookup"><span data-stu-id="8b68b-104">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="8b68b-105">編譯器正在進行參考比較。</span><span class="sxs-lookup"><span data-stu-id="8b68b-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="8b68b-106">如果您想要比較字串值，請將運算式的右邊轉換為 `type`。</span><span class="sxs-lookup"><span data-stu-id="8b68b-106">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="8b68b-107">下列範例會產生 CS0253：</span><span class="sxs-lookup"><span data-stu-id="8b68b-107">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
