# <a name="compiler-error-cs0150"></a><span data-ttu-id="4c748-103">編譯器錯誤 CS0150</span><span class="sxs-lookup"><span data-stu-id="4c748-103">Compiler Error CS0150</span></span>

<span data-ttu-id="4c748-104">必須是常數值</span><span class="sxs-lookup"><span data-stu-id="4c748-104">A constant value is expected</span></span>  
  
 <span data-ttu-id="4c748-105">找到必須有常數的變數。</span><span class="sxs-lookup"><span data-stu-id="4c748-105">A variable was found where a constant was expected.</span></span> <span data-ttu-id="4c748-106">如需詳細資訊，請參閱 [switch](../language-reference/keywords/switch.md)。</span><span class="sxs-lookup"><span data-stu-id="4c748-106">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="4c748-107">下列範例會產生 CS0150：</span><span class="sxs-lookup"><span data-stu-id="4c748-107">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="4c748-108">以變數值指定並以陣列初始設定式初始化陣列大小時，也會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="4c748-108">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="4c748-109">若要移除此錯誤，請以一或多個個別陳述式初始化陣列。</span><span class="sxs-lookup"><span data-stu-id="4c748-109">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
