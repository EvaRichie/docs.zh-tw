## <a name="example"></a><span data-ttu-id="a22f0-110">範例</span><span class="sxs-lookup"><span data-stu-id="a22f0-110">Example</span></span>  

 <span data-ttu-id="a22f0-111">在這個程式碼中，部分類別定義都在單一檔案中，但它們也可以分散到多個檔案中。</span><span class="sxs-lookup"><span data-stu-id="a22f0-111">In this code, the partial class definitions are all in a single file, but they could also be spread across multiple files.</span></span>

    interface IFace1 { }
