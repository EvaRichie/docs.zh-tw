# <a name="compiler-error-cs1556"></a>編譯器錯誤 CS1556

為 Main 方法所指定的 'construct' 必須為有效的類別或結構  
  
 傳遞的識別碼不是類別名稱的 [-main](../language-reference/compiler-options/main-compiler-option.md) 編譯器選項。
