# <a name="compiler-error-cs1514"></a>編譯器錯誤 CS1514

必須是 {  
  
 編譯器必須有左大括號 (`{`)，但找不到。  
  
 下列範例會產生 CS1514：  
  
```csharp  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
