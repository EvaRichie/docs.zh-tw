# <a name="compiler-error-cs0685"></a><span data-ttu-id="d9d2d-103">編譯器錯誤 CS0685</span><span class="sxs-lookup"><span data-stu-id="d9d2d-103">Compiler Error CS0685</span></span>
<span data-ttu-id="d9d2d-104">Conditional 成員 'member' 不能有 out 參數</span><span class="sxs-lookup"><span data-stu-id="d9d2d-104">Conditional member 'member' cannot have an out parameter</span></span>  
  
 <span data-ttu-id="d9d2d-105">在方法上使用 <xref:System.Diagnostics.ConditionalAttribute> 屬性時，該方法可能沒有 out 參數。</span><span class="sxs-lookup"><span data-stu-id="d9d2d-105">When using the <xref:System.Diagnostics.ConditionalAttribute> attribute on a method, that method may not have an out parameter.</span></span> <span data-ttu-id="d9d2d-106">這是因為在方法呼叫編譯為 Nothing 的情況下，不會定義用於 out 參數的變數值。</span><span class="sxs-lookup"><span data-stu-id="d9d2d-106">This is because the value of the variable used for the out parameter would not be defined in the case that the method call is compiled to nothing.</span></span> <span data-ttu-id="d9d2d-107">若要避免這個錯誤，請從條件式方法宣告中移除 out 參數，或不要使用條件式屬性。</span><span class="sxs-lookup"><span data-stu-id="d9d2d-107">To avoid this error, remove the out parameter from the conditional method declaration, or don't use the Conditional Attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d9d2d-108">範例</span><span class="sxs-lookup"><span data-stu-id="d9d2d-108">Example</span></span>  
 <span data-ttu-id="d9d2d-109">下列範例會產生 CS0685：</span><span class="sxs-lookup"><span data-stu-id="d9d2d-109">The following sample generates CS0685:</span></span>

    [Conditional("DEBUG")]
