# <a name="compiler-error-cs0167"></a>編譯器錯誤 CS0167

委派 'delegate' 遺漏 Invoke 方法  
  
 您已匯入和使用受管理的程式 (一個使用以其他編譯器建立的 .NET 執行時間) 。 該編譯器允許格式不正確的 [委派](../language-reference/builtin-types/reference-types.md)。 因此，無法使用 `Invoke` 方法。 如需詳細資訊，請參閱 [委派](../programming-guide/delegates/index.md)中定義的介面的私用 C++ 專屬實作。
