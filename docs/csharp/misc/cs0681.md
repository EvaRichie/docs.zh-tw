# <a name="compiler-error-cs0681"></a>編譯器錯誤 CS0681
修飾詞 'abstract' 在欄位上無效。 請嘗試改用屬性  
  
 無法將欄位設為抽象。 不過，您可以讓抽象屬性存取欄位。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0681：  
  
```csharp  
// CS0681.cs  
// compile with: /target:library  
abstract class C  
{  
    abstract int num;  // CS0681  
}  
```  
  
## <a name="example"></a>範例  
 請改嘗試下列程式碼：  
  
```csharp  
// CS0681b.cs  
// compile with: /target:library  
abstract class C  
{  
    public abstract int num  
    {  
       get;  
       set;  
    }  
}  
```
