# <a name="compiler-error-cs0542"></a><span data-ttu-id="cfd82-103">編譯器錯誤 CS0542</span><span class="sxs-lookup"><span data-stu-id="cfd82-103">Compiler Error CS0542</span></span>
<span data-ttu-id="cfd82-104">'user-defined type'：成員名稱不能與它的封入類型名稱相同</span><span class="sxs-lookup"><span data-stu-id="cfd82-104">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="cfd82-105">類別或結構的成員不能有相同名稱的類別或結構，除非該成員是建構函式。</span><span class="sxs-lookup"><span data-stu-id="cfd82-105">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="cfd82-106">下列範例會產生 CS0542：</span><span class="sxs-lookup"><span data-stu-id="cfd82-106">The following sample generates CS0542:</span></span>  
  
```csharp  
// CS0542.cs  
class C  
{  
    public int C;  
}  
```  
  
 <span data-ttu-id="cfd82-107">如果您不小心將傳回類型放在建構函式中 (實際上會將其轉換成一般的方法)，就可能造成這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="cfd82-107">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="cfd82-108">下列範例會產生 CS0542；因為 `F` 具有傳回類型，所以是一種方法，不是建構函式：</span><span class="sxs-lookup"><span data-stu-id="cfd82-108">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp  
// CS0542.cs  
class F  
{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="cfd82-109">如果您的類別為 'Item'，並具有宣告為 `this`的索引子，則可能會收到這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="cfd82-109">If your class is named 'Item' and has an indexer declared as `this`, you may get this error.</span></span> <span data-ttu-id="cfd82-110">發出程式碼中的預設索引子名稱為 'Item'，造成衝突。</span><span class="sxs-lookup"><span data-stu-id="cfd82-110">A default indexer is given the name 'Item' in the emitted code, creating the conflict.</span></span>  
  
```csharp  
// CS0542b.cs  
class Item  
{  
   public int this[int i]  // CS0542  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```
