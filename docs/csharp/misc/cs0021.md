## <a name="example"></a><span data-ttu-id="715ea-109">範例</span><span class="sxs-lookup"><span data-stu-id="715ea-109">Example</span></span>

<span data-ttu-id="715ea-110">下列 c + + 範例會編譯為 .dll 檔案。</span><span class="sxs-lookup"><span data-stu-id="715ea-110">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="715ea-111">請注意， [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) 屬性已標記為批註，以產生錯誤。</span><span class="sxs-lookup"><span data-stu-id="715ea-111">Note that the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>

```cpp
// CPP0021.cpp
// compile with: /clr /LD
using namespace System::Reflection;
// Uncomment the following line to resolve
//[DefaultMember("myItem")]
public ref class MyClassMC
{
        public:
        property int myItem[int]
        {
            int get(int i){  return 5; }
            void set(int i, int value) {}
        }
};
```

<span data-ttu-id="715ea-112">下列 c # 範例會呼叫 .dll 檔案。</span><span class="sxs-lookup"><span data-stu-id="715ea-112">The following C# example calls the .dll file.</span></span> <span data-ttu-id="715ea-113">它會嘗試透過索引子存取類別，但因為沒有任何成員宣告為預設索引子，所以會產生錯誤。</span><span class="sxs-lookup"><span data-stu-id="715ea-113">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="715ea-114">您可以藉由取消批註行來解決錯誤 `[DefaultMember("myItem")]` 。</span><span class="sxs-lookup"><span data-stu-id="715ea-114">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>

```csharp

    public static void Main()
    {
        MyClassMC myMC = new MyClassMC();

    }
}
```
