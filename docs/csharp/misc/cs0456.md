<span data-ttu-id="48bf3-104">類型參數 '類型參數名稱 1' 具有 'struct' 條件約束，因此 '類型參數名稱 1' 不能作為 '類型參數名稱 2' 的條件約束</span><span class="sxs-lookup"><span data-stu-id="48bf3-104">Type parameter 'Type Parameter Name 1' has the 'struct' constraint so 'Type Parameter Name 1' cannot be used as a constraint for 'Type Parameter Name 2'</span></span>

## <a name="example"></a><span data-ttu-id="48bf3-108">範例</span><span class="sxs-lookup"><span data-stu-id="48bf3-108">Example</span></span>  

 <span data-ttu-id="48bf3-109">下列範例會產生 CS0456。</span><span class="sxs-lookup"><span data-stu-id="48bf3-109">The following sample generates CS0456.</span></span>  
  
```csharp  
// CS0456.cs  
// compile with: /target:library  
public class GenericsErrors  
{  
   public class G5<T> where T : struct  
   {  
      public class N<U> where U : T {}   // CS0456  
      public class N2<U> where U : struct {}   // OK  
   }  
}  
```
