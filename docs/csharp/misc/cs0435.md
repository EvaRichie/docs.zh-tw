   public class A { }

```  
  
 <span data-ttu-id="df513-110">然後，編譯這個檔案：</span><span class="sxs-lookup"><span data-stu-id="df513-110">Then, compile this file:</span></span>

{  
   public class A { }  
}

   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }

```
