<span data-ttu-id="d1d0f-104">'類型參數': 屬性引數不可使用類型參數</span><span class="sxs-lookup"><span data-stu-id="d1d0f-104">'type parameter': an attribute argument cannot use type parameters</span></span>  
  
 <span data-ttu-id="d1d0f-105">使用了類型參數作為屬性引數，這是不允許的做法。</span><span class="sxs-lookup"><span data-stu-id="d1d0f-105">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="d1d0f-106">請使用非泛型類型。</span><span class="sxs-lookup"><span data-stu-id="d1d0f-106">Use a non-generic type.</span></span>

```csharp

{  
   public MyAttribute(System.Type t)  
   {  
   }

   [MyAttribute(typeof(G<T>))]  // CS0416

   {  
   }  
  
}
```
