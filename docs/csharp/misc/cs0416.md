# <a name="compiler-error-cs0416"></a><span data-ttu-id="a63ab-103">編譯器錯誤 CS0416</span><span class="sxs-lookup"><span data-stu-id="a63ab-103">Compiler Error CS0416</span></span>

<span data-ttu-id="a63ab-104">'類型參數': 屬性引數不可使用類型參數</span><span class="sxs-lookup"><span data-stu-id="a63ab-104">'type parameter': an attribute argument cannot use type parameters</span></span>  
  
 <span data-ttu-id="a63ab-105">使用了類型參數作為屬性引數，這是不允許的做法。</span><span class="sxs-lookup"><span data-stu-id="a63ab-105">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="a63ab-106">請使用非泛型類型。</span><span class="sxs-lookup"><span data-stu-id="a63ab-106">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="a63ab-107">下列範例會產生 CS0416：</span><span class="sxs-lookup"><span data-stu-id="a63ab-107">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}  
  
class G<T>  
{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
