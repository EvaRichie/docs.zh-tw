# <a name="compiler-error-cs0243"></a><span data-ttu-id="a54e5-103">編譯器錯誤 CS0243</span><span class="sxs-lookup"><span data-stu-id="a54e5-103">Compiler error CS0243</span></span>

<span data-ttu-id="a54e5-104">Conditional 屬性不能用在 'method' 上，因為它是一個覆寫方法</span><span class="sxs-lookup"><span data-stu-id="a54e5-104">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="a54e5-105"><xref:System.Diagnostics.ConditionalAttribute>使用[override](../language-reference/keywords/override.md)關鍵字標記的方法上不允許屬性。</span><span class="sxs-lookup"><span data-stu-id="a54e5-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="a54e5-106">如需詳細資訊，請參閱 [瞭解使用 Override 和 New 關鍵字](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)的時機。</span><span class="sxs-lookup"><span data-stu-id="a54e5-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="a54e5-107">編譯器永遠不會系結到覆寫方法。</span><span class="sxs-lookup"><span data-stu-id="a54e5-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="a54e5-108">它只會系結至基底方法，而 common language runtime 會適當地呼叫覆寫。</span><span class="sxs-lookup"><span data-stu-id="a54e5-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="a54e5-109">下列程式碼會產生 CS0243：</span><span class="sxs-lookup"><span data-stu-id="a54e5-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
