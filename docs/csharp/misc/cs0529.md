 <span data-ttu-id="b53ca-105">[介面](../language-reference/keywords/interface.md) 的繼承清單包含本身的直接或間接參考。</span><span class="sxs-lookup"><span data-stu-id="b53ca-105">The inheritance list for an [interface](../language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="b53ca-106">介面無法繼承自本身。</span><span class="sxs-lookup"><span data-stu-id="b53ca-106">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="b53ca-107">下列範例會產生 CS0529：</span><span class="sxs-lookup"><span data-stu-id="b53ca-107">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c

   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
