 <span data-ttu-id="bc104-105">嘗試覆寫屬性的其中一個存取子方法失敗，因為無法覆寫存取子。</span><span class="sxs-lookup"><span data-stu-id="bc104-105">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="bc104-106">此錯誤可能會在下列情況中發生：</span><span class="sxs-lookup"><span data-stu-id="bc104-106">This error can occur if:</span></span>

## <a name="example"></a><span data-ttu-id="bc104-111">範例</span><span class="sxs-lookup"><span data-stu-id="bc104-111">Example</span></span>  

 <span data-ttu-id="bc104-112">下列範例會產生 CS0546，因為基底類別未宣告屬性的 set 存取子。</span><span class="sxs-lookup"><span data-stu-id="bc104-112">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
