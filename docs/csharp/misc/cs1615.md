# <a name="compiler-error-cs1615"></a><span data-ttu-id="6c096-103">編譯器錯誤 CS1615</span><span class="sxs-lookup"><span data-stu-id="6c096-103">Compiler Error CS1615</span></span>

<span data-ttu-id="6c096-104">傳遞引數 'number' 時不應該包含 'keyword' 關鍵字</span><span class="sxs-lookup"><span data-stu-id="6c096-104">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="6c096-105">使用了 `ref` 或 **out** 的其中一個關鍵字，但函式未接受該引數的 `ref` 或 **out** 參數。</span><span class="sxs-lookup"><span data-stu-id="6c096-105">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="6c096-106">若要解決此錯誤，請移除不正確的關鍵字，並使用符合函式宣告的適當關鍵字，如果有的話。</span><span class="sxs-lookup"><span data-stu-id="6c096-106">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="6c096-107">下列範例會產生 C1615：</span><span class="sxs-lookup"><span data-stu-id="6c096-107">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
