## <a name="example"></a><span data-ttu-id="d8851-107">範例</span><span class="sxs-lookup"><span data-stu-id="d8851-107">Example</span></span>  

 <span data-ttu-id="d8851-108">下列範例會產生 CS0643：</span><span class="sxs-lookup"><span data-stu-id="d8851-108">The following sample generates CS0643:</span></span>  
  
```csharp  
// CS0643.cs  
using System;  
using System.Runtime.InteropServices;  
  
[AttributeUsage(AttributeTargets.Class)]  
public class MyAttribute : Attribute  
{  
    public MyAttribute()  
    {  
    }  
  
    public int x;  
}  
  
[MyAttribute(x = 5, x = 6)]   // CS0643, error setting x twice  
// try the following line instead  
// [MyAttribute(x = 5)]  
class MyClass  
{  
}  
  
public class MainClass  
{  
    public static void Main ()  
    {  
    }  
}  
```
