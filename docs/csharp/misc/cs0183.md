# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="729d8-103">編譯器警告 (層級 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="729d8-103">Compiler Warning (level 1) CS0183</span></span>

<span data-ttu-id="729d8-104">指定的運算式一律為提供的 ('type') 類型</span><span class="sxs-lookup"><span data-stu-id="729d8-104">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="729d8-105">如果條件陳述式一律評估為 **true**，則不需要條件陳述式。</span><span class="sxs-lookup"><span data-stu-id="729d8-105">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="729d8-106">嘗試使用 **is** 運算子來評估類型時，會發生這個警告。</span><span class="sxs-lookup"><span data-stu-id="729d8-106">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="729d8-107">如果評估為實值類型，則不需要檢查。</span><span class="sxs-lookup"><span data-stu-id="729d8-107">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="729d8-108">下列範例會產生 CS0183：</span><span class="sxs-lookup"><span data-stu-id="729d8-108">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);
   }  
}  
```
