## <a name="example"></a><span data-ttu-id="6e339-108">範例</span><span class="sxs-lookup"><span data-stu-id="6e339-108">Example</span></span>  
 <span data-ttu-id="6e339-109">下列範例會產生 CS0191：</span><span class="sxs-lookup"><span data-stu-id="6e339-109">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
