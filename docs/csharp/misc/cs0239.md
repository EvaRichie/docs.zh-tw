 <span data-ttu-id="e136c-105">成員無法覆[寫](../language-reference/keywords/override.md)[密封](../language-reference/keywords/sealed.md)的繼承成員。</span><span class="sxs-lookup"><span data-stu-id="e136c-105">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="e136c-106">如需詳細資訊，請參閱 [Checked 和 Unchecked](../language-reference/keywords/checked-and-unchecked.md)。</span><span class="sxs-lookup"><span data-stu-id="e136c-106">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="e136c-107">下列範例會產生 CS0239：</span><span class="sxs-lookup"><span data-stu-id="e136c-107">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
