# <a name="compiler-error-cs0220"></a><span data-ttu-id="56c87-103">編譯器錯誤 CS0220</span><span class="sxs-lookup"><span data-stu-id="56c87-103">Compiler Error CS0220</span></span>
<span data-ttu-id="56c87-104">檢查模式下，作業於編譯時期溢位</span><span class="sxs-lookup"><span data-stu-id="56c87-104">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="56c87-105">[checked](../language-reference/keywords/checked.md)偵測到一項作業，這是預設值，並且會導致資料遺失。</span><span class="sxs-lookup"><span data-stu-id="56c87-105">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="56c87-106">請更正指派的輸入，或使用 [unchecked](../language-reference/keywords/unchecked.md) 來解決這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="56c87-106">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="56c87-107">如需詳細資訊，請參閱 [Checked 和 Unchecked](../language-reference/keywords/checked-and-unchecked.md)。</span><span class="sxs-lookup"><span data-stu-id="56c87-107">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="56c87-108">下列範例會產生 CS0220：</span><span class="sxs-lookup"><span data-stu-id="56c87-108">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
