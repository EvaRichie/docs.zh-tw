<span data-ttu-id="965d3-104">無法在匿名方法、lambda 運算式或查詢運算式內的 ref 或 out 參數 ' parameter ' 中使用</span><span class="sxs-lookup"><span data-stu-id="965d3-104">Cannot use in ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="965d3-105">如果您在 `in` `ref` `out` 匿名方法區塊內使用、或參數，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="965d3-105">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="965d3-106">要避免這個錯誤，請使用區域變數或其他建構。</span><span class="sxs-lookup"><span data-stu-id="965d3-106">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="965d3-107">下列範例會產生 C1628：</span><span class="sxs-lookup"><span data-stu-id="965d3-107">The following sample generates CS1628:</span></span>  
  
```csharp  
// CS1628.cs

class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
