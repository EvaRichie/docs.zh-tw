# <a name="compiler-error-cs0070"></a><span data-ttu-id="76a97-103">編譯器錯誤 CS0070</span><span class="sxs-lookup"><span data-stu-id="76a97-103">Compiler Error CS0070</span></span>

<span data-ttu-id="76a97-104">事件 'event' 只能出現在 += 或 -= 的左邊 (除非從類型 'type' 中使用)</span><span class="sxs-lookup"><span data-stu-id="76a97-104">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="76a97-105">[事件](../language-reference/keywords/event.md) 在其定義所在的類別之外只能加減參考。</span><span class="sxs-lookup"><span data-stu-id="76a97-105">Outside of the class it is defined in, an [event](../language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="76a97-106">如需詳細資訊，請參閱[事件](../programming-guide/events/index.md)。</span><span class="sxs-lookup"><span data-stu-id="76a97-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="76a97-107">下列範例會產生 CS0070：</span><span class="sxs-lookup"><span data-stu-id="76a97-107">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
