# <a name="compiler-error-cs1929"></a><span data-ttu-id="0d5e4-102">編譯器錯誤 CS1929</span><span class="sxs-lookup"><span data-stu-id="0d5e4-102">Compiler Error CS1929</span></span>

<span data-ttu-id="0d5e4-103">執行個體引數：無法從 'typeA' 轉換為 'typeB'。</span><span class="sxs-lookup"><span data-stu-id="0d5e4-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="0d5e4-104">當您嘗試從不會擴充的類別叫用擴充方法時，就會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="0d5e4-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="0d5e4-105">在此處所顯示的範例中，擴充方法是針對衍生類別 `A`而定義，但不是針對基底類別 `B`。</span><span class="sxs-lookup"><span data-stu-id="0d5e4-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0d5e4-106">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="0d5e4-106">To correct this error</span></span>  
  
1. <span data-ttu-id="0d5e4-107">請在您必須叫用之處為類型建立新的擴充方法，或是將呼叫移到現有方法會擴充之類型的物件。</span><span class="sxs-lookup"><span data-stu-id="0d5e4-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0d5e4-108">範例</span><span class="sxs-lookup"><span data-stu-id="0d5e4-108">Example</span></span>  

 <span data-ttu-id="0d5e4-109">下列程式碼會產生 CS1928 和 CS1929：</span><span class="sxs-lookup"><span data-stu-id="0d5e4-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
