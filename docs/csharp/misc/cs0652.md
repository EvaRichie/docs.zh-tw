 <span data-ttu-id="85a90-105">編譯器偵測到常數與變數之間的比較，其中常數位於變數的範圍外。</span><span class="sxs-lookup"><span data-stu-id="85a90-105">The compiler detected a comparison between a constant and a variable where the constant is out of the range of the variable.</span></span>

```csharp  
// CS0652.cs  
// compile with: /W:2  
public class Class1  
{  
   private static byte i = 0;  
   public static void Main()  
   {  
      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte  
         i = 0;  
   }  
}  
```
