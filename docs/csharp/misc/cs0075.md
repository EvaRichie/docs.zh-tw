# <a name="compiler-error-cs0075"></a><span data-ttu-id="d0a4e-103">編譯器錯誤 CS0075</span><span class="sxs-lookup"><span data-stu-id="d0a4e-103">Compiler Error CS0075</span></span>
<span data-ttu-id="d0a4e-104">要做轉型的負值必須置於括號之內</span><span class="sxs-lookup"><span data-stu-id="d0a4e-104">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="d0a4e-105">如果您使用會識別預先定義之類型的關鍵字來將類型轉型，則不需要括弧。</span><span class="sxs-lookup"><span data-stu-id="d0a4e-105">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="d0a4e-106">否則，您必須加上括弧，因為 (x) –y 不會視為轉型運算式。</span><span class="sxs-lookup"><span data-stu-id="d0a4e-106">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="d0a4e-107">取自 C# 規格的第 7.6.6 節：</span><span class="sxs-lookup"><span data-stu-id="d0a4e-107">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="d0a4e-108">*從去除混淆規則開始，如果 x 和 y 是識別碼、 (x) y、 (x) # B4 y) ，且 (x) # A8-y) 是轉型運算式，但即使 x 識別類型， (x) -y 也是一樣。但是，如果 x 是識別預先定義之類型 (例如 int) 的關鍵字，則四種形式都是轉換運算式 (，因為這類關鍵字本身不能是運算式本身的) 。*</span><span class="sxs-lookup"><span data-stu-id="d0a4e-108">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="d0a4e-109">下列程式碼會產生 CS0075：</span><span class="sxs-lookup"><span data-stu-id="d0a4e-109">The following code generates CS0075:</span></span>  
  
```csharp  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
