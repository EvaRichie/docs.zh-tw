# <a name="compiler-error-cs1940"></a><span data-ttu-id="ed9df-102">編譯器錯誤 CS1940</span><span class="sxs-lookup"><span data-stu-id="ed9df-102">Compiler Error CS1940</span></span>

<span data-ttu-id="ed9df-103">為來源類型 'type' 找到多個查詢模式實作。</span><span class="sxs-lookup"><span data-stu-id="ed9df-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="ed9df-104">模稜兩可的 'method' 呼叫。</span><span class="sxs-lookup"><span data-stu-id="ed9df-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="ed9df-105">已定義查詢方法的多個實作定義，編譯器無法區別哪一個最適用於查詢時，就會產生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="ed9df-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="ed9df-106">在下列範例中，兩個版本的 `Select` 都具有相同的簽章，因為它們都接受一個 `int` 作為輸入參數，而且以 `int` 作為傳回值。</span><span class="sxs-lookup"><span data-stu-id="ed9df-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

1. <span data-ttu-id="ed9df-108">請為每個方法只提供一個實作。</span><span class="sxs-lookup"><span data-stu-id="ed9df-108">Provide only one implementation for each method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ed9df-109">範例</span><span class="sxs-lookup"><span data-stu-id="ed9df-109">Example</span></span>  

 <span data-ttu-id="ed9df-110">下列程式碼會產生 CS1940：</span><span class="sxs-lookup"><span data-stu-id="ed9df-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
