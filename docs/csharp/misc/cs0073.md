# <a name="compiler-error-cs0073"></a><span data-ttu-id="54770-103">編譯器錯誤 CS0073</span><span class="sxs-lookup"><span data-stu-id="54770-103">Compiler Error CS0073</span></span>
<span data-ttu-id="54770-104">add 或 remove 存取子必須具有主體</span><span class="sxs-lookup"><span data-stu-id="54770-104">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="54770-105">**事件** 定義中的 **加入** 或 [移除](../language-reference/keywords/event.md) 關鍵字必須有主體。</span><span class="sxs-lookup"><span data-stu-id="54770-105">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="54770-106">如需詳細資訊，請參閱[事件](../programming-guide/events/index.md)。</span><span class="sxs-lookup"><span data-stu-id="54770-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="54770-107">下列範例會產生 CS0073：</span><span class="sxs-lookup"><span data-stu-id="54770-107">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
