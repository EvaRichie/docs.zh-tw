# <a name="compiler-error-cs1621"></a>編譯器錯誤 CS1621
在匿名方法或 Lambda 運算式內不可使用 yield 陳述式  
  
您不能在[匿名方法](../language-reference/operators/delegate-operator.md)或[lambda 運算式](../language-reference/operators/lambda-expressions.md)中使用[yield](../language-reference/keywords/yield.md)語句。
  
下列範例會產生 CS1621：
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
