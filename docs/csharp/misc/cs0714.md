# <a name="compiler-error-cs0714"></a>編譯器錯誤 CS0714
'靜態類型': 靜態類別不能實作介面  
  
 介面可以定義物件上的非靜態方法，因此可能不是由靜態類別實作。 若要解決這個錯誤，請確定您的類別未嘗試實作任何介面。  
  
## <a name="example"></a>範例  
 下列範例會產生 CS0714：  
  
```csharp  
// CS0714.cs  
interface I  
{  
}  
  
public static class C : I  // CS0714  
{  
    public static void Main()  
    {  
    }  
}  
```
