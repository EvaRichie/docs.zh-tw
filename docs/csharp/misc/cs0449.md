# <a name="compiler-error-cs0449"></a><span data-ttu-id="efbe5-103">編譯器錯誤 CS0449</span><span class="sxs-lookup"><span data-stu-id="efbe5-103">Compiler Error CS0449</span></span>
<span data-ttu-id="efbe5-104">class' 或 'struct' 條件約束必須在所有其他條件約束之前</span><span class="sxs-lookup"><span data-stu-id="efbe5-104">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="efbe5-105">泛型類型或方法之類型參數的條件約束必須以特定順序發生： `class` 或 `struct` 必須是第一個 (如果存在)，接著是任何介面條件約束，最後是任何建構函式條件約束。</span><span class="sxs-lookup"><span data-stu-id="efbe5-105">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="efbe5-106">這個錯誤是 `class` 或 `struct` 條件約束未第一個出現所造成。</span><span class="sxs-lookup"><span data-stu-id="efbe5-106">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="efbe5-107">若要解決這個錯誤，請重新排列條件約束子句。</span><span class="sxs-lookup"><span data-stu-id="efbe5-107">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="efbe5-108">範例</span><span class="sxs-lookup"><span data-stu-id="efbe5-108">Example</span></span>  
 <span data-ttu-id="efbe5-109">下列範例會產生 CS0449。</span><span class="sxs-lookup"><span data-stu-id="efbe5-109">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
