# <a name="compiler-error-cs1024"></a>編譯器錯誤 CS1024

必須是前置處理器指示詞  
  
 行開頭的井字號 (#)，但後續字串不是有效的 [前置處理器指示詞](../language-reference/preprocessor-directives/index.md)。  
  
 下列範例會產生 CS1024：  
  
```csharp  
// CS1024.cs  
#import System   // CS1024  
```
