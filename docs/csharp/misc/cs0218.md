# <a name="compiler-error-cs0218"></a><span data-ttu-id="16dde-103">編譯器錯誤 CS0218</span><span class="sxs-lookup"><span data-stu-id="16dde-103">Compiler Error CS0218</span></span>
<span data-ttu-id="16dde-104">類型 ('type') 必須包含運算子 true 和運算子 false 的宣告</span><span class="sxs-lookup"><span data-stu-id="16dde-104">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="16dde-105">如果使用者定義型別多載了 [& 運算子](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) 或 [&#124; 運算子](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-)，則也必須定義 [true 和 false](../language-reference/operators/true-false-operators.md) 運算子，才能定義最短運算 [&& 運算子](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) 或 [&#124;&#124; 運算子](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) 。</span><span class="sxs-lookup"><span data-stu-id="16dde-105">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="16dde-106">下列範例會產生 CS0218：</span><span class="sxs-lookup"><span data-stu-id="16dde-106">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
