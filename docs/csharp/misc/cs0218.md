# <a name="compiler-error-cs0218"></a>編譯器錯誤 CS0218
類型 ('type') 必須包含運算子 true 和運算子 false 的宣告  
  
如果使用者定義型別多載了 [& 運算子](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) 或 [&#124; 運算子](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-)，則也必須定義 [true 和 false](../language-reference/operators/true-false-operators.md) 運算子，才能定義最短運算 [&& 運算子](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) 或 [&#124;&#124; 運算子](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) 。
  
 下列範例會產生 CS0218：  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
