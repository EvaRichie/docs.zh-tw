# <a name="compiler-error-cs0663"></a><span data-ttu-id="c5501-103">編譯器錯誤 CS0663</span><span class="sxs-lookup"><span data-stu-id="c5501-103">Compiler Error CS0663</span></span>

<span data-ttu-id="c5501-104">不可定義只有 ref 和 out 不同的多載方法。</span><span class="sxs-lookup"><span data-stu-id="c5501-104">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="c5501-105">不允許只有在參數上使用的、 [ref](../language-reference/keywords/ref.md)和[out](../language-reference/keywords/out-parameter-modifier.md) [不同的方法](../language-reference/keywords/in-parameter-modifier.md)。</span><span class="sxs-lookup"><span data-stu-id="c5501-105">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="c5501-106">下列範例會產生 CS0663：</span><span class="sxs-lookup"><span data-stu-id="c5501-106">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
