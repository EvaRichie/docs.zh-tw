## <a name="example"></a><span data-ttu-id="4a51c-107">範例</span><span class="sxs-lookup"><span data-stu-id="4a51c-107">Example</span></span>  

 <span data-ttu-id="4a51c-108">下列範例會產生 CS0406。</span><span class="sxs-lookup"><span data-stu-id="4a51c-108">The following sample generates CS0406.</span></span>  
  
```csharp  
// CS0406.cs

interface I {}  
class C {}  
class D<T> where T : I, C {}   // CS0406  
class D2<T> where T : C, I {}   // OK  
```
