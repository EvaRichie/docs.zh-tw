# <a name="compiler-error-cs0538"></a><span data-ttu-id="a74b0-103">編譯器錯誤 CS0538</span><span class="sxs-lookup"><span data-stu-id="a74b0-103">Compiler Error CS0538</span></span>
<span data-ttu-id="a74b0-104">在明確介面宣告中的 'name' 不是介面</span><span class="sxs-lookup"><span data-stu-id="a74b0-104">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="a74b0-105">嘗試明確宣告 [介面](../language-reference/keywords/interface.md)，但未指定介面。</span><span class="sxs-lookup"><span data-stu-id="a74b0-105">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="a74b0-106">下列範例會產生 CS0538：</span><span class="sxs-lookup"><span data-stu-id="a74b0-106">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
