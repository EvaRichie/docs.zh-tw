# <a name="compiler-error-cs0517"></a>編譯器錯誤 CS0517

'class' 沒有基底類別且無法呼叫基底建構函式  
  
 只有當 .NET 執行時間編譯物件類別的原始程式碼（這是唯一沒有基類的類別）時，才會發生 CS0517。
