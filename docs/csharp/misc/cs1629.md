# <a name="compiler-error-cs1629"></a><span data-ttu-id="efd8f-103">編譯器錯誤 CS1629</span><span class="sxs-lookup"><span data-stu-id="efd8f-103">Compiler Error CS1629</span></span>

<span data-ttu-id="efd8f-104">Unsafe 程式碼不可出現在迭代器中</span><span class="sxs-lookup"><span data-stu-id="efd8f-104">Unsafe code may not appear in iterators</span></span>  
  
 <span data-ttu-id="efd8f-105">C# 語言規格不容許迭代器中有 Unsafe 程式碼。</span><span class="sxs-lookup"><span data-stu-id="efd8f-105">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="efd8f-106">下列範例會產生 CS1629：</span><span class="sxs-lookup"><span data-stu-id="efd8f-106">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe
using System.Collections.Generic;  
class C
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
