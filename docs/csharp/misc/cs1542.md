# <a name="compiler-error-cs1542"></a>編譯器錯誤 CS1542

無法將 'dll' 加入至此組件，因為本身已經是組件；請使用 '/R' 選項代替  
  
 使用- [addmodule](../language-reference/compiler-options/addmodule-compiler-option.md) 編譯器選項所參考的檔案不是以 [-target： module](../language-reference/compiler-options/target-module-compiler-option.md)所建立;使用 [-reference](../language-reference/compiler-options/reference-compiler-option.md) 可參考此編譯中的檔案。
