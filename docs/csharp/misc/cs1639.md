# <a name="compiler-error-cs1639"></a>編譯器錯誤 CS1639
介面 'interface' 的 Managed coclass 包裝函式類別簽章 'signature'，不是有效的類別名稱簽章  
  
 如果介面的中繼資料無效，會發生這個錯誤。 如果使用 C# 或支援的 .NET 語言來產生介面，則這應該絕不會發生。 請與介面廠商確認，或確認已順利產生介面組件。
