## <a name="example"></a><span data-ttu-id="98f9f-108">範例</span><span class="sxs-lookup"><span data-stu-id="98f9f-108">Example</span></span>  
 <span data-ttu-id="98f9f-109">下列範例會產生 CS0241。</span><span class="sxs-lookup"><span data-stu-id="98f9f-109">The following sample generates CS0241.</span></span> <span data-ttu-id="98f9f-110">此外，這個範例還會示範如何使用多載來模擬具有預設引數的方法。</span><span class="sxs-lookup"><span data-stu-id="98f9f-110">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {
      B x = new B();  
      x.Test();  
   }  
}  
```
