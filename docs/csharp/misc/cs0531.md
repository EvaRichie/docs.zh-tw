# <a name="compiler-error-cs0531"></a><span data-ttu-id="eea2f-103">編譯器錯誤 CS0531</span><span class="sxs-lookup"><span data-stu-id="eea2f-103">Compiler Error CS0531</span></span>

<span data-ttu-id="eea2f-104">'member' : 介面成員不能有定義</span><span class="sxs-lookup"><span data-stu-id="eea2f-104">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="eea2f-105">[介面](../language-reference/keywords/interface.md) 中宣告的方法必須實作在繼承自它的類別中，而不是在介面本身實作。</span><span class="sxs-lookup"><span data-stu-id="eea2f-105">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="eea2f-106">下列範例會產生 CS0531：</span><span class="sxs-lookup"><span data-stu-id="eea2f-106">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
