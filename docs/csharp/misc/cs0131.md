## <a name="example"></a><span data-ttu-id="99324-109">範例</span><span class="sxs-lookup"><span data-stu-id="99324-109">Example</span></span>  

 <span data-ttu-id="99324-110">下列範例會產生 CS0131。</span><span class="sxs-lookup"><span data-stu-id="99324-110">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```

    public static int Main()
