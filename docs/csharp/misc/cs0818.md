# <a name="compiler-error-cs0818"></a>編譯器錯誤 CS0818

隱含類型區域變數必須初始化  
  
 宣告的同時，隱含類型的區域變數必須以一個值初始化。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1. 請將值指派給變數，或是指定明確的類型。  
  
## <a name="example"></a>範例  

 下列程式碼會產生 CS0818：  
  
```csharp  
// cs0818.cs  
class A  
{  
    public static int Main()  
    {  
        var a; // CS0818  
        return -1;  
    }  
}  
```
