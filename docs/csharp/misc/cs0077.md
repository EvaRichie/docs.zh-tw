# <a name="compiler-error-cs0077"></a><span data-ttu-id="de5dc-103">編譯器錯誤 CS0077</span><span class="sxs-lookup"><span data-stu-id="de5dc-103">Compiler Error CS0077</span></span>
<span data-ttu-id="de5dc-104">as 運算子必須和參考類型或可為 Null 的類型一起使用 ('int' 是不可為 Null 的實值類型)。</span><span class="sxs-lookup"><span data-stu-id="de5dc-104">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="de5dc-105">傳遞了 [實值類型](../language-reference/operators/type-testing-and-cast.md#as-operator) 給 [as](../language-reference/builtin-types/value-types.md)運算子。</span><span class="sxs-lookup"><span data-stu-id="de5dc-105">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/builtin-types/value-types.md).</span></span> <span data-ttu-id="de5dc-106">因為 `as` 可以傳回 [null](../language-reference/keywords/null.md)，所以只能傳遞 [參考型別](../language-reference/keywords/reference-types.md) 或可 [為 null 的實值型](../language-reference/builtin-types/nullable-value-types.md)別。</span><span class="sxs-lookup"><span data-stu-id="de5dc-106">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed a [reference type](../language-reference/keywords/reference-types.md) or a [nullable value type](../language-reference/builtin-types/nullable-value-types.md).</span></span>
  
 <span data-ttu-id="de5dc-107">下列範例會產生 CS0077：</span><span class="sxs-lookup"><span data-stu-id="de5dc-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
