# <a name="compiler-error-cs0443"></a>編譯器錯誤 CS0443

語法錯誤，應為值  
  
 當您參考未指定陣列索引值的陣列時，就會發生這個錯誤。  
  
## <a name="example"></a>範例  

 下列程式碼會產生 CS0443。  
  
```csharp  
// CS0443.cs
using System;
class MyClass
{  
    public static void Main()
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}
    }  
}  
```
