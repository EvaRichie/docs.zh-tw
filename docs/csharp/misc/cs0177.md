 <span data-ttu-id="52357-105">未在方法主體中指派值給以 [out](../language-reference/keywords/out-parameter-modifier.md) 關鍵字標記的參數。</span><span class="sxs-lookup"><span data-stu-id="52357-105">A parameter marked with the [out](../language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="52357-106">如需詳細資訊，請參閱 [傳遞參數](../programming-guide/classes-and-structs/passing-parameters.md)</span><span class="sxs-lookup"><span data-stu-id="52357-106">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md)</span></span>

```csharp  
   }  
}  
```
