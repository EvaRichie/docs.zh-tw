# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="99886-103">編譯器警告 (層級 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="99886-103">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="99886-104">'class' 覆寫了 Object.Equals(object o)，但並沒有覆寫 Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="99886-104">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="99886-105">編譯器偵測到方法的覆寫， <xref:System.Object.Equals%2A?displayProperty=nameWithType> 但沒有方法的覆寫 <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> 。</span><span class="sxs-lookup"><span data-stu-id="99886-105">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="99886-106">的覆寫 <xref:System.Object.Equals%2A> 表示您也要覆寫 <xref:System.Object.GetHashCode%2A> 。</span><span class="sxs-lookup"><span data-stu-id="99886-106">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="99886-107">下列程式碼會產生 CS0659：</span><span class="sxs-lookup"><span data-stu-id="99886-107">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
