<span data-ttu-id="9b3ae-104">如果運算式樹狀架構的類型引數 'type' 不是委派類型，無法將 Lambda 轉換為運算式樹狀架構。</span><span class="sxs-lookup"><span data-stu-id="9b3ae-104">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="9b3ae-105">如果 Lambda 運算式轉換成運算式樹狀架構，運算式樹狀架構的引數必須具有委派類型。</span><span class="sxs-lookup"><span data-stu-id="9b3ae-105">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="9b3ae-106">此外，Lambda 運算式必須能夠轉換成委派類型。</span><span class="sxs-lookup"><span data-stu-id="9b3ae-106">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>

## <a name="example"></a><span data-ttu-id="9b3ae-109">範例</span><span class="sxs-lookup"><span data-stu-id="9b3ae-109">Example</span></span>

```csharp

using System.Linq.Expressions;  
  
public class C

    public static int Main()

        Expression<int> e = x => x + 1; // CS0835

    }  
}  
```
