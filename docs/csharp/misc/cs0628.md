# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="995e4-103">編譯器警告 (層級 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="995e4-103">Compiler Warning (level 4) CS0628</span></span>

<span data-ttu-id="995e4-104">'member': 在密封類別中宣告了新的 protected 成員</span><span class="sxs-lookup"><span data-stu-id="995e4-104">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="995e4-105">[sealed](../language-reference/keywords/sealed.md) 類別不能引入 [protected](../language-reference/keywords/protected.md) 成員，因為沒有其他類別可以繼承自 `sealed` 類別，並且使用 `protected` 成員。</span><span class="sxs-lookup"><span data-stu-id="995e4-105">A [sealed](../language-reference/keywords/sealed.md) class cannot introduce a [protected](../language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="995e4-106">下列範例會產生 CS0628：</span><span class="sxs-lookup"><span data-stu-id="995e4-106">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
