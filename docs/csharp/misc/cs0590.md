# <a name="compiler-error-cs0590"></a>編譯器錯誤 CS0590

使用者定義的運算子無法傳回 void  
  
 使用者定義運算子的目的是要傳回物件。  
  
 下列範例會產生 CS0590：  
  
```csharp  
// CS0590.cs  
namespace x  
{  
   public class a  
   {  
      public static void operator+(a A1, a A2)   // CS0590  
      {  
      }  
  
      // try the following user-defined operator  
      /*  
      public static a operator+(a A1, a A2)  
      {  
         return A2;  
      }  
      */  
  
      public static int Main()  
      {  
         return 1;  
      }  
   }  
}  
```
