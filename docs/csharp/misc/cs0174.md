# <a name="compiler-error-cs0174"></a>編譯器錯誤 CS0174

'base' 參考需要基底類別  
  
 只有當 .NET 執行時間編譯類別的原始程式碼 `System.Object` （這是唯一沒有基類的類別）時，才會發生此錯誤。
