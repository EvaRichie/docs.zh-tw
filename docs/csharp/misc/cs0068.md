 <span data-ttu-id="67279-105">介面中的事件不能有初始設定式。</span><span class="sxs-lookup"><span data-stu-id="67279-105">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="67279-106">如需詳細資訊，請參閱[介面](../programming-guide/interfaces/index.md)。</span><span class="sxs-lookup"><span data-stu-id="67279-106">For more information, see [Interfaces](../programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="67279-107">下列範例會產生 CS0068：</span><span class="sxs-lookup"><span data-stu-id="67279-107">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
