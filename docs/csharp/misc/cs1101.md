# <a name="compiler-error-cs1101"></a><span data-ttu-id="5796a-103">編譯器錯誤 CS1101</span><span class="sxs-lookup"><span data-stu-id="5796a-103">Compiler Error CS1101</span></span>

<span data-ttu-id="5796a-104">參數修飾詞 'ref' 不可與 'this' 一起使用。</span><span class="sxs-lookup"><span data-stu-id="5796a-104">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="5796a-105">當 `this` 關鍵字修改靜態方法的第一個參數時，它會通知編譯器，該方法是一個擴充方法。</span><span class="sxs-lookup"><span data-stu-id="5796a-105">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="5796a-106">擴充方法的第一個參數上不需要也不允許其他修飾詞。</span><span class="sxs-lookup"><span data-stu-id="5796a-106">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5796a-107">範例</span><span class="sxs-lookup"><span data-stu-id="5796a-107">Example</span></span>  

 <span data-ttu-id="5796a-108">下列範例會產生 CS1101：</span><span class="sxs-lookup"><span data-stu-id="5796a-108">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
