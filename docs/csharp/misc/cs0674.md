# <a name="compiler-error-cs0674"></a><span data-ttu-id="2c5c5-103">編譯器錯誤 CS0674</span><span class="sxs-lookup"><span data-stu-id="2c5c5-103">Compiler Error CS0674</span></span>
<span data-ttu-id="2c5c5-104">請勿使用 'System.ParamArrayAttribute'。</span><span class="sxs-lookup"><span data-stu-id="2c5c5-104">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="2c5c5-105">請改用 'params' 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="2c5c5-105">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="2c5c5-106">C# 編譯器不允許使用 <xref:System.ParamArrayAttribute?displayProperty=nameWithType>；請改用 [params](../language-reference/keywords/params.md) 。</span><span class="sxs-lookup"><span data-stu-id="2c5c5-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="2c5c5-107">下列範例會產生 CS0674：</span><span class="sxs-lookup"><span data-stu-id="2c5c5-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
