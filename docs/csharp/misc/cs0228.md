# <a name="compiler-error-cs0228"></a>編譯器錯誤 CS0228

'type' 不包含 'member' 的定義，或不可存取  
  
 您可能已取代系統版本的 **System.Hashtable**、 **System.String**或 **System.Array**，且取代未包含 `member`。  
  
 否則，您可能需要修復或重新安裝 Visual Studio。
