## <a name="example"></a><span data-ttu-id="f278f-105">範例</span><span class="sxs-lookup"><span data-stu-id="f278f-105">Example</span></span>  

 <span data-ttu-id="f278f-106">下列範例會產生 CS1909。</span><span class="sxs-lookup"><span data-stu-id="f278f-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
