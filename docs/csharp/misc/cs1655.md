# <a name="compiler-error-cs1655"></a><span data-ttu-id="92eb4-103">編譯器錯誤 CS1655</span><span class="sxs-lookup"><span data-stu-id="92eb4-103">Compiler Error CS1655</span></span>
<span data-ttu-id="92eb4-104">無法將 'variable' 的欄位當做 ref 或 out 引數傳遞，因為它是 'readonly variable type'</span><span class="sxs-lookup"><span data-stu-id="92eb4-104">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="92eb4-105">如果您嘗試將 [foreach](../language-reference/keywords/foreach-in.md) 變數、 [using](../language-reference/keywords/using-statement.md) 變數或 [fixed](../language-reference/keywords/fixed-statement.md) 變數的成員當做 ref 或 out 引數傳遞給函式，則會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="92eb4-105">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="92eb4-106">因為這些變數在這些內容中視為唯讀，所以這是不允許的作業。</span><span class="sxs-lookup"><span data-stu-id="92eb4-106">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="92eb4-107">下列範例會產生 CS1655：</span><span class="sxs-lookup"><span data-stu-id="92eb4-107">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
