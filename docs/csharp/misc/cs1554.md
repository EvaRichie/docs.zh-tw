# <a name="compiler-error-cs1554"></a><span data-ttu-id="3b138-103">編譯器錯誤 CS1554</span><span class="sxs-lookup"><span data-stu-id="3b138-103">Compiler Error CS1554</span></span>

<span data-ttu-id="3b138-104">宣告無效;使用 ' \<type> operator op ( ... '相反</span><span class="sxs-lookup"><span data-stu-id="3b138-104">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="3b138-105">多載 [運算子](../language-reference/operators/operator-overloading.md) 的傳回類型必須出現在關鍵字之前 `operator` 。</span><span class="sxs-lookup"><span data-stu-id="3b138-105">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>

```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
