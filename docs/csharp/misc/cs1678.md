# <a name="compiler-error-cs1678"></a>編譯器錯誤 CS1678

參數 'number' 宣告為類型 'type1'，但應該是 'type2'  
  
 當匿名方法中的參數類型和轉換方法的委派宣告不同時，就會發生這個錯誤。  
  
 下列範例會產生 CS1678：  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors
{  
   static void Main()
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
