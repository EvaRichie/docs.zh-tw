 <span data-ttu-id="471a5-106">您已搭配使用 "::" 與參考類型的別名。</span><span class="sxs-lookup"><span data-stu-id="471a5-106">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="471a5-107">若要解決這個錯誤，請使用 "." 運算子。</span><span class="sxs-lookup"><span data-stu-id="471a5-107">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="471a5-108">下列範例會產生 CS0431：</span><span class="sxs-lookup"><span data-stu-id="471a5-108">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
