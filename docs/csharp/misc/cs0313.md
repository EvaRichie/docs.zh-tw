# <a name="compiler-error-cs0313"></a><span data-ttu-id="43224-103">編譯器錯誤 CS0313</span><span class="sxs-lookup"><span data-stu-id="43224-103">Compiler Error CS0313</span></span>
<span data-ttu-id="43224-104">類型 'type1' 不能作為泛型類型或方法 'type2' 中的類型參數 'parameter name'。</span><span class="sxs-lookup"><span data-stu-id="43224-104">The type 'type1' cannot be used as type parameter 'parameter name' in the generic type or method 'type2'.</span></span> <span data-ttu-id="43224-105">可為 Null 的類型 'type1' 無法滿足 'type2' 的條件約束。</span><span class="sxs-lookup"><span data-stu-id="43224-105">The nullable type 'type1' does not satisfy the constraint of 'type2'.</span></span> <span data-ttu-id="43224-106">可為 Null 的類型無法滿足任何介面條件約束。</span><span class="sxs-lookup"><span data-stu-id="43224-106">Nullable types cannot satisfy any interface constraints.</span></span>  
  
 <span data-ttu-id="43224-107">可為 null 的實值型別不等於它的不可為 null 的對應專案。</span><span class="sxs-lookup"><span data-stu-id="43224-107">A nullable value type is not equivalent to its non-nullable counterpart.</span></span> <span data-ttu-id="43224-108">在下列範例中， `ImplStruct` 滿足 `BaseInterface` 條件約束，但 `ImplStruct?` 不滿足，因為 `Nullable<ImplStruct>` 未實作 `BaseInterface`。</span><span class="sxs-lookup"><span data-stu-id="43224-108">In the example that follows, `ImplStruct` satisfies the `BaseInterface` constraint but `ImplStruct?` does not because `Nullable<ImplStruct>` does not implement `BaseInterface`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="43224-109">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="43224-109">To correct this error</span></span>  
  
1. <span data-ttu-id="43224-110">使用下面的程式碼作為範例，其中一種解決方案是指定一般 `ImplStruct` 作為 `TestMethod`呼叫中的第一個類型引數。</span><span class="sxs-lookup"><span data-stu-id="43224-110">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="43224-111">然後，修改 `TestMethod` ，以在其傳回陳述式中建立可為 Null 的 `Implstruct` 版本：</span><span class="sxs-lookup"><span data-stu-id="43224-111">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>  
  
    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="43224-112">範例</span><span class="sxs-lookup"><span data-stu-id="43224-112">Example</span></span>

<span data-ttu-id="43224-113">下列程式碼會產生 CS0313：</span><span class="sxs-lookup"><span data-stu-id="43224-113">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
