## <a name="example"></a><span data-ttu-id="c9318-111">範例</span><span class="sxs-lookup"><span data-stu-id="c9318-111">Example</span></span>  

 <span data-ttu-id="c9318-112">下列範例會產生 CS1059，因為 `i` 是常數，而非變數，而 `E` 是其項目也是常數值的 `Enum` 類型。</span><span class="sxs-lookup"><span data-stu-id="c9318-112">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
