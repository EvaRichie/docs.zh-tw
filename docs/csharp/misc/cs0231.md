 <span data-ttu-id="371be-105">[params](../language-reference/keywords/params.md) 參數支援數目不定的引數，且必須在所有其他參數之後。</span><span class="sxs-lookup"><span data-stu-id="371be-105">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="371be-106">如需詳細資訊，請參閱[方法](../programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="371be-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="371be-107">下列範例會產生 CS0231：</span><span class="sxs-lookup"><span data-stu-id="371be-107">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```
