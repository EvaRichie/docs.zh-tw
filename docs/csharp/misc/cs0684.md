# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="b1cc2-103">編譯器警告 (層級 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="b1cc2-103">Compiler Warning (level 1) CS0684</span></span>

<span data-ttu-id="b1cc2-104">'interface' 介面標記為 'CoClassAttribute'，但沒有標記為 'ComImportAttribute'</span><span class="sxs-lookup"><span data-stu-id="b1cc2-104">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="b1cc2-105">如果您在介面指定 **CoClassAttribute** ，則也必須指定 **ComImportAttribute**。</span><span class="sxs-lookup"><span data-stu-id="b1cc2-105">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="b1cc2-106">下列範例會產生 CS0684：</span><span class="sxs-lookup"><span data-stu-id="b1cc2-106">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
