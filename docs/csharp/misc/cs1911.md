# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="d8a82-102">編譯器警告 (層級 1) CS1911</span><span class="sxs-lookup"><span data-stu-id="d8a82-102">Compiler Warning (level 1) CS1911</span></span>

<span data-ttu-id="d8a82-103">透過匿名方法、Lambda 運算式、查詢運算式或迭代器的 'base' 關鍵字存取成員 'name'，會導致無法驗證的程式碼。</span><span class="sxs-lookup"><span data-stu-id="d8a82-103">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="d8a82-104">考慮將存取移入所包含類型的 Helper 方法。</span><span class="sxs-lookup"><span data-stu-id="d8a82-104">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="d8a82-105">以迭代器或匿名方法之方法主體內的 `base` 關鍵字呼叫虛擬函式，會導致無法驗證的程式碼。</span><span class="sxs-lookup"><span data-stu-id="d8a82-105">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="d8a82-106">無法驗證的程式碼無法在部分信任的環境中執行。</span><span class="sxs-lookup"><span data-stu-id="d8a82-106">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="d8a82-107">CS1911 的解決方法之一，是將虛擬函式呼叫移至 Helper 函式。</span><span class="sxs-lookup"><span data-stu-id="d8a82-107">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d8a82-108">範例</span><span class="sxs-lookup"><span data-stu-id="d8a82-108">Example</span></span>  

 <span data-ttu-id="d8a82-109">下列範例會產生 CS1911。</span><span class="sxs-lookup"><span data-stu-id="d8a82-109">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
