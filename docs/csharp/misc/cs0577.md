<span data-ttu-id="e3daf-104">Conditional 屬性在 'function' 上無效，因為它是建構函式、解構函式、運算子或明確介面實作</span><span class="sxs-lookup"><span data-stu-id="e3daf-104">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>

```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
