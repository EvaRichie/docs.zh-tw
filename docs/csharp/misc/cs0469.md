 <span data-ttu-id="1455d-105">當您使用 `goto case`時，必須有從 goto 案例的值到參數類型的隱含轉換。</span><span class="sxs-lookup"><span data-stu-id="1455d-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1455d-106">範例</span><span class="sxs-lookup"><span data-stu-id="1455d-106">Example</span></span>  
 <span data-ttu-id="1455d-107">下列範例會產生 CS0469。</span><span class="sxs-lookup"><span data-stu-id="1455d-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
