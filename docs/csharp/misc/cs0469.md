# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="299a8-103">編譯器警告 (層級 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="299a8-103">Compiler Warning (level 2) CS0469</span></span>

<span data-ttu-id="299a8-104">'goto case' 值未隱含轉換成類型 'type'</span><span class="sxs-lookup"><span data-stu-id="299a8-104">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="299a8-105">當您使用 `goto case`時，必須有從 goto 案例的值到參數類型的隱含轉換。</span><span class="sxs-lookup"><span data-stu-id="299a8-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="299a8-106">範例</span><span class="sxs-lookup"><span data-stu-id="299a8-106">Example</span></span>  

 <span data-ttu-id="299a8-107">下列範例會產生 CS0469。</span><span class="sxs-lookup"><span data-stu-id="299a8-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
