## <a name="example"></a><span data-ttu-id="467b3-110">範例</span><span class="sxs-lookup"><span data-stu-id="467b3-110">Example</span></span>  
 <span data-ttu-id="467b3-111">下列範例會產生 CS0279：</span><span class="sxs-lookup"><span data-stu-id="467b3-111">The following example generates CS0279:</span></span>

{  
    IEnumerator IEnumerable.GetEnumerator()

    }
