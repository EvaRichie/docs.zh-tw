# <a name="compiler-error-cs1650"></a><span data-ttu-id="fef1f-103">編譯器錯誤 CS1650</span><span class="sxs-lookup"><span data-stu-id="fef1f-103">Compiler Error CS1650</span></span>
<span data-ttu-id="fef1f-104">無法對靜態唯讀欄位 'identifier' 的欄位進行指派 (除非在靜態建構函式或變數初始設定式)</span><span class="sxs-lookup"><span data-stu-id="fef1f-104">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="fef1f-105">當您嘗試修改不允許修改的唯讀和靜態欄位成員時，就會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="fef1f-105">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="fef1f-106">若要解決這個錯誤，請將唯讀欄位的指派限制為建構函式或變數初始設定式，或從欄位的宣告中移除 `readonly` 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="fef1f-106">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
