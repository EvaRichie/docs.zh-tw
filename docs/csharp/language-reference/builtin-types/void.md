# <a name="void-c-reference"></a>void (c # 參考) 

您可以使用 `void` 做為[方法](../../programming-guide/classes-and-structs/methods.md)的傳回型別 (或[區域](../../programming-guide/classes-and-structs/local-functions.md)函式) 來指定方法不會傳回值。

[!code-csharp[void method](snippets/VoidType.cs#VoidExample)]

您也可以使用 `void` 做為參考型別，宣告未知類型的指標。 如需詳細資訊，請參閱[指標類型](../../programming-guide/unsafe-code-pointers/pointer-types.md)。

您不能使用 `void` 做為變數的類型。
