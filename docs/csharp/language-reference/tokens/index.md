# <a name="c-special-characters"></a>C# 特殊字元

特殊字元是預先定義的內容字元，這類字元會修改將它們加在前面的程式元素 (常值字串、識別項或屬性名稱)。 C# 支援下列特殊字元：

- [@](./verbatim.md)，逐字識別碼字元。

- [$](./interpolated.md)，插補的字串字元。
