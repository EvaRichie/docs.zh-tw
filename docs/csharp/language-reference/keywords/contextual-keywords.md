# <a name="contextual-keywords-c-reference"></a>內容關鍵字 (C# 參考)

內容關鍵字可用來在程式碼中提供特定的意義，但它不是 C# 中的保留字。 本節介紹下列內容關鍵字︰  
  
|關鍵字|描述|  
|-------------|-----------------|  
|[新增](./add.md)|定義用戶端程式碼訂閱事件時所叫用的自訂事件存取子。|  
|[async](./async.md)|指出修改的方法、Lambda 運算式或匿名方法為非同步。|  
|[await](../operators/await.md)|除非等候的工作完成，否則會暫止非同步方法。|  
|[動態](../builtin-types/reference-types.md)|定義參考型別，以讓發生它的作業略過編譯時期類型檢查。|  
|[get](./get.md)|定義屬性或索引子的存取子方法。|  
|[全球](../operators/namespace-alias-qualifier.md)|全域命名空間的別名，否則為未命名。|  
|[部分](./partial-type.md)|定義相同編譯單位內的部分類別、結構和介面。|  
|[remove](./remove.md)|定義用戶端程式碼解除與事件的訂閱時所叫用的自訂事件存取子。|  
|[set](./set.md)|定義屬性或索引子的存取子方法。|  
|[value](./value.md)|用來設定存取子，以及新增或移除事件處理常式。|  
|[無 功](./var.md)|可讓編譯器決定方法範圍內所宣告的變數類型。|  
|[當](when.md)|指定 `catch` 區塊的篩選條件或 `switch` 陳述式的 `case` 標籤。|
|[where](./where-generic-type-constraint.md)|將條件約束新增至泛型宣告。 (另請參閱 [where](./where-clause.md))。|  
|[yield](./yield.md)|用於迭代器區塊中，以將值傳回給列舉值物件或表示反覆項目結尾。|  
  
 C# 3.0 中引進的所有查詢關鍵字也都是內容。 如需詳細資訊，請參閱[查詢關鍵字 (LINQ)](./query-keywords.md)。
