# <a name="default-c-reference"></a>default (C# 參考)

`default` 關鍵字能以兩種方式使用：

- 在[ `switch` 語句](switch.md)中指定預設標籤。
- 作為[預設運算子或常值](../operators/default.md)以產生型別的預設值。
