# <a name="endregion-c-reference"></a>#endregion (C# 參考)

`#endregion` 會標示 [#region](./preprocessor-region.md) 區塊的結尾。 例如：  
  
```csharp
#region MyClass definition  
class MyClass
{  
    static void Main()
    {  
    }  
}  
#endregion  
```
