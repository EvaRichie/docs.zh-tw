## <a name="example"></a><span data-ttu-id="bcb27-111">範例</span><span class="sxs-lookup"><span data-stu-id="bcb27-111">Example</span></span>  

 <span data-ttu-id="bcb27-112">下列範例會產生 CS0178。</span><span class="sxs-lookup"><span data-stu-id="bcb27-112">The following sample generates CS0178.</span></span>  
  
```csharp  
// CS0178.cs  
class MyClass  
{  
   public static void Main()  
   {  
      int a = new int[5][,][][5;   // CS0178  
      int[,] b = new int[3,2];   // OK  
  
      int[][] c = new int[10][];  
      c[0] = new int[5][5];   // CS0178  
      c[0] = new int[2];   // OK  
      c[1] = new int[2]{1,2};   // OK  
   }  
}  
```
