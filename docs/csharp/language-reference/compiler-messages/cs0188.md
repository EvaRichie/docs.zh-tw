# <a name="compiler-error-cs0188"></a><span data-ttu-id="1cd9e-103">編譯器錯誤 CS0188</span><span class="sxs-lookup"><span data-stu-id="1cd9e-103">Compiler Error CS0188</span></span>

<span data-ttu-id="1cd9e-104">指定其所有欄位前，無法使用 'this' 物件</span><span class="sxs-lookup"><span data-stu-id="1cd9e-104">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="1cd9e-105">建構函式必須先指派 `struct` 中的所有欄位，才能在 `struct` 中呼叫方法。</span><span class="sxs-lookup"><span data-stu-id="1cd9e-105">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="1cd9e-106">如果您在嘗試初始化結構函式中的屬性時看到此錯誤，解決方案是直接初始化該屬性的支援欄位，而不是屬性本身。</span><span class="sxs-lookup"><span data-stu-id="1cd9e-106">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="1cd9e-107">範例</span><span class="sxs-lookup"><span data-stu-id="1cd9e-107">Example</span></span>

 <span data-ttu-id="1cd9e-108">下列範例會產生 CS0188：</span><span class="sxs-lookup"><span data-stu-id="1cd9e-108">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
