# <a name="compiler-error-cs1019"></a><span data-ttu-id="1a5e9-103">編譯器錯誤 CS1019</span><span class="sxs-lookup"><span data-stu-id="1a5e9-103">Compiler Error CS1019</span></span>

<span data-ttu-id="1a5e9-104">必須是可多載的一元運算子</span><span class="sxs-lookup"><span data-stu-id="1a5e9-104">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="1a5e9-105">已宣告看起來像是可多載的一元運算子物件，但運算子遺失或位在簽章的錯誤位置中。</span><span class="sxs-lookup"><span data-stu-id="1a5e9-105">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="1a5e9-106">「一元運算子」是在單一運算元上作業的運算子。</span><span class="sxs-lookup"><span data-stu-id="1a5e9-106">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="1a5e9-107">例如，`++` 就是一元運算子。</span><span class="sxs-lookup"><span data-stu-id="1a5e9-107">For example, `++` is a unary operator.</span></span> <span data-ttu-id="1a5e9-108">您可以使用 `operator` 關鍵字，指定運算子操作類型的單一參數，來多載一些一元運算子。</span><span class="sxs-lookup"><span data-stu-id="1a5e9-108">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="1a5e9-109">例如，如果您想要為使用者定義的類別 `Temperature` 多載運算子 `++`，以便寫入 `Temperature++`，您可以下列方式定義它︰</span><span class="sxs-lookup"><span data-stu-id="1a5e9-109">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="1a5e9-110">當您收到此錯誤時，您已宣告看起來像是可多載的一元運算子物件，但運算子本身遺失或位在簽章的錯誤位置中。</span><span class="sxs-lookup"><span data-stu-id="1a5e9-110">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="1a5e9-111">如果從上例的簽章中移除 `++`，就會產生 CS1019。</span><span class="sxs-lookup"><span data-stu-id="1a5e9-111">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="1a5e9-112">下列程式碼會產生 CS1019：</span><span class="sxs-lookup"><span data-stu-id="1a5e9-112">The following code generates CS1019:</span></span>

   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;
   }  
  
   public static void Main()  
   {  
   }
