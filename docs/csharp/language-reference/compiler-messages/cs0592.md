## <a name="example"></a><span data-ttu-id="85e4d-109">範例</span><span class="sxs-lookup"><span data-stu-id="85e4d-109">Example</span></span>

 <span data-ttu-id="85e4d-110">下列範例會產生 CS0592：</span><span class="sxs-lookup"><span data-stu-id="85e4d-110">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
