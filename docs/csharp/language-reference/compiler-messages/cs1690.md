# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="78b56-103">編譯器警告 (層級 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="78b56-103">Compiler Warning (level 1) CS1690</span></span>

<span data-ttu-id="78b56-104">存取 'member' 上的成員可能會造成執行階段例外狀況，因為其為 marshal-by-reference 類別的欄位</span><span class="sxs-lookup"><span data-stu-id="78b56-104">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="78b56-105">當您嘗試在衍生自 <xref:System.MarshalByRefObject> 的類別成員上呼叫方法、屬性或索引子，而且此成員為實值型別時，就會發生此警告。</span><span class="sxs-lookup"><span data-stu-id="78b56-105">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="78b56-106">繼承自 `MarshalByRefObject` 的物件通常用於以傳址方式跨應用程式定義域進行封送處理。</span><span class="sxs-lookup"><span data-stu-id="78b56-106">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="78b56-107">如果有任何程式碼嘗試在應用程式域中直接存取這類物件的實值型別成員， <xref:System.InvalidOperationException> 就會發生執行時間。</span><span class="sxs-lookup"><span data-stu-id="78b56-107">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="78b56-108">若要解決這個警告，請先將成員複製到區域變數，再於該變數上呼叫方法。</span><span class="sxs-lookup"><span data-stu-id="78b56-108">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="78b56-109">下列範例會產生 CS1690：</span><span class="sxs-lookup"><span data-stu-id="78b56-109">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {
        AppDomain domain = AppDomain.CreateDomain("MyDomain");
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;
    }
}
```
