# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="37275-103">編譯器警告 (層級 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="37275-103">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="37275-104">只有未命名陣列類型有差異的多載方法 'method' 不符合 CLS 標準</span><span class="sxs-lookup"><span data-stu-id="37275-104">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="37275-105">如果您有使用不規則陣列 (Jagged Array) 的多載方法，而方法簽章的唯一不同只在於陣列的元素類型，便會發生這個錯誤。</span><span class="sxs-lookup"><span data-stu-id="37275-105">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="37275-106">若要避免這個錯誤，請用矩形陣列取代不規則陣列、使用額外的參數來分辨函式呼叫、重新命名一或多個多載方法，或者如果不需要符合 CLS 標準，可移除 <xref:System.CLSCompliantAttribute> 屬性。</span><span class="sxs-lookup"><span data-stu-id="37275-106">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="37275-107">如需 CLS 合規性的詳細資訊，請參閱[語言獨立性以及與語言無關的元件](../../../standard/language-independence-and-language-independent-components.md)。</span><span class="sxs-lookup"><span data-stu-id="37275-107">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](../../../standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="37275-108">範例</span><span class="sxs-lookup"><span data-stu-id="37275-108">Example</span></span>  
 <span data-ttu-id="37275-109">下列範例會產生 CS3007：</span><span class="sxs-lookup"><span data-stu-id="37275-109">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
