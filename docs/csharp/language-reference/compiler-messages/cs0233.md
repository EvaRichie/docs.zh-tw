# <a name="compiler-error-cs0233"></a><span data-ttu-id="6ca72-103">編譯器錯誤 CS0233</span><span class="sxs-lookup"><span data-stu-id="6ca72-103">Compiler Error CS0233</span></span>

<span data-ttu-id="6ca72-104">'identifier' 沒有預先定義的大小，因此只能在 unsafe 內容中使用 sizeof</span><span class="sxs-lookup"><span data-stu-id="6ca72-104">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="6ca72-105">若沒有 [unsafe](../keywords/unsafe.md) 內容，則 [sizeof](../operators/sizeof.md) 運算子便只能針對大小在編譯時間為常數的型別使用。</span><span class="sxs-lookup"><span data-stu-id="6ca72-105">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="6ca72-106">若您收到這項錯誤，請使用 unsafe 內容。</span><span class="sxs-lookup"><span data-stu-id="6ca72-106">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="6ca72-107">下列範例會產生 CS0233：</span><span class="sxs-lookup"><span data-stu-id="6ca72-107">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
