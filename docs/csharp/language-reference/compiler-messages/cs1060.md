## <a name="example"></a><span data-ttu-id="612e4-112">範例</span><span class="sxs-lookup"><span data-stu-id="612e4-112">Example</span></span>  

 <span data-ttu-id="612e4-113">因為類別類型 `U` 是 `struct S` 的成員，但永遠不會進行初始化，所以下列程式碼會產生 CS1060。</span><span class="sxs-lookup"><span data-stu-id="612e4-113">The following code generates CS1060 because the class type `U` is a member of the `struct S` but is never initialized.</span></span>

```csharp  
namespace CS1060  
{
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }
}  
```
