## <a name="example"></a><span data-ttu-id="c0634-107">範例</span><span class="sxs-lookup"><span data-stu-id="c0634-107">Example</span></span>  
 <span data-ttu-id="c0634-108">下列多檔案範例參考使用兩個不同外部別名的組件。</span><span class="sxs-lookup"><span data-stu-id="c0634-108">The following multifile sample references an assembly using two different external aliases.</span></span> <span data-ttu-id="c0634-109">第一個範例會建立舊版程式碼，以建立組件 CS1701_d。</span><span class="sxs-lookup"><span data-stu-id="c0634-109">This first sample builds the older version of the code that creates assembly CS1701_d.</span></span>

// compile with: /target:library /out:cs1701_d.dll /keyfile:mykey.snk  
using System.Reflection;  
[assembly:AssemblyVersion("1.0")]  
public class A {  
   public void M1() {}  
}  
  
public class C1 {}  
```  
  
## <a name="example"></a><span data-ttu-id="c0634-110">範例</span><span class="sxs-lookup"><span data-stu-id="c0634-110">Example</span></span>  
 <span data-ttu-id="c0634-111">這是建立新版組件 CS1701_d 的程式碼。</span><span class="sxs-lookup"><span data-stu-id="c0634-111">This is the code that creates the newer version of assembly CS1701_d.</span></span> <span data-ttu-id="c0634-112">請注意，它會編譯到與舊版不同的目錄中，由於輸出檔案具有相同名稱，因此必須這樣做。</span><span class="sxs-lookup"><span data-stu-id="c0634-112">Note that it compiles into a different directory than the older version, necessary since the output files have the same names.</span></span>

public class A {  
   public void M2() {}  
   public void M1() {}  
}  
  
public class C2 {}  
public class C1 {}  
```  
  
## <a name="example"></a><span data-ttu-id="c0634-113">範例</span><span class="sxs-lookup"><span data-stu-id="c0634-113">Example</span></span>  
 <span data-ttu-id="c0634-114">此範例會設定外部別名 A1 和 A2。</span><span class="sxs-lookup"><span data-stu-id="c0634-114">This sample sets up the external aliases A1 and A2.</span></span>  
  
```csharp  
// CS1701_c.cs  
// compile with: /target:library /reference:A2=c:\\cs1701_d.dll /reference:A1=cs1701_d.dll  
  
extern alias A1;  
extern alias A2;  
// using System;  
using a1 = A1::A;  
using a2 = A2::A;  
  
public class Ref {  
   public static a1 A1() { return new a1(); }  
   public static a2 A2() { return new a2(); }  
  
   public static A1::C1 M1() { return new A1::C1(); }  
   public static A2::C2 M2() { return new A2::C2(); }  
}  
```
