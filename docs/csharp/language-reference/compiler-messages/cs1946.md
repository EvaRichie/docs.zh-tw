# <a name="compiler-error-cs1946"></a><span data-ttu-id="dbc54-102">編譯器錯誤 CS1946</span><span class="sxs-lookup"><span data-stu-id="dbc54-102">Compiler Error CS1946</span></span>

<span data-ttu-id="dbc54-103">匿名方法運算式無法轉換為運算式樹狀架構。</span><span class="sxs-lookup"><span data-stu-id="dbc54-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="dbc54-104">[匿名方法](../operators/delegate-operator.md)代表一組語句，但是運算式樹狀架構不得包含語句。</span><span class="sxs-lookup"><span data-stu-id="dbc54-104">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="dbc54-105">因此，無法以運算式樹狀架構表示匿名方法。</span><span class="sxs-lookup"><span data-stu-id="dbc54-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="dbc54-106">若要更正此錯誤，請將匿名方法變更為 [Lambda 運算式](../operators/lambda-expressions.md)。</span><span class="sxs-lookup"><span data-stu-id="dbc54-106">To correct this error, change the anonymous method to a [lambda expression](../operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="dbc54-107">範例</span><span class="sxs-lookup"><span data-stu-id="dbc54-107">Example</span></span>

<span data-ttu-id="dbc54-108">下列範例會產生 CS1946：</span><span class="sxs-lookup"><span data-stu-id="dbc54-108">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
