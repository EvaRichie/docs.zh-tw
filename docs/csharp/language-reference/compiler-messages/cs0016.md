# <a name="compiler-error-cs0016"></a>編譯器錯誤 CS0016

無法寫入輸出檔 'file' — 'reason'

 編譯器無法寫入輸出檔案。 請檢查檔案路徑，確定檔案存在。 如果該位置已有之前建置的檔案，請確定該檔案可供寫入，且目前沒有處理序鎖定該檔案。 例如，請確定當您嘗試建置時，您的可執行檔並未執行。
