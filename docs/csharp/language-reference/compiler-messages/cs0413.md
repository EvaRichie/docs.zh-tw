# <a name="compiler-error-cs0413"></a><span data-ttu-id="c3b31-103">編譯器錯誤 CS0413</span><span class="sxs-lookup"><span data-stu-id="c3b31-103">Compiler Error CS0413</span></span>
<span data-ttu-id="c3b31-104">型別參數 'type parameter' 不能配合 'as' 運算子使用，因為它沒有類別類型條件約束也沒有 'class' 條件約束</span><span class="sxs-lookup"><span data-stu-id="c3b31-104">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
<span data-ttu-id="c3b31-105">如果泛型型別使用 [as](../operators/type-testing-and-cast.md#as-operator)運算子，但該泛型型別沒有類別類型條件約束，就會發生此錯誤。</span><span class="sxs-lookup"><span data-stu-id="c3b31-105">This error occurs if a generic type uses the [as](../operators/type-testing-and-cast.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="c3b31-106">`as` 運算子只能用於參考和可為 Null 的實值型別，因此型別參數必須受到條件約束，以確保它不是實值型別。</span><span class="sxs-lookup"><span data-stu-id="c3b31-106">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="c3b31-107">若要避免這個錯誤，請使用類別類型條件約束或參考型別條件約束。</span><span class="sxs-lookup"><span data-stu-id="c3b31-107">To avoid this error, use a class type constraint or a reference type constraint.</span></span>
  
<span data-ttu-id="c3b31-108">這是因為 `as` 運算子可能會傳回 `null`，但這不是實值型別的可能值。除非型別參數是類別類型條件約束或參考型別條件約束，否則就必須將型別參數視為實值型別。</span><span class="sxs-lookup"><span data-stu-id="c3b31-108">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>
  
## <a name="example"></a><span data-ttu-id="c3b31-109">範例</span><span class="sxs-lookup"><span data-stu-id="c3b31-109">Example</span></span>

<span data-ttu-id="c3b31-110">下列範例會產生 CS0413。</span><span class="sxs-lookup"><span data-stu-id="c3b31-110">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
