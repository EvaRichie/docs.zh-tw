# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="6ec51-103">編譯器警告 (層級 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="6ec51-103">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="6ec51-104">'member1' 隱藏了繼承的成員 'member2'。</span><span class="sxs-lookup"><span data-stu-id="6ec51-104">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="6ec51-105">如果是刻意要隱藏，請使用 new 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="6ec51-105">Use the new keyword if hiding was intended.</span></span>

 <span data-ttu-id="6ec51-106">所宣告的變數名稱與基底類別中的變數相同。</span><span class="sxs-lookup"><span data-stu-id="6ec51-106">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="6ec51-107">不過，未使用 [new](../keywords/new-modifier.md) 關鍵字。</span><span class="sxs-lookup"><span data-stu-id="6ec51-107">However, the [new](../keywords/new-modifier.md) keyword was not used.</span></span> <span data-ttu-id="6ec51-108">此警告通知您應該使用 **new**；宣告變數的方式就像已將 **new** 用於宣告一樣。</span><span class="sxs-lookup"><span data-stu-id="6ec51-108">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>

 <span data-ttu-id="6ec51-109">下列範例會產生 CS0108：</span><span class="sxs-lookup"><span data-stu-id="6ec51-109">The following sample generates CS0108:</span></span>

```csharp
// CS0108.cs
// compile with: /W:2
using System;

namespace x
{
    public class clx
    {
        public int i = 1;
    }

    public class cly : clx
    {
        public static int i = 2;   // CS0108, use the new keyword
        // Use the following line instead:
        // public static new int i = 2;

      public static void Main()
      {
         Console.WriteLine(i);
      }
   }
}
```
