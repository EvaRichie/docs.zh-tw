# <a name="compiler-error-cs0826"></a><span data-ttu-id="8afe9-103">編譯器錯誤 CS0826</span><span class="sxs-lookup"><span data-stu-id="8afe9-103">Compiler Error CS0826</span></span>
<span data-ttu-id="8afe9-104">找不到隱含型別陣列的最佳類型。</span><span class="sxs-lookup"><span data-stu-id="8afe9-104">No best type found for implicitly typed array.</span></span>  
  
 <span data-ttu-id="8afe9-105">陣列元素的類型必須全部相同，或是可以根據編譯器所使用的型別推斷規則，隱含轉換成相同類型。</span><span class="sxs-lookup"><span data-stu-id="8afe9-105">Array elements must all be the same type or implicitly convertible to the same type according to the type inference rules used by the compiler.</span></span> <span data-ttu-id="8afe9-106">最佳類型必須是陣列運算式中現有的其中一種類型。</span><span class="sxs-lookup"><span data-stu-id="8afe9-106">The best type must be one of the types present in the array expression.</span></span> <span data-ttu-id="8afe9-107">元素不會轉換成新的類型，例如 `object`。</span><span class="sxs-lookup"><span data-stu-id="8afe9-107">Elements will not be converted to a new type such as `object`.</span></span> <span data-ttu-id="8afe9-108">若是隱含型別陣列，編譯器必須根據指派給陣列的元素類型來推斷陣列類型。</span><span class="sxs-lookup"><span data-stu-id="8afe9-108">For an implicitly typed array, the compiler must infer the array type based on the type of elements assigned to it.</span></span>

## <a name="example"></a><span data-ttu-id="8afe9-113">範例</span><span class="sxs-lookup"><span data-stu-id="8afe9-113">Example</span></span>  
 <span data-ttu-id="8afe9-114">因為陣列元素的類型並不是都相同，而且編譯器的型別推斷邏輯找不到單一最佳類型，所以下列程式碼會產生 CS0826：</span><span class="sxs-lookup"><span data-stu-id="8afe9-114">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>  
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
