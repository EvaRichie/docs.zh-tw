<span data-ttu-id="5a53e-104">'Property name' 不是標記成 abstract 或 extern，因此必須宣告主體。</span><span class="sxs-lookup"><span data-stu-id="5a53e-104">'Property name' must declare a body because it is not marked abstract or extern.</span></span> <span data-ttu-id="5a53e-105">自動實作屬性必須同時定義 get 與 set 存取子。</span><span class="sxs-lookup"><span data-stu-id="5a53e-105">Automatically implemented properties must define both get and set accessors.</span></span>  
  
 <span data-ttu-id="5a53e-106">除非規則屬性標記成 `abstract` 或`extern`，或是 `partial` 類型的成員，否則必須提供主體。</span><span class="sxs-lookup"><span data-stu-id="5a53e-106">Unless a regular property is marked as `abstract` or `extern`, or is a member of a `partial` type, it must supply a body.</span></span> <span data-ttu-id="5a53e-107">自動實作屬性未提供存取子主體，但必須同時指定這兩個存取子。</span><span class="sxs-lookup"><span data-stu-id="5a53e-107">Auto-implemented properties do not provide accessor bodies, but they must specify both accessors.</span></span> <span data-ttu-id="5a53e-108">若要建立唯讀自動實作屬性，請將 set 存取子設為 `private`。</span><span class="sxs-lookup"><span data-stu-id="5a53e-108">To create a read-only auto-implemented property, make the set accessor `private`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5a53e-109">更正這個錯誤</span><span class="sxs-lookup"><span data-stu-id="5a53e-109">To correct this error</span></span>  
  
1. <span data-ttu-id="5a53e-110">提供遺漏的主體或存取子，或是在其上和 (或) 其封入類型上使用 [abstract](../keywords/abstract.md)、[extern](../keywords/extern.md) 或 [partial (類型)](../keywords/partial-type.md) 修飾詞。</span><span class="sxs-lookup"><span data-stu-id="5a53e-110">Supply the missing body or accessor or else use the [abstract](../keywords/abstract.md), [extern](../keywords/extern.md), or [partial (Type)](../keywords/partial-type.md) modifiers on it and/or its enclosing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5a53e-111">範例</span><span class="sxs-lookup"><span data-stu-id="5a53e-111">Example</span></span>  

 <span data-ttu-id="5a53e-112">下列範例會產生 CS0840：</span><span class="sxs-lookup"><span data-stu-id="5a53e-112">The following example generates CS0840:</span></span>

{  
    public int myProp { get; } // CS0840
