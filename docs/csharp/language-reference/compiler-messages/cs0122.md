## <a name="example"></a><span data-ttu-id="739ed-109">範例</span><span class="sxs-lookup"><span data-stu-id="739ed-109">Example</span></span>

 <span data-ttu-id="739ed-110">下列範例會產生 CS0122：</span><span class="sxs-lookup"><span data-stu-id="739ed-110">The following sample generates CS0122:</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
