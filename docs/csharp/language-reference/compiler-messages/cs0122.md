# <a name="compiler-error-cs0122"></a>編譯器錯誤 CS0122

'member' 的保護層級導致無法對其進行存取

 類別成員的[存取修飾詞](../keywords/index.md)可防止存取成員。 如需詳細資訊，請參閱[存取修飾詞](../../programming-guide/classes-and-structs/access-modifiers.md)。

 其中一個原因 (未於下列範例中顯示) 可能是省略 Friend 組件目標的 **/out** 編譯器旗標。 如需詳細資訊，請參閱 [Friend 元件](../../../standard/assembly/friend.md) 和 [-Out (c # 編譯器選項) ](../compiler-options/out-compiler-option.md)。

## <a name="example"></a>範例

 下列範例會產生 CS0122：

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
