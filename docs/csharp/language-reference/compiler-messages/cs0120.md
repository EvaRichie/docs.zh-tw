<span data-ttu-id="3ccd0-104">需要有物件參考才能使用非靜態欄位、方法或屬性 'member'</span><span class="sxs-lookup"><span data-stu-id="3ccd0-104">An object reference is required for the nonstatic field, method, or property 'member'</span></span>

 <span data-ttu-id="3ccd0-105">若要使用非靜態欄位、方法或屬性，您必須先建立物件執行個體。</span><span class="sxs-lookup"><span data-stu-id="3ccd0-105">In order to use a non-static field, method, or property, you must first create an object instance.</span></span> <span data-ttu-id="3ccd0-106">如需靜態方法的詳細資訊，請參閱[靜態類別和靜態類別成員](../../programming-guide/classes-and-structs/static-classes-and-static-class-members.md)。</span><span class="sxs-lookup"><span data-stu-id="3ccd0-106">For more information about static methods, see [Static Classes and Static Class Members](../../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span> <span data-ttu-id="3ccd0-107">如需建立類別執行個體的詳細資訊，請參閱[執行個體建構函式](../../programming-guide/classes-and-structs/instance-constructors.md)。</span><span class="sxs-lookup"><span data-stu-id="3ccd0-107">For more information about creating instances of classes, see [Instance Constructors](../../programming-guide/classes-and-structs/instance-constructors.md).</span></span>

 <span data-ttu-id="3ccd0-108">下列範例會產生 CS0120：</span><span class="sxs-lookup"><span data-stu-id="3ccd0-108">The following sample generates CS0120:</span></span>

```csharp
// CS0120_1.cs
public class MyClass
{
    // Non-static field.
    public int i;
    // Non-static method.
    public void f() {}
    // Non-static property.
    int Prop
    {
        get
        {
            return 1;
        }
    }

    public static void Main()
    {
        i = 10;   // CS0120
        f();   // CS0120
        int p = Prop;   // CS0120
        // Try the following lines instead.
        // var mc = new MyClass();
        // mc.i = 10;
        // mc.f();
        // int p = mc.Prop;
    }
}
```

 <span data-ttu-id="3ccd0-109">如果從靜態方法呼叫非靜態方法，也會產生 CS0120，如下所示：</span><span class="sxs-lookup"><span data-stu-id="3ccd0-109">CS0120 will also be generated if there is a call to a non-static method from a static method, as follows:</span></span>

```csharp
// CS0120_2.cs
// CS0120 expected
using System;

public class MyClass
{
    public static void Main()  
    {  
        TestCall();   // CS0120
        // To call a non-static method from Main,
        // first create an instance of the class.
        // Use the following two lines instead:
        // var anInstanceofMyClass = new MyClass();
        // anInstanceofMyClass.TestCall();
   }

   public void TestCall()
   {
   }
}
```

 <span data-ttu-id="3ccd0-110">同樣地，除非您明確提供類別執行個體，否則靜態方法無法呼叫執行個體方法，如下所示：</span><span class="sxs-lookup"><span data-stu-id="3ccd0-110">Similarly, a static method cannot call an instance method unless you explicitly give it an instance of the class, as follows:</span></span>

```csharp
// CS0120_3.cs
using System;

public class MyClass
{
   public static void Main()
   {
      DoIt("Hello There");   // CS0120
   }
  
   private void DoIt(string sText)
   // You could also add the keyword static to the method definition.
   // private static void DoIt(string sText)
   {
      Console.WriteLine(sText);
   }
}
```
