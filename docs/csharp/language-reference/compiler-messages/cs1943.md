# <a name="compiler-error-cs1943"></a><span data-ttu-id="2dd5b-103">編譯器錯誤 CS1943</span><span class="sxs-lookup"><span data-stu-id="2dd5b-103">Compiler Error CS1943</span></span>

<span data-ttu-id="2dd5b-104">在具有來源類型 'type' 之查詢運算式內後續的 from 子句中不允許有類型 'type' 的運算式。</span><span class="sxs-lookup"><span data-stu-id="2dd5b-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="2dd5b-105">呼叫 'method' 時發生型別推斷失敗。</span><span class="sxs-lookup"><span data-stu-id="2dd5b-105">Type inference failed in the call to 'method'.</span></span>

## <a name="example"></a><span data-ttu-id="2dd5b-110">範例</span><span class="sxs-lookup"><span data-stu-id="2dd5b-110">Example</span></span>  

 <span data-ttu-id="2dd5b-111">下列程式碼會產生 CS1943：</span><span class="sxs-lookup"><span data-stu-id="2dd5b-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
