# <a name="compiler-error-cs8400"></a>編譯器錯誤 CS8400

C # 8.0 中不提供功能「功能」。 請使用語言版本 \<version> 或更高版本。

## <a name="to-correct-this-error"></a>更正這個錯誤

請將您的專案升級至錯誤中所述的至少版本。
