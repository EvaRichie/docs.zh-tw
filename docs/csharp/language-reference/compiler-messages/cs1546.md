## <a name="example"></a><span data-ttu-id="b82e1-109">範例</span><span class="sxs-lookup"><span data-stu-id="b82e1-109">Example</span></span>  

 <span data-ttu-id="b82e1-110">此程式碼範例包含編譯成 .dll 的 .cpp 檔案和使用該 .dll 的 .cs 檔案。</span><span class="sxs-lookup"><span data-stu-id="b82e1-110">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="b82e1-111">下列程式碼用於 .dll 檔案，並定義 .cs 檔案中程式碼存取的屬性。</span><span class="sxs-lookup"><span data-stu-id="b82e1-111">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
