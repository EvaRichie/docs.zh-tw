# <a name="compiler-error-cs8403"></a>編譯器錯誤 CS8403

具有 iterator 區塊的方法 ' method ' 必須是 ' async '，才能傳回 ' {IAsyncEnumerable \<T> |IAsyncEnumerator \<T> } '

## <a name="to-correct-this-error"></a>更正這個錯誤

使用修飾詞標記您的方法 `async` 。
