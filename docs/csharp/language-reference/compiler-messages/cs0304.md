# <a name="compiler-error-cs0304"></a><span data-ttu-id="315e8-103">編譯器錯誤 CS0304</span><span class="sxs-lookup"><span data-stu-id="315e8-103">Compiler Error CS0304</span></span>

<span data-ttu-id="315e8-104">無法建立變數類型 'type' 的執行個體，的因為沒有 new() 限制式。</span><span class="sxs-lookup"><span data-stu-id="315e8-104">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="315e8-105">當您實作泛型類別，而且想要使用 `new` 關鍵字來建立為型別參數 `T` 提供之任何類型的新執行個體時，您必須在類別宣告中將 [new() 限制式](../keywords/new-constraint.md)套用至 `T`，如下例所示。</span><span class="sxs-lookup"><span data-stu-id="315e8-105">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../keywords/new-constraint.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="315e8-106">`new()` 限制式強制執行型別安全的方式，是保證為 `T` 提供的任何具象類型皆有預設值：無參數的建構函式。</span><span class="sxs-lookup"><span data-stu-id="315e8-106">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="315e8-107">如果當 `T` 未指定 `new()` 限制式時，您嘗試在類別主體中使用 `new` 運算子來建立型別參數 `T` 的執行個體，就會發生 CS0304。</span><span class="sxs-lookup"><span data-stu-id="315e8-107">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="315e8-108">若用戶端程式碼嘗試使用沒有無參數建構函式的型別具現化泛型類別，則該程式碼會產生[編譯器錯誤 CS0310](./cs0310.md)。</span><span class="sxs-lookup"><span data-stu-id="315e8-108">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](./cs0310.md).</span></span>  
  
 <span data-ttu-id="315e8-109">下列範例會產生 CS0304。</span><span class="sxs-lookup"><span data-stu-id="315e8-109">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="315e8-110">類別的方法中也不允許 `new` 運算子。</span><span class="sxs-lookup"><span data-stu-id="315e8-110">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="315e8-111">若要避免此錯誤，請使用 `new()` 限制式宣告類別，如下例所示。</span><span class="sxs-lookup"><span data-stu-id="315e8-111">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
