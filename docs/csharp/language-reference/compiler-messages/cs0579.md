# <a name="compiler-error-cs0579"></a><span data-ttu-id="21bd2-103">編譯器錯誤 CS0579</span><span class="sxs-lookup"><span data-stu-id="21bd2-103">Compiler Error CS0579</span></span>
<span data-ttu-id="21bd2-104">重複的 'attribute' 屬性</span><span class="sxs-lookup"><span data-stu-id="21bd2-104">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="21bd2-105">相同的屬性只能指定一次，除非屬性在其 [AttributeUsage](../attributes/general.md) 中指定 **AllowMultiple=true**。</span><span class="sxs-lookup"><span data-stu-id="21bd2-105">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../attributes/general.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="21bd2-106">範例</span><span class="sxs-lookup"><span data-stu-id="21bd2-106">Example</span></span>  
 <span data-ttu-id="21bd2-107">下列範例會產生 CS0579。</span><span class="sxs-lookup"><span data-stu-id="21bd2-107">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
