# <a name="-fullpaths-c-compiler-options"></a>-fullpaths (C# 編譯器選項)

**-Fullpaths**選項會在列出編譯錯誤和警告時，讓編譯器指定檔案的完整路徑。  
  
## <a name="syntax"></a>語法  
  
```console  
-fullpaths  
```  
  
## <a name="remarks"></a>備註  

 根據預設，編譯造成的錯誤和警告會指定發現錯誤的檔案名稱。 **-Fullpaths**選項會讓編譯器指定檔案的完整路徑。  
  
 Visual Studio 不提供這個編譯器選項，您亦無法以程式設計方式變更。
