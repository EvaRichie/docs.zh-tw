# <a name="language-integrated-query-linq"></a><span data-ttu-id="34f27-103">Language Integrated Query (LINQ)</span><span class="sxs-lookup"><span data-stu-id="34f27-103">Language Integrated Query (LINQ)</span></span>

<span data-ttu-id="34f27-104">Language Integrated Query (LINQ) 是一組以直接將查詢功能整合至 C# 語言為基礎之技術的名稱。</span><span class="sxs-lookup"><span data-stu-id="34f27-104">Language-Integrated Query (LINQ) is the name for a set of technologies based on the integration of query capabilities directly into the C# language.</span></span> <span data-ttu-id="34f27-105">傳統上，資料查詢是以簡單的字串表示，既不會在編譯時進行類型檢查，也不支援 IntelliSense。</span><span class="sxs-lookup"><span data-stu-id="34f27-105">Traditionally, queries against data are expressed as simple strings without type checking at compile time or IntelliSense support.</span></span> <span data-ttu-id="34f27-106">此外，您還必須針對每種資料來源類型學習不同的查詢語言：SQL 資料庫、XML 文件、各種 Web 服務等等。</span><span class="sxs-lookup"><span data-stu-id="34f27-106">Furthermore, you have to learn a different query language for each type of data source: SQL databases, XML documents, various Web services, and so on.</span></span> <span data-ttu-id="34f27-107">透過 LINQ，查詢會是第一級語言建構，和類別、方法及事件相同。</span><span class="sxs-lookup"><span data-stu-id="34f27-107">With LINQ, a query is a first-class language construct, just like classes, methods, events.</span></span> <span data-ttu-id="34f27-108">您可以使用語言關鍵字和熟悉的運算子，針對強型別的物件集合撰寫查詢。</span><span class="sxs-lookup"><span data-stu-id="34f27-108">You write queries against strongly typed collections of objects by using language keywords and familiar operators.</span></span> <span data-ttu-id="34f27-109">LINQ 技術系列會針對物件 (LINQ to Object)、關聯式資料庫 (LINQ to SQL) 與 XML (LINQ to XML)，提供一致的查詢體驗。</span><span class="sxs-lookup"><span data-stu-id="34f27-109">The LINQ family of technologies provides a consistent query experience for objects (LINQ to Objects), relational databases (LINQ to SQL), and XML (LINQ to XML).</span></span>

<span data-ttu-id="34f27-110">對於撰寫查詢的開發人員來說，LINQ 最明顯的「語言整合」部分就是查詢運算式。</span><span class="sxs-lookup"><span data-stu-id="34f27-110">For a developer who writes queries, the most visible "language-integrated" part of LINQ is the query expression.</span></span> <span data-ttu-id="34f27-111">查詢運算式是以宣告式「查詢語法」\*\* 撰寫。</span><span class="sxs-lookup"><span data-stu-id="34f27-111">Query expressions are written in a declarative *query syntax*.</span></span> <span data-ttu-id="34f27-112">透過使用查詢語法，您就可以利用最少的程式碼，針對資料來源執行篩選、排序及分組作業。</span><span class="sxs-lookup"><span data-stu-id="34f27-112">By using query syntax, you can perform filtering, ordering, and grouping operations on data sources with a minimum of code.</span></span> <span data-ttu-id="34f27-113">您可以使用相同的基本查詢運算式模式來查詢和轉換 SQL 資料庫中的資料、ADO.NET 資料集、XML 檔和資料流程，以及 .NET 集合。</span><span class="sxs-lookup"><span data-stu-id="34f27-113">You use the same basic query expression patterns to query and transform data in SQL databases, ADO.NET Datasets, XML documents and streams, and .NET collections.</span></span>

<span data-ttu-id="34f27-114">您可以使用 C# 針對下列項目撰寫 LINQ 查詢：SQL Server 資料庫、XML 文件、ADO.NET 資料集，以及支援 <xref:System.Collections.IEnumerable> 或泛型 <xref:System.Collections.Generic.IEnumerable%601> 介面的任何物件集合。</span><span class="sxs-lookup"><span data-stu-id="34f27-114">You can write LINQ queries in C# for SQL Server databases, XML documents, ADO.NET Datasets, and any collection of objects that supports <xref:System.Collections.IEnumerable> or the generic <xref:System.Collections.Generic.IEnumerable%601> interface.</span></span> <span data-ttu-id="34f27-115">也有協力廠商針對許多 Web 服務和其他資料庫實作提供 LINQ 支援。</span><span class="sxs-lookup"><span data-stu-id="34f27-115">LINQ support is also provided by third parties for many Web services and other database implementations.</span></span>

## <a name="query-expression-overview"></a><span data-ttu-id="34f27-120">查詢運算式概觀</span><span class="sxs-lookup"><span data-stu-id="34f27-120">Query expression overview</span></span>

- <span data-ttu-id="34f27-121">查詢運算式可以用來查詢並轉換來自任何已啟用 LINQ 之資料來源的資料。</span><span class="sxs-lookup"><span data-stu-id="34f27-121">Query expressions can be used to query and to transform data from any LINQ-enabled data source.</span></span> <span data-ttu-id="34f27-122">例如，單一查詢可以從 SQL 資料庫擷取資料，並產生 XML 資料流做為輸出。</span><span class="sxs-lookup"><span data-stu-id="34f27-122">For example, a single query can retrieve data from a SQL database, and produce an XML stream as output.</span></span>
- <span data-ttu-id="34f27-123">查詢運算式很容易學習，因為其中使用許多熟悉的 C# 語言建構。</span><span class="sxs-lookup"><span data-stu-id="34f27-123">Query expressions are easy to master because they use many familiar C# language constructs.</span></span>
- <span data-ttu-id="34f27-124">查詢運算式中的變數皆為強型別，不過您在很多情況下並不需要明確提供型別，因為編譯器能夠自行推斷它。</span><span class="sxs-lookup"><span data-stu-id="34f27-124">The variables in a query expression are all strongly typed, although in many cases you do not have to provide the type explicitly because the compiler can infer it.</span></span> <span data-ttu-id="34f27-125">如需詳細資訊，請參閱[LINQ 查詢作業中的類型關聯](type-relationships-in-linq-query-operations.md)性。</span><span class="sxs-lookup"><span data-stu-id="34f27-125">For more information, see [Type relationships in LINQ query operations](type-relationships-in-linq-query-operations.md).</span></span>

- <span data-ttu-id="34f27-128">在編譯期間，查詢運算式會根據 C# 規格中提出的規則，轉換成「標準查詢運算子」方法呼叫。</span><span class="sxs-lookup"><span data-stu-id="34f27-128">At compile time, query expressions are converted to Standard Query Operator method calls according to the rules set forth in the C# specification.</span></span> <span data-ttu-id="34f27-129">所有可使用查詢語法表示的查詢，也都可以利用方法語法來表示。</span><span class="sxs-lookup"><span data-stu-id="34f27-129">Any query that can be expressed by using query syntax can also be expressed by using method syntax.</span></span> <span data-ttu-id="34f27-130">不過，在大多數情況下，查詢語法較容易閱讀且更簡潔。</span><span class="sxs-lookup"><span data-stu-id="34f27-130">However, in most cases query syntax is more readable and concise.</span></span> <span data-ttu-id="34f27-131">如需詳細資訊，請參閱 [C# 語言規格](~/_csharplang/spec/expressions.md#query-expressions)和[標準查詢運算子概觀](standard-query-operators-overview.md)。</span><span class="sxs-lookup"><span data-stu-id="34f27-131">For more information, see [C# language specification](~/_csharplang/spec/expressions.md#query-expressions) and [Standard query operators overview](standard-query-operators-overview.md).</span></span>

- <span data-ttu-id="34f27-138">根據查詢所套用的型別，可將查詢運算式編譯為運算式樹狀結構或委派。</span><span class="sxs-lookup"><span data-stu-id="34f27-138">Query expressions can be compiled to expression trees or to delegates, depending on the type that the query is applied to.</span></span> <span data-ttu-id="34f27-139"><xref:System.Collections.Generic.IEnumerable%601> 查詢會編譯成委派。</span><span class="sxs-lookup"><span data-stu-id="34f27-139"><xref:System.Collections.Generic.IEnumerable%601> queries are compiled to delegates.</span></span> <span data-ttu-id="34f27-140"><xref:System.Linq.IQueryable> 和 <xref:System.Linq.IQueryable%601> 查詢會編譯成運算式樹狀架構。</span><span class="sxs-lookup"><span data-stu-id="34f27-140"><xref:System.Linq.IQueryable> and <xref:System.Linq.IQueryable%601> queries are compiled to expression trees.</span></span> <span data-ttu-id="34f27-141">如需詳細資訊，請參閱[運算式樹狀結構](../../../expression-trees.md)。</span><span class="sxs-lookup"><span data-stu-id="34f27-141">For more information, see [Expression trees](../../../expression-trees.md).</span></span>
