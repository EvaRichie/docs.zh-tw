# <a name="using-namespaces-c-programming-guide"></a>使用命名空間 (C# 程式設計手冊)

- `N1` 命名空間是全域命名空間的成員。 其完整名稱是 `N1`。  
  
- `N2` 命名空間是 `N1` 的成員。 其完整名稱是 `N1.N2`。  
  
- `C1` 類別是 `N1` 的成員。 其完整名稱是 `N1.C1`。
