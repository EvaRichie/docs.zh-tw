# <a name="code-c-programming-guide"></a>\<code>（C # 程式設計手冊）

## <a name="syntax"></a>語法

```xml
<code>content</code>
```

## <a name="parameters"></a>參數

- `content`

  您要標記為程式碼的文字。

## <a name="remarks"></a>備註

`<code>`標記是用來表示多行程式碼。 用 [\<c>](./code-inline.md) 來表示描述中的單行文字應標記為程式碼。

使用[-doc](../../language-reference/compiler-options/doc-compiler-option.md)進行編譯，以處理檔案的檔批註。

## <a name="example"></a>範例

如需 [\<example>](./example.md) 如何使用標記的範例，請參閱文章 `<code>` 。
