- <span data-ttu-id="e5375-114">Azure DevOps Services 和 Team Foundation Server 包含第一個和協力廠商擴充功能的強大且成長的生態系統，可讓您輕鬆地為微服務建立 CI、組建、測試、傳遞和發行管理管線。</span><span class="sxs-lookup"><span data-stu-id="e5375-114">Azure DevOps Services and Team Foundation Server include a powerful and growing ecosystem of first and third-party extensions with which you easily can construct a CI, build, test, delivery, and release management pipeline for microservices.</span></span>
