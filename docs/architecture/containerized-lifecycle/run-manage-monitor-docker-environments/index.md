# <a name="run-manage-and-monitor-docker-production-environments"></a>執行、管理和監視 Docker 生產環境

願景：企業應用程式需要以高可用性和高延展性執行；IT 作業需要能夠管理與監視環境和應用程式本身。

容器化 Docker 應用程式生命週期中的最後一個部分集中於如何在可擴充、高可用性 (HA) 生產環境中執行、管理和監視應用程式。

在生產環境 (基礎結構架構和平台技術) 中執行容器化應用程式的方式十分相關，且以本電子書第 1 章中所述的所選架構和開發平台為基礎。

本章檢查來自 Microsoft 和其他廠商的特定產品及技術，讓您有效地執行可延展的 HA 分散式應用程式，以及從 IT 觀點管理和監視它們的方式。
