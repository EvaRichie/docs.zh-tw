<span data-ttu-id="4bbe7-147">您可以將現有的應用程式轉換為 Cloud-Optimized 應用程式 (，而不需要重新架構或撰寫) 的程式碼，即可獲得下列優點：</span><span class="sxs-lookup"><span data-stu-id="4bbe7-147">You can get the following benefits by converting an existing application to a Cloud-Optimized application (without rearchitecting or coding):</span></span>
