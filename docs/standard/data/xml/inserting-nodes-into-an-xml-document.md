# <a name="inserting-nodes-into-an-xml-document"></a>將節點插入 XML 文件
您可藉由插入新節點來修改文件。 節點可以是新建立的、存在於同一文件中，或是從其他文件匯入的。 如需建立新節點的詳細資訊，請參閱[在 DOM 中建立新節點](create-new-nodes-in-the-dom.md)。 如需複製現有節點的詳細資訊，請參閱[複製現有節點](copy-existing-nodes.md)、[將現有節點從一個文件複製到另一個文件](copying-existing-nodes-from-one-document-to-another.md)或[複製文件片段](copying-document-fragments.md)。
