# <a name="extending-xslt-style-sheets"></a>延伸 XSLT 樣式表

本節說明擴充 XSLT 功能的不同方法。 您可使用 <xref:System.Xml.Xsl.XsltArgumentList> 類別，加入擴充物件或參數。 然後可從樣式表呼叫該擴充物件或參數。 此外，您還可使用 `msxsl:script` 項目將指令碼區塊嵌入樣式表中。
