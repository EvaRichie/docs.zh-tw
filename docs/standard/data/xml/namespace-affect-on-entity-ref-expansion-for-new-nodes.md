# <a name="namespace-affect-on-entity-reference-expansion-for-new-nodes-containing-elements-and-attributes"></a><span data-ttu-id="df1f9-102">命名空間對包含項目和屬性的新節點之實體參考擴充的影響</span><span class="sxs-lookup"><span data-stu-id="df1f9-102">Namespace Affect on Entity Reference Expansion for New Nodes Containing Elements and Attributes</span></span>
<span data-ttu-id="df1f9-103">因為實體宣告的內容幾乎可包含任何內容，所以內容所包含的項目可能包括 `<!ENTITY aname "<elem>test</elem>">`。</span><span class="sxs-lookup"><span data-stu-id="df1f9-103">Because the content of an entity declaration can contain almost anything, there is a possibility that the content could contain an element like `<!ENTITY aname "<elem>test</elem>">`.</span></span>  
  
 <span data-ttu-id="df1f9-104">剖析 XML 時，`&aname;` 在剖析期間不會以其取代內容進行擴充。</span><span class="sxs-lookup"><span data-stu-id="df1f9-104">When the XML is parsed, `&aname;` is not expanded with its replacement content at parse time.</span></span> <span data-ttu-id="df1f9-105">因為項目的命名空間解析不會發生，所以 XML 不會擴充，除非節點置於文件中。</span><span class="sxs-lookup"><span data-stu-id="df1f9-105">The expansion of the XML is not done because resolution of the namespace for the element cannot occur until the node is placed in the document.</span></span> <span data-ttu-id="df1f9-106">此時才會知道範圍內的命名空間是什麼。</span><span class="sxs-lookup"><span data-stu-id="df1f9-106">Until that time, there is no knowledge of what namespace is in scope.</span></span> <span data-ttu-id="df1f9-107">當節點置於文件中時，會進行命名空間解析，而且產生的實體內容會剖析進它的適當節點。</span><span class="sxs-lookup"><span data-stu-id="df1f9-107">When the node is put into the document, then the namespace resolution occurs, and the resulting entity content is parsed into its appropriate nodes.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="df1f9-108">一旦擴充發生在新建的實體參考節點上，它就不會再發生了。</span><span class="sxs-lookup"><span data-stu-id="df1f9-108">Once the expansion occurs on a newly created entity reference node, it never reoccurs.</span></span> <span data-ttu-id="df1f9-109">因此，項目的取代文字中所使用的命名空間會在設定父節點時產生繫結。</span><span class="sxs-lookup"><span data-stu-id="df1f9-109">Therefore, the namespaces used in the replacement text for the element are bound at the time that the parent node is set.</span></span> <span data-ttu-id="df1f9-110">然而，在您移除命名空間並將其插入其他地方，或者在以 **CloneNode** 方法複製的實體參考節點上時，會為現有實體參考節點變更命名空間。</span><span class="sxs-lookup"><span data-stu-id="df1f9-110">However, the namespace can be changed for existing entity reference nodes when you remove and insert them somewhere else, or on entity reference nodes that are cloned with the **CloneNode** method.</span></span>
