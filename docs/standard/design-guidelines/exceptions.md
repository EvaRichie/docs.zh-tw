# <a name="design-guidelines-for-exceptions"></a>例外狀況的設計方針

例外狀況處理具有許多優於以傳回值為基礎之錯誤報表的優點。 良好的架構設計可協助應用程式開發人員實現例外狀況的優點。 本節將討論例外狀況的優點，並提供有效使用這些功能的指導方針。
