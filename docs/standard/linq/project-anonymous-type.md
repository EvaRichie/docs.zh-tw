<span data-ttu-id="c223b-105">在某些情況下，您可能會想要將查詢投射至新的型別，但是查詢會是您只用于新型別的專案。</span><span class="sxs-lookup"><span data-stu-id="c223b-105">In some cases, you might want to project a query to a new type, but the query would be your only use for the new type.</span></span> <span data-ttu-id="c223b-106">您可以投影至匿名型別，而不是建立型別。</span><span class="sxs-lookup"><span data-stu-id="c223b-106">Rather than create the type, you can project to an anonymous type.</span></span> <span data-ttu-id="c223b-107">匿名型別提供一個方便的方式，在物件中封裝一組唯讀屬性，而不需要先明確地定義型別。</span><span class="sxs-lookup"><span data-stu-id="c223b-107">Anonymous types provide a convenient way to encapsulate a set of read-only properties in an object without having to explicitly define a type first.</span></span> <span data-ttu-id="c223b-108">如果您撰寫的查詢會在子句中建立匿名型別的物件 `select` ，則查詢會傳回 <xref:System.Collections.IEnumerable> 型別的。</span><span class="sxs-lookup"><span data-stu-id="c223b-108">If you write a query that creates an object of an anonymous type in the `select` clause, the query returns an <xref:System.Collections.IEnumerable> of the type.</span></span>

## <a name="example-project-an-anonymous-type-by-creating-objects-in-the-select-clause"></a><span data-ttu-id="c223b-115">範例：在子句中建立物件來投影匿名型別 `select`</span><span class="sxs-lookup"><span data-stu-id="c223b-115">Example: Project an anonymous type by creating objects in the `select` clause</span></span>

<span data-ttu-id="c223b-116">在這個範例中，`select` 子句會規劃匿名型別。</span><span class="sxs-lookup"><span data-stu-id="c223b-116">In this example, the `select` clause projects an anonymous type.</span></span> <span data-ttu-id="c223b-117">然後，此範例會使用 `var` 來建立 `IEnumerable` 物件。</span><span class="sxs-lookup"><span data-stu-id="c223b-117">The example then uses `var` to create the `IEnumerable` object.</span></span> <span data-ttu-id="c223b-118">在 `foreach` 迴圈中，反覆運算變數會變成在查詢運算式中建立之匿名型別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="c223b-118">Within the `foreach` loop, the iteration variable becomes an instance of the anonymous type created in the query expression.</span></span>

<span data-ttu-id="c223b-119">此範例會使用 XML [檔範例 xml 檔：客戶和訂單](sample-xml-file-customers-orders.md)。</span><span class="sxs-lookup"><span data-stu-id="c223b-119">This example uses XML document [Sample XML file: Customers and orders](sample-xml-file-customers-orders.md).</span></span>

```csharp
XElement custOrd = XElement.Load("CustomersOrders.xml");
var custList =
    from el in custOrd.Element("Customers").Elements("Customer")
    select new {
        CustomerID = (string)el.Attribute("CustomerID"),
        CompanyName = (string)el.Element("CompanyName"),
        ContactName = (string)el.Element("ContactName")
    };
foreach (var cust in custList)
    Console.WriteLine("{0}:{1}:{2}", cust.CustomerID, cust.CompanyName, cust.ContactName);
```

```vb
Dim custOrd As XElement = XElement.Load("CustomersOrders.xml")
Dim custList = _
    From el In custOrd.<Customers>.<Customer> _
    Select New With { _
        .CustomerID = el.@<CustomerID>, _
        .CompanyName = el.<CompanyName>.Value, _
        .ContactName = el.<ContactName>.Value _
    }
For Each cust In custList
    Console.WriteLine("{0}:{1}:{2}", cust.CustomerID, cust.CompanyName, cust.ContactName)
Next
```
