 <span data-ttu-id="26609-105">如果您不熟悉將屬性套用至 Managed 程式碼的工作，請參閱[使用屬性擴充中繼資料](../attributes/index.md)。</span><span class="sxs-lookup"><span data-stu-id="26609-105">If you are unfamiliar with the task of applying attributes to managed code, see [Extending Metadata Using Attributes](../attributes/index.md).</span></span> <span data-ttu-id="26609-106">就像其他自訂屬性一樣，您可以將 Interop 專屬屬性 (attribute) 套用至類型、方法、屬性 (property)、參數、欄位和其他成員。</span><span class="sxs-lookup"><span data-stu-id="26609-106">Like other custom attributes, you can apply interop-specific attributes to types, methods, properties, parameters, fields, and other members.</span></span>
