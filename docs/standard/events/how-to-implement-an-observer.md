# <a name="how-to-implement-an-observer"></a>如何：實作觀察器

觀察者設計模式需要觀察者和提供者之間的分區，其中觀察者會註冊通知，而提供者會監視資料並將通知傳送到一個或多個觀察者。 本主題討論如何建立觀察者。 相關主題為[操作說明：實作提供者](how-to-implement-a-provider.md)，討論如何建立提供者。
