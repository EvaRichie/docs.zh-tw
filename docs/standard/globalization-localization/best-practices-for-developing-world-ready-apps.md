4. <span data-ttu-id="7dd18-116">啟用您的應用程式使用 <xref:System.Text> 命名空間中的編碼類別來回讀取和寫入各種編碼方式的資料。</span><span class="sxs-lookup"><span data-stu-id="7dd18-116">Enable your application to read and write data to and from a variety of encodings by using the encoding classes in the <xref:System.Text> namespace.</span></span> <span data-ttu-id="7dd18-117">請勿假設 ASCII 資料。</span><span class="sxs-lookup"><span data-stu-id="7dd18-117">Do not assume ASCII data.</span></span> <span data-ttu-id="7dd18-118">請假設在使用中輸入文字的地方提供國際字元。</span><span class="sxs-lookup"><span data-stu-id="7dd18-118">Assume that international characters will be supplied anywhere a user can enter text.</span></span> <span data-ttu-id="7dd18-119">例如，應用程式應該會接受在伺服器名稱、目錄、檔名和 URL 中使用國際字元。</span><span class="sxs-lookup"><span data-stu-id="7dd18-119">For example, the application should accept international characters in server names, directories, file names, user names, and URLs.</span></span>

5. <span data-ttu-id="7dd18-120">在使用 <xref:System.Text.UTF8Encoding> 類別時，基於安全理由，請使用這個類別所提供的錯誤偵測功能。</span><span class="sxs-lookup"><span data-stu-id="7dd18-120">When using the <xref:System.Text.UTF8Encoding> class, for security reasons, use the error detection feature offered by this class.</span></span> <span data-ttu-id="7dd18-121">若要開啟錯誤偵測功能，請使用採用 `throwOnInvalidBytes` 參數的建構函式，並將此參數的值設定為 `true`，以建立此類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="7dd18-121">To turn on the error detection feature, create an instance of the class using the constructor that takes a `throwOnInvalidBytes` parameter and set the value of this parameter to `true`.</span></span>

6. <span data-ttu-id="7dd18-122">如果可能的話，請將字串當做完整的字串處理，而不是連續的個別字串。</span><span class="sxs-lookup"><span data-stu-id="7dd18-122">Whenever possible, handle strings as entire strings instead of as a series of individual characters.</span></span> <span data-ttu-id="7dd18-123">在排序或搜尋子字串時，這點特別重要。</span><span class="sxs-lookup"><span data-stu-id="7dd18-123">This is especially important when sorting or searching for substrings.</span></span> <span data-ttu-id="7dd18-124">它將可避免發生剖析組合字元的相關問題。</span><span class="sxs-lookup"><span data-stu-id="7dd18-124">This will prevent problems associated with parsing combined characters.</span></span> <span data-ttu-id="7dd18-125">您也可利用 <xref:System.Globalization.StringInfo?displayProperty=nameWithType> 類別，便無須使用單一字元，而是以單位來使用文字。</span><span class="sxs-lookup"><span data-stu-id="7dd18-125">You can also work with units of text rather than single characters by using the <xref:System.Globalization.StringInfo?displayProperty=nameWithType> class.</span></span>

7. <span data-ttu-id="7dd18-126">使用 <xref:System.Drawing> 命名空間提供類別來顯示文字。</span><span class="sxs-lookup"><span data-stu-id="7dd18-126">Display text using the classes provided by the <xref:System.Drawing> namespace.</span></span>

8. <span data-ttu-id="7dd18-127">為了作業系統的一致性，請勿使用使用者設定來覆寫 <xref:System.Globalization.CultureInfo>。</span><span class="sxs-lookup"><span data-stu-id="7dd18-127">For consistency across operating systems, do not allow user settings to override <xref:System.Globalization.CultureInfo>.</span></span> <span data-ttu-id="7dd18-128">請使用採用 `CultureInfo` 參數的 `useUserOverride` 建構函式，並將此參數設定為 `false`。</span><span class="sxs-lookup"><span data-stu-id="7dd18-128">Use the `CultureInfo` constructor that accepts a `useUserOverride` parameter and set it to `false`.</span></span>

9. <span data-ttu-id="7dd18-129">使用國際資料，在國際作業系統版本上測試您的應用程式功能。</span><span class="sxs-lookup"><span data-stu-id="7dd18-129">Test your application functionality on international operating system versions, using international data.</span></span>

10. <span data-ttu-id="7dd18-130">如果安全性決策是根據字串比較或大小寫變更作業的結果而定，則請使用不區分文化特性 (Culture) 的字串作業。</span><span class="sxs-lookup"><span data-stu-id="7dd18-130">If a security decision is based on the result of a string comparison or case change operation, use a culture-insensitive string operation.</span></span> <span data-ttu-id="7dd18-131">這種作法可以確保結果不會受 `CultureInfo.CurrentCulture` 的值所影響。</span><span class="sxs-lookup"><span data-stu-id="7dd18-131">This practice ensures that the result is not affected by the value of `CultureInfo.CurrentCulture`.</span></span> <span data-ttu-id="7dd18-132">如需示範區分文化特性的字串比較如何產生不一致結果的範例，請參閱[使用字串之最佳作法](../base-types/best-practices-strings.md)的「[使用目前文化特性的字串比較](../base-types/best-practices-strings.md#string-comparisons-that-use-the-current-culture)」一節。</span><span class="sxs-lookup"><span data-stu-id="7dd18-132">See the ["String Comparisons that Use the Current Culture"](../base-types/best-practices-strings.md#string-comparisons-that-use-the-current-culture) section of [Best Practices for Using Strings](../base-types/best-practices-strings.md) for an example that demonstrates how culture-sensitive string comparisons can produce inconsistent results.</span></span>

## <a name="localization-best-practices"></a><span data-ttu-id="7dd18-133">當地語系化最佳做法</span><span class="sxs-lookup"><span data-stu-id="7dd18-133">Localization best practices</span></span>

1. <span data-ttu-id="7dd18-134">將所有可當地語系化資源移到個別僅含資源的 DLL。</span><span class="sxs-lookup"><span data-stu-id="7dd18-134">Move all localizable resources to separate resource-only DLLs.</span></span> <span data-ttu-id="7dd18-135">可當地語系化的資源包括使用者介面項目，例如字串、錯誤訊息、對話方塊、功能表和內嵌物件資源。</span><span class="sxs-lookup"><span data-stu-id="7dd18-135">Localizable resources include user interface elements, such as strings, error messages, dialog boxes, menus, and embedded object resources.</span></span>

2. <span data-ttu-id="7dd18-136">請勿將字串或使用者介面資源寫在程式中。</span><span class="sxs-lookup"><span data-stu-id="7dd18-136">Do not hardcode strings or user interface resources.</span></span>

3. <span data-ttu-id="7dd18-137">請勿將不可當地語系化的資源放入僅含資源的 DLL。</span><span class="sxs-lookup"><span data-stu-id="7dd18-137">Do not put nonlocalizable resources into the resource-only DLLs.</span></span> <span data-ttu-id="7dd18-138">它將造成轉譯器的混淆。</span><span class="sxs-lookup"><span data-stu-id="7dd18-138">This causes confusion for translators.</span></span>

4. <span data-ttu-id="7dd18-139">請勿使用在執行階段建置和來自連結字詞的複合字串。</span><span class="sxs-lookup"><span data-stu-id="7dd18-139">Do not use composite strings that are built at run time from concatenated phrases.</span></span> <span data-ttu-id="7dd18-140">複合字串很難進行當地語系化，因為它們通常會使用不適用於所有語言的英文文法順序。</span><span class="sxs-lookup"><span data-stu-id="7dd18-140">Composite strings are difficult to localize because they often assume an English grammatical order that does not apply to all languages.</span></span>

5. <span data-ttu-id="7dd18-141">避免模糊的建構，例如 "Empty Folder"，其中的字串將依據字串元件擔任的文法角色，而進行不同的轉譯。</span><span class="sxs-lookup"><span data-stu-id="7dd18-141">Avoid ambiguous constructs such as "Empty Folder" where the strings can be translated differently depending on the grammatical roles of the string components.</span></span> <span data-ttu-id="7dd18-142">例如，"empty" 可以是動詞或形容詞，在某些語言中會有不同的轉譯結果，例如義大利文或法文。</span><span class="sxs-lookup"><span data-stu-id="7dd18-142">For example, "empty" can be either a verb or an adjective, which can lead to different translations in languages such as Italian or French.</span></span>

6. <span data-ttu-id="7dd18-143">避免在您的應用程式中使用含有文字的影像和圖示。</span><span class="sxs-lookup"><span data-stu-id="7dd18-143">Avoid using images and icons that contain text in your application.</span></span> <span data-ttu-id="7dd18-144">它們進行當地語系化的成本過高。</span><span class="sxs-lookup"><span data-stu-id="7dd18-144">They are expensive to localize.</span></span>

7. <span data-ttu-id="7dd18-145">請保留充分的空間，以便在應用程式介面中擴展字串長度。</span><span class="sxs-lookup"><span data-stu-id="7dd18-145">Allow plenty of room for the length of strings to expand in the user interface.</span></span> <span data-ttu-id="7dd18-146">在某些語言中，字詞所需要的空間可能比在其他語言多百分之 50-75。</span><span class="sxs-lookup"><span data-stu-id="7dd18-146">In some languages, phrases can require 50-75 percent more space than they need in other languages.</span></span>

8. <span data-ttu-id="7dd18-147">使用 <xref:System.Resources.ResourceManager?displayProperty=nameWithType> 類別來根據文化特性擷取資源。</span><span class="sxs-lookup"><span data-stu-id="7dd18-147">Use the <xref:System.Resources.ResourceManager?displayProperty=nameWithType> class to retrieve resources based on culture.</span></span>

9. <span data-ttu-id="7dd18-148">使用 [Visual Studio](https://visualstudio.microsoft.com/vs/?utm_medium=microsoft&utm_source=docs.microsoft.com&utm_campaign=inline+link) 建立 [Windows Forms] 對話方塊，如此就能使用 [Windows Forms 資源編輯器 (Winres.exe)](../../framework/tools/winres-exe-windows-forms-resource-editor.md) 將對話方塊當地語系化。</span><span class="sxs-lookup"><span data-stu-id="7dd18-148">Use [Visual Studio](https://visualstudio.microsoft.com/vs/?utm_medium=microsoft&utm_source=docs.microsoft.com&utm_campaign=inline+link) to create Windows Forms dialog boxes so they can be localized using the [Windows Forms Resource Editor (Winres.exe)](../../framework/tools/winres-exe-windows-forms-resource-editor.md).</span></span> <span data-ttu-id="7dd18-149">請不要以手動方式編碼 Windows Form 對話方塊。</span><span class="sxs-lookup"><span data-stu-id="7dd18-149">Do not code Windows Forms dialog boxes by hand.</span></span>

10. <span data-ttu-id="7dd18-150">進行專業當地語系化 (轉譯)。</span><span class="sxs-lookup"><span data-stu-id="7dd18-150">Arrange for professional localization (translation).</span></span>

11. <span data-ttu-id="7dd18-151">如需建立和當地語系化資源的完整描述，請參閱[應用程式中的資源](../../framework/resources/index.md)。</span><span class="sxs-lookup"><span data-stu-id="7dd18-151">For a complete description of creating and localizing resources, see [Resources in Applications](../../framework/resources/index.md).</span></span>
