 <span data-ttu-id="f1351-116">當您建立 XML Web 服務，可同時使用方法上的兩組屬性。</span><span class="sxs-lookup"><span data-stu-id="f1351-116">When you're creating an XML Web service, you can use both sets of attributes on the methods.</span></span> <span data-ttu-id="f1351-117">在下列程式碼範例中，名為 的類別包含兩個 XML Web 服務方法： 及 。</span><span class="sxs-lookup"><span data-stu-id="f1351-117">In the following code example, the class named `MyService` contains two XML Web service methods, `MyLiteralMethod` and `MyEncodedMethod`.</span></span> <span data-ttu-id="f1351-118">兩種方法都執行相同功能：傳回 `Order` 類別的執行個體。</span><span class="sxs-lookup"><span data-stu-id="f1351-118">Both methods perform the same function: returning an instance of the `Order` class.</span></span> <span data-ttu-id="f1351-119">在 `Order` 類別中，<xref:System.Xml.Serialization.XmlTypeAttribute> 以及 <xref:System.Xml.Serialization.SoapTypeAttribute> 屬性 (attribute) 都套用至 `OrderID` 欄位，而且兩個屬性 (attribute) 的 `ElementName` 屬性 (property) 都設為不同值。</span><span class="sxs-lookup"><span data-stu-id="f1351-119">In the `Order` class, the <xref:System.Xml.Serialization.XmlTypeAttribute> and the <xref:System.Xml.Serialization.SoapTypeAttribute> attributes are both applied to the `OrderID` field, and both attributes have their `ElementName` property set to different values.</span></span>

```csharp  
<%@ WebService Language="C#" Class="MyService" %>  
using System;  
using System.Web.Services;  
using System.Web.Services.Protocols;  
using System.Xml.Serialization;  
public class Order {  
    // Both types of attributes can be applied. Depending on which type  
    // the method used, either one will affect the call.  
    [SoapElement(ElementName = "EncodedOrderID")]  
    [XmlElement(ElementName = "LiteralOrderID")]  
    public String OrderID;  
}  
public class MyService {  
    [WebMethod][SoapDocumentMethod]  
    public Order MyLiteralMethod(){  
        Order myOrder = new Order();  
        return myOrder;  
    }  
    [WebMethod][SoapRpcMethod]  
    public Order MyEncodedMethod(){  
        Order myOrder = new Order();  
        return myOrder;  
    }  
}  
```
