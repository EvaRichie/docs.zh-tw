<span data-ttu-id="dbd78-339">下列範例說明如何在程式碼中註冊篩選規則：</span><span class="sxs-lookup"><span data-stu-id="dbd78-339">The following example shows how to register filter rules in code:</span></span>

```csharp
class Program
{
    static Task Main(string[] args) =>
        CreateHostBuilder(args).Build().RunAsync();

    static IHostBuilder CreateHostBuilder(string[] args) =>
        Host.CreateDefaultBuilder(args)
            .ConfigureLogging(logging =>
               logging.AddFilter("System", LogLevel.Debug)
                  .AddFilter<DebugLoggerProvider>("Microsoft", LogLevel.Information)
                  .AddFilter<ConsoleLoggerProvider>("Microsoft", LogLevel.Trace));
}
```

<span data-ttu-id="dbd78-340">`logging.AddFilter("System", LogLevel.Debug)` 指定 `System` 類別目錄和記錄層級 `Debug` 。</span><span class="sxs-lookup"><span data-stu-id="dbd78-340">`logging.AddFilter("System", LogLevel.Debug)` specifies the `System` category and log level `Debug`.</span></span> <span data-ttu-id="dbd78-341">篩選會套用至所有提供者，因為未設定特定的提供者。</span><span class="sxs-lookup"><span data-stu-id="dbd78-341">The filter is applied to all providers because a specific provider was not configured.</span></span>

<span data-ttu-id="dbd78-342">`AddFilter<DebugLoggerProvider>("Microsoft", LogLevel.Information)` 指定：</span><span class="sxs-lookup"><span data-stu-id="dbd78-342">`AddFilter<DebugLoggerProvider>("Microsoft", LogLevel.Information)` specifies:</span></span>

- <span data-ttu-id="dbd78-343">`Debug`記錄提供者。</span><span class="sxs-lookup"><span data-stu-id="dbd78-343">The `Debug` logging provider.</span></span>
- <span data-ttu-id="dbd78-344">記錄層級 `Information` 和更新版本。</span><span class="sxs-lookup"><span data-stu-id="dbd78-344">Log level `Information` and higher.</span></span>
- <span data-ttu-id="dbd78-345">開頭為的所有分類 `"Microsoft"` 。</span><span class="sxs-lookup"><span data-stu-id="dbd78-345">All categories starting with `"Microsoft"`.</span></span>
