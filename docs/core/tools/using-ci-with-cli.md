```powershell

$ProgressPreference="SilentlyContinue"

# $LocalDotnet is the path to the locally-installed SDK to ensure the
#   correct version of the tools are executed.

$LocalDotnet = "$InstallDir/dotnet"
```

<span data-ttu-id="d283d-145">您可以在指令碼結尾處提供建置程序的實作。</span><span class="sxs-lookup"><span data-stu-id="d283d-145">You provide the implementation for your build process at the end of the script.</span></span> <span data-ttu-id="d283d-146">指令碼會取得工具，然後執行您的建置程序。</span><span class="sxs-lookup"><span data-stu-id="d283d-146">The script acquires the tools and then executes your build process.</span></span> <span data-ttu-id="d283d-147">針對 UNIX 電腦，以下 Bash 指令碼會以類似的方式執行 PowerShell 指令碼中描述的動作：</span><span class="sxs-lookup"><span data-stu-id="d283d-147">For UNIX machines, the following bash script performs the actions described in the PowerShell script in a similar manner:</span></span>
