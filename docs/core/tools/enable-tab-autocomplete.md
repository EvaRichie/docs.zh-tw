<span data-ttu-id="fd2da-128">將下列程式碼新增至您的設定檔：</span><span class="sxs-lookup"><span data-stu-id="fd2da-128">Add the following code to your profile:</span></span>

# PowerShell parameter completion shim for the dotnet CLI
Register-ArgumentCompleter -Native -CommandName dotnet -ScriptBlock {
     param($commandName, $wordToComplete, $cursorPosition)
         dotnet complete --position $cursorPosition "$wordToComplete" | ForEach-Object {

         }
 }

## <a name="bash"></a><span data-ttu-id="fd2da-129">Bash</span><span class="sxs-lookup"><span data-stu-id="fd2da-129">bash</span></span>

<span data-ttu-id="fd2da-130">若要將 tab 鍵自動完成新增至 .NET Core CLI 的 **bash** 殼層，請將下列程式碼新增至您的 `.bashrc` 檔案：</span><span class="sxs-lookup"><span data-stu-id="fd2da-130">To add tab completion to your **bash** shell for the .NET Core CLI, add the following code to your `.bashrc` file:</span></span>

```bash
# bash parameter completion for the dotnet CLI

_dotnet_bash_complete()
{
  local word=${COMP_WORDS[COMP_CWORD]}

  local completions
  completions="$(dotnet complete --position "${COMP_POINT}" "${COMP_LINE}" 2>/dev/null)"
  if [ $? -ne 0 ]; then
    completions=""
  fi

  COMPREPLY=( $(compgen -W "$completions" -- "$word") )
}

complete -f -F _dotnet_bash_complete dotnet
```
