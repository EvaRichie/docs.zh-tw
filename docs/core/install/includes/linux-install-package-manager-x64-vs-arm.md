> [!IMPORTANT]
> 只有 **x64** 架構支援安裝套件管理員。 其他架構（例如 **ARM**）必須透過其他方法（例如，使用嵌入式管理單元、安裝程式腳本或手動二進位安裝）安裝 .net。

如需在沒有套件管理員的情況下安裝 .NET 的詳細資訊，請參閱下列其中一篇文章：

- [安裝具有嵌入式管理單元的 .NET SDK 或 .NET 執行時間。](../linux-snap.md)
- [使用腳本安裝 .NET SDK 或 .NET 執行時間。](../linux-scripted-manual.md#scripted-install)
- [手動安裝 .NET SDK 或 .NET 執行時間。](../linux-scripted-manual.md#manual-install)
