使用封裝管理員來管理 .NET 的安裝時，如果您先前已安裝預覽版本，可能會遇到衝突。 套件管理員可能會將非預覽版本解釋為舊版 .NET。 若要安裝非預覽版本，請先卸載預覽版本。 如需卸載 .NET 的詳細資訊，請參閱 [如何移除 .Net 執行時間和 SDK](../remove-runtime-sdk-versions.md?pivots=os-linux#uninstall-net)。
