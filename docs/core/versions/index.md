## <a name="semantic-versioning"></a><span data-ttu-id="78524-144">語意化版本控制系統</span><span class="sxs-lookup"><span data-stu-id="78524-144">Semantic versioning</span></span>

<span data-ttu-id="78524-145">.NET Core *執行階段* 大致上遵循 [語意式版本控制 (SemVer)](https://semver.org/)並採用 `MAJOR.MINOR.PATCH` 版本控制，使用版本號碼的不同部分來描述變更的程度和類型。</span><span class="sxs-lookup"><span data-stu-id="78524-145">The .NET Core *Runtime* roughly adheres to [Semantic Versioning (SemVer)](https://semver.org/), adopting the use of `MAJOR.MINOR.PATCH` versioning, using the various parts of the version number to describe the degree and type of change.</span></span>
