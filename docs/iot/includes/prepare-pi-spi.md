## <a name="prepare-the-raspberry-pi"></a>準備 Raspberry Pi

使用 `raspi-config` 命令以確定已啟用下列兩項服務：

- SSH
- SPI

如需的詳細資訊 `raspi-config` ，請參閱 [Raspberry Pi 檔](https://www.raspberrypi.org/documentation/configuration/raspi-config.md) <span class="docon docon-navigate-external x-hidden-focus"></span> 。
