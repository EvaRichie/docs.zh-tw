### <a name="example"></a><span data-ttu-id="a5e77-129">範例</span><span class="sxs-lookup"><span data-stu-id="a5e77-129">Example</span></span>

```csharp

// dotnet_style_prefer_auto_properties = false

{
    get
    {
        return age;
    }
}
```
