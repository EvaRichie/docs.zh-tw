## <a name="example"></a><span data-ttu-id="fb073-122">範例</span><span class="sxs-lookup"><span data-stu-id="fb073-122">Example</span></span>

```csharp
enum E
{
    A,
    B
}

class C
{
    // Code with violations
    int M(E e)
    {
        // IDE0010: Add missing cases
        return e switch
        {
            E.A => 0,
            _ => -1,
        };
    }

    // Fixed code
    int M(E e)
    {
        return e switch
        {
            E.A => 0,
            E.B => 1,
            _ => -1,
        };
    }
}
```
