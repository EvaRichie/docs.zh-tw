## <a name="how-to-fix-violations"></a><span data-ttu-id="3e26b-117">如何修正違規</span><span class="sxs-lookup"><span data-stu-id="3e26b-117">How to fix violations</span></span>

<span data-ttu-id="3e26b-118">若要修正違規，請移除完成項定義。</span><span class="sxs-lookup"><span data-stu-id="3e26b-118">To fix the violation, remove the finalizer definition.</span></span>

```csharp
class DerivedClass <T> : MemoryManager<T>
{
    public override bool Dispose(bool disposing)
    {
        if (disposing)
        {
            _handle.Dispose();
        }
    }

    ...

    // Violation occurs, remove the finalizer to fix the warning.
    ~DerivedClass() => Dispose(false);
}
```
