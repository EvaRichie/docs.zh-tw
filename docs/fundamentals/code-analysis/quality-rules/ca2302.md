# <a name="ca2302-ensure-binaryformatterbinder-is-set-before-calling-binaryformatterdeserialize"></a>CA2302：呼叫 BinaryFormatter.Deserialize 之前，請務必先設定 BinaryFormatter.Binder

| | 值 |
|-|-|
| **規則識別碼** |CA2302|
| **類別** |Microsoft.Security|
| **修正程式中斷或未中斷** |非中斷|

## <a name="cause"></a>原因

<xref:System.Runtime.Serialization.Formatters.Binary.BinaryFormatter?displayProperty=nameWithType>呼叫或參考了還原序列化方法，且 <xref:System.Runtime.Serialization.Formatters.Binary.BinaryFormatter.Binder> 屬性可能是 null。

這項規則類似于 [CA2301](ca2301.md)，但分析無法判斷 <xref:System.Runtime.Serialization.Formatters.Binary.BinaryFormatter.Binder> 是否肯定為 null。

根據預設，此規則會分析整個程式碼基底，但這是 [可](#configure-code-to-analyze)設定的。

> [!WARNING]
> 使用 SerializationBinder 限制類型無法防止所有攻擊。 如需詳細資訊，請參閱 [BinaryFormatter 安全性指南](../../../standard/serialization/binaryformatter-security-guide.md)。

## <a name="rule-description"></a>規則描述

[!INCLUDE[insecure-deserializers-description](~/includes/code-analysis/insecure-deserializers-description.md)]

<xref:System.Runtime.Serialization.Formatters.Binary.BinaryFormatter?displayProperty=nameWithType>當可能為 null 時，此規則會尋找還原序列化方法呼叫或參考 <xref:System.Runtime.Serialization.Formatters.Binary.BinaryFormatter.Binder> 。 如果您想要在不使用任何 <xref:System.Runtime.Serialization.Formatters.Binary.BinaryFormatter> 屬性的情況下允許任何還原序列化 <xref:System.Runtime.Serialization.Formatters.Binary.BinaryFormatter.Binder> ，請停用此規則並 [CA2301](ca2301.md)，然後啟用 [規則 [CA2300](ca2300.md)]。

## <a name="pseudo-code-examples"></a>虛擬程式碼範例

### <a name="violation-1"></a>違規1

```csharp
using System;
using System.IO;
using System.Runtime.Serialization;
using System.Runtime.Serialization.Formatters.Binary;

public class BookRecordSerializationBinder : SerializationBinder
{
    public override Type BindToType(string assemblyName, string typeName)
    {
        // One way to discover expected types is through testing deserialization
        // of **valid** data and logging the types used.

        ////Console.WriteLine($"BindToType('{assemblyName}', '{typeName}')");

        if (typeName == "BookRecord" || typeName == "AisleLocation")
        {
            return null;
        }
        else
        {
            throw new ArgumentException("Unexpected type", nameof(typeName));
        }
    }
}

[Serializable]
public class BookRecord
{
    public string Title { get; set; }
    public AisleLocation Location { get; set; }
}

[Serializable]
public class AisleLocation
{
    public char Aisle { get; set; }
    public byte Shelf { get; set; }
}

public class Binders
{
    public static SerializationBinder BookRecord =
        new BookRecordSerializationBinder();
}

public class ExampleClass
{
    public BookRecord DeserializeBookRecord(byte[] bytes)
    {
        BinaryFormatter formatter = new BinaryFormatter();
        formatter.Binder = Binders.BookRecord;
        using (MemoryStream ms = new MemoryStream(bytes))
        {
            return (BookRecord)formatter.Deserialize(ms);    // CA2302 violation
        }
    }
}
```
