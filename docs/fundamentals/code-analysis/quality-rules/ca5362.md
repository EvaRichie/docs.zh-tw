## <a name="cause"></a><span data-ttu-id="e6b30-111">原因</span><span class="sxs-lookup"><span data-stu-id="e6b30-111">Cause</span></span>

## <a name="rule-description"></a><span data-ttu-id="e6b30-113">規則描述</span><span class="sxs-lookup"><span data-stu-id="e6b30-113">Rule description</span></span>

<span data-ttu-id="e6b30-117">這項規則不一定表示有弱點，但只會在還原序列化的物件圖形中標示潛在的參考週期。</span><span class="sxs-lookup"><span data-stu-id="e6b30-117">This rule doesn't necessarily mean there's a vulnerability, but just flags potential reference cycles in deserialized object graphs.</span></span>

## <a name="pseudo-code-examples"></a><span data-ttu-id="e6b30-126">虛擬程式碼範例</span><span class="sxs-lookup"><span data-stu-id="e6b30-126">Pseudo-code examples</span></span>

### <a name="potential-reference-cycle-violation"></a><span data-ttu-id="e6b30-127">可能的參考迴圈違規</span><span class="sxs-lookup"><span data-stu-id="e6b30-127">Potential reference cycle violation</span></span>

```csharp
using System;

[Serializable()]
class ExampleClass
{
    public ExampleClass ExampleProperty {get; set;}

    public int NormalProperty {get; set;}
}

class AnotherClass
{
    // The argument passed by could be `JsonConvert.DeserializeObject<ExampleClass>(untrustedData)`.
    public void AnotherMethod(ExampleClass ec)
    {
        while(ec != null)
        {
            Console.WriteLine(ec.ToString());
            ec = ec.ExampleProperty;
        }
    }
}
```

### <a name="solution"></a><span data-ttu-id="e6b30-128">解決方案</span><span class="sxs-lookup"><span data-stu-id="e6b30-128">Solution</span></span>

```csharp
using System;

[Serializable()]
class ExampleClass
{
    [NonSerialized]
    public ExampleClass ExampleProperty {get; set;}

    public int NormalProperty {get; set;}
}

class AnotherClass
{
    // The argument passed by could be `JsonConvert.DeserializeObject<ExampleClass>(untrustedData)`.
    public void AnotherMethod(ExampleClass ec)
    {
        while(ec != null)
        {
            Console.WriteLine(ec.ToString());
            ec = ec.ExampleProperty;
        }
    }
}
```
