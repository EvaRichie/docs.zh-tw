## <a name="cause"></a><span data-ttu-id="bee72-111">原因</span><span class="sxs-lookup"><span data-stu-id="bee72-111">Cause</span></span>

<span data-ttu-id="bee72-112">使用 <xref:System.Net.Http.HttpClient?displayProperty=fullName> 時，提供平臺特定的處理常式 (<xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> 或 <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName>) 其 `CheckCertificateRevocationList` 屬性未設定為， `true` 將會允許將已撤銷的憑證接受 <xref:System.Net.Http.HttpClient> 為有效。</span><span class="sxs-lookup"><span data-stu-id="bee72-112">Using <xref:System.Net.Http.HttpClient?displayProperty=fullName> while providing a platform specific handler (<xref:System.Net.Http.WinHttpHandler?displayProperty=fullName> or <xref:System.Net.Http.HttpClientHandler?displayProperty=fullName>) whose `CheckCertificateRevocationList` property is not set to `true` will allow revoked certificates to be accepted by the <xref:System.Net.Http.HttpClient> as valid.</span></span>

<span data-ttu-id="bee72-113">這項規則類似于 [CA5400](ca5400.md)，但分析可判斷 `CheckCertificateRevocationList` 屬性是否為絕對 `false` 或未設定。</span><span class="sxs-lookup"><span data-stu-id="bee72-113">This rule is similar to [CA5400](ca5400.md), but analysis can determine that the `CheckCertificateRevocationList` property is definitely `false` or not set.</span></span>
