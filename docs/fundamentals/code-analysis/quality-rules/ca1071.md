## <a name="how-to-fix-violations"></a><span data-ttu-id="2e302-118">如何修正違規</span><span class="sxs-lookup"><span data-stu-id="2e302-118">How to fix violations</span></span>

<span data-ttu-id="2e302-119">若要修正違規，請覆寫基類 <xref:System.IEquatable%601.Equals%2A> 方法，並藉由叫用執行來執行它 <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> 。</span><span class="sxs-lookup"><span data-stu-id="2e302-119">To fix a violation, override the base class <xref:System.IEquatable%601.Equals%2A> method and implement it by invoking the <xref:System.IEquatable%601.Equals%2A?displayProperty=fullName> implementation.</span></span>
<span data-ttu-id="2e302-120">例如，下列兩個程式碼片段會顯示規則違規以及如何修正此問題：</span><span class="sxs-lookup"><span data-stu-id="2e302-120">For example, the following two code snippets show a violation of the rule and how to fix it:</span></span>

```csharp
using System;

public class B : IEquatable<B>
{
    public B(int value)
    {
        Value = value;
    }

    public int Value { get; }

    public virtual bool Equals(B other) => Value == other.Value;

    public sealed override bool Equals(object obj) => Equals(obj as B);
}

public sealed class C : B, IEquatable<C>
{
    public C(string name, int value)
        : base(value)
    {
        Name = name;
    }

    public string Name { get; }

    public bool Equals(C other) => base.Equals(other) && Name == other.Name;
}
```

<span data-ttu-id="2e302-121">下列程式碼片段顯示具有違規的相同兩個類別。</span><span class="sxs-lookup"><span data-stu-id="2e302-121">The following code snippet shows the same two classes with the violation addressed.</span></span>

```csharp
using System;

public class B : IEquatable<B>
{
    public B(int value)
    {
        Value = value;
    }

    public int Value { get; }

    public virtual bool Equals(B other) => Value == other.Value;

    public sealed override bool Equals(object obj) => Equals(obj as B);
}

public sealed class C : B, IEquatable<C>
{
    public C(string name, int value)
        : base(value)
    {
        Name = name;
    }

    public string Name { get; }

    public bool Equals(C other) => base.Equals(other) && Name == other.Name;

    // This line fixes the violation.
    public sealed override bool Equals(B other) => Equals(other as C);
}
```
