## <a name="choosing-a-hosting-option"></a><span data-ttu-id="2e877-121">選擇託管選項</span><span class="sxs-lookup"><span data-stu-id="2e877-121">Choosing a hosting option</span></span>

<span data-ttu-id="2e877-122">Azure 中的託管可分為三類。</span><span class="sxs-lookup"><span data-stu-id="2e877-122">Hosting in Azure can be divided into three categories.</span></span>

* <span data-ttu-id="2e877-123">**基礎結構即服務 (IaaS)**：有了 IaaS，您可以佈建所需要的虛擬機器，以及相關聯的網路和儲存體元件。</span><span class="sxs-lookup"><span data-stu-id="2e877-123">**Infrastructure-as-a-Service (IaaS)**: With IaaS, you provision the virtual machines you need along with associated network and storage components.</span></span> <span data-ttu-id="2e877-124">然後您會部署想放在這些 VM 上的任何軟體和應用程式。</span><span class="sxs-lookup"><span data-stu-id="2e877-124">You then deploy whatever software and applications you want onto those VMs.</span></span> <span data-ttu-id="2e877-125">此模型最接近傳統內部部署環境，不同之處在於 Microsoft 會管理基礎結構。</span><span class="sxs-lookup"><span data-stu-id="2e877-125">This model is the closest to a traditional on-premises environment except that Microsoft manages the infrastructure.</span></span> <span data-ttu-id="2e877-126">您還是可以管理個別的 VM，包括作業系統、自訂軟體和安全性更新。</span><span class="sxs-lookup"><span data-stu-id="2e877-126">You still manage the individual VMs, including the operating system, custom software, and security updates.</span></span>

* <span data-ttu-id="2e877-127">**平台即服務 (PaaS)**：PaaS 提供受控的託管環境，您可以在其中部署應用程式而不需要管理 VM 或網路資源。</span><span class="sxs-lookup"><span data-stu-id="2e877-127">**Platform-as-a-Service (PaaS)**: PaaS provides a managed hosting environment where you deploy your application without needing to manage VMs or networking resources.</span></span> <span data-ttu-id="2e877-128">例如，不要建立個別的 VM，您可以指定執行個體計數，而服務將會佈建、設定和管理所需的資源。</span><span class="sxs-lookup"><span data-stu-id="2e877-128">For example, instead of creating individual VMs, you specify an instance count, and the service will provision, configure, and manage the necessary resources.</span></span> <span data-ttu-id="2e877-129">Azure App Service 是 PaaS 服務的範例。</span><span class="sxs-lookup"><span data-stu-id="2e877-129">Azure App Service is an example of a PaaS service.</span></span>
  
* <span data-ttu-id="2e877-130">**函式即服務 (FaaS)**：FaaS 通常稱為無伺服器運算，能比 PaaS 更進一步地分擔託管環境的考量。</span><span class="sxs-lookup"><span data-stu-id="2e877-130">**Functions-as-a-Service (FaaS)**: Commonly referred to as serverless computing, FaaS goes even further than PaaS in abstracting the concerns of the hosting environment.</span></span> <span data-ttu-id="2e877-131">您不用建立計算執行個體並將程式碼部署到其中，只需部署您的程式碼，服務即會自動執行程式碼。</span><span class="sxs-lookup"><span data-stu-id="2e877-131">Instead of creating compute instances and then deploying code to those instances, you deploy your code and the service automatically runs it.</span></span> <span data-ttu-id="2e877-132">您也不需要管理計算資源。</span><span class="sxs-lookup"><span data-stu-id="2e877-132">You don't need to administer the compute resources.</span></span> <span data-ttu-id="2e877-133">平台會根據處理流量時所需的任何層級，無縫地相應增加或減少您的程式碼，而且只有當程式碼執行時才需支付費用。</span><span class="sxs-lookup"><span data-stu-id="2e877-133">The platform seamlessly scales your code up or down to whatever level necessary to handle the traffic, and you pay only when your code is running.</span></span> <span data-ttu-id="2e877-134">Azure Functions 即是 FaaS 服務。</span><span class="sxs-lookup"><span data-stu-id="2e877-134">Azure Functions is a FaaS service.</span></span>
