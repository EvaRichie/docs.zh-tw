## <a name="the-yield-keyword"></a><span data-ttu-id="eaa85-130">`yield!` 關鍵字</span><span class="sxs-lookup"><span data-stu-id="eaa85-130">The `yield!` keyword</span></span>

<span data-ttu-id="eaa85-131">有時候，您可能會想要將一連串的專案包含在另一個序列中。</span><span class="sxs-lookup"><span data-stu-id="eaa85-131">Sometimes, you may wish to include a sequence of elements into another sequence.</span></span> <span data-ttu-id="eaa85-132">若要在另一個序列中包含順序，您必須使用 `yield!` 關鍵字：</span><span class="sxs-lookup"><span data-stu-id="eaa85-132">To include a sequence within another sequence, you'll need to use the `yield!` keyword:</span></span>

```fsharp

        yield! seq { 1; 2; 3; 4; 5}
}
```

<span data-ttu-id="eaa85-133">另一個思考方式 `yield!` 是，它會壓平合併內部序列，然後將其包含在包含序列中。</span><span class="sxs-lookup"><span data-stu-id="eaa85-133">Another way of thinking of `yield!` is that it flattens an inner sequence and then includes that in the containing sequence.</span></span>

<span data-ttu-id="eaa85-134">`yield!`在運算式中使用時，所有其他單一值都必須使用 `yield` 關鍵字：</span><span class="sxs-lookup"><span data-stu-id="eaa85-134">When `yield!` is used in an expression, all other single values must use the `yield` keyword:</span></span>

        yield x
        yield! seq { for i in 1..x -> i}

<span data-ttu-id="eaa85-135">只有 `x` 在上述範例中指定，才會產生序列，而不會產生任何值。</span><span class="sxs-lookup"><span data-stu-id="eaa85-135">Specifying only `x` in the previous example will result in the sequence generating no values.</span></span>
