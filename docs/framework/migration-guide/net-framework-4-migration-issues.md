# <a name="net-framework-4-migration-issues"></a>.NET Framework 4 移轉問題

本文說明 .NET Framework 3.5 版 Service Pack 1 和 .NET Framework 第4版之間的遷移問題，包括修正、標準合規性和安全性的變更，以及根據客戶意見反應的變更。 大多數變更都不需要您的應用程式進行任何設計修改。 對於可能涉及修改的人，請參閱資料表中的 **建議變更** 資料行。 值得注意的變更會依區域細分，例如，ASP.NET 和 Windows Presentation Foundation (WPF) 。

如需本文中問題的更高層級總覽，請參閱 [.NET Framework 4 的遷移指南](/previous-versions/dotnet/netframework-4.0/ff657133(v=vs.100))。

如需新功能的詳細資訊，請參閱 [.NET Framework 4 的新](/previous-versions/dotnet/netframework-4.0/ms171868(v=vs.100))功能。
