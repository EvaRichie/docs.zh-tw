# <a name="data-contracts"></a>資料合約
本節包含的範例會示範 Windows Communication Foundation （WCF）中的資料合約。  
  
## <a name="in-this-section"></a>本節內容  
 [基本資料合約](basic-data-contract.md)  
 示範如何實作資料合約。  
  
 [DataContractSerializer 範例](datacontractserializer-sample.md)  
 示範執行資料合約類別之一般序列化與還原序列化服務的 <xref:System.Runtime.Serialization.DataContractSerializer>。  
  
 [已知型別](known-types.md)  
 示範如何在資料合約中指定有關衍生型別的資訊。  
  
 [物件參考](object-references.md)  
 示範如何以傳址 (By Reference) 方式在伺服器和用戶端之間傳遞物件。  
  
 [POCO 支援](poco-support.md)  
 示範對未標記型別的序列化支援，此種型別就是尚未套用序列化屬性的型別，有時亦稱為「簡單的 CLR 物件」(Plain Old CLR Object，POCO) 型別。  
  
 [使用序列化繫結器](usage-of-serialization-binder.md)  
 此範例示範如何使用 <xref:System.Runtime.Serialization.SerializationBinder> 變更序列化時一般類型的版本。
