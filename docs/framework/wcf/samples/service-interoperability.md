# <a name="service-interoperability"></a>服務互通性

本節包含的範例示範 Windows Communication Foundation (WCF) 和其他服務技術之間的互動。  
  
## <a name="in-this-section"></a>本節內容  

 [利用 COM 用戶端使用 WCF Moniker](using-the-wcf-moniker-with-com-clients.md)  
 示範如何使用 WCF 服務標記將 Web 服務整合至 COM 型開發環境，例如 Microsoft Office Visual Basic for Applications (Office VBA) 或 Visual Basic 6.0。  
  
 [含 WCF 服務的 ASMX 用戶端](asmx-client-with-a-wcf-service.md)  
 示範如何使用 WCF 建立服務，然後從非 WCF 用戶端（例如，.ASMX 用戶端）存取服務。
