## <a name="chunking-programming-model"></a><span data-ttu-id="5b91b-153">區塊處理程式設計模型</span><span class="sxs-lookup"><span data-stu-id="5b91b-153">Chunking Programming Model</span></span>

<span data-ttu-id="5b91b-154">服務開發人員可以將 `ChunkingBehavior` 屬性套用至合約內的作業，即可指定要區塊處理的訊息。</span><span class="sxs-lookup"><span data-stu-id="5b91b-154">Service developers can specify which messages are to be chunked by applying the `ChunkingBehavior` attribute to operations within the contract.</span></span> <span data-ttu-id="5b91b-155">屬性 (Attribute) 會公開 `AppliesTo` 屬性 (Property)，讓開發人員指定是否要將區塊處理套用至輸入訊息、輸出訊息或兩者皆套用。</span><span class="sxs-lookup"><span data-stu-id="5b91b-155">The attribute exposes an `AppliesTo` property that allows the developer to specify whether chunking applies to the input message, the output message or both.</span></span> <span data-ttu-id="5b91b-156">下列範例會示範 `ChunkingBehavior` 屬性的用法：</span><span class="sxs-lookup"><span data-stu-id="5b91b-156">The following example shows the usage of `ChunkingBehavior` attribute:</span></span>

```csharp

interface ITestService
{
    [OperationContract]
    [ChunkingBehavior(ChunkingAppliesTo.Both)]
    Stream EchoStream(Stream stream);

    [OperationContract]
    [ChunkingBehavior(ChunkingAppliesTo.OutMessage)]
    Stream DownloadStream();

    [OperationContract(IsOneWay=true)]
    [ChunkingBehavior(ChunkingAppliesTo.InMessage)]
    void UploadStream(Stream stream);

}
```
