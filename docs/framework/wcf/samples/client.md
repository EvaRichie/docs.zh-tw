# <a name="client"></a>用戶端
本節包含示範 Windows Communication Foundation （WCF）用戶端應用程式的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [用戶端互通性](client-interoperability.md)  
 示範 WCF 與其他技術之間的互通性。  
  
 [位址標頭](address-headers.md)  
 示範用戶端如何使用 WCF 將參考參數傳遞至服務。  
  
 [通道處理站](channel-factory.md)  
 示範用戶端應用程式如何使用 <xref:System.ServiceModel.ChannelFactory> 類別而不是所產生的用戶端來建立通道。  
  
 [預期的例外狀況](expected-exceptions.md)  
 示範如何在使用型別用戶端時攔截預期的例外狀況。  
  
 [擷取中繼資料](retrieve-metadata.md)  
 示範如何將會動態擷取服務之中繼資料的用戶端實作成選擇要與其通訊的端點。  
  
 [使用關閉和中止發行 WCF 用戶端資源](use-close-abort-release-wcf-client-resources.md)  
 說明使用具型別用戶端時，為什麼不能使用 C# "using" 陳述式自動清除資源。  
  
 [型別用戶端](typed-client.md)  
 示範如何從由[System.servicemodel 中繼資料公用程式工具（Svcutil）](../servicemodel-metadata-utility-tool-svcutil-exe.md)產生的具型別用戶端取得資訊。
