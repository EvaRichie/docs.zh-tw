# <a name="transport-extensibility"></a>傳輸擴充性
本節包含的範例會示範 Windows Communication Foundation （WCF）中的自訂傳輸。  
  
## <a name="in-this-section"></a>本節內容  
 [UDP 啟用](udp-activation.md)  
 示範如何使用 Windows 進程啟用服務（WAS）來擴充[Transport： UDP](transport-udp.md)範例，以支援進程啟用。  
  
 [傳輸：自訂跨 UDP 異動](transport-custom-transactions-over-udp-sample.md)  
 示範如何擴充 UDP 傳輸範例以支援自訂異動流程，並示範 <xref:System.ServiceModel.Channels.TransactionMessageProperty> 屬性的使用方式。  
  
 [傳輸：WSE 3.0 TCP 互通性](transport-wse-3-0-tcp-interoperability.md)  
 這個範例會示範如何將 TCP 雙工會話實作為自訂 Windows Communication Foundation （WCF）傳輸。
