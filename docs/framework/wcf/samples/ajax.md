本節包含示範非同步 JavaScript 與 XML 功能的範例。

 [JSONP](jsonp.md)  
 示範如何在 WCF REST 服務中支援 JSON with Padding (JSONP)。  
  
 [JSON 序列化](json-serialization.md)  
 示範如何使用 <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer> 來序列化與還原序列化採用「JavaScript 物件標記法」(JavaScript Object Notation，JSON) 格式的資料。  
  
 [基本 AJAX 服務](basic-ajax-service.md)  
 示範如何使用 WCF 來建立基本的 ASP.NET 非同步 JavaScript 和 XML （AJAX）服務。

 [無組態的 AJAX 服務](ajax-service-without-configuration.md)  
 示範如何使用 WCF 來建立基本的 ASP.NET 非同步 JavaScript 和 XML （AJAX）服務。  
  
 [使用複雜類型的 AJAX 服務](ajax-service-using-complex-types-sample.md)  
 示範如何使用 WCF 來建立 ASP.NET 非同步 JavaScript 和 XML （AJAX）服務，以建立複雜類型的實例。  
  
 [使用 JSON 和 XML 的 AJAX 服務](ajax-service-with-json-and-xml-sample.md)
