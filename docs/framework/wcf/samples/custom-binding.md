# <a name="custom-binding"></a>自訂繫結
本節包含示範使用端點項目之自訂繫結屬性的範例。  
  
## <a name="in-this-section"></a>本節內容  
 [命令式自訂繫結](custom-binding-imperative.md)  
 示範如何撰寫命令式程式碼來定義和使用自訂系結，而不使用設定檔或 WCF 產生的用戶端。  
  
 [自訂繫結傳輸和編碼](custom-binding-transport-and-encoding.md)  
 示範如何使用各種傳輸與訊息編碼項目來設定自訂繫結。

 [HTTPS 上的自訂繫結可靠工作階段](custom-binding-reliable-session-over-https.md)  
 示範透過可靠工作階段使用 SSL 傳輸安全性。
