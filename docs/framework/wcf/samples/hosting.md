# <a name="hosting"></a>裝載

本節包含的範例示範如何 (WCF) 服務裝載 Windows Communication Foundation。  
  
## <a name="in-this-section"></a>本節內容  

 [Windows Process Activation](windows-process-activation.md)  
 示範透過網路通訊協定啟動服務。  
  
 [SystemWebRouting 整合範例](systemwebrouting-integration-sample.md)  
 示範裝載層與 <xref:System.Web.Routing> 命名空間中的類別整合。  
  
 [ASP.NET 相容性](aspnet-compatibility.md)  
 示範如何在 WCF 中啟用 ASP.NET 相容性模式。  
  
 [使用內嵌程式碼的 IIS 裝載](iis-hosting-using-inline-code.md)

 [Windows 服務主機](windows-service-host.md)  
 示範裝載于 managed Windows 服務中的 WCF 服務。  
  
 [自我裝載](self-host.md)  
 示範如何在主控台應用程式中實作自我裝載的服務。
