# <a name="behaviors"></a>行為

本節包含的範例示範 Windows Communication Foundation (WCF) 服務行為。  
  
## <a name="in-this-section"></a>本節內容  

 [並行](concurrency.md)  
 示範搭配 <xref:System.ServiceModel.ServiceBehaviorAttribute> 列舉使用 <xref:System.ServiceModel.ConcurrencyMode>，以控制服務的執行個體要循序處理或並行處理訊息。  
  
 [預設服務行為](default-service-behavior.md)  
 示範如何設定服務行為設定。  
  
 [執行個體](instancing.md)  
 示範執行個體行為設定，此設定會控制如何建立可回應用戶端需求的服務類別執行個體。  
  
 [中繼資料發行行為](metadata-publishing-behavior.md)  
 示範如何控制服務的中繼資料發行功能。  
  
 [服務異動行為](service-transaction-behavior.md)  
 示範如何使用用戶端協調的交易，以及設定 <xref:System.ServiceModel.ServiceBehaviorAttribute> 和 <xref:System.ServiceModel.OperationBehaviorAttribute>，以控制服務交易行為。  
  
 [服務偵錯行為](service-debug-behavior.md)  
 示範如何設定服務偵錯行為設定。  
  
 [節流](throttling.md)  
 示範節流控制項的用法。  
  
 [行為安全性](behavior-security.md)  
 示範如何設定服務行為安全性。
