# <a name="missingnecessarykeyusage"></a>MissingNecessaryKeyUsage

識別碼：156  
  
 嚴重性：錯誤  
  
 分類：TransactionBridge  
  
## <a name="description"></a>描述  

 這個事件指出，指定之主體名稱和指紋的身分識別憑證未在其 KeyUsages 中提供特定資料。 它也會列出處理序名稱和處理序識別碼。
