# <a name="messageloggingon"></a>MessageLoggingOn

識別碼：116  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## <a name="description"></a>描述  

 訊息記錄已開啟。 敏感資訊即使已在網路上加密，仍然可以記錄。 例如，訊息本文。 此事件會列出 AppID、例外狀況、處理序名稱和處理序識別碼。
