# <a name="servicestartfailed"></a>ServiceStartFailed

識別碼：164  
  
 嚴重性：錯誤  
  
 分類：SMSvcHost  
  
## <a name="description"></a>描述  

 這個事件表示啟動服務的要求失敗。 此事件會列出錯誤碼、處理序名稱和處理序識別碼。
