# <a name="failfastexception"></a>FailFastException

識別碼：103  
  
 嚴重性：錯誤  
  
 分類：追蹤  
  
## <a name="description"></a>描述  

 在 FailFast 期間擲回例外狀況。 此事件會列出例外狀況、處理序名稱和處理序識別碼。
