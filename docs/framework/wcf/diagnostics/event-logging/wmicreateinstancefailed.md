# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed

識別碼：123  
  
 嚴重性：錯誤  
  
 分類：ServiceModel  
  
## <a name="description"></a>描述  

 這個事件指出 WMI CreateInstance 要求沒有處理。 此事件會列出類別名稱、例外狀況、處理序名稱與處理序識別碼。
