# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp

識別碼：160  
  
 嚴重性：錯誤  
  
 分類：SMSvcHost  
  
## <a name="description"></a>描述  

 此事件表示在指定的通訊協定中，指定的啟用服務發生錯誤，但在嘗試接聽指定的網站時，會暫時停用網站的通訊協定。 此事件會列出 URL、狀態、例外狀況、進程名稱和處理序識別碼。
