# <a name="queued-poison-messages"></a>佇列的有害訊息

計數器名稱：佇列的有害訊息。  
  
## <a name="description"></a>描述  

 此服務佇列傳輸標示為有害的訊息數目。  
  
 如需有關何時將訊息標示為有害的詳細資訊，請參閱 [有害訊息處理](../../feature-details/poison-message-handling.md)。
