# <a name="transactions-flowed"></a>流動的交易數

計數器名稱：流動的交易數  
  
## <a name="description"></a>描述  

 流動至此作業的交易數。 每當傳送給作業的訊息中包含交易識別碼時，此計數器就會遞增。
