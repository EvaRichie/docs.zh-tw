# <a name="queued-poison-messages-per-second"></a>每秒佇列的有害訊息數
計數器名稱：每秒佇列的有害訊息數。  
  
## <a name="description"></a>描述  
 此服務佇列傳輸標示為有害的每秒訊息數量。  
  
 此計數器是效能計數器類型 [PERF_COUNTER_COUNTER](/previous-versions/windows/it-pro/windows-server-2003/cc740048(v=ws.10))，其值是使用下列公式來計算。  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
