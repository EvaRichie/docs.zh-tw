# <a name="transacted-operations-aborted-per-second"></a>每秒中止的交易作業數

計數器名稱：每秒中止的交易作業數。  
  
## <a name="description"></a>描述  

 此服務中每秒中止的異動作業數。  
  
 此計數器是效能計數器類型 [PERF_COUNTER_COUNTER](/previous-versions/windows/it-pro/windows-server-2003/cc740048(v=ws.10))，其值是使用下列公式來計算。  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
