# <a name="service-transactions-flowed-per-second"></a>服務：每秒流動的異動數
計數器名稱：每秒流動的交易數。  
  
## <a name="description"></a>描述  
 每秒流動至此服務中作業的交易數。  
  
 此計數器是效能計數器類型 [PERF_COUNTER_COUNTER](/previous-versions/windows/it-pro/windows-server-2003/cc740048(v=ws.10))，其值是使用下列公式來計算。  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
