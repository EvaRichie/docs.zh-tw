# <a name="calls-failed"></a>失敗的呼叫數

計數器名稱：失敗的呼叫數  
  
## <a name="description"></a>描述  

 此作業中包含未處理之例外狀況的呼叫數。 此計數器會在此作業中每次有未處理的例外狀況時遞增。
