# <a name="queued-rejected-messages"></a>佇列拒絕的訊息數

計數器名稱：拒絕的佇列訊息。  
  
## <a name="description"></a>描述  

 此服務佇列傳輸所拒絕的訊息數量。  
  
 如需拒絕訊息時的詳細資訊，請參閱 [有害訊息處理](../../feature-details/poison-message-handling.md)。
