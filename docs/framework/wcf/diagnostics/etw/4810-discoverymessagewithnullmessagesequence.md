# <a name="4810---discoverymessagewithnullmessagesequence"></a>4810 - DiscoveryMessageWithNullMessageSequence

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|4810|  
|關鍵字|探索|  
|層級|警告|  
|通路|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  

 當發現訊息已被 DiscoveryClient 丟棄時，此事件發出，因為它沒有 DiscoveryMessageSequence 屬性。  
  
## <a name="message"></a>訊息  

 messageId='%2' 的 %1 訊息已遭 DiscoveryClient 丟棄，因為它沒有 DiscoveryMessageSequence 屬性。
