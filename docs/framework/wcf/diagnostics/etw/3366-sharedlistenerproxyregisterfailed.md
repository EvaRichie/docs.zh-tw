# <a name="3366---sharedlistenerproxyregisterfailed"></a>3366 - SharedListenerProxyRegisterFailed

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3366|  
|關鍵字|ActivationServices|  
|層級|錯誤|  
|通路|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>描述  

 當 SharedListenerProxy 登錄失敗時，就會發出此事件。  
  
## <a name="message"></a>訊息  

 SharedListenerProxy 登錄失敗，狀態為 '%1'。
