# <a name="4806---discoverymessagewithinvalidrelatestooroperationcompleted"></a>4806 - DiscoveryMessageWithInvalidRelatesToOrOperationCompleted

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|4806|  
|關鍵字|探索|  
|層級|警告|  
|通路|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  

 當對應的作業已完成，或是 relatesTo 值無效，導致探索訊息已由 DiscoveryClient 丟棄時，就會發出此事件。  
  
## <a name="message"></a>訊息  

 messageId='%2' 且 relatesTo='%3' 的 %1 訊息已由 DiscoveryClient 丟棄，因為對應的 %4 作業已完成，或是 relatesTo 值無效。
