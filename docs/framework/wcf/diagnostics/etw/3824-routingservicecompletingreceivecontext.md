## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3824|  
|關鍵字|RoutingServices|  
|層級|「詳細資訊」|

## <a name="description"></a>描述

## <a name="message"></a>訊息  

 路由服務正在為 ID：'%1' 的訊息呼叫 ReceiveContext.Complete。
