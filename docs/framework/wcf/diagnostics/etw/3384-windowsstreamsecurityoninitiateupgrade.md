# <a name="3384---windowsstreamsecurityoninitiateupgrade"></a>3384 - WindowsStreamSecurityOnInitiateUpgrade

|||  
|-|-|  
|識別碼|3384|  
|關鍵字|安全性|  
|層級|「詳細資訊」|  
|通路|Microsoft-Windows-Application Server-Applications/Analytic|
