|||

|識別碼|4820|  
|關鍵字|探索|

## <a name="description"></a>描述  

 當 SynchronizationContext 已由 DiscoveryClientChannel 重設為其原始值時，就會發出此事件。  
  
## <a name="message"></a>訊息
