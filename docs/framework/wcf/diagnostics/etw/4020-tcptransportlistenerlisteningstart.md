# <a name="4020---tcptransportlistenerlisteningstart"></a>4020 - TcpTransportListenerListeningStart

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|4020|  
|關鍵字|ActivationServices|  
|層級|「詳細資訊」|  
|通路|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>描述  

 當 TCP 傳輸接聽項開始接聽時，就會發出此事件。  
  
## <a name="message"></a>訊息  

 開始在 URI：'%1' 上接聽 Tcp 傳輸接聽項。
