# <a name="3827---routingservicetransmitfailed"></a>3827 - RoutingServiceTransmitFailed

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3827|  
|關鍵字|RoutingServices|  
|層級|警告|  
|通路|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  

 當路由服務傳輸失敗時，就會發出此事件。  
  
## <a name="message"></a>訊息  

 傳送至 '%1' 時，路由服務失敗。
