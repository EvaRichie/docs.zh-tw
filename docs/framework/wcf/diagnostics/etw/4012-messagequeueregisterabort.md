# <a name="4012---messagequeueregisterabort"></a>4012 - MessageQueueRegisterAbort

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|4012|  
|關鍵字|ActivationServices|  
|層級|錯誤|  
|通路|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>描述  

 當訊息佇列登錄中止時，就會發出此事件。  
  
## <a name="message"></a>訊息  

 URI：'%2' 的訊息佇列登錄已中止，狀態為：'%1'。
