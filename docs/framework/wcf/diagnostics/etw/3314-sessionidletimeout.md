# <a name="3314---sessionidletimeout"></a>3314 - SessionIdleTimeout

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3314|  
|關鍵字|ServiceModel|  
|層級|錯誤|  
|通路|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>描述  

 當工作階段因閒置通道而中止時，就會發出此事件。  
  
## <a name="message"></a>訊息  

 工作階段正在中止閒置的 URI 通道：'%1'。
