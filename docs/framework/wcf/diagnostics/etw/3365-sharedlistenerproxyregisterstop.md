# <a name="3365---sharedlistenerproxyregisterstop"></a>3365 - SharedListenerProxyRegisterStop

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3365|  
|關鍵字|ActivationServices|  
|層級|「詳細資訊」|
