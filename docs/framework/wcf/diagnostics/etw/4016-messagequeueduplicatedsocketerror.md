# <a name="4016---messagequeueduplicatedsocketerror"></a>4016 - MessageQueueDuplicatedSocketError

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|4016|  
|關鍵字|ActivationServices|  
|層級|錯誤|  
|通路|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>描述  

 當訊息佇列複製通訊端失敗時，就會發出此事件。  
  
## <a name="message"></a>訊息  

 訊息佇列無法複製通訊端。
