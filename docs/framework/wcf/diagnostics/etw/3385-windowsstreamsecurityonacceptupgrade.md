# <a name="3385---windowsstreamsecurityonacceptupgrade"></a>3385 - WindowsStreamSecurityOnAcceptUpgrade

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|3385|  
|關鍵字|安全性|  
|層級|「詳細資訊」|  
|通路|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>描述  

 當 Windows 資料流安全性接受安全性升級時，就會發出此事件。  
  
## <a name="message"></a>訊息  

 Windows 接受升級的資料流安全性。
