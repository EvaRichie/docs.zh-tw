# <a name="message-security-with-issued-tokens"></a>發行之權杖的訊息安全性
發行的權杖用戶端會啟用 Windows Communication Foundation （WCF）中的聯合安全性案例。  
  
 聯合安全性可在獨立網域之間建立信任關係。 每個網域都會集中進行安全性權杖服務中的驗證與授權作業。 當您在兩個此類服務之間建立起信任關係時，每個網域的成員都能使用另一個網域的服務。  
  
 如需聯合案例的詳細說明，請參閱[同盟和發行的權杖](federation-and-issued-tokens.md)。
