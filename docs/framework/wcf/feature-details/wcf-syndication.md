# <a name="wcf-syndication"></a>WCF 新聞訂閱

Windows Communication Foundation (WCF) 可讓您輕鬆地使用 Atom、RSS 或其他自訂格式的新聞訂閱摘要，讓您可以讀取和建立它們，並將它們公開在服務端點上。 本節的各主題會詳細描述新聞訂閱的程式設計模型。  
  
## <a name="in-this-section"></a>本節內容  

 [WCF 新聞訂閱概觀](wcf-syndication-overview.md)  
 提供 WCF 提供的新聞訂閱支援總覽。  
  
 [新聞訂閱架構](architecture-of-syndication.md)  
 描述物件模型中的類別以及新聞訂閱的可擴充性。  
  
 [WCF 新聞訂閱物件模型對應到 Atom 和 RSS 的方式](how-the-wcf-syndication-object-model-maps-to-atom-and-rss.md)  
 描述摘要在 WCF 新聞訂閱物件模型中的呈現方式，以及轉換至 RSS 和 Atom 摘要的方法。  
  
 [作法：建立基本 RSS 摘要](how-to-create-a-basic-rss-feed.md)  
 顯示如何建立可產生基本 RSS 摘要的服務。  
  
 [作法：建立基本 Atom 摘要](how-to-create-a-basic-atom-feed.md)  
 顯示如何建立可產生基本 ATOM 摘要的服務。  
  
 [作法：將摘要同時公開為 Atom 和 RSS](how-to-expose-a-feed-as-both-atom-and-rss.md)  
 顯示如何建立可產生 ATOM 和 RSS 相同摘要的服務。  
  
 [新聞訂閱擴充性](syndication-extensibility.md)  
 說明將自訂項目和屬性加入至新聞訂閱摘要的方法。
