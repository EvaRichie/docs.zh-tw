## <a name="the-channel-stack"></a><span data-ttu-id="41823-244">通道堆疊</span><span class="sxs-lookup"><span data-stu-id="41823-244">The Channel Stack</span></span>

 <span data-ttu-id="41823-246">如同之前所述，通道堆疊負責將傳出 <xref:System.ServiceModel.Channels.Message> 執行個體轉換成某些動作 (例如在網路上傳送封包)，或是將某些動作 (例如接收網路封包) 轉換成傳入 `Message` 執行個體。</span><span class="sxs-lookup"><span data-stu-id="41823-246">As stated before, the channel stack is responsible for converting outgoing <xref:System.ServiceModel.Channels.Message> instances into some action (such as sending packets over the network), or converting some action (such as receiving network packets) into incoming `Message` instances.</span></span>
