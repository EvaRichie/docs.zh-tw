## <a name="limitations-of-using-certain-types-in-partial-trust-mode"></a><span data-ttu-id="3dbee-137">在部分信任模式中使用特定型別的限制</span><span class="sxs-lookup"><span data-stu-id="3dbee-137">Limitations of Using Certain Types in Partial Trust Mode</span></span>

<span data-ttu-id="3dbee-138">下列是在部分信任模式案例中使用特定型別的限制清單：</span><span class="sxs-lookup"><span data-stu-id="3dbee-138">The following is a list of limitations when using certain types in partial trust mode scenarios:</span></span>

- <span data-ttu-id="3dbee-139">若要在部分信任程式碼中，透過 <xref:System.Runtime.Serialization.ISerializable> 序列化或還原序列化可實作 <xref:System.Runtime.Serialization.DataContractSerializer> 的型別需要 <xref:System.Security.Permissions.SecurityPermissionAttribute.SerializationFormatter%2A> 和 <xref:System.Security.Permissions.SecurityPermissionAttribute.UnmanagedCode%2A> 權限。</span><span class="sxs-lookup"><span data-stu-id="3dbee-139">To serialize or deserialize a type that implements <xref:System.Runtime.Serialization.ISerializable> in partially-trusted code using the <xref:System.Runtime.Serialization.DataContractSerializer> requires the <xref:System.Security.Permissions.SecurityPermissionAttribute.SerializationFormatter%2A> and <xref:System.Security.Permissions.SecurityPermissionAttribute.UnmanagedCode%2A> permissions.</span></span>

- <span data-ttu-id="3dbee-140">在[部分信任](partial-trust.md)模式下執行 WCF 程式碼時，不支援欄位的序列化和還原序列化 `readonly` （ `public` 和兩者 `private` ）。</span><span class="sxs-lookup"><span data-stu-id="3dbee-140">When running WCF code in [Partial Trust](partial-trust.md) mode, the serialization and deserialization of `readonly` fields (both `public` and `private`) is not supported.</span></span> <span data-ttu-id="3dbee-141">這是因為產生的 IL 無法加以驗證，因此需要較高的權限。</span><span class="sxs-lookup"><span data-stu-id="3dbee-141">This is because the generated IL is unverifiable and therefore requires elevated permissions.</span></span>

- <span data-ttu-id="3dbee-142">部分信任環境同時支援 <xref:System.Runtime.Serialization.DataContractSerializer> 和 <xref:System.Xml.Serialization.XmlSerializer> 。</span><span class="sxs-lookup"><span data-stu-id="3dbee-142">Both the <xref:System.Runtime.Serialization.DataContractSerializer> and the <xref:System.Xml.Serialization.XmlSerializer> are supported in a partial trust environment.</span></span> <span data-ttu-id="3dbee-143">然而， <xref:System.Runtime.Serialization.DataContractSerializer> 的使用需視下列情況而定：</span><span class="sxs-lookup"><span data-stu-id="3dbee-143">However, use of the <xref:System.Runtime.Serialization.DataContractSerializer> is subject to the following conditions:</span></span>

  - <span data-ttu-id="3dbee-144">所有可序列化的 `[DataContract]` 型別必須是公用的。</span><span class="sxs-lookup"><span data-stu-id="3dbee-144">All serializable `[DataContract]` types must be public.</span></span>

  - <span data-ttu-id="3dbee-145">`[DataMember]` 型別中所有可序列化的 `[DataContract]` 欄位或屬性必須具有公用和讀/寫性質。</span><span class="sxs-lookup"><span data-stu-id="3dbee-145">All serializable `[DataMember]` fields or properties in a `[DataContract]` type must be public and read/write.</span></span> <span data-ttu-id="3dbee-146">`readonly`在部分信任的應用程式中執行 WCF 時，不支援欄位的序列化和還原序列化。</span><span class="sxs-lookup"><span data-stu-id="3dbee-146">The serialization and deserialization of `readonly` fields is not supported when running WCF in a partially-trusted application.</span></span>

  - <span data-ttu-id="3dbee-147">支援 `[Serializable]`/`ISerializable]` 程式設計模型。</span><span class="sxs-lookup"><span data-stu-id="3dbee-147">The `[Serializable]`/`ISerializable]` programming model is not supported in a partial trust environment.</span></span>

  - <span data-ttu-id="3dbee-148">已知型別必須在程式碼或電腦層級組態 (`Machine.config`) 中指定。</span><span class="sxs-lookup"><span data-stu-id="3dbee-148">Known types must be specified in code or machine-level configuration (`Machine.config`).</span></span> <span data-ttu-id="3dbee-149">為了安全起見，已知型別無法在應用程式層級的組態中指定。</span><span class="sxs-lookup"><span data-stu-id="3dbee-149">Known types cannot be specified in application-level configuration for security reasons.</span></span>

- <span data-ttu-id="3dbee-150">在部分信任環境中，可實作 <xref:System.Runtime.Serialization.IObjectReference> 的型別會擲回例外狀況，因為 <xref:System.Runtime.Serialization.IObjectReference.GetRealObject%2A> 方法需要安全性權限 `[SecurityPermission(SecurityAction.LinkDemand, Flags=SecurityPermissionFlag.SerializationFormatter)]`。</span><span class="sxs-lookup"><span data-stu-id="3dbee-150">Types that implement <xref:System.Runtime.Serialization.IObjectReference> throw an exception in a partially-trusted environment because the <xref:System.Runtime.Serialization.IObjectReference.GetRealObject%2A> method requires the security permission `[SecurityPermission(SecurityAction.LinkDemand, Flags=SecurityPermissionFlag.SerializationFormatter)]`.</span></span>
