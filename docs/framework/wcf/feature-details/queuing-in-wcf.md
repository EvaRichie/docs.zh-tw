#### <a name="exactlyonce-and-durable-properties"></a><span data-ttu-id="6c152-152">ExactlyOnce 和 Durable 屬性</span><span class="sxs-lookup"><span data-stu-id="6c152-152">ExactlyOnce and Durable Properties</span></span>
