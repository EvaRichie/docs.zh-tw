# <a name="metadata"></a>中繼資料

Windows Communication Foundation (WCF) 會提供用來匯出、發佈、取出和匯入服務中繼資料的基礎結構。 WCF 服務會使用中繼資料來描述如何與服務的端點互動，如此一來，工具（例如 Svcutil.exe）就可以自動產生用戶端程式代碼來存取服務。
