# <a name="metadata"></a>中繼資料
Windows Communication Foundation （WCF）提供匯出、發行、抓取和匯入服務中繼資料的基礎結構。 WCF 服務會使用中繼資料來描述如何與服務的端點互動，讓工具（例如 Svcutil）可以自動產生用戶端程式代碼來存取服務。
