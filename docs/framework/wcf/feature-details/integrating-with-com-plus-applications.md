# <a name="integrating-with-com-applications"></a>整合 COM+ 應用程式
Windows Communication Foundation （WCF）提供豐富的環境來建立分散式應用程式。 如果您在 COM + 裝載的元件架構應用程式邏輯中有大量的投資，您可以使用 WCF 來擴充現有的邏輯，而不需要重寫它。 本節中的主題描述如何搭配 WCF 使用 COM +。

 [整合 COM+ 應用程式概觀](integrating-with-com-plus-applications-overview.md)  
 對何時及如何整合 COM+ 元件提供概觀。  
  
 [如何：使用 COM+ 服務模型組態工具](how-to-use-the-com-service-model-configuration-tool.md)  
 說明如何使用 COM + 服務模型設定命令列工具（ComSvcConfig）來設定您想要公開為 WCF 服務的應用程式介面。  
  
 [HOW TO：設定 COM+ 服務設定](how-to-configure-com-service-settings.md)  
 說明如何將 COM + 物件設定為 WCF 服務。  
  
 [HOW TO：部署 COM+ 整合應用程式](how-to-deploy-a-com-integration-application.md)  
 解釋如何移動 COM+ 整合應用程式。
