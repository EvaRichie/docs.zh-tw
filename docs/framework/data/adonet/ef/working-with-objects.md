# <a name="working-with-objects"></a>處理物件
Entity Framework 可讓您查詢、插入、更新和刪除資料，而這些資料是以具類型的 common language runtime （ (CLR) 物件，也就是實體類型的實例）來表示。 這些實體類型代表在概念模型中定義的實體。 Entity Framework 會將概念模型中定義的實體和關聯性對應至資料來源。 Entity Framework 提供可執行下列動作的工具：將從資料來源傳回的資料具體化為物件;追蹤對物件所做的變更;處理平行存取;將物件變更傳播回資料來源;以及將物件系結至控制項。  
  
 如需使用最新版 Entity Framework 中物件的詳細資訊，請參閱使用 [物件](/previous-versions/gg696163(v=vs.103))。
