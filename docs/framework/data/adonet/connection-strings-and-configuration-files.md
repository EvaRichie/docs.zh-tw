## <a name="working-with-application-configuration-files"></a><span data-ttu-id="77f67-108">使用應用程式組態檔</span><span class="sxs-lookup"><span data-stu-id="77f67-108">Working with Application Configuration Files</span></span>

### <a name="the-connectionstrings-section"></a><span data-ttu-id="77f67-112">connectionStrings 區段</span><span class="sxs-lookup"><span data-stu-id="77f67-112">The connectionStrings Section</span></span>  

 <span data-ttu-id="77f67-113">連接字串可以用索引鍵/值對的方式，儲存在應用程式組態檔 **configuration** 項目的 **connectionStrings** 區段中。</span><span class="sxs-lookup"><span data-stu-id="77f67-113">Connection strings can be stored as key/value pairs in the **connectionStrings** section of the **configuration** element of an application configuration file.</span></span> <span data-ttu-id="77f67-114">子項目包含 **add**、**clear** 和 **remove**。</span><span class="sxs-lookup"><span data-stu-id="77f67-114">Child elements include **add**, **clear**, and **remove**.</span></span>  
  
 <span data-ttu-id="77f67-115">下列組態檔片段示範儲存連接字串的結構描述和語法。</span><span class="sxs-lookup"><span data-stu-id="77f67-115">The following configuration file fragment demonstrates the schema and syntax for storing a connection string.</span></span> <span data-ttu-id="77f67-116">**name** 屬性是用於唯一識別連接字串的名稱，可藉此在執行階段擷取該連接字串。</span><span class="sxs-lookup"><span data-stu-id="77f67-116">The **name** attribute is a name that you provide to uniquely identify a connection string so that it can be retrieved at run time.</span></span> <span data-ttu-id="77f67-117">**providerName** 是 .NET Framework 資料提供者的非變異名稱，登錄於 machine.config 檔案。</span><span class="sxs-lookup"><span data-stu-id="77f67-117">The **providerName** is the invariant name of the .NET Framework data provider, which is registered in the machine.config file.</span></span>  
  
```xml  
<?xml version='1.0' encoding='utf-8'?>  
  <configuration>  
    <connectionStrings>  
      <clear />  
      <add name="Name"
       providerName="System.Data.ProviderName"
       connectionString="Valid Connection String;" />  
    </connectionStrings>  
  </configuration>  
```  
  
> [!NOTE]
> <span data-ttu-id="77f67-118">您可將部分連接字串儲存在組態檔中，然後在執行階段使用 <xref:System.Data.Common.DbConnectionStringBuilder> 類別 (Class) 加以完成。</span><span class="sxs-lookup"><span data-stu-id="77f67-118">You can save part of a connection string in a configuration file and use the <xref:System.Data.Common.DbConnectionStringBuilder> class to complete it at run time.</span></span> <span data-ttu-id="77f67-119">在您無法預先知道連接字串的項目，或者不想將機密資訊儲存在組態檔中時，這種方法很有用。</span><span class="sxs-lookup"><span data-stu-id="77f67-119">This is useful in scenarios where you do not know elements of the connection string ahead of time, or when you do not want to save sensitive information in a configuration file.</span></span> <span data-ttu-id="77f67-120">如需詳細資訊，請參閱[連接字串建置器](connection-string-builders.md)。</span><span class="sxs-lookup"><span data-stu-id="77f67-120">For more information, see [Connection String Builders](connection-string-builders.md).</span></span>
