<span data-ttu-id="bf7c2-104">查詢是指從資料來源中擷取資料的運算式。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-104">A query is an expression that retrieves data from a data source.</span></span> <span data-ttu-id="bf7c2-105">查詢通常會以特定的查詢語言來表示，例如 SQL 用於關聯式資料庫，而 XQuery 用於 XML。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-105">Queries are usually expressed in a specialized query language, such as SQL for relational databases and XQuery for XML.</span></span> <span data-ttu-id="bf7c2-106">因此，開發人員必須針對他們所查詢的每種資料來源或資料格式，學習新的查詢語言。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-106">Therefore, developers have had to learn a new query language for each type of data source or data format that they query.</span></span> <span data-ttu-id="bf7c2-107">Language-Integrated Query (LINQ) 提供了一種較簡單且一致的模型，可處理各種資料來源和格式的資料。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-107">Language-Integrated Query (LINQ) offers a simpler, consistent model for working with data across various kinds of data sources and formats.</span></span> <span data-ttu-id="bf7c2-108">在 LINQ 查詢中，您一定會使用程式設計物件。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-108">In a LINQ query, you always work with programming objects.</span></span>  
  
 <span data-ttu-id="bf7c2-109">LINQ 查詢作業由三個動作構成：取得資料來源、建立查詢和執行查詢。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-109">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>  
  
 <span data-ttu-id="bf7c2-110">執行泛型介面的資料來源 <xref:System.Collections.Generic.IEnumerable%601> 可以透過 LINQ 查詢。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-110">Data sources that implement the <xref:System.Collections.Generic.IEnumerable%601> generic interface can be queried through LINQ.</span></span> <span data-ttu-id="bf7c2-111">在上呼叫會傳回 <xref:System.Data.DataTableExtensions.AsEnumerable%2A> <xref:System.Data.DataTable> 物件，它會實 <xref:System.Collections.Generic.IEnumerable%601> 作為 LINQ to DataSet 查詢的資料來源。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-111">Calling <xref:System.Data.DataTableExtensions.AsEnumerable%2A> on a <xref:System.Data.DataTable> returns an object which implements the generic <xref:System.Collections.Generic.IEnumerable%601> interface, which serves as the data source for LINQ to DataSet queries.</span></span>  
  
 <span data-ttu-id="bf7c2-112">在此查詢中，您可以精確地指定想要從資料來源中擷取的資訊。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-112">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="bf7c2-113">此外，查詢也可以指定該項資訊傳回之前應該如何排序、分組和成形。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-113">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="bf7c2-114">在 LINQ 中，查詢會儲存在變數內。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-114">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="bf7c2-115">如果查詢設計成傳回值的序列 (Sequence)，查詢變數本身就必須是可列舉的型別。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-115">If the query is designed to return a sequence of values, the query variable itself must be a enumerable type.</span></span> <span data-ttu-id="bf7c2-116">這個查詢變數不會採取任何動作，也不會傳回任何資料。它只會儲存查詢資訊。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-116">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="bf7c2-117">在您建立查詢之後，必須執行該查詢以便擷取任何資料。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-117">After you create a query you must execute that query to retrieve any data.</span></span>  
  
 <span data-ttu-id="bf7c2-118">在傳回值序列的查詢中，查詢變數本身絕不會保存查詢結果，只會儲存查詢命令而已。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-118">In a query that returns a sequence of values, the query variable itself never holds the query results and only stores the query commands.</span></span> <span data-ttu-id="bf7c2-119">查詢的執行會延後，直到在 `foreach` 或 `For Each` 迴圈 (Loop) 中反覆查看查詢變數為止。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-119">Execution of the query is deferred until the query variable is iterated over in a `foreach` or `For Each` loop.</span></span> <span data-ttu-id="bf7c2-120">這稱為「*延後執行*」;也就是說，查詢執行會在查詢經過一段時間之後發生。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-120">This is called *deferred execution*; that is, query execution occurs some time after the query is constructed.</span></span> <span data-ttu-id="bf7c2-121">這表示，您可以依照想要的頻率來執行查詢。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-121">This means that you can execute a query as often as you want to.</span></span> <span data-ttu-id="bf7c2-122">例如，當您擁有一個正由其他應用程式更新的資料庫時，這就很有用。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-122">This is useful when, for example, you have a database that is being updated by other applications.</span></span> <span data-ttu-id="bf7c2-123">您可以在應用程式中建立擷取最新資訊的查詢，然後重複執行此查詢，以便每次都傳回更新的資訊。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-123">In your application, you can create a query to retrieve the latest information and repeatedly execute the query, returning the updated information every time.</span></span>  
  
 <span data-ttu-id="bf7c2-124">相較於傳回值序列的延後查詢，傳回單一子句值的查詢會立即執行。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-124">In contrast to deferred queries, which return a sequence of values, queries that return a singleton value are executed immediately.</span></span> <span data-ttu-id="bf7c2-125">單一子句查詢的某些範例包括 <xref:System.Linq.Enumerable.Count%2A>、<xref:System.Linq.Enumerable.Max%2A>、<xref:System.Linq.Enumerable.Average%2A> 和 <xref:System.Linq.Enumerable.First%2A>。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-125">Some examples of singleton queries are <xref:System.Linq.Enumerable.Count%2A>, <xref:System.Linq.Enumerable.Max%2A>, <xref:System.Linq.Enumerable.Average%2A>, and <xref:System.Linq.Enumerable.First%2A>.</span></span> <span data-ttu-id="bf7c2-126">這些查詢會立即執行，因為系統需要查詢結果來計算單一子句結果。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-126">These execute immediately because the query results are required to calculate the singleton result.</span></span> <span data-ttu-id="bf7c2-127">例如，若要尋找查詢結果的平均值，您必須執行查詢，如此平均函式才有輸入資料可處理。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-127">For example, in order to find the average of the query results the query must be executed so that the averaging function has input data to work with.</span></span> <span data-ttu-id="bf7c2-128">此外，您也可以針對查詢使用 <xref:System.Linq.Enumerable.ToList%2A> 或 <xref:System.Linq.Enumerable.ToArray%2A> 方法，強制立即執行不會產生單一子句值的查詢。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-128">You can also use the <xref:System.Linq.Enumerable.ToList%2A> or <xref:System.Linq.Enumerable.ToArray%2A> methods on a query to force immediate execution of a query that does not produce a singleton value.</span></span> <span data-ttu-id="bf7c2-129">當您想要快取查詢的結果時，這些強制立即執行的技巧就很有用。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-129">These techniques to force immediate execution can be useful when you want to cache the results of a query.</span></span>
  
## <a name="queries"></a><span data-ttu-id="bf7c2-130">查詢</span><span class="sxs-lookup"><span data-stu-id="bf7c2-130">Queries</span></span>  
 <span data-ttu-id="bf7c2-131">LINQ to DataSet 查詢可以用兩種不同的語法來撰寫：查詢運算式語法和以方法為基礎的查詢語法。</span><span class="sxs-lookup"><span data-stu-id="bf7c2-131">LINQ to DataSet queries can be formulated in two different syntaxes: query expression syntax and method-based query syntax.</span></span>
