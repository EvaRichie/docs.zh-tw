# <a name="multiple-active-result-sets-mars"></a>Multiple Active Result Set (MARS)

Multiple Active Result Set (MARS) 是可允許在單一連線中執行多個批次作業的功能。 在先前版本中，針對單一連線，一次只能執行一個批次。 使用 MARS 執行多個批次，並非意味著同時執行作業。
