<span data-ttu-id="72005-103">使用 <xref:System.Data.Linq.DataLoadOptions.LoadWith%2A> 方法可指定與您主要目標有關、應該同時擷取的資料。</span><span class="sxs-lookup"><span data-stu-id="72005-103">Use the <xref:System.Data.Linq.DataLoadOptions.LoadWith%2A> method to specify which data related to your main target should be retrieved at the same time.</span></span> <span data-ttu-id="72005-104">例如，如果您預先得知需要客戶訂單的相關資訊，則可以使用 <xref:System.Data.Linq.DataLoadOptions.LoadWith%2A>，以確保在擷取客戶資訊的同時也會擷取訂單資訊。</span><span class="sxs-lookup"><span data-stu-id="72005-104">For example, if you know you will need information about customers' orders, you can use <xref:System.Data.Linq.DataLoadOptions.LoadWith%2A> to make sure that the order information is retrieved at the same time as the customer information.</span></span> <span data-ttu-id="72005-105">這種方法只要存取一次資料庫，就可以同時取得兩個資訊集。</span><span class="sxs-lookup"><span data-stu-id="72005-105">This approach results in only one trip to the database for both sets of information.</span></span>
