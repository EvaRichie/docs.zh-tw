# <a name="query-examples"></a>查詢範例

本節提供一般查詢的 Visual Basic 和 c # 範例 [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] 。 使用 Visual Studio 的開發人員可以在範例一節所提供的範例解決方案中找到更多範例。 如需詳細資訊，請參閱 [範例](samples.md)。  
  
> [!IMPORTANT]
> *資料庫* 通常用於檔中的程式碼範例 [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] 。 *db* 假設為 *Northwind* 類別的實例，而該類別繼承自 <xref:System.Data.Linq.DataContext> 。
