# <a name="sort-elements-in-a-sequence"></a>在序列中排序項目

使用 <xref:System.Linq.Enumerable.OrderBy%2A> 運算子，可以根據一個或多個索引鍵來排序序列。  
  
> [!NOTE]
> [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] 是設計來支援以簡單的基本型別（例如 `string` 、等）排序 `int` 。 但不支援複雜多重值類別 (如匿名型別) 的排序， 也不支援 `byte` 資料型別。
