<span data-ttu-id="6c817-103">本主題描述 LINQ to SQL 查詢的類型，這些查詢可從 <xref:System.Data.Linq.DataContext> 所管理的識別快取傳回物件。</span><span class="sxs-lookup"><span data-stu-id="6c817-103">This topic describes the types of LINQ to SQL queries that return an object from the identity cache that is managed by the <xref:System.Data.Linq.DataContext>.</span></span>  
  
 <span data-ttu-id="6c817-104">在 LINQ to SQL 中，<xref:System.Data.Linq.DataContext> 用來管理物件的其中一種方式就是在執行查詢時，將物件識別記錄在識別快取中。</span><span class="sxs-lookup"><span data-stu-id="6c817-104">In LINQ to SQL, one of the ways in which the <xref:System.Data.Linq.DataContext> manages objects is by logging object identities in an identity cache as queries are executed.</span></span> <span data-ttu-id="6c817-105">在某些情況下，LINQ to SQL 會先嘗試從識別快取擷取物件，然後再執行資料庫的查詢。</span><span class="sxs-lookup"><span data-stu-id="6c817-105">In some cases, LINQ to SQL will attempt to retrieve an object from the identity cache before executing a query in the database.</span></span>  
  
 <span data-ttu-id="6c817-106">一般而言，為了讓 LINQ to SQL 查詢從識別快取傳回物件，此查詢必須以物件的主索引鍵為基礎，而且必須傳回單一物件。</span><span class="sxs-lookup"><span data-stu-id="6c817-106">In general, for a LINQ to SQL query to return an object from the identity cache, the query must be based on the primary key of an object and must return a single object.</span></span> <span data-ttu-id="6c817-107">特別是，此查詢必須採用下列其中一種一般格式。</span><span class="sxs-lookup"><span data-stu-id="6c817-107">In particular, the query must be in one of the general forms shown below.</span></span>

 <span data-ttu-id="6c817-113">在這些一般格式中，`Function1`、`Function2` 和 `predicate` 的定義方式如下。</span><span class="sxs-lookup"><span data-stu-id="6c817-113">In these general forms, `Function1`, `Function2`, and `predicate` are defined as follows.</span></span>  
  
 <span data-ttu-id="6c817-114">`Function1` 可以是下列任何項目：</span><span class="sxs-lookup"><span data-stu-id="6c817-114">`Function1` can be any of the following:</span></span>

- <xref:System.Linq.Queryable.FirstOrDefault%2A>

- <xref:System.Linq.Queryable.SingleOrDefault%2A>  
  
 <span data-ttu-id="6c817-115">`Function2` 可以是下列任何項目：</span><span class="sxs-lookup"><span data-stu-id="6c817-115">`Function2` can be any of the following:</span></span>  
  
- <xref:System.Linq.Queryable.First%2A>  
  
- <xref:System.Linq.Queryable.FirstOrDefault%2A>  
  
- <xref:System.Linq.Queryable.Single%2A>

 <span data-ttu-id="6c817-116">`predicate` 必須是物件之主索引鍵屬性設定為常數值的運算式。</span><span class="sxs-lookup"><span data-stu-id="6c817-116">`predicate` must be an expression in which the object's primary key property is set to a constant value.</span></span> <span data-ttu-id="6c817-117">如果物件具有多個屬性所定義的主索引鍵，每個主索引鍵屬性都必須設定為常數值。</span><span class="sxs-lookup"><span data-stu-id="6c817-117">If an object has a primary key defined by more than one property, each primary key property must be set to a constant value.</span></span> <span data-ttu-id="6c817-118">下面是 `predicate` 必須採用的格式範例：</span><span class="sxs-lookup"><span data-stu-id="6c817-118">The following are examples of the form `predicate` must take:</span></span>
