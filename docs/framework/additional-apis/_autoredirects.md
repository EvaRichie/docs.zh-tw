# <a name="httpwebrequest_autoredirects-field"></a>HttpWebRequest。 \_AutoRedirects 欄位

`HttpWebRequest._AutoRedirects`是 <xref:System.Int32> ，它會反映為此所做的重新導向數目 <xref:System.Net.HttpWebRequest> 。

## <a name="syntax"></a>Syntax  
  
```csharp  
private int _AutoRedirects
```

> [!WARNING]
> `HttpWebRequest._AutoRedirects`欄位是內部的，不適合直接在程式碼中使用。
>
> 在任何情況下，Microsoft 不支援在生產應用程式中使用此欄位。

## <a name="requirements"></a>需求

**命名空間：** <xref:System.Net>

**元件：** 系統（在 System.dll 中）

**.NET Framework 版本：** 自2.0 開始提供。
