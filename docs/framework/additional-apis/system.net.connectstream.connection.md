# <a name="connectstreamconnection-property"></a>ConnectStream 連接屬性

取得連接。

## <a name="syntax"></a>Syntax

```csharp
internal Connection Connection { get; }
```

## <a name="property-value"></a>屬性值

`System.Net.Connection`

連接物件。

## <a name="remarks"></a>備註

> [!WARNING]
> `ConnectStream.Connection`屬性是內部的，而且不適合直接在程式碼中使用。
>
> 在任何情況下，Microsoft 不支援在生產應用程式中使用此屬性。
