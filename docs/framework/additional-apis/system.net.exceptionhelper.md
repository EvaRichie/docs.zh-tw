# <a name="exceptionhelper-class"></a>ExceptionHelper 類別

提供具有標準化錯誤訊息的例外狀況。 這個類別無法被繼承。

```csharp
internal static class ExceptionHelper
```

> [!WARNING]
> 這個類別是內部的，不適合直接在程式碼中使用。
>
> 在任何情況下，Microsoft 不支援在生產應用程式中使用此類別。

## <a name="webpermissionunrestricted-field"></a>WebPermissionUnrestricted 欄位

指定應用程式可以連接到網際網路資源。

```csharp
internal static readonly WebPermission WebPermissionUnrestricted
```
