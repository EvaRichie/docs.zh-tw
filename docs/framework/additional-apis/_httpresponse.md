# <a name="httpwebrequest_httpresponse-field"></a>HttpWebRequest。 \_HttpResponse 欄位

`HttpWebRequest._HttpResponse`是， <xref:System.Net.HttpWebResponse> 其中包含來自 HTTP 要求的 HTTP 回應詳細資料。 這可能是在 `null` 收到 HTTP 回應之前。

## <a name="syntax"></a>Syntax
  
```csharp  
internal HttpWebResponse _HttpResponse
```

> [!WARNING]
> `HttpWebRequest._HttpResponse`欄位是內部的，而不是直接在程式碼中使用。
>
> 在任何情況下，Microsoft 不支援在生產應用程式中使用此欄位。

## <a name="requirements"></a>需求

**命名空間：** <xref:System.Net>

**元件：** 系統（在 System.dll 中）

**.NET Framework 版本：** 自2.0 開始提供。
