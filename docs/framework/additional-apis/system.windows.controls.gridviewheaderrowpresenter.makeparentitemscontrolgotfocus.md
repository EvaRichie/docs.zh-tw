# <a name="makeparentitemscontrolgotfocus-method"></a>MakeParentItemsControlGotFocus 方法

將焦點提供給專案的父控制項。 如果父控制項是，則會將 <xref:System.Windows.Controls.ListBox> 焦點放在中最近存取的專案 <xref:System.Windows.Controls.ListBox> 。

```csharp
internal void MakeParentItemsControlGotFocus()
```

> [!WARNING]
> 這個方法是內部方法，不適合直接在程式碼中使用。
>
> 在任何情況下，Microsoft 不支援在生產應用程式中使用此方法。
