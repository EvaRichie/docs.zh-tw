## <a name="iehostdll-and-ieexecexe"></a>IEHost.dll 和 IEExec.exe

已從 .NET Framework 中移除 IEHost.dll 和 IEExec.exe 元件。 其所有類型和成員都已經過時，而且在 .NET Framework 4.5 和更新版本中不受支援。 這些組件之前是用來裝載 Windows Form 控制項以及在 Internet Explorer 中執行可執行檔。 這項技術的替代做法包括 ClickOnce、XAML 瀏覽器應用程式 (XBAP) 和 Microsoft Silverlight。
