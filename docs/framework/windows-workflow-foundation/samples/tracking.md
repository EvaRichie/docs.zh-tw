# <a name="tracking"></a>追蹤

本節包含的範例會示範 Windows Workflow Foundation (WF) 中的工作流程追蹤。

## <a name="in-this-section"></a>本節內容

 [自訂追蹤](custom-tracking.md) 示範如何建立自訂追蹤參與者，以及將追蹤資料的內容寫入主控台。

 將[事件追蹤至 Windows 中的事件追蹤](tracking-events-into-event-tracing-in-windows.md)示範如何 [!INCLUDE[wf1](../../../../includes/wf1-md.md)] 在工作流程服務上啟用追蹤，並在 Windows (ETW) 的事件追蹤中發出追蹤事件。

 [SQL 追蹤](sql-tracking.md) 示範如何撰寫自訂的 SQL 追蹤參與者，以將追蹤記錄寫入至 SQL 資料庫。
