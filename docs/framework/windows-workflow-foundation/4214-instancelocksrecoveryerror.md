# <a name="4214---instancelocksrecoveryerror"></a>4214 - InstanceLocksRecoveryError

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|4214|  
|關鍵字|WFInstanceStore|  
|層級|錯誤|  
|通路|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  

 復原執行個體鎖定失敗，因為發生例外狀況。  
  
## <a name="message"></a>訊息  

 復原執行個體鎖定失敗，因為發生下列例外狀況
