# <a name="migration-activity-in-wf"></a>在 WF 中移轉活動

[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] 提供 <xref:System.Activities.Statements.Interop> 用來執行活動的活動，這些活動是根據的工作流程內的活動所衍生 <xref:System.Activities.Activity> 。 如需詳細資訊，請參閱 [遷移指引](migration-guidance.md) 一節。  
  
> [!NOTE]
> <xref:System.Activities.Statements.Interop>除非工作流程的專案將 **目標 Framework** 設定設為 **.net Framework 4** 或更高版本，否則活動不會出現在工作流程設計工具工具箱中。
