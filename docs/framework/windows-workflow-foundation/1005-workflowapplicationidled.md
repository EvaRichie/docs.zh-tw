# <a name="1005---workflowapplicationidled"></a>1005 - WorkflowApplicationIdled

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|1005|  
|關鍵字|WFRuntime|  
|層級|資訊|  
|通路|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  

 表示工作流程應用程式已閒置。

 WorkflowApplication Id：'%1' 閒置。
