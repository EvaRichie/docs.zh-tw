# <a name="connection-string-and-connection-string-name"></a>連接字串與連接字串名稱

**連接字串** 屬性會指定 SQL 工作流程實例存放區用來連接至持續性資料庫的連接字串。 這個參數是選擇性參數。 [**連接字串名稱**] 屬性會指定 SQL 工作流程實例存放區用來連接持續性資料庫的命名連接字串名稱。 這個參數是選擇性參數。 如果您不想讓 SQL 工作流程實例存放區使用預設的命名連接字串 **DefaultSqlWorkflowInstanceStoreConnectionString**，則應該指定 [連接字串名稱] 屬性的值或 [連接字串] 屬性。
