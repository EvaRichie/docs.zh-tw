# <a name="windows-workflow-foundation-data-model"></a>Windows Workflow Foundation 資料模型

Windows Workflow Foundation 資料模型是由三種概念組成：變數、引數和運算式。 變數表示資料的存放區，而引數表示進出活動之資料的流程。 引數會使用可參照變數的運算式來繫結 (指派值)。
