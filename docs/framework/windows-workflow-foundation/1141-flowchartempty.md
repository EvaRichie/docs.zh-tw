# <a name="1141---flowchartempty"></a>1141 - FlowchartEmpty

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|1141|  
|關鍵字|WFActivities|  
|層級|警告|  
|通路|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>描述  

 表示流程圖已執行，但沒有任何節點。  
  
## <a name="message"></a>訊息  

 流程圖 '%1' - 已執行，但沒有任何節點。
