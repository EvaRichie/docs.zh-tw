# <a name="2027---cacherootmetadatastart"></a>2027 - CacheRootMetadataStart

## <a name="properties"></a>屬性  
  
|||  
|-|-|  
|識別碼|2027|  
|關鍵字|WFRuntime|  
|層級|「詳細資訊」|  
|通路|Microsoft-Windows-Application Server-Applications/Debug|
