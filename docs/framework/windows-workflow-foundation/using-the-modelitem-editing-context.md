## <a name="subscribing-to-a-service"></a><span data-ttu-id="0fb86-119">訂閱服務</span><span class="sxs-lookup"><span data-stu-id="0fb86-119">Subscribing to a service</span></span>

```csharp

{  
    if (!subscribed)  
    {  
        this.Context.Services.Subscribe<IMyService>(  
            servInstance =>  
            {  
                listBox1.ItemsSource = servInstance.GetValues(this.ModelItem.Properties["DisplayName"].ComputedValue.ToString());  
            }  
            );  
        subscribed = true;
    }  
}  
```
