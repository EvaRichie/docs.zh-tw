# <a name="how-to-enable-and-disable-automatic-binding-redirection"></a><span data-ttu-id="49427-102">作法：啟用和停用自動繫結重新導向</span><span class="sxs-lookup"><span data-stu-id="49427-102">How to: Enable and Disable Automatic Binding Redirection</span></span>

<span data-ttu-id="49427-103">當您在以 .NET Framework 4.5.1 和更新版本為目標的 Visual Studio 中編譯應用程式時，可能會自動將系結重新導向新增至應用程式佈建檔，以覆寫元件統一。</span><span class="sxs-lookup"><span data-stu-id="49427-103">When you compile apps in Visual Studio that target the .NET Framework 4.5.1 and later versions, binding redirects may be automatically added to the app configuration file to override assembly unification.</span></span> <span data-ttu-id="49427-104">如果您的應用程式或其元件參考相同組件的多個版本，即使您在應用程式的組態檔中手動指定繫結重新導向，仍會加入繫結重新導向。</span><span class="sxs-lookup"><span data-stu-id="49427-104">Binding redirects are added if your app or its components reference more than one version of the same assembly, even if you manually specify binding redirects in the configuration file for your app.</span></span> <span data-ttu-id="49427-105">自動系結重新導向功能會影響以 .NET Framework 4.5.1 或更新版本為目標的桌面應用程式和 web 應用程式，雖然 web 應用程式的行為稍有不同。</span><span class="sxs-lookup"><span data-stu-id="49427-105">The automatic binding redirection feature affects desktop apps and web apps that target the .NET Framework 4.5.1 or a later version, although the behavior is slightly different for a web app.</span></span> <span data-ttu-id="49427-106">如果您現有的應用程式是以舊版 .NET Framework 為目標，您可以啟用自動系結重新導向，如果您想要手動編寫系結重新導向，可以停用這項功能。</span><span class="sxs-lookup"><span data-stu-id="49427-106">You can enable automatic binding redirection if you have existing apps that target previous versions of the .NET Framework, or you can disable this feature if you want to manually author binding redirects.</span></span>

<span data-ttu-id="49427-108">針對以 .NET Framework 4.5.1 和更新版本為目標的 Windows 桌面應用程式，預設會啟用自動系結重新導向。</span><span class="sxs-lookup"><span data-stu-id="49427-108">Automatic binding redirects are enabled by default for Windows desktop apps that target the .NET Framework 4.5.1 and later versions.</span></span> <span data-ttu-id="49427-109">編譯應用程式時，會將系結重新導向新增至輸出設定（**app.config**）檔案，並覆寫可能會發生的元件統一。</span><span class="sxs-lookup"><span data-stu-id="49427-109">The binding redirects are added to the output configuration (**app.config**) file when the app is compiled and override the assembly unification that might otherwise take place.</span></span> <span data-ttu-id="49427-110">未修改來源**app.config**檔案。</span><span class="sxs-lookup"><span data-stu-id="49427-110">The source **app.config** file is not modified.</span></span> <span data-ttu-id="49427-111">您可以藉由修改應用程式的專案檔，或在 Visual Studio 的專案屬性中取消勾選核取方塊，來停用這項功能。</span><span class="sxs-lookup"><span data-stu-id="49427-111">You can disable this feature by modifying the project file for the app or by deselecting a checkbox in the project's properties in Visual Studio.</span></span>
