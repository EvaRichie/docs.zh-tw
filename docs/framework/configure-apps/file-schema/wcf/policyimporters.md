# \<policyImporters>

這個組態項目會指定所有原則匯入工具，這個匯入工具控制匯入有關繫結的自訂原則判斷提示。 每個子項目都是 <的 `policyImporter`>，可用來搜尋有關系結功能的自訂原則判斷提示，以及附加自訂繫結項目，以執行判斷提示所需的功能。
