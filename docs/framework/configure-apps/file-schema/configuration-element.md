# <a name="configuration-element"></a>\<configuration> 項目

通用語言執行平台和 .NET Framework 應用程式所使用之每個組態檔中的根項目。

**\<configuration>**

## <a name="syntax"></a>語法

```xml
<configuration>
  <!-- Configuration settings -->
</configuration>
```

## <a name="attributes"></a>屬性

無

## <a name="parent-element"></a>父元素

無

## <a name="remarks"></a>備註

每個設定檔都必須剛好包含一個 **\<configuration>** 元素。
