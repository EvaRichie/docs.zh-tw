# <a name="itypelibresolver-interface"></a>ITypeLibResolver 介面

提供 [ResolveTypeLib 方法](resolvetypelib-method.md)，此方法會解析類型程式庫的檔案路徑。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[ResolveTypeLib 方法](resolvetypelib-method.md)|傳回類型程式庫的完整路徑，以解析其簡單名稱。|
