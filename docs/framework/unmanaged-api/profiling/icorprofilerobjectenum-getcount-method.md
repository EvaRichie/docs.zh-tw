# <a name="icorprofilerobjectenumgetcount-method"></a>ICorProfilerObjectEnum::GetCount 方法
取得集合中凍結物件的總數。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetCount (  
    [out] ULONG   *pcelt  
);  
```  
  
## <a name="parameters"></a>參數  
 `pcelt`  
 脫銷集合中凍結物件數目的指標。  
  
 在 .NET Framework 版本 3.5 Service Pack 1 （SP1）和更新版本中，這個方法一律會傳回零。
