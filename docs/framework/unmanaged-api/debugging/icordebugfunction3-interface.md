# <a name="icordebugfunction3-interface"></a>ICorDebugFunction3 介面
[.NET Framework 4.5.2 與更新版本提供支援]  
  
 以邏輯方式延伸 ICorDebugFunction 介面，以提供對 ReJIT 要求之程式碼的存取。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetActiveReJitRequestILCode 方法](icordebugfunction3-getactiverejitrequestilcode-method.md)|從使用中的 ReJIT 要求取得包含 IL 的 [ICorDebugILCode](icordebugilcode-interface.md) 介面指標。|
