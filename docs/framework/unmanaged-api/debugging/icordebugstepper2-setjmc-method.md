# <a name="icordebugstepper2setjmc-method"></a>ICorDebugStepper2::SetJMC 方法

設定值，這個值會指定此 ICorDebugStepper 是否只透過應用程式開發人員所撰寫的程式碼進行步驟。 此程式也稱為「我的程式碼 (JMC) 的偵錯工具。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetJMC (  
    [in] BOOL    fIsJMCStepper  
);  
```  
  
## <a name="parameters"></a>參數  

 `fIsJMCStepper`  
 在若設定為 `true` ，則只會透過應用程式開發人員所撰寫的程式碼進行步驟; 否則，請將設定為 `false` 。
