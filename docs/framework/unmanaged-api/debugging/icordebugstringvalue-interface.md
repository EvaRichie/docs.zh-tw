# <a name="icordebugstringvalue-interface"></a>ICorDebugStringValue 介面

## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetLength 方法](icordebugstringvalue-getlength-method.md)|取得這個所參考字串中的字元數 `ICorDebugStringValue` 。|
