# <a name="icordebugobjectenumnext-method"></a>ICorDebugObjectEnum::Next 方法

從目前位置開始，取得列舉中指定物件數目 (Rva) 的相對虛擬位址。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Next (  
    [in] ULONG celt,  
    [out, size_is(celt), length_is(*pceltFetched)]
        CORDB_ADDRESS objects[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>參數  

 `celt`  
 [in] 要擷取的物件數目。  
  
 `objects`  
 擴展指標的陣列，每個指標都會指向 CORDB_ADDRESS 物件。  
  
 `pceltFetched`  
 擴展實際傳回的物件數目指標。 如果是1，則這個值可能是 null `celt` 。
