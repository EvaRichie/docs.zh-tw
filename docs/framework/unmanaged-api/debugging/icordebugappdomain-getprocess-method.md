# <a name="icordebugappdomaingetprocess-method"></a>ICorDebugAppDomain::GetProcess 方法
取得包含應用程式域的進程。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetProcess (  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppProcess`  
 脫銷代表進程之 ICorDebugProcess 物件的位址指標。
