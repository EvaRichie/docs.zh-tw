# <a name="icordebugstepperdeactivate-method"></a>ICorDebugStepper::Deactivate 方法
導致此 ICorDebugStepper 取消它所收到的最後一個步驟命令。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Deactivate ();  
```  
  
## <a name="remarks"></a>備註  
 在最近收到的步驟命令取消之後，可能會發出新的逐步執行命令。
