# <a name="icordebugvaluebreakpointgetvalue-method"></a>ICorDebugValueBreakpoint::GetValue 方法

取得 "ICorDebugValue" 物件的介面指標，該物件表示設定中斷點的物件值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetValue (  
    [out] ICorDebugValue   **ppValue  
);  
```  
  
## <a name="parameters"></a>參數  

 `ppValue`  
 擴展物件位址的指標 `ICorDebugValue` 。
