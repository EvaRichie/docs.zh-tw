# <a name="icordebugcontrollerisrunning-method"></a>ICorDebugController::IsRunning 方法

取得值，這個值表示進程中的執行緒目前是否可自由執行。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```  
  
## <a name="parameters"></a>參數  

 `pbRunning`  
 擴展值的指標， `true` 如果進程中的執行緒可自由執行，則為，否則為 `false` 。
