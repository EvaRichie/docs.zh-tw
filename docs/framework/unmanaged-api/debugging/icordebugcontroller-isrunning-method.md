# <a name="icordebugcontrollerisrunning-method"></a>ICorDebugController::IsRunning 方法
取得值，指出進程中的執行緒目前是否可自由執行。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```  
  
## <a name="parameters"></a>參數  
 `pbRunning`  
 脫銷值的指標，如果進程中`true`的執行緒是自由執行，則為，否則為`false`。
