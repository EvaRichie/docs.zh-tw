# <a name="icordebugevalisactive-method"></a>ICorDebugEval::IsActive 方法

取得值，這個值會指出這個 ICorDebugEval 物件目前是否正在執行。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsActive (  
    [out] BOOL              *pbActive  
);  
```  
  
## <a name="parameters"></a>參數  

 `pbActive`  
 擴展指出此評估是否為作用中之值的指標。
