# <a name="icordebugmodule3-interface"></a>ICorDebugModule3 介面
建立動態模組的符號讀取器。  
  
## <a name="syntax"></a>語法  
  
```cpp  
interface ICorDebugModule3 : IUnknown  
{  
    HRESULT CreateReaderForInMemorySymbols  
      (  
      [in] REFIID riid,  
      [out][iid_is(riid)] void **  ppObj  
      );  
};  
```  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[ICorDebugModule3::CreateReaderForInMemorySymbols 方法](icordebugmodule3-createreaderforinmemorysymbols-method.md)|建立動態模組的符號讀取器（通常是[ISymUnmanagedReader 介面](../diagnostics/isymunmanagedreader-interface.md)）。|  
  
## <a name="remarks"></a>備註  
 這個介面會以邏輯方式擴充 "ICorDebugModule" 和 "ICorDebugModule2" 介面。  
  
> [!NOTE]
> 這個介面不支援跨電腦或跨處理序的遠端呼叫。

 **平台：** 請參閱[系統需求](../../get-started/system-requirements.md)。  
  
 **標頭：** CorDebug.idl、CorDebug.h

 **.NET Framework 版本：** 4.5、4、3.5 SP1
