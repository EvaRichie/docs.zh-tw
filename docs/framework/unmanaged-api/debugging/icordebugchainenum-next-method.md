# <a name="icordebugchainenumnext-method"></a>ICorDebugChainEnum::Next 方法
從列舉中取得指定數目的 ICorDebugChain 實例，從目前位置開始。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Next (  
    [in] ULONG  celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugChain *chains[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>參數  
 `celt`  
 在要抓取的`ICorDebugChain`實例數目。  
  
 `chains`  
 脫銷指標陣列，其中每一個都會指向代表一個鏈`ICorDebugChain`的物件。  
  
 `pceltFetched`  
 脫銷實際傳回之`ICorDebugChain`實例數目的指標。 如果`celt`是一個，這個值可能會是 null。
