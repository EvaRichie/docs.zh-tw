# <a name="icordebugilframe3-interface"></a>ICorDebugILFrame3 介面

提供封裝函式傳回值的方法。 `ICorDebugILFrame3` 是 ICorDebugILFrame 和 ICorDebugILFrame2 介面的邏輯擴充。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetReturnValueForILOffset 方法](icordebugilframe3-getreturnvalueforiloffset-method.md)|取得 ICorDebugValue 物件，該物件會封裝函式的傳回值。|  
  
## <a name="remarks"></a>備註  
  
> [!NOTE]
> 這個介面不支援跨電腦或跨處理序的遠端呼叫。
