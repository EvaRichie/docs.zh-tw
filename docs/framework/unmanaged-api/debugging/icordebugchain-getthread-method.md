# <a name="icordebugchaingetthread-method"></a>ICorDebugChain::GetThread 方法

取得此呼叫鏈所屬的實體執行緒。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetThread (  
    [out] ICorDebugThread    **ppThread  
);  
```  
  
## <a name="parameters"></a>參數  

 `ppThread`  
 擴展ICorDebugThread 物件的指標，該物件代表此呼叫鏈所屬的實體執行緒。
