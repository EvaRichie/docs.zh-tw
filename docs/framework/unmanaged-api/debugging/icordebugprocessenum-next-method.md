# <a name="icordebugprocessenumnext-method"></a>ICorDebugProcessEnum::Next 方法
從列舉中取得指定數目的 ICorDebugProcess 實例，從目前位置開始。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Next (  
    [in]  ULONG celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugProcess *processes[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>參數  
 `celt`  
 在要抓取的 `ICorDebugProcess` 實例數目。  
  
 `processes`  
 脫銷指標陣列，其中每一個 `ICorDebugProcess` 都會指向代表進程的物件。  
  
 `pceltFetched`  
 脫銷實際傳回之實例數目的指標 `ICorDebugProcess` 。 如果是一個，這個值可能會是 null `celt` 。
