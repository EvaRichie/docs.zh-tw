這個方法尚未在目前的 .NET Framework 版本中執行。  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT GetFunctionFromRVA(  
    [in]  CORDB_ADDRESS      rva,  
    [out] ICorDebugFunction  **ppFunction

```
