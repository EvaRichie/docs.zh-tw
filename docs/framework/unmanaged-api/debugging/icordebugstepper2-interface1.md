# <a name="icordebugstepper2-interface"></a>ICorDebugStepper2 介面

僅 (JMC) 的偵錯工具，提供程式碼的支援。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[SetJMC 方法](icordebugstepper2-setjmc-method.md)|設定值，這個值會指定此 ICorDebugStepper 是否只透過應用程式開發人員所撰寫的程式碼進行步驟。|  
  
## <a name="remarks"></a>備註  
  
> [!NOTE]
> 這個介面不支援跨電腦或跨處理序的遠端呼叫。
