# <a name="icordebugboxvaluegetobject-method"></a>ICorDebugBoxValue::GetObject 方法

取得已裝箱的值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetObject (  
    [out] ICorDebugObjectValue **ppObject  
);  
```  
  
## <a name="parameters"></a>參數  

 `ppObject`  
 擴展ICorDebugObjectValue 物件位址的指標，該物件表示已框出的值。
