# <a name="icordebugclassgetmodule-method"></a>ICorDebugClass::GetModule 方法

取得定義這個類別的模組。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetModule (  
    [out] ICorDebugModule    **pModule  
);  
```  
  
## <a name="parameters"></a>參數  

 `pModule`  
 擴展ICorDebugModule 物件位址的指標，該物件表示定義此類別的模組。
