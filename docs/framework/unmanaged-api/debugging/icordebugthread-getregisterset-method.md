# <a name="icordebugthreadgetregisterset-method"></a>ICorDebugThread::GetRegisterSet 方法

取得與這個 ICorDebugThread 物件之使用中部分相關聯之註冊集的介面指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetRegisterSet (  
    [out] ICorDebugRegisterSet **ppRegisters  
);  
```  
  
## <a name="parameters"></a>參數  

 `ppRegisters`  
 擴展 [ICorDebugRegisterSet](icordebugregisterset-interface.md) 介面物件位址的指標，該物件代表此執行緒之使用中部分的暫存器集。
