# <a name="icordebugappdomainattach-method"></a>ICorDebugAppDomain::Attach 方法
將偵錯工具附加至應用程式域。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Attach ();  
```  
  
## <a name="remarks"></a>備註  
 偵錯工具必須附加至應用程式域，才能接收事件和啟用應用程式域的偵測。
