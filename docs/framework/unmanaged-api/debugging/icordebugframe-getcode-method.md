# <a name="icordebugframegetcode-method"></a>ICorDebugFrame::GetCode 方法
取得與這個堆疊框架相關聯之程式碼的指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetCode (  
    [out] ICorDebugCode      **ppCode  
);  
```  
  
## <a name="parameters"></a>參數  
 `ppCode`  
 脫銷ICorDebugCode 物件位址的指標，表示與此框架相關聯的程式碼。
