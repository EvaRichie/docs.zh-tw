# <a name="icordebugarrayvaluegetcount-method"></a>ICorDebugArrayValue::GetCount 方法

取得陣列中的元素總數。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetCount (  
    [out] ULONG32 *pnCount  
);  
```  
  
## <a name="parameters"></a>參數  

 `pnCount`  
 擴展陣列中元素總數的指標。
