# <a name="icordebugarrayvaluegetrank-method"></a>ICorDebugArrayValue::GetRank 方法

取得陣列的維度數目。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>參數  

 `pnRank`  
 擴展此物件中之維度數目的指標 `ICorDebugArrayValue` 。
