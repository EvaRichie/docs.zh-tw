# <a name="icordebugreferencevaluegetvalue-method"></a>ICorDebugReferenceValue::GetValue 方法
取得受參考物件目前的記憶體位址。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetValue (  
    [out] CORDB_ADDRESS   *pValue  
);  
```  
  
## <a name="parameters"></a>參數  
 `pValue`  
 脫銷值的指標 `CORDB_ADDRESS` ，指定這個 ICorDebugReferenceValue 物件所指向之物件的位址。
