# <a name="icordebugthreadenum-interface"></a>ICorDebugThreadEnum 介面
會執行 ICorDebugEnum 方法並列舉 ICorDebugThread 陣列。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[下一個方法](icordebugthreadenum-next-method.md)|`ICorDebugThread`從列舉中取得指定的實例數目，從目前位置開始。|
