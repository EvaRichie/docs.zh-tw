# <a name="icorpublishprocessgetprocessid-method"></a>ICorPublishProcess::GetProcessID 方法

取得此進程的作業系統識別碼。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetProcessID (  
    [out] unsigned   *pid  
);  
```  
  
## <a name="parameters"></a>參數  

 `pid`  
 擴展這個 [ICorPublishProcess](icorpublishprocess-interface.md) 物件所表示之進程的識別碼指標。
