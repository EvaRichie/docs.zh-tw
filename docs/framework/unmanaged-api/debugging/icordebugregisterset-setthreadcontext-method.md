# <a name="icordebugregistersetsetthreadcontext-method"></a>ICorDebugRegisterSet::SetThreadContext 方法

`SetThreadContext` 不是在 .NET Framework 版本2.0 中執行。 請不要呼叫此方法。  
  
> [!NOTE]
> 使用較高層級的 operation [ICorDebugNativeFrame：： SetIP](icordebugnativeframe-setip-method.md) 來設定執行緒的內容。  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT SetThreadContext (  
    [in] ULONG32 contextSize,  
    [in, length_is(contextSize),  
         size_is(contextSize)] BYTE context[]  
);  
```

 **.NET Framework 版本：** 1.1、1。0
