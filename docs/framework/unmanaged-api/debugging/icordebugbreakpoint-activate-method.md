# <a name="icordebugbreakpointactivate-method"></a>ICorDebugBreakpoint::Activate 方法
設定這個`ICorDebugBreakpoint`的作用中狀態。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Activate (  
    [in] BOOL bActive  
);  
```  
  
## <a name="parameters"></a>參數  
 `bActive`  
 在將此值設定`true`為，以將狀態指定為作用中;否則，請將此值`false`設定為。
