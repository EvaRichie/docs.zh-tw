# <a name="icordebugbreakpointenumnext-method"></a>ICorDebugBreakpointEnum::Next 方法

從目前位置開始，取得列舉中指定的 ICorDebugBreakpoint 實例數目。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Next (  
    [in] ULONG  celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugBreakpoint *breakpoints[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>參數  

 `celt`  
 在 `ICorDebugBreakpoint` 要取出的實例數目。  
  
 `breakpoints`  
 擴展指標的陣列，每個指標 `ICorDebugBreakpoint` 都會指向代表中斷點的物件。  
  
 `pceltFetched`  
 擴展實際傳回之實例數目的指標 `ICorDebugBreakpoint` 。 如果是1，則這個值可能是 null `celt` 。
