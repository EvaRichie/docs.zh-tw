# <a name="icordebugprocess5gettypefortypeid-method"></a>ICorDebugProcess5::GetTypeForTypeID 方法
將類型識別碼轉換成 ICorDebugType 值。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetTypeForTypeID(  
    [in] COR_TYPEID id, [  
    out] ICorDebugType **ppType  
);  
```  
  
## <a name="parameters"></a>參數  
 `id`  
 在類型識別碼。  
  
 `ppType`  
 脫銷ICorDebugType 物件位址的指標。  
  
## <a name="remarks"></a>備註  
 在某些情況下，傳回類型識別碼的方法可能會傳回 null `COR_TYPEID` 值。 如果將這個值當做 `id` 引數傳遞，此 `GetTypeForTypeID` 方法將會失敗並傳回 `E_FAIL` 。
