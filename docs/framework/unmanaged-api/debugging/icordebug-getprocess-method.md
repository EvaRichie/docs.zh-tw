# <a name="icordebuggetprocess-method"></a>ICorDebug::GetProcess 方法
取得指定進程之 "ICorDebugProcess" 實例的指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetProcess (  
    [in] DWORD               dwProcessId,  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
## <a name="parameters"></a>參數  
 `dwProcessId`  
 在進程的識別碼。  
  
 `ppProcess`  
 脫銷指定進程之`ICorDebugProcess`實例位址的指標。
