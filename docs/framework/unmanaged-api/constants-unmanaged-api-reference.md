# <a name="constants-unmanaged-api-reference"></a>常數 (Unmanaged API 參考)

本主題說明 CorSym 中定義的語言類型、語言廠商和檔案類型常數。  
  
## <a name="language-type-constants"></a>語言類型常數  

 下表顯示語言類型常數，表示可識別程式設計語言的 Guid。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_LanguageType_C|表示 C 語言。|  
|CorSym_LanguageType_CPlusPlus|指出 c + + 語言。|  
|CorSym_LanguageType_CSharp|表示 c # 語言。|  
|CorSym_LanguageType_Basic|表示基礎語言。|  
|CorSym_LanguageType_JAVA|表示 JAVA 語言。|  
|CorSym_LanguageType_Cobol|表示 COBOL 語言。|  
|CorSym_LanguageType_Pascal|表示 Pascal 語言。|  
|CorSym_LanguageType_ILAssembly|指出 Microsoft 中繼語言 (MSIL) 元件程式碼。|  
|CorSym_LanguageType_JScript|指出 JScript 語言。|  
|CorSym_LanguageType_SMC|指出 SMC 語言。|  
|CorSym_LanguageType_MCPlusPlus|指出針對 .NET Framework 啟用的 c + + 語言。|  
  
## <a name="language-vendor-constants"></a>語言廠商常數  

 下表顯示語言廠商常數，表示可識別程式設計語言廠商的 Guid。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_LanguageVendor_Microsoft|表示 Microsoft。|  
  
## <a name="document-type-constants"></a>檔案類型常數  

 下表顯示檔案類型常數，表示可識別檔案類型的 Guid。  
  
|符號|描述|  
|------------|-----------------|  
|CorSym_DocumentType_Text|表示文字檔。|  
|CorSym_DocumentType_MC|表示非文字檔。|
