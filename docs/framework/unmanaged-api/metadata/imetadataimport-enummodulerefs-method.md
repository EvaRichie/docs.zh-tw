# <a name="imetadataimportenummodulerefs-method"></a>IMetaDataImport::EnumModuleRefs 方法

列舉代表已匯入的模組之 ModuleRef 語彙基元。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT EnumModuleRefs (  
   [in, out] HCORENUM     *phEnum,  
   [out]     mdModuleRef  rModuleRefs[],  
   [in]      ULONG        cMax,  
   [out]     ULONG        *pcModuleRefs  
);  
```  
  
## <a name="parameters"></a>參數  

 `phEnum`  
 [in，out]列舉值的指標。 此方法的第一個呼叫必須是 Null。  
  
 `rModuleRefs`  
 擴展用來儲存 ModuleRef 標記的陣列。  
  
 `cMax`  
 [in] `rModuleRefs` 陣列的大小上限。  
  
 `pcModuleRefs`  
 擴展傳回的 ModuleRef 權杖數目 `rModuleRefs` 。  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|`S_OK`|`EnumModuleRefs` 傳回成功。|  
|`S_FALSE`|沒有要列舉的權杖。 在此情況下， `pcModuleRefs` 是零。|
