# <a name="imetadataimportisglobal-method"></a>IMetaDataImport::IsGlobal 方法
取得一個值，用來表示指定中繼資料語彙基元所代表的欄位、方法或類型值是否具有全域範圍。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsGlobal (  
   [in]  mdToken     pd,  
   [out] int         *pbGlobal  
);  
```  
  
## <a name="parameters"></a>參數  
 `pd`  
 在代表類型、欄位或方法的元資料標記。  
  
 `pbGlobal`  
 [out] 1，如果物件具有全域範圍，則為，否則為0（零）。
