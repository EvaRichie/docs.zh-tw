# <a name="imetadataemitsave-method"></a>IMetaDataEmit::Save 方法

將目前範圍中的所有中繼資料儲存至檔案的指定位址。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Save (
    [in]  LPCWSTR     szFile,
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>參數  

 `wzFile`  
 在要儲存的檔案名。 如果這個值為 null，記憶體中的複本將會儲存到最後使用的位置。  
  
 `dwSaveFlags`  
 [in] 保留。 必須為零。
