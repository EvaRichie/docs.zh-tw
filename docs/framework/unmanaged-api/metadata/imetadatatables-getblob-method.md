# <a name="imetadatatablesgetblob-method"></a>IMetaDataTables::GetBlob 方法

取得在指定的資料行索引上 (BLOB) 之二進位大型物件的指標。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetBlob (  
    [in]  ULONG          ixBlob,  
    [out] ULONG          *pcbData,  
    [out] const void     **ppData  
);  
```  
  
## <a name="parameters"></a>參數  

 `ixBlob`  
 在要從中取得的記憶體位址 `ppData` 。  
  
 `pcbData`  
 擴展大小的指標，以位元組為單位 `ppData` 。  
  
 `ppData`  
 擴展已抓取之二進位資料指標的指標。
