# <a name="imetadataimportgetuserstring-method"></a>IMetaDataImport::GetUserString 方法

取得指定中繼資料語彙基元所代表的常值字串。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetUserString (  
   [in]   mdString    stk,  
   [out]  LPWSTR      szString,  
   [in]   ULONG       cchString,  
   [out]  ULONG       *pchString  
);  
```  
  
## <a name="parameters"></a>參數  

 `stk`  
 在要傳回相關聯之字串的字串標記。  
  
 `szString`  
 擴展要求的字串複本。  
  
 `cchString`  
 在要求的大小上限（以寬字元為單位） `szString` 。  
  
 `pchString`  
 擴展傳回之的大小（以寬字元為單位） `szString` 。
