# <a name="imetadataemitsavetostream-method"></a>IMetaDataEmit::SaveToStream 方法

將目前範圍中的所有中繼資料儲存至指定的 `IStream` 。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SaveToStream (
    [in]  IStream     *pIStream,  
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>參數  

 `pIStream`  
 在要儲存到其中的可寫入資料流程。  
  
 `dwSaveFlags`  
 [in] 保留。 必須為零。
