# <a name="imetadataerroronerror-method"></a>IMetaDataError::OnError 方法
提供中繼資料合併期間發生之錯誤的通知。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT OnError (  
    [in] HRESULT   hrError,
    [in] mdToken   token  
);  
```  
  
## <a name="parameters"></a>參數  
 `hrError`  
 在傳回給呼叫方法的 HRESULT 錯誤值。  
  
 `token`  
 在發生錯誤時，所合併之程式碼物件的元資料標記。
