# <a name="iceegengetimaptokeniface-method"></a>ICeeGen::GetIMapTokenIface 方法
取得指定之標記所參考的介面。  
  
 這個方法已過時，不應使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetIMapTokenIface (  
    [in, out] IUnknown   **pIMapToken  
);  
```  
  
## <a name="parameters"></a>參數  
 `pIMapToken`  
 [in、out]要傳回之介面的元資料標記。
