# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken 方法

從目前的中繼資料範圍中刪除指定的標記。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT DeleteToken (
    [in]  mdToken     tkObj
);  
```  
  
## <a name="parameters"></a>參數  

 `tkObj`  
 在要刪除的標記。
