# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString 方法

取得指定之常值字串的元資料標記。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT DefineUserString (
    [in]  LPCWSTR     szString,
    [in]  ULONG       cchString,
    [out] mdString    *pstk
);  
```  
  
## <a name="parameters"></a>參數  

 `szString`  
 在要儲存的使用者字串。  
  
 `cchString`  
 在中寬字元的計數 `szString` 。  
  
 `pstk`  
 擴展指派的字串標記。
