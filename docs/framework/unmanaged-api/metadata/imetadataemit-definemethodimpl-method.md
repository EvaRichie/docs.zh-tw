# <a name="imetadataemitdefinemethodimpl-method"></a>IMetaDataEmit::DefineMethodImpl 方法

建立用來執行繼承自介面之方法的定義，並將標記傳回至該方法執行定義。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT DefineMethodImpl (
    [in]  mdTypeDef         td,
    [in]  mdToken           tkBody,
    [in]  mdToken           tkDecl  
);  
```  
  
## <a name="parameters"></a>參數  

 `td`  
 在 `mdTypedef` 執行類別的 token。  
  
 `tkBody`  
 在程式 `mdMethodDef` `mdMemberRef` 代碼主體的或標記。  
  
 `tkDecl`  
 在 `mdMethodDef` `mdMemberRef` 正在執行之介面方法的或標記。
