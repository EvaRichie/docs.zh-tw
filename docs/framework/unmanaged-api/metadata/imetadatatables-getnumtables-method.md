# <a name="imetadatatablesgetnumtables-method"></a>IMetaDataTables::GetNumTables 方法
取得目前實例範圍中的資料表數目 `IMetaDataTables` 。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetNumTables (  
    [out]  ULONG   *pcTables  
);  
```  
  
## <a name="parameters"></a>參數  
 `pcTables`  
 脫銷目前實例範圍中的資料表數目指標。
