# <a name="ceesectionrelocextra-union"></a>CeeSectionRelocExtra 等位
表示[ICeeGen](iceegen-interface.md)介面用來重新放置區段的位址位移。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef union  {  
    USHORT highAdj;  
} CeeSectionRelocExtra;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`highAdj`|區段的上方位址調整。|
