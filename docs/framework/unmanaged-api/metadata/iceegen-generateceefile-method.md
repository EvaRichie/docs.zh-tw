# <a name="iceegengenerateceefile-method"></a>ICeeGen::GenerateCeeFile 方法

產生程式碼基底檔案，其中包含目前載入至此 [ICeeGen](iceegen-interface.md) 介面的程式碼基底。  
  
 這個方法已經過時，不應該使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GenerateCeeFile ();  
```  
  
## <a name="parameters"></a>參數  

 `GenerateCeeFile` 不接受任何參數。
