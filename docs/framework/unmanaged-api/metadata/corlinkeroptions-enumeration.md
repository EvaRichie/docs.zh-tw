# <a name="corlinkeroptions-enumeration"></a>CorLinkerOptions 列舉

指定旗標，以選取中繼資料連結器的選項。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum CorLinkerOptions {  
    MDAssembly          = 0x00000000,  
    MDNetModule         = 0x00000001,  
} CorLinkerOptions;  
```  
  
## <a name="members"></a>成員  
  
|member|描述|  
|------------|-----------------|  
|`MDAssembly`|私用類型和全域函式並不會保留。|  
|`MDNetModule`|私用類型和全域函式會保留下來。|
