# <a name="cvstruct-structure"></a>CVStruct 結構

包含在安裝模組或複合映像時，所使用的資訊。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```  
  
## <a name="members"></a>成員  
  
|member|描述|  
|------------|-----------------|  
|主要|主要版本組建編號。|  
|Minor|次要版本組建編號。|  
|Sub|子組建編號。|  
|組建|組建編號。|
