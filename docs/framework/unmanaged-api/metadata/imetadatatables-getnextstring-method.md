# <a name="imetadatatablesgetnextstring-method"></a>IMetaDataTables::GetNextString 方法

取得目前資料表資料行中下一個字串的索引。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetNextString (
    [in]  ULONG   ixString,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>參數  

 `ixString`  
 在字串資料表資料行中的索引值。  
  
 `pNext`  
 擴展資料行中下一個字串的索引指標。
