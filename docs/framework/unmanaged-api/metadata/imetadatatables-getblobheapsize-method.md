# <a name="imetadatatablesgetblobheapsize-method"></a>IMetaDataTables::GetBlobHeapSize 方法

取得二進位大型物件 (BLOB) 堆積的大小（以位元組為單位）。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetBlobHeapSize (  
    [out] ULONG     *pcbBlobs  
);
```  
  
## <a name="parameters"></a>參數  

 `pcbBlobs`  
 擴展BLOB 堆積大小的指標，以位元組為單位。
