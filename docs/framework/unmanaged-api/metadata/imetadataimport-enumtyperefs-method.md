# <a name="imetadataimportenumtyperefs-method"></a>IMetaDataImport::EnumTypeRefs 方法

列舉在目前中繼資料範圍中定義的 TypeRef 語彙基元。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT EnumTypeRefs (  
   [in, out] HCORENUM    *phEnum,
   [out] mdTypeRef       rTypeRefs[],  
   [in]  ULONG           cMax,
   [out] ULONG           *pcTypeRefs  
);  
```  
  
## <a name="parameters"></a>參數  

 `phEnum`  
 [in，out]列舉值的指標。 此方法的第一個呼叫必須是 Null。  
  
 `rTypeRefs`  
 擴展用來儲存 TypeRef 標記的陣列。  
  
 `cMax`  
 [in] `rTypeRefs` 陣列的大小上限。  
  
 `pcTypeRefs`  
 擴展傳回之 TypeRef token 數目的指標 `rTypeRefs` 。  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|`S_OK`|`EnumTypeRefs` 傳回成功。|  
|`S_FALSE`|沒有要列舉的權杖。 在此情況下， `pcTypeRefs` 是零。|  
  
## <a name="remarks"></a>備註  

 TypeRef 標記代表類型的參考。
