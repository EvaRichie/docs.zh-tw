# <a name="iceegentruncatesection-method"></a>ICeeGen::TruncateSection 方法

依指定的長度截斷指定的程式碼區段。  
  
 這個方法已經過時，不應該使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT TruncateSection (  
    [in]  HCEESECTION     section,  
    [in]  ULONG           len  
);  
```  
  
## <a name="parameters"></a>參數  

 `section`  
 在要截斷的區段。  
  
 `len`  
 在要截斷區段的長度（以位元組為單位）。  
  
## <a name="remarks"></a>備註  

 `TruncateSection`只有當您有不是由其他方法處理的特殊區段需求時，才需要呼叫。
