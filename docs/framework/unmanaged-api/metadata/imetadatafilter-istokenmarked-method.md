# <a name="imetadatafilteristokenmarked-method"></a>IMetaDataFilter::IsTokenMarked 方法
取得值，指出指定的元資料標記是否已標示為已處理。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsTokenMarked (  
    [in]  mdToken  tk,
    [out] BOOL     *pIsMarked  
);  
```  
  
## <a name="parameters"></a>參數  
 `tk`  
 在要檢查處理標記的 token。  
  
 `pIsMarked`  
 脫銷如果已處理，則值為， `true` `tk` 否則為 `false` 。
