# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA 方法

設定指定方法的相對虛擬位址。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,
    [in]  ULONG        ulRVA
);  
```  
  
## <a name="parameters"></a>參數  

 `md`  
 在目標方法或方法執行的標記。  
  
 `ulRVA`  
 在程式碼或資料區域的位址。
