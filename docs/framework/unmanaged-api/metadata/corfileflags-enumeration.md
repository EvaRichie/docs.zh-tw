# <a name="corfileflags-enumeration"></a>CorFileFlags 列舉

包含值，這些值會描述在呼叫 [IMetaDataAssemblyEmit：:D efinefile](imetadataassemblyemit-definefile-method.md)中定義的檔案類型。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum CorFileFlags {  
  
    ffContainsMetaData      =   0x0000,  
    ffContainsNoMetaData    =   0x0001  
  
} CorFileFlags;  
```  
  
## <a name="members"></a>成員  
  
|member|描述|  
|------------|-----------------|  
|`ffContainsMetaData`|指出檔案不是資源檔。|  
|`ffContainsNoMetaData`|指出檔案（可能為資源檔）不包含中繼資料。|
