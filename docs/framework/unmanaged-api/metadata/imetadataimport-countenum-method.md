# <a name="imetadataimportcountenum-method"></a>IMetaDataImport::CountEnum 方法
取得列舉中指定的列舉值所抓取的元素數目。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT CountEnum (  
   [in]  HCORENUM    hEnum,
   [out] ULONG       *pulCount  
);  
```  
  
## <a name="parameters"></a>參數  
 `hEnum`  
 在列舉值的控制碼。  
  
 `pulCount`  
 脫銷列舉的元素數目。  
  
## <a name="remarks"></a>備註  
 所指定的控制碼 `hEnum` 是從先前的 `Enum` *名稱*呼叫取得（例如， [IMetaDataImport：： EnumTypeDefs](imetadataimport-enumtypedefs-method.md)）。
