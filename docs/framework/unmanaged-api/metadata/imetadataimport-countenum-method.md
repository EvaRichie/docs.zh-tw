# <a name="imetadataimportcountenum-method"></a>IMetaDataImport::CountEnum 方法

取得列舉中由指定列舉值抓取的元素數目。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT CountEnum (  
   [in]  HCORENUM    hEnum,
   [out] ULONG       *pulCount  
);  
```  
  
## <a name="parameters"></a>參數  

 `hEnum`  
 在列舉值的控制碼。  
  
 `pulCount`  
 擴展列舉的元素數目。  
  
## <a name="remarks"></a>備註  

 指定的控制碼 `hEnum` 是從先前的 `Enum` *名稱* 呼叫取得 (例如， [IMetaDataImport：： EnumTypeDefs](imetadataimport-enumtypedefs-method.md)) 。
