# <a name="imetadataimportgetmodulefromscope-method"></a>IMetaDataImport::GetModuleFromScope 方法

取得目前中繼資料範圍中所參考之模組的元資料標記。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetModuleFromScope (  
   [out] mdModule    *pmd  
);  
```  
  
## <a name="parameters"></a>參數  

 `pmd`  
 擴展Token 的指標，代表目前中繼資料範圍中所參考的模組。
