# <a name="imetadatatablesgetstring-method"></a>IMetaDataTables::GetString 方法
從目前參考範圍的資料表資料行中，取得位於指定索引位置的字串。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetString (
    [in]  ULONG       ixString,  
    [out] const char  **ppString  
);  
```  
  
## <a name="parameters"></a>參數  
 `ixString`  
 在要開始搜尋下一個值的索引。  
  
 `ppString`  
 脫銷傳回之字串值指標的指標。
