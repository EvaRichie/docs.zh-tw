# <a name="iceegenallocatemethodbuffer-method"></a>ICeeGen::AllocateMethodBuffer 方法

針對方法建立指定大小的緩衝區，並取得方法的相對虛擬位址。  
  
 這個方法已經過時，不應該使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT AllocateMethodBuffer (
    [in]  ULONG    cchBuffer,
    [out] UCHAR    **lpBuffer,  
    [out] ULONG    *RVA  
);  
```  
  
## <a name="parameters"></a>參數  

 `cchBuffer`  
 在要建立之緩衝區的長度。  
  
 `lpBuffer`  
 擴展傳回的緩衝區。  
  
 `RVA`  
 擴展方法的相對虛擬位址。
