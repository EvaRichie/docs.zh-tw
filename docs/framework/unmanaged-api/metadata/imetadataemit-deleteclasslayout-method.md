# <a name="imetadataemitdeleteclasslayout-method"></a>IMetaDataEmit::DeleteClassLayout 方法

終結指定標記所表示之型別的類別配置中繼資料簽章。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT DeleteClassLayout (  
    [in]  mdTypeDef   td  
);  
```  
  
## <a name="parameters"></a>參數  

 `td`  
 在 `mdTypeDef` 元資料標記，代表將刪除其類別配置的型別。
