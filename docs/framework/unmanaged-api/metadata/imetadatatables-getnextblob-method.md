# <a name="imetadatatablesgetnextblob-method"></a>IMetaDataTables::GetNextBlob 方法

取得資料表中下一個二進位大型物件 (BLOB) 的索引。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetNextBlob (  
    [in]  ULONG   ixBlob,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>參數  

 `ixBlob`  
 在從 Blob 的資料行傳回的索引。  
  
 `pNext`  
 擴展下一個 BLOB 的索引指標。
