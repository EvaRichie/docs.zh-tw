# <a name="imetadataemit2savedeltatomemory-method"></a>IMetaDataEmit2::SaveDeltaToMemory 方法

將目前的編輯後繼續會話的變更儲存至記憶體。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SaveDeltaToMemory (  
    [out] void        *pbData,
    [in]  ULONG       cbData  
);  
```  
  
## <a name="parameters"></a>參數  

 `pbData`  
 擴展開始寫入中繼資料差異的位址。  
  
 `cbData`  
 在變更的大小。 使用 [IMetaDataEmit2：： GetDeltaSaveSize](imetadataemit2-getdeltasavesize-method.md) 來判斷大小。
