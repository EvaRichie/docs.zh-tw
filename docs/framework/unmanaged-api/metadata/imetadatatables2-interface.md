# <a name="imetadatatables2-interface"></a>IMetaDataTables2 介面

擴充 [IMetaDataTables](imetadatatables-interface.md) ，以包含處理中繼資料資料流程的方法。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetMetaDataStorage 方法](imetadatatables2-getmetadatastorage-method.md)|取得儲存在指定區段中之中繼資料的大小和內容。|  
|[GetMetaDataStreamInfo 方法](imetadatatables2-getmetadatastreaminfo-method.md)|取得位於指定索引處之中繼資料資料流程的名稱、大小和內容。|
