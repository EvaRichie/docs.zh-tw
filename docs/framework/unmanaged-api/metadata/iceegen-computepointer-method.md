# <a name="iceegencomputepointer-method"></a>ICeeGen::ComputePointer 方法

判斷指定程式碼區段的緩衝區。  
  
 這個方法已經過時，不應該使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT ComputePointer (  
    [in]  HCEESECTION  section,  
    [in]  ULONG        RVA,
    [out] UCHAR        **lpBuffer  
);  
```  
  
## <a name="parameters"></a>參數  

 `section`  
 在要傳回緩衝區的程式碼區段。  
  
 `RVA`  
 在取得指標之方法的相對虛擬位址。  
  
 `lpBuffer`  
 擴展傳回之緩衝區的指標。
