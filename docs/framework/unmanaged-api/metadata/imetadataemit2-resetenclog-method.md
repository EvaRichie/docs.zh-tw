# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog 方法

重設編輯後繼續記錄，並啟動新的會話。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>參數  

 `ResetENCLog` 不接受任何參數。
