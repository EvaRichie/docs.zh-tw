# <a name="iceegengetstring-method"></a>ICeeGen::GetString 方法

取得儲存在指定相對虛擬位址的字串。  
  
 這個方法已經過時，不應該使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetString (  
    [in]  ULONG      RVA,
    [out] LPWSTR     *lpString  
);  
```  
  
## <a name="parameters"></a>參數  

 `RVA`  
 在要傳回之字串的相對虛擬位址。  
  
 `lpString`  
 擴展傳回的字串。
