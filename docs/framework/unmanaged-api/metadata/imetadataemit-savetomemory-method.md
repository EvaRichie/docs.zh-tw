# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory 方法
將目前範圍中的所有中繼資料儲存至指定的記憶體區域。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SaveToMemory (
    [out]  void        *pbData,
    [in]   ULONG       cbData
);  
```  
  
## <a name="parameters"></a>參數  
 `pbData`  
 脫銷要開始寫入中繼資料的位址。  
  
 `cbData`  
 在配置的記憶體大小（以位元組為單位）。
