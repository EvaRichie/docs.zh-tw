# <a name="imetadataimportenummethodimpls-method"></a>IMetaDataImport::EnumMethodImpls 方法

列舉代表指定類型方法的 MethodBody 和 MethodDeclaration 語彙基元。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT EnumMethodImpls (  
   [in, out] HCORENUM    *phEnum,
   [in]      mdTypeDef   td,
   [out]     mdToken     rMethodBody[],
   [out]     mdToken     rMethodDecl[],
   [in]      ULONG       cMax,
   [in]      ULONG       *pcTokens  
);  
```  
  
## <a name="parameters"></a>參數  

 `phEnum`  
 [in，out]列舉值的指標。 此方法的第一個呼叫必須是 Null。  
  
 `td`  
 在要列舉其方法的型別之 TypeDef 標記。  
  
 `rMethodBody`  
 擴展要儲存 MethodBody 標記的陣列。  
  
 `rMethodDecl`  
 擴展要儲存 MethodDeclaration 標記的陣列。  
  
 `cMax`  
 在和陣列的大小上限 `rMethodBody` `rMethodDecl` 。  
  
 `pcTokens`  
 在和中傳回的實際方法數目 `rMethodBody` `rMethodDecl` 。  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|`S_OK`|`EnumMethodImpls` 傳回成功。|  
|`S_FALSE`|沒有要列舉的方法標記。 在此情況下， `pcTokens` 是零。|
