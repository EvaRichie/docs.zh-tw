# <a name="cor_native_link-structure"></a>COR_NATIVE_LINK 結構
包含用來連結原生程式碼的資訊。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef struct
{  
    BYTE        m_linkType;  
    BYTE        m_flags;  
    mdMemberRef m_entryPoint;  
} COR_NATIVE_LINK;  
```  
  
## <a name="members"></a>成員  
  
|成員|描述|  
|------------|-----------------|  
|`m_linkType`|要以機器碼連結的類型。 這個值是其中一個[CorNativeLinkType](cornativelinktype-enumeration.md)值。|  
|`m_flags`|連結器在連結機器碼時所使用的旗標。 這個值是其中一個[CorNativeLinkFlags](cornativelinkflags-enumeration.md)值。|  
|`m_entryPoint`|表示進入點的 MemberRef 元資料標記。 格式為 `lib:entrypoint`。|
