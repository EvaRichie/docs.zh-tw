# <a name="imetadataimportgetnestedclassprops-method"></a>IMetaDataImport::GetNestedClassProps 方法
取得 <xref:System.Type> 指定之嵌套型別之父系的 TypeDef token。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetNestedClassProps (  
   [in]   mdTypeDef      tdNestedClass,  
   [out]  mdTypeDef      *ptdEnclosingClass  
);  
```  
  
## <a name="parameters"></a>參數  
 `tdNestedClass`  
 在TypeDef token，代表要傳回之 <xref:System.Type> 父類別 token 的。  
  
 `ptdEnclosingClass`  
 脫銷所連結之的 TypeDef token 的指標 <xref:System.Type> `tdNestedClass` 。
