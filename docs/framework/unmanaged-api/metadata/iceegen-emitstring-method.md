# <a name="iceegenemitstring-method"></a>ICeeGen::EmitString 方法

將指定的字串發出至程式碼基底。  
  
 這個方法已經過時，不應該使用。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT EmitString (  
    [in]  LPWSTR    lpString,  
    [out] ULONG     *RVA  
);  
```  
  
## <a name="parameters"></a>參數  

 `lpString`  
 在要發出的字串。  
  
 `RVA`  
 擴展發出之字串的相對虛擬位址。
