# <a name="iassemblyenum-interface"></a>IAssemblyEnum 介面

表示物件陣列的列舉值 `IAssemblyName` 。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Clone 方法](iassemblyenum-clone-method.md)|建立這個物件的淺層複本 `IAssemblyEnum` 。|  
|[GetNextAssembly 方法](iassemblyenum-getnextassembly-method.md)|取得 `IAssemblyName` 包含在此物件中下一個的指標 `IAssemblyEnum` 。|  
|[Reset 方法](iassemblyenum-reset-method.md)|將此 `IAssemblyEnum` 物件重設為其開始位置。|
