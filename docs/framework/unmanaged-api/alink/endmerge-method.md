# <a name="endmerge-method"></a>EndMerge 方法

指出所有自訂屬性都已合併到發出範圍中。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT EndMerge(  
    mdAssembly AssemblyID  
) PURE;  
```  
  
## <a name="parameters"></a>參數  

 `AssemblyID`  
 元件的識別碼。  
  
## <a name="return-value"></a>傳回值  

 如果方法成功，則傳回 S_OK。  
  
## <a name="requirements"></a>需求  

 需要 alink。h
