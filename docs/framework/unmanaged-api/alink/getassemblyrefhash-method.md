# <a name="getassemblyrefhash-method"></a>GetAssemblyRefHash 方法

抓取指定元件的雜湊 blob。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetAssemblyRefHash(  
    mdToken FileToken,  
    const void** ppvHash,  
    DWORD* pcbHash  
) PURE;  
```  
  
## <a name="parameters"></a>參數  

 `FileToken`  
 雜湊將參考的元件識別碼。  
  
 `ppvHash`  
 接收產生的雜湊 blob。  
  
 `pcbHash`  
 接收雜湊 blob 的大小（以位元組為單位）。  
  
## <a name="return-value"></a>傳回值  

 如果方法成功，則傳回 S_OK。  
  
## <a name="requirements"></a>需求  

 需要 alink。h
