# <a name="iclrreferenceassemblyenum-interface"></a>ICLRReferenceAssemblyEnum 介面

提供的方法可讓主機使用 common language runtime (CLR) 內部的元件身分識別資料，來操作檔案或資料流程所參考的元件集合，而不需要建立或瞭解這些身分識別。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Get 方法](iclrreferenceassemblyenum-get-method.md)|取得提供的索引處的元件識別。|
