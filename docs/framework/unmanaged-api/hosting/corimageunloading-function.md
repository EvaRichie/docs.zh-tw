# <a name="_corimageunloading-function"></a>_CorImageUnloading 函式
卸載受管理的模組映射時，通知載入器。  
  
 此函數並未實作。 如果呼叫，它會傳回 E_NOTIMPL。  
  
## <a name="syntax"></a>語法  
  
```cpp  
STDAPI (VOID) _CorImageUnloading(
   [in] PVOID* ImageBase  
);  
```  
  
## <a name="parameters"></a>參數  
 `ImageBase`  
 在要卸載之影像起始位置的指標。
