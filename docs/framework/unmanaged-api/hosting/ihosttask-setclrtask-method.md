# <a name="ihosttasksetclrtask-method"></a>IHostTask::SetCLRTask 方法
將 `ICLRTask` 實例與目前的[IHostTask](ihosttask-interface.md)實例產生關聯。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT SetCLRTask (  
    [in] ICLRTask *pCLRTask  
);  
```  
  
## <a name="parameters"></a>參數  
 `pCLRTask`  
 在要 `ICLRTask` 與目前實例相關聯之實例的介面指標 `IHostTask` 。  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|S_OK|`SetCLRTask`已成功傳回。|  
|HOST_E_CLRNOTAVAILABLE|Common language runtime （CLR）尚未載入進程中，或 CLR 處於無法執行 managed 程式碼或成功處理呼叫的狀態。|  
|HOST_E_TIMEOUT|呼叫超時。|  
|HOST_E_NOT_OWNER|呼叫端沒有擁有鎖定。|  
|HOST_E_ABANDONED|已封鎖的執行緒或光纖在等候時取消了事件。|  
|E_FAIL|發生不明的嚴重失敗。 當方法傳回 E_FAIL 時，CLR 就無法在進程內使用。 對裝載方法的後續呼叫會傳回 HOST_E_CLRNOTAVAILABLE。|  
  
## <a name="remarks"></a>備註  
 CLR 會呼叫 `SetCLRTask` `ICLRTask` ，將實例與目前的 `IHostTask` 實例產生關聯，這是由呼叫[IHostTaskManager：： CreateTask](ihosttaskmanager-createtask-method.md)所建立。
