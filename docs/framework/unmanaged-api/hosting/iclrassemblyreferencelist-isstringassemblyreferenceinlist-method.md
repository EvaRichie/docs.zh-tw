# <a name="iclrassemblyreferencelistisstringassemblyreferenceinlist-method"></a>ICLRAssemblyReferenceList::IsStringAssemblyReferenceInList 方法

取得值，這個值會指出提供的名稱是否符合清單中的元件名稱。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsStringAssemblyReferenceInList (  
    [in] LPCWSTR pwzAssemblyName  
);  
```  
  
## <a name="parameters"></a>參數  

 `pwzAssemblyName`  
 在要搜尋之元件的名稱。  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|S_OK|字串會出現在清單中。|  
|S_FALSE|字串不會出現在清單中。|  
|E_FAIL|發生未知的嚴重失敗。 在方法傳回 E_FAIL 之後，就無法在進程中使用 common language runtime。 對裝載方法的後續呼叫會傳回 HOST_E_CLRNOTAVAILABLE。|
