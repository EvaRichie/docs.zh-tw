# <a name="iclrerrorreportingmanagergetbucketparametersforcurrentexception-method"></a>ICLRErrorReportingManager::GetBucketParametersForCurrentException 方法

取得目前例外狀況在呼叫執行緒上的 Watson 值區。  
  
 值 *區是與* 相同程式碼缺失相關的錯誤資料集合。 *Watson* 指的是一組技術，用來收集和分析與例外狀況相關聯的資料。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetBucketParametersForCurrentException(  
    [out] BucketParameters *pParams  
);  
```  
  
## <a name="parameters"></a>參數  

 `pParams`  
 擴展 [BucketParameters](bucketparameters-structure.md) 結構的指標，其中包含例外狀況的錯誤資料。
