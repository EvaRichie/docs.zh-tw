# <a name="icorruntimehostmapfile-method"></a>ICorRuntimeHost::MapFile 方法
將指定的檔案對應到記憶體中。 這個方法已過時。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT MapFile(  
    [in]  HANDLE    hFile,  
    [out] HMODULE*  hMapAddress  
);  
```  
  
## <a name="parameters"></a>參數  
 `hFile`  
 在要對應之檔案的控制碼。  
  
 `hMapAddress`  
 脫銷要開始對應檔案的起始記憶體位址。
