# <a name="iclrassemblyreferencelistisassemblyreferenceinlist-method"></a>ICLRAssemblyReferenceList::IsAssemblyReferenceInList 方法

取得值，這個值會指出提供的指標是否參考清單中的元件。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT IsAssemblyReferenceInList (  
    [in] IUnknown *pName  
);  
```  
  
## <a name="parameters"></a>參數  

 `pName`  
 在要搜尋之元件的介面指標。 有效的值為類型 `IAssemblyName` 或 `IReferenceIdentity` 。  
  
## <a name="return-value"></a>傳回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|S_OK|字串會出現在清單中。|  
|S_FALSE|字串不會出現在清單中。|  
|E_FAIL|發生未知的嚴重失敗。 在方法傳回 E_FAIL 之後，就無法在進程中使用 common language runtime。 對裝載方法的後續呼叫會傳回 HOST_E_CLRNOTAVAILABLE。|
