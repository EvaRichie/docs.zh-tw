# <a name="igchostcontrol-interface"></a>IGCHostControl 介面

提供一種方法，可讓垃圾收集行程要求主控制項變更虛擬記憶體的限制。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[RequestVirtualMemLimit 方法](igchostcontrol-requestvirtualmemlimit-method.md)|要求主機變更虛擬記憶體的限制。|
