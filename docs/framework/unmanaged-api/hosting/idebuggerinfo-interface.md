# <a name="idebuggerinfo-interface"></a>IDebuggerInfo 介面

提供方法，以取得有關偵錯工具狀態的資訊。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[IsDebuggerAttached 方法](idebuggerinfo-isdebuggerattached-method.md)|取得值，這個值會指出 managed 偵錯工具是否已附加至此進程。|
