## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[IsDebuggerAttached 方法](idebuggerinfo-isdebuggerattached-method.md)|取得值，指出 managed 偵錯工具是否附加至此進程。|
