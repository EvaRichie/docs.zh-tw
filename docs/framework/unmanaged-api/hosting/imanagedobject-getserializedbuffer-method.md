# <a name="imanagedobjectgetserializedbuffer-method"></a>IManagedObject::GetSerializedBuffer 方法

取得此 managed 物件的字串表示。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT GetSerializedBuffer (  
    [out] BSTR *pBSTR  
);  
```  
  
## <a name="parameters"></a>參數  

 `pBSTR`  
 擴展字串的指標，該字串為已序列化的物件。  
  
## <a name="remarks"></a>備註  

 方法會將物件序列化， `GetSerializedBuffer` 以便將其封送處理至用戶端。
