# <a name="rundll32shimw-function"></a>RunDll32ShimW 函式
執行指定命令。  
  
 此函式在 .NET Framework 4 中已被取代。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT RunDll32ShimW (  
    [in] HWND        hwnd,  
    [in] HINSTANCE   hinst,  
    [in] LPCWSTR     lpszCmdLine,  
    [in] int         nCmdShow  
);  
```  
  
## <a name="parameters"></a>參數  
 `hwnd`  
 在將顯示命令輸出之視窗的控制碼。  
  
 `hinst`  
 在包含命令之程式庫的控制碼。  
  
 `lpszCmdLine`  
 在字串，指定要執行的命令。  
  
 `nCmdShow`  
 在整數，指定輸出視窗的顯示模式。
