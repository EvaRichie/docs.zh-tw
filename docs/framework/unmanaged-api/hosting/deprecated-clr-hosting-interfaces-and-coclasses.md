# <a name="deprecated-clr-hosting-interfaces-and-coclasses"></a>已被取代的 CLR 裝載介面和 Coclass

本節說明非受控主機可用來將 .NET Framework 1.0 版和1.1 版中的 common language runtime (CLR) 整合至其應用程式的介面。 這些介面會提供方法，讓主機在進程中設定和載入執行時間。  
  
## <a name="in-this-section"></a>本節內容  

 IAppDomainSetup  
 提供可供主機設定的方法 <xref:System.AppDomain> 。  
  
 [ICeeFileGen 類別](iceefilegen-class.md)  
  (已淘汰的) 會提供功能，以 (PE) 檔建立原生可攜式可執行檔。  
  
 [ICorRuntimeHost 介面](icorruntimehost-interface.md)  
 提供用來設定 CLR 設定的主機方法。
