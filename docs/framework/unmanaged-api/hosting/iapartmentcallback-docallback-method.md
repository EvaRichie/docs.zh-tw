# <a name="iapartmentcallbackdocallback-method"></a>IApartmentCallback::DoCallback 方法

在公寓內執行指定的函式。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT _stdcall DoCallback(  
    [in] SIZE_T pFunc,  
    [in] SIZE_T pData  
);  
```  
  
## <a name="parameters"></a>參數  

 `pFunc`  
 在要在單元內執行之函式的指標。  
  
 `pData`  
 在函數之引數的指標。
