# <a name="iobjecthandle-interface"></a>IObjectHandle 介面

提供方法來解除包裝從間接取值的封送處理物件。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Unwrap 方法](iobjecthandle-unwrap-method.md)|從間接取值解除封送處理傳值的物件。|
