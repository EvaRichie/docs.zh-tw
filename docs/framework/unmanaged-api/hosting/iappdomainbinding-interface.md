# <a name="iappdomainbinding-interface"></a>IAppDomainBinding 介面
提供通用語言執行時間（CLR）所呼叫的方法，以通知主應用程式已建立應用程式域。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[OnAppDomain 方法](iappdomainbinding-onappdomain-method.md)|由 common language runtime （CLR）呼叫，以通知主機已建立應用程式域。|
