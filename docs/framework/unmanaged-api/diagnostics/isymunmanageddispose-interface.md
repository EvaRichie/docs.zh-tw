# <a name="isymunmanageddispose-interface"></a>ISymUnmanagedDispose 介面

## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Destroy 方法](isymunmanageddispose-destroy-method.md)|導致基礎物件釋放所有的內部參考，並在任何後續的方法呼叫上傳回失敗。|
