# <a name="diagnostics-symbol-store-structures"></a>診斷符號存放區結構

本主題說明診斷符號存放區 API 所使用的非受控結構。  
  
## <a name="in-this-section"></a>本節內容  

 [CALL_ID 結構](call-id-structure.md)  
 提供有關正在呼叫的函式之偵錯工具的資訊。  
  
 [SYMLINEDELTA 結構](symlinedelta-structure.md)  
 提供有關方法的符號處理常式資訊，這些方法是在編輯時移動的。  
  
 [USER_THREAD 結構](user-thread-structure.md)  
 提供有關執行緒的資訊給偵錯工具。
