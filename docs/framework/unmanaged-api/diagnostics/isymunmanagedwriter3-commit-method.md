# <a name="isymunmanagedwriter3commit-method"></a>ISymUnmanagedWriter3::Commit 方法

認可到目前為止寫入資料流程的變更。  
  
## <a name="syntax"></a>語法  
  
```cpp  
HRESULT Commit();  
```  
  
## <a name="return-value"></a>傳回值  

 如果方法成功，則為 S_OK;否則，E_FAIL 或其他一些錯誤碼。  
  
## <a name="requirements"></a>需求  

 **標頭：** CorSym .idl、CorSym。h
