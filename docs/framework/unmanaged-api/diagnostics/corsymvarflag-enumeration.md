# <a name="corsymvarflag-enumeration"></a>CorSymVarFlag 列舉
指出變數是否由編譯器產生。  
  
## <a name="syntax"></a>語法  
  
```cpp  
typedef enum CorSymVarFlag
{  
    VAR_IS_COMP_GEN = 1  
} CorSymVarFlag;  
```  
  
## <a name="members"></a>成員  
  
|成員|說明|  
|------------|-----------------|  
|`VAR_IS_COMP_GEN`|表示指定的變數是編譯器產生的。|  
  
## <a name="requirements"></a>需求  
 **標頭：** CorSym .idl，CorSym。h
