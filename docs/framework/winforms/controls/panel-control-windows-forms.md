# <a name="panel-control-windows-forms"></a>Panel 控制項 (Windows Form)
Windows Forms `Panel` 控制項是用來為其他控制項提供可識別的群組。 一般而言，您可以使用面板依函式細分表單。 `Panel`控制項與 <xref:System.Windows.Forms.GroupBox> 控制項類似; 不過，只有 `Panel` 控制項可以有捲軸，而且只有控制項才會 <xref:System.Windows.Forms.GroupBox> 顯示標題。
