# <a name="how-to-align-multiple-controls-on-windows-forms"></a>如何：對齊 Windows Form 上的多個控制項

若要將使用者介面的配置標準化 (Windows 應用程式的 UI) ，您可以使用單一命令來放置控制項群組。

## <a name="to-align-multiple-controls-on-a-form"></a>對齊表單上的多個控制項

1. 在 Visual Studio 中，開啟包含您要在**Windows Form 設計工具**中放置之控制項的表單。

2. 選取您要對齊的控制項。 您選取的第一個控制項是其他專案將會對齊的主要控制項。

3. 在 [**格式**] 功能表上，選取 [**對齊**]，然後選取可用的七個選項之一。
