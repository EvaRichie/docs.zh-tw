### <a name="fixed-a-hang-when-listbox-contains-duplicate-value-types"></a>已修正當 ListBox 包含重複實值型別時的停止回應問題

| 名稱    | 值       |
|:--------|:------------|
| 範圍   |主要|
|版本|4.8|
|類型|執行階段|
