### <a name="security-identitymodel-nuget-package-versions-updated"></a>安全性：已更新 Microsoft.identitymodel NuGet 套件版本

下列套件已更新至版本6.6.0：

- [Microsoft.identitymodel 記錄](https://www.nuget.org/packages/Microsoft.IdentityModel.Logging)
- [Microsoft.identitymodel 通訊協定. OpenIdConnect](https://www.nuget.org/packages/Microsoft.IdentityModel.Protocols.OpenIdConnect)
- [Microsoft.identitymodel 通訊協定. WsFederation](https://www.nuget.org/packages/Microsoft.IdentityModel.Protocols.WsFederation)
- [System.IdentityModel.Tokens.Jwt](https://www.nuget.org/packages/System.IdentityModel.Tokens.Jwt)

#### <a name="version-introduced"></a>引進的版本

5.0 Preview 7

#### <a name="old-behavior"></a>舊的行為

使用的套件版本是5.5.0。

#### <a name="new-behavior"></a>新的行為

如需套件版本之間變更的詳細資訊，請參閱 [6.6.0 版本](https://github.com/AzureAD/azure-activedirectory-identitymodel-extensions-for-dotnet/releases/tag/6.6.0)資訊。

#### <a name="reason-for-change"></a>變更的原因

已更新套件以利用基礎程式庫中的增強功能。

#### <a name="recommended-action"></a>建議的動作

套件更新不會對 ASP.NET Core 引進公用 API 變更。 不過，封裝本身可能會有重大變更。

#### <a name="category"></a>類別

ASP.NET Core

#### <a name="affected-apis"></a>受影響的 API

無
