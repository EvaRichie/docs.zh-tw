> [!NOTE]
> .NET Framework 4.8 是 .NET Framework 的最新版本，而且不會再發行其他版本。 不過，.NET Framework 將 [繼續提供](https://devblogs.microsoft.com/dotnet/?s=%22.net+framework%22) 每月安全性和可靠性錯誤修正的服務。 此外，它仍會繼續隨附于 Windows，而且沒有計劃將它移除。 您不需要遷移 .NET Framework 的應用程式，但對於新的開發，請使用 [.net 5.0 或更新版本](~/docs/core/introduction.md)。
